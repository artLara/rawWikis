<p>Clasificación tradicional:<sup class="reference separada" id="cite_ref-2"><a href="#cite_note-2"><span class="corchete-llamada">[</span>2<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Clasificación alternativa:<sup class="reference separada" id="cite_ref-Brusca2016_3-0"><a href="#cite_note-Brusca2016-3"><span class="corchete-llamada">[</span>3<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>En la <a href="/wiki/Taxonom%C3%ADa" title="Taxonomía">clasificación científica de los seres vivos</a>, los <b>animales</b> (<b>Animalia</b>) o <b>metazoos</b> (<b>Metazoa</b>) constituyen un <a href="/wiki/Reino_(biolog%C3%ADa)" title="Reino (biología)">reino</a> que reúne un amplio grupo de organismos que son <a href="/wiki/Eukaryota" title="Eukaryota">eucariotas</a>, <a href="/wiki/Nutrici%C3%B3n_heter%C3%B3trofa" title="Nutrición heterótrofa">heterótrofos</a>, <a class="mw-redirect" href="/wiki/Pluricelular" title="Pluricelular">pluricelulares</a> y <a href="/wiki/Tejido_(biolog%C3%ADa)" title="Tejido (biología)">tisulares</a> (excepto los <a href="/wiki/Porifera" title="Porifera">poríferos</a>). Se caracterizan por su <a class="mw-redirect" href="/wiki/Locomoci%C3%B3n" title="Locomoción">amplia capacidad de movimiento</a>, por no tener <a href="/wiki/Cloroplasto" title="Cloroplasto">cloroplasto</a> (aunque hay excepciones, como en el caso de <i><a href="/wiki/Elysia_chlorotica" title="Elysia chlorotica">Elysia chlorotica</a></i>) ni <a href="/wiki/Pared_celular" title="Pared celular">pared celular</a>, y por su <a class="mw-redirect" href="/wiki/Desarrollo_embrionario" title="Desarrollo embrionario">desarrollo embrionario</a>; que atraviesa una fase de <a href="/wiki/Bl%C3%A1stula" title="Blástula">blástula</a> y determina un plan corporal fijo (aunque muchas especies pueden sufrir una <a href="/wiki/Metamorfosis_(biolog%C3%ADa)" title="Metamorfosis (biología)">metamorfosis</a> posterior como los <a href="/wiki/Arthropoda" title="Arthropoda">artrópodos</a>). Los animales forman un <a href="/wiki/Monofil%C3%A9tico" title="Monofilético">grupo natural</a> estrechamente emparentado con los <a href="/wiki/Fungi" title="Fungi">hongos</a> (reino <a href="/wiki/Fungi" title="Fungi">Fungi</a>). Animalia es uno de los cinco <a class="mw-redirect" href="/wiki/Reinos_naturales_(biolog%C3%ADa)" title="Reinos naturales (biología)">reinos</a> del dominio <a href="/wiki/Eukaryota" title="Eukaryota">Eukaryota</a>, y a él pertenece el <a href="/wiki/Homo_sapiens" title="Homo sapiens">ser humano</a>. La parte de la <a href="/wiki/Biolog%C3%ADa" title="Biología">biología</a> que estudia los animales es la <a href="/wiki/Zoolog%C3%ADa" title="Zoología">zoología</a>.
</p>
<p>Los <a href="/wiki/Filo" title="Filo">filos</a> animales más conocidos aparecen en el registro <a href="/wiki/F%C3%B3sil" title="Fósil">fósil</a> durante la denominada <a href="/wiki/Explosi%C3%B3n_c%C3%A1mbrica" title="Explosión cámbrica">explosión cámbrica</a>, sucedida en los mares hace unos 542 a 530 millones de años. Los animales se dividen en varios subgrupos, algunos de los cuales son <a class="mw-redirect" href="/wiki/Vertebrados" title="Vertebrados">vertebrados</a>: (<a href="/wiki/Aves" title="Aves">aves</a>, <a href="/wiki/Mammalia" title="Mammalia">mamíferos</a>, <a href="/wiki/Amphibia" title="Amphibia">anfibios</a>, <a href="/wiki/Reptilia" title="Reptilia">reptiles</a>, <a class="mw-redirect" href="/wiki/Pisces" title="Pisces">peces</a>) e <a href="/wiki/Invertebrado" title="Invertebrado">invertebrados</a>: <a href="/wiki/Arthropoda" title="Arthropoda">artrópodos</a> (<a href="/wiki/Insecta" title="Insecta">insectos</a>, <a href="/wiki/Arachnida" title="Arachnida">arácnidos</a>, <a href="/wiki/Myriapoda" title="Myriapoda">miriápodos</a>, <a href="/wiki/Crustacea" title="Crustacea">crustáceos</a>), <a href="/wiki/Annelida" title="Annelida">anélidos</a> (<a href="/wiki/Lumbricidae" title="Lumbricidae">lombrices</a>, <a href="/wiki/Hirudinea" title="Hirudinea">sanguijuelas</a>), <a href="/wiki/Mollusca" title="Mollusca">moluscos</a> (<a href="/wiki/Bivalvia" title="Bivalvia">bivalvos</a>, <a href="/wiki/Gastropoda" title="Gastropoda">gasterópodos</a>, <a href="/wiki/Cephalopoda" title="Cephalopoda">cefalópodos</a>), <a href="/wiki/Porifera" title="Porifera">poríferos</a> (<a class="mw-redirect" href="/wiki/Esponjas" title="Esponjas">esponjas</a>), <a href="/wiki/Cnidaria" title="Cnidaria">cnidarios</a> (<a class="mw-redirect" href="/wiki/Medusa_(animal)" title="Medusa (animal)">medusas</a>, <a class="mw-redirect" href="/wiki/Polipo" title="Polipo">pólipos</a>, <a href="/wiki/Coral" title="Coral">corales</a>), <a href="/wiki/Echinodermata" title="Echinodermata">equinodermos</a> (<a href="/wiki/Asteroidea" title="Asteroidea">estrellas de mar</a>), <a href="/wiki/Nematoda" title="Nematoda">nematodos</a> (gusanos cilíndricos), <a href="/wiki/Platyhelminthes" title="Platyhelminthes">platelmintos</a> (gusanos planos), etc.
</p>
<p>La movilidad es la característica más llamativa de los organismos de este reino, pero no es exclusiva del grupo, lo que da lugar a que sean designados a menudo como animales ciertos organismos, los llamados <a href="/wiki/Protozoo" title="Protozoo">protozoos</a>, que pertenecen al reino <a href="/wiki/Protista" title="Protista">Protista</a>.
</p>
<p>En el siguiente esquema se muestran las características comunes a todos los animales:
</p>
<p>Con pocas excepciones, la más notable la de las esponjas (<a href="/wiki/Filo" title="Filo">filo</a> <a href="/wiki/Porifera" title="Porifera">Porifera</a>), los animales presentan <a href="/wiki/Tejido_(biolog%C3%ADa)" title="Tejido (biología)">tejidos</a> diferenciados y especializados. Estos incluyen <a href="/wiki/M%C3%BAsculo" title="Músculo">músculos</a>, que pueden contraerse para controlar el movimiento, y un <a href="/wiki/Sistema_nervioso" title="Sistema nervioso">sistema nervioso</a>, que envía y procesa señales. Suele haber también una cámara digestiva interna, con una o dos aberturas. Los animales con este tipo de organización son conocidos como <a href="/wiki/Eumetazoa" title="Eumetazoa">eumetazoos</a>, en contraposición a los <a href="/wiki/Parazoa" title="Parazoa">parazoos</a> y <a href="/wiki/Mesozoa" title="Mesozoa">mesozoos</a>, que son niveles de organización más simples ya que carecen de algunas de las características mencionadas.
</p>
<p>Todos los animales tienen <a class="mw-redirect" href="/wiki/C%C3%A9lula_(biolog%C3%ADa)" title="Célula (biología)">células</a> <a class="mw-redirect" href="/wiki/Eucarionte" title="Eucarionte">eucariontes</a>, rodeadas de una <a href="/wiki/Matriz_extracelular" title="Matriz extracelular">matriz extracelular</a> característica compuesta de <a href="/wiki/Col%C3%A1geno" title="Colágeno">colágeno</a> y <a href="/wiki/Glucoprote%C3%ADna" title="Glucoproteína">glucoproteínas</a> elásticas. Esta puede <a href="/wiki/Calcio" title="Calcio">calcificarse</a> para formar estructuras como <a href="/wiki/Concha" title="Concha">conchas</a>, <a href="/wiki/Hueso" title="Hueso">huesos</a> y <a href="/wiki/Esp%C3%ADcula" title="Espícula">espículas</a>. Durante el desarrollo del animal se crea un armazón relativamente flexible por el que las células se pueden mover y reorganizarse, haciendo posibles estructuras más complejas. Esto contrasta con otros organismos <a class="mw-redirect" href="/wiki/Pluricelular" title="Pluricelular">pluricelulares</a> como las <a href="/wiki/Vegetal" title="Vegetal">plantas</a> y los <a class="mw-redirect" href="/wiki/Hongo" title="Hongo">hongos</a>, que desarrollan un crecimiento progresivo.
</p>
<p>Los animales llevan a cabo las siguientes funciones esenciales: alimentación, respiración, circulación, excreción, respuesta, movimiento y reproducción:
</p>
<p>La mayoría de los animales no pueden absorber comida; la ingieren. Los animales han evolucionado de diversas formas para alimentarse. Los <a class="mw-redirect" href="/wiki/Herb%C3%ADvoros" title="Herbívoros">herbívoros</a> comen plantas, los <a href="/wiki/Carn%C3%ADvoro" title="Carnívoro">carnívoros</a> comen otros animales; y los <a href="/wiki/Omn%C3%ADvoro" title="Omnívoro">omnívoros</a> se alimentan tanto de plantas como de animales. Los <a class="mw-redirect" href="/wiki/Detrit%C3%ADvoros" title="Detritívoros">detritívoros</a> comen material vegetal y animal en descomposición. Los comedores por filtración son animales acuáticos que cuelan minúsculos organismos que flotan en el agua. Los animales también forman relaciones simbióticas, en las que dos especies viven en estrecha asociación mutua. Por ejemplo, un <a href="/wiki/Parasitismo" title="Parasitismo">parásito</a> es un tipo de simbionte que vive dentro o sobre otro organismo, el huésped. El parásito se alimenta del huésped y lo daña.<sup class="reference separada" id="cite_ref-miller_4-0"><a href="#cite_note-miller-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>No importa si viven en el agua o en la tierra, todos los animales respiran; esto significa que pueden tomar <a href="/wiki/Ox%C3%ADgeno" title="Oxígeno">oxígeno</a> y despedir <a href="/wiki/Di%C3%B3xido_de_carbono" title="Dióxido de carbono">dióxido de carbono</a>. Gracias a sus cuerpos muy simples y de delgadas paredes, algunos animales utilizan la difusión de estas sustancias a través de la <a href="/wiki/Piel" title="Piel">piel</a>. Sin embargo, la mayoría de los animales han evolucionado complejos tejidos y sistemas orgánicos para la respiración.<sup class="reference separada" id="cite_ref-miller_4-1"><a href="#cite_note-miller-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Muchos animales acuáticos pequeños, como algunos gusanos, utilizan solo la difusión para transportar oxígeno y moléculas de nutrientes a todas sus células, y recoger de ellas los productos de desecho. La difusión basta porque estos animales apenas tienen un espesor de unas cuantas células. Sin embargo, los animales más grandes poseen algún tipo de <a class="mw-redirect" href="/wiki/Sistema_circulatorio" title="Sistema circulatorio">sistema circulatorio</a> para desplazar sustancias por el interior de sus cuerpos.<sup class="reference separada" id="cite_ref-miller_4-2"><a href="#cite_note-miller-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Un producto de desecho primario de las células es el <a href="/wiki/Amon%C3%ADaco" title="Amoníaco">amoníaco</a>, sustancia venenosa que contiene nitrógeno. La acumulación de amoniaco y otros productos de desecho podrían matar a un animal. La mayoría de los animales poseen un sistema excretor que bien elimina amoniaco o bien lo transforma en una sustancia menos tóxica que se elimina del cuerpo. Gracias a que eliminan los desechos metabólicos, los sistemas excretores ayudan a mantener la <a href="/wiki/Homeostasis" title="Homeostasis">homeostasis</a>. Los sistemas excretores varían, desde células que bombean agua fuera del cuerpo hasta órganos complejos como riñones.<sup class="reference separada" id="cite_ref-miller_4-3"><a href="#cite_note-miller-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Los animales usan células especializadas, llamadas células nerviosas, para responder a los sucesos de su <a href="/wiki/Nicho_ecol%C3%B3gico" title="Nicho ecológico">medio ambiente</a>. En la mayoría de los animales, las células nerviosas están conectadas entre sí para formar un sistema nervioso. Algunas células llamadas receptores, responden a sonidos, luz y otros estímulos externos. Otras células nerviosas procesan información y determinan la respuesta del animal. La organización de las células nerviosas dentro del cuerpo cambia dramáticamente de un fílum a otro.<sup class="reference separada" id="cite_ref-miller_4-4"><a href="#cite_note-miller-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Algunos animales adultos permanecen fijos en un sitio. Aunque muchos tienen movilidad. Sin embargo, tanto los fijos como los más veloces normalmente poseen músculos o tejidos musculares que se acortan para generar fuerza. La contracción muscular permite que los animales movibles se desplacen, a menudo en combinación con una estructura llamada <a href="/wiki/Esqueleto" title="Esqueleto">esqueleto</a>. Los músculos también ayudan a los animales, aun los más sedentarios, a comer y bombear agua y otros líquidos fuera del cuerpo.<sup class="reference separada" id="cite_ref-miller_4-5"><a href="#cite_note-miller-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La mayoría de los animales se reproducen sexualmente mediante la producción de <a class="mw-redirect" href="/wiki/Gametos" title="Gametos">gametos</a> haploides. La reproducción sexual ayuda a crear y mantener la diversidad genética de una población. Por consiguiente, ayuda a mejorar la capacidad de una especie para evolucionar con los cambios del medio ambiente. Muchos <a class="mw-redirect" href="/wiki/Invertebrados" title="Invertebrados">invertebrados</a> también pueden reproducirse asexualmente. La <a href="/wiki/Reproducci%C3%B3n_asexual" title="Reproducción asexual">reproducción asexual</a> da origen a descendiente genéticamente idénticos a los progenitores. Esta forma de reproducción permite que los animales aumenten rápidamente en cantidad.<sup class="reference separada" id="cite_ref-miller_4-6"><a href="#cite_note-miller-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Los animales han sido estudiados desde la antigüedad, y aún hoy, la clasificación animal se muestra cambiante, pues depende de los estudios que revelan constantemente información novedosa. Los grupos animales se definieron sobre la base de sus caracteres biológicos, morfológicos y ultraestructurales; sin embargo, la filogenia del siglo <span style="font-variant:small-caps;text-transform:lowercase">XXI</span> está basada principalmente en el estudio filogenómico molecular del <a class="mw-redirect" href="/wiki/ADN_mitocondrial" title="ADN mitocondrial">ADN mitocondrial</a>, ribosómico y nuclear, lo que ha determinado también cambios importantes. </p>
<p>La clasificación de Hyman (1940), que ha estado en vigencia hasta hace poco (Margulis &amp; Chapman, 2009), ha sido invalidada por los estudios filogenéticos moleculares contemporáneos, ya que se demostró que grupos como los <a class="mw-redirect" href="/wiki/Acelomados" title="Acelomados">acelomados</a>, <a class="mw-redirect" href="/wiki/Pseudocelomados" title="Pseudocelomados">pseudocelomados</a>, <a class="mw-redirect" href="/wiki/Celomados" title="Celomados">celomados</a> y <a href="/wiki/Esquizocelomados" title="Esquizocelomados">esquizocelomados</a> son en realidad grupos artificiales (<a href="/wiki/Polifil%C3%A9tico" title="Polifilético">polifiléticos</a>).<sup class="reference separada" id="cite_ref-17"><a href="#cite_note-17"><span class="corchete-llamada">[</span>17<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El reino animal se subdivide en una serie de grandes grupos denominados <a href="/wiki/Filo" title="Filo">filos</a> (el equivalente a las <a href="/wiki/Divisi%C3%B3n_(biolog%C3%ADa)" title="División (biología)">divisiones</a> del <a class="mw-redirect" href="/wiki/Reino_vegetal" title="Reino vegetal">reino vegetal</a>); cada uno responde a un tipo de organización bien definido, aunque hay algunos de afiliación controvertida. En el siguiente cuadro, se enumeran los filos animales y sus principales características:
</p>
<p>En esta tabla no figuran los filos <a href="/wiki/Echiura" title="Echiura">Echiura</a>, <a class="mw-redirect" href="/wiki/Pogonophora" title="Pogonophora">Pogonophora</a>, <a href="/wiki/Sipuncula" title="Sipuncula">Sipuncula</a> y <a href="/wiki/Orthonectida" title="Orthonectida">Orthonectida</a> los cuales han sido reclasificados en <a href="/wiki/Annelida" title="Annelida">Annelida</a>,<sup class="reference separada" id="cite_ref-shiffer_23-0"><a href="#cite_note-shiffer-23"><span class="corchete-llamada">[</span>23<span class="corchete-llamada">]</span></a></sup>​ y el filo <a href="/wiki/Acanthocephala" title="Acanthocephala">Acanthocephala</a> el cual fue reclasificado en <a href="/wiki/Rotifera" title="Rotifera">Rotifera</a>.<sup class="reference separada" id="cite_ref-24"><a href="#cite_note-24"><span class="corchete-llamada">[</span>24<span class="corchete-llamada">]</span></a></sup>​ El filo <a href="/wiki/Myxozoa" title="Myxozoa">Myxozoa</a> ha sido reclasificado en <a href="/wiki/Cnidaria" title="Cnidaria">Cnidaria</a> y el filo <a href="/wiki/Monoblastozoa" title="Monoblastozoa">Monoblastozoa</a> es de dudosa existencia.
</p>
<p>Mientras que en las <a class="mw-redirect" href="/wiki/Plantas" title="Plantas">plantas</a> se conocen varias series de formas que conducen de la organización <a class="mw-redirect" href="/wiki/Unicelular" title="Unicelular">unicelular</a> a la <a class="mw-redirect" href="/wiki/Pluricelular" title="Pluricelular">pluricelular</a>, en el Reino Animal se sabe muy poco sobre la transición entre <a href="/wiki/Protozoo" title="Protozoo">protozoos</a> y metazoos. Dicha transición no está documentada por <a href="/wiki/F%C3%B3sil" title="Fósil">fósiles</a> y las formas recientes supuestamente intermedias tampoco nos ayudan demasiado.
</p>
<p>En este campo de la transición pueden mencionarse, por una parte, a <i><a href="/wiki/Proterospongia" title="Proterospongia">Proterospongia</a></i>, <a class="mw-redirect" href="/wiki/Choanoflagellata" title="Choanoflagellata">coanoflagelado</a> marino y <a class="mw-redirect" href="/wiki/Planct%C3%B3nico" title="Planctónico">planctónico</a> que forma una masa gelatinosa con <a href="/wiki/Coanocito" title="Coanocito">coanocitos</a> en la parte exterior y células <a href="/wiki/Ameboide" title="Ameboide">ameboides</a> en el interior, y por otra al pequeño organismo marino <i><a class="mw-redirect" href="/wiki/Trichoplax_adhaerens" title="Trichoplax adhaerens">Trichoplax adhaerens</a></i> (filo <a href="/wiki/Placozoa" title="Placozoa">placozoos</a>) que forma una placa cerrada por <a href="/wiki/Epitelio" title="Epitelio">epitelio</a> pavimentoso en la parte dorsal y cilíndrico en la parte central, y presenta en la cavidad interior células en forma de estrella; se reproduce por yemas flageladas y huevos. Otra forma sencilla de metazoo es <i><a class="mw-redirect" href="/wiki/Xenoturbellida" title="Xenoturbellida">Xenoturbella</a></i>, que vive sobre los fondos fangosos del mar. Tienen algunos centímetros de largo y forma de hoja, una boca ventral que conduce a un estómago en forma de saco. Entre la <a class="new" href="/w/index.php?title=Epidermis_(invertebrados)&amp;action=edit&amp;redlink=1" title="Epidermis (invertebrados) (aún no redactado)">epidermis</a> y el <a href="/wiki/Intestino" title="Intestino">intestino</a> existe una capa de <a href="/wiki/Tejido_conjuntivo" title="Tejido conjuntivo">tejido conjuntivo</a> con un tubo muscular longitudinal y células musculares en el <a class="mw-redirect" href="/wiki/Mes%C3%A9nquima" title="Mesénquima">mesénquima</a>; en la parte basal de la epidermis existe un plexo nervioso y en la parte anterior presenta un <a href="/wiki/Estatocisto" title="Estatocisto">estatocisto</a>; produce <a href="/wiki/%C3%93vulo" title="Óvulo">óvulos</a> y <a href="/wiki/Espermatozoide" title="Espermatozoide">espermatozoides</a>, estos idénticos a los de diferentes metazoos primitivos. Su posición sistemática es incierta, habiéndose propuesto como miembro de un filo independiente (<a class="mw-redirect" href="/wiki/Xenoturbellida" title="Xenoturbellida">xenoturbélidos</a>), a emplazar tal vez en la base de los <a href="/wiki/Deuterostomia" title="Deuterostomia">deuteróstomos</a>. Por lo que respecta a los <a href="/wiki/Mesozoa" title="Mesozoa">mesozoos</a>, ya no son considerados un estado de transición entre <a href="/wiki/Protista" title="Protista">protistas</a> y <a class="mw-redirect" href="/wiki/Metazoos" title="Metazoos">metazoos</a>; su modo de vida parásito parece que les condujo a una reducción y simplificación extremas a partir de vermes <a class="mw-redirect" href="/wiki/Acelomados" title="Acelomados">acelomados</a>.
</p>
<p>Por tanto, se debe recurrir a la <a href="/wiki/Morfolog%C3%ADa_(biolog%C3%ADa)" title="Morfología (biología)">morfología</a>, <a href="/wiki/Fisiolog%C3%ADa" title="Fisiología">fisiología</a> y <a href="/wiki/Ontogenia" title="Ontogenia">ontogenia</a> comparadas de los metazoos para poder reconstruir esta etapa de la <a href="/wiki/Evoluci%C3%B3n_biol%C3%B3gica" title="Evolución biológica">evolución</a>. Los datos obtenidos con <a href="/wiki/Microscopio_electr%C3%B3nico" title="Microscopio electrónico">microscopía electrónica</a> y análisis moleculares han apagado antiguas controversias sobre el origen de los metazoos. En este sentido, parece definitivamente rechazada la hipótesis sobre un origen <a href="/wiki/Polifil%C3%A9tico" title="Polifilético">polifilético</a>; incluso los <a href="/wiki/Placozoa" title="Placozoa">placozoos</a> y los <a href="/wiki/Mesozoa" title="Mesozoa">mesozoos</a>, considerados a veces como originados directa e independientemente de los <a href="/wiki/Protista" title="Protista">protistas</a>, parecen a la luz de los nuevos datos claramente metazoos. <p><b>Animalia</b>
</p>
<p>Los primeros <a href="/wiki/F%C3%B3sil" title="Fósil">fósiles</a> que podrían representar a animales corresponden a <i><a href="/wiki/Otavia" title="Otavia">Otavia</a></i> hallado en <a href="/wiki/Namibia" title="Namibia">Namibia</a> de entre 760-550 millones de años. Estos fósiles se interpretan como <a href="/wiki/Porifera" title="Porifera">esponjas</a> tempranas.<sup class="reference separada" id="cite_ref-Brain_1-1"><a href="#cite_note-Brain-1"><span class="corchete-llamada">[</span>1<span class="corchete-llamada">]</span></a></sup>​ Sin embargo estudios que usan <a href="/wiki/Reloj_molecular" title="Reloj molecular">relojes moleculares</a> estiman el origen de los animales entre unos 850 millones de años durante la glaciación del <a href="/wiki/Per%C3%ADodo_Criog%C3%A9nico" title="Período Criogénico">Criogénico</a>-<a href="/wiki/Per%C3%ADodo_T%C3%B3nico" title="Período Tónico">Tónico</a>.<sup class="reference separada" id="cite_ref-royalsociety_27-0"><a href="#cite_note-royalsociety-27"><span class="corchete-llamada">[</span>27<span class="corchete-llamada">]</span></a></sup>​ 
</p>
<p>Los animales más antiguos que se conocen aparecen hacia el final del <a href="/wiki/Prec%C3%A1mbrico" title="Precámbrico">Precámbrico</a>, hace alrededor de 580 millones de años, y se les conoce como <a href="/wiki/Vendobionta" title="Vendobionta">vendobiontes</a> o la <a href="/wiki/Biota_del_periodo_Ediac%C3%A1rico" title="Biota del periodo Ediacárico">biota del periodo Ediacárico</a>.<sup class="reference separada" id="cite_ref-roseMaloof_28-0"><a href="#cite_note-roseMaloof-28"><span class="corchete-llamada">[</span>28<span class="corchete-llamada">]</span></a></sup>​ No obstante, son muy difíciles de relacionar con los fósiles posteriores. Algunos de estos organismos podrían ser los precursores de los filos modernos, pero también podrían ser grupos separados, y es posible que no fueran realmente animales en sentido estricto. Entre los primeros animales conocidos estarían <i><a href="/wiki/Cyclomedusa" title="Cyclomedusa">Cyclomedusa</a></i>, <i><a href="/wiki/Charnia" title="Charnia">Charnia</a></i>, <i><a class="new" href="/w/index.php?title=Charniodiscus&amp;action=edit&amp;redlink=1" title="Charniodiscus (aún no redactado)">Charniodiscus</a></i>, <i><a class="new" href="/w/index.php?title=Parvancorina&amp;action=edit&amp;redlink=1" title="Parvancorina (aún no redactado)">Parvancorina</a></i>, <i><a class="new" href="/w/index.php?title=Annulatubus&amp;action=edit&amp;redlink=1" title="Annulatubus (aún no redactado)">Annulatubus</a></i>, <i><a href="/wiki/Spriggina" title="Spriggina">Spriggina</a></i>, etc.
</p>
<p>Aparte de ellos, muchos filos conocidos de animales hicieron una aparición más o menos simultánea durante el período <a href="/wiki/C%C3%A1mbrico" title="Cámbrico">Cámbrico</a>, hace cerca de 570 millones de años. Todavía se discute si este evento, llamado <a href="/wiki/Explosi%C3%B3n_c%C3%A1mbrica" title="Explosión cámbrica">explosión cámbrica</a>, representa una rápida divergencia entre diferentes grupos o un cambio de condiciones que facilitó la fosilización. Algunos ejemplos serían <i><a href="/wiki/Wiwaxia" title="Wiwaxia">Wiwaxia</a></i>, <i><a href="/wiki/Pikaia" title="Pikaia">Pikaia</a></i>, <i><a href="/wiki/Hallucigenia" title="Hallucigenia">Hallucigenia</a></i>, <i><a class="mw-redirect" href="/wiki/Opabinidae" title="Opabinidae">Opabinia</a></i>, etc.
</p>
<p>Entre los ancestros de grupos posteriores destaca <i><a href="/wiki/Anomalocaris" title="Anomalocaris">Anomalocaris</a></i>, del <a href="/wiki/C%C3%A1mbrico" title="Cámbrico">Cámbrico</a>, como posible ancestro de diversos grupos de <a class="mw-redirect" href="/wiki/Artr%C3%B3podos" title="Artrópodos">artrópodos</a>, por su cuerpo segmentado, evolucionado de <i><a href="/wiki/Opabinia" title="Opabinia">Opabinia</a></i> y otros similares. Los <a class="mw-redirect" href="/wiki/Cordados" title="Cordados">cordados</a> podrían tener relación con <i><a href="/wiki/Pikaia" title="Pikaia">Pikaia</a></i>.
</p>
<p>En cuanto a la evolución de los filos, tradicionalmente los animales se clasificaron por simetría y su nivel de complejidad en grupos como <a href="/wiki/Radiata" title="Radiata">Radiata</a>, <a href="/wiki/Mesozoa" title="Mesozoa">Mesozoa</a>, <a href="/wiki/Acoelomata" title="Acoelomata">Acoelomata</a>, <a href="/wiki/Coelomata" title="Coelomata">Coelomata</a>, y <a href="/wiki/Pseudocoelomata" title="Pseudocoelomata">Pseudocoelomata</a> que resultaron ser <a href="/wiki/Polifil%C3%A9tico" title="Polifilético">polifiléticos</a>. Actualmente no está bien claro como fue el último ancestro común de todos los animales ("<a class="mw-redirect" href="/wiki/Urmetazoa" title="Urmetazoa">Urmetazoa</a>"), todos los análisis moleculares respaldan la teoría colonial que afirma que los animales surgieron de la unión colonial de <a href="/wiki/Protozoo" title="Protozoo">protozoos</a> similares a los <a class="mw-redirect" href="/wiki/Coanoflagelado" title="Coanoflagelado">coanoflagelados</a> que es la teoría ampliamente aceptada para explicar el origen de los animales. Parece poco probable que los animales hayan surgido de un único ancestro en común dado a la falta de <a href="/wiki/Homolog%C3%ADa_(biolog%C3%ADa)" title="Homología (biología)">homología</a> entre los <a href="/wiki/Porifera" title="Porifera">poríferos</a> y los animales verdaderos (<a href="/wiki/Eumetazoa" title="Eumetazoa">Eumetazoa</a>), por lo que es probable que hayan surgido dos veces de las colonias de coanoflagelados. Los <a class="mw-redirect" href="/wiki/Por%C3%ADferos" title="Poríferos">poríferos</a> son muy diferentes de los eumetazoos puesto que carecen de tejidos, sistema nervioso, notocordio y son generalmente inmóviles, de hecho Haeckel las clasificaba en el reino <a href="/wiki/Protista" title="Protista">Protista</a>. Según los análisis moleculares el grupo más basal de los eumetazoos es <a href="/wiki/Ctenophora" title="Ctenophora">Ctenophora</a> lo que sugiere que los bilaterios, <a class="mw-redirect" href="/wiki/Cnidario" title="Cnidario">cnidarios</a> y <a class="mw-redirect" href="/wiki/Placozoo" title="Placozoo">placozoos</a> evolucionaron de ancestros similares a los ctenóforos. El origen y el ancestro de los <a class="mw-redirect" href="/wiki/Bilaterio" title="Bilaterio">bilaterios</a> es un tema oscuro, pero los análisis moleculares han favorecido dos hipótesis clásicas: la hipótesis "<a href="/wiki/Planulozoa" title="Planulozoa">Planulozoa</a>" considera que los bilaterios surgieron de la derivación pedomórfica de la larva de un cnidario y según este punto de vista los gusanos <a class="mw-redirect" href="/wiki/Acelomado" title="Acelomado">acelomados</a> (<a class="mw-redirect" href="/wiki/Platelminto" title="Platelminto">platelmintos</a>, <a class="mw-redirect" href="/wiki/Nemertino" title="Nemertino">nemertinos</a> y <a class="mw-redirect" href="/wiki/Xenacelomorfo" title="Xenacelomorfo">xenacelomorfos</a>) serían los bilaterales más primitivos derivándose posteriormente de ellos celomados y pseudocelomados. Esta hipótesis fue respaldada por algunos estudios, pero a la vez criticada por otros, dado que los xenacelomorfos como platelmintos, son linajes muy reducidos, ha habido cambios constantes en el <a href="/wiki/Genoma" title="Genoma">genoma</a> y <a href="/wiki/Mutaci%C3%B3n" title="Mutación">mutan</a> más rápidamente que cualquier tipo de animal conocido, así que sugerir que estos linajes ocupan posiciones en los árboles filogenéticos podría significar caer en la <a href="/wiki/Atracci%C3%B3n_de_ramas_largas" title="Atracción de ramas largas">atracción de ramas largas</a>. Por el contrario, la otra hipótesis "<a class="mw-redirect" href="/wiki/Urbilateria" title="Urbilateria">Urbilateria</a>" considera que el ancestro de los bilaterios fue un celomado (es decir, complejo) y con características de deuteróstomo derivándose los restantes celomados, los pseudocelomados y acelomados. Esta hipótesis es la más probable para el primer ancestro bilateral. Dentro de Bilateria dos clados han sido bien establecidos <a href="/wiki/Protostomia" title="Protostomia">Protostomia</a> respaldado por todos los análisis moleculares y <a href="/wiki/Deuterostomia" title="Deuterostomia">Deuterostomia</a> respaldado por la mayoría de los análisis moleculares, aunque algunos más recientes han cuestionado la monofilia de los deuteróstomos sugiriendo que son <a href="/wiki/Parafil%C3%A9tico" title="Parafilético">parafiléticos</a>. Estos se definen por el desarrollo embrionario, en protóstomos la boca se desarrolla primero y el ano después, en cambio en los deuteróstomos es lo contrario el ano primero y la boca por último. Los análisis moleculares han demostrado que los pseudocelomados y acelomados (exceptuando los xenacelomorfos) son protóstomos reducidos y que evolucionaron de ancestros celomados en diferentes líneas evolutivas por lo que son grupos <a href="/wiki/Polifil%C3%A9tico" title="Polifilético">polifiléticos</a>. Estos filos se han clasificados junto con los <a class="mw-redirect" href="/wiki/An%C3%A9lido" title="Anélido">anélidos</a>, <a class="mw-redirect" href="/wiki/Molusco" title="Molusco">moluscos</a>, <a class="mw-redirect" href="/wiki/Artr%C3%B3podo" title="Artrópodo">artrópodos</a>, en los clados <a href="/wiki/Spiralia" title="Spiralia">Spiralia</a> y <a href="/wiki/Ecdysozoa" title="Ecdysozoa">Ecdysozoa</a> respectivamente. Antiguamente se creía que los anélidos estaban más estrechamente con los artrópodos y que los <a class="mw-redirect" href="/wiki/Lofoforado" title="Lofoforado">lofoforados</a> con <a class="mw-redirect" href="/wiki/Quetognatos" title="Quetognatos">quetognatos</a> eran deuteróstomos, pero los análisis moleculares han demostrado que estos últimos pertenecen Spiralia junto con anélidos, moluscos, etc. La posición de los entoproctos que fue incierta durante muchos también se resolvió en Spiralia. El clado Spiralia contiene los (<a class="mw-redirect" href="/wiki/Moluscos" title="Moluscos">moluscos</a>, <a class="mw-redirect" href="/wiki/An%C3%A9lidos" title="Anélidos">anélidos</a>, <a class="mw-redirect" href="/wiki/Lofoforados" title="Lofoforados">lofoforados</a>, <a class="mw-redirect" href="/wiki/Quetognatos" title="Quetognatos">quetognatos</a>, <a class="mw-redirect" href="/wiki/Platelmintos" title="Platelmintos">platelmintos</a>, <a class="mw-redirect" href="/wiki/Nemertinos" title="Nemertinos">nemertinos</a>, <a class="mw-redirect" href="/wiki/Gastrotricos" title="Gastrotricos">gastrotricos</a>, <a class="mw-redirect" href="/wiki/Rot%C3%ADferos" title="Rotíferos">rotíferos</a>, <a class="mw-redirect" href="/wiki/Entoproctos" title="Entoproctos">entoproctos</a>, <a class="mw-redirect" href="/wiki/Cicli%C3%B3foros" title="Ciclióforos">ciclióforos</a>, <a class="mw-redirect" href="/wiki/Gnatostom%C3%BAlidos" title="Gnatostomúlidos">gnatostomúlidos</a>, <a class="mw-redirect" href="/wiki/Micrognatozoos" title="Micrognatozoos">micrognatozoos</a>, <a class="mw-redirect" href="/wiki/Rombozoos" title="Rombozoos">rombozoos</a>), y el clado Ecdysozoa (<a class="mw-redirect" href="/wiki/Artr%C3%B3podo" title="Artrópodo">artrópodos</a>, <a class="mw-redirect" href="/wiki/Tard%C3%ADgrados" title="Tardígrados">tardígrados</a>, <a class="mw-redirect" href="/wiki/Onic%C3%B3foros" title="Onicóforos">onicóforos</a>, <a class="mw-redirect" href="/wiki/Nematodos" title="Nematodos">nematodos</a>, <a class="mw-redirect" href="/wiki/Nematomorfos" title="Nematomorfos">nematomorfos</a>, <a class="mw-redirect" href="/wiki/Quinorrincos" title="Quinorrincos">quinorrincos</a>, <a class="mw-redirect" href="/wiki/Priap%C3%BAlidos" title="Priapúlidos">priapúlidos</a> y <a class="mw-redirect" href="/wiki/Loric%C3%ADferos" title="Loricíferos">loricíferos</a>). Estos clados se han definido por la segmentación espiral del huevo y la muda de cutícula (<a href="/wiki/Ecdisis" title="Ecdisis">ecdisis</a>). En los análisis moleculares los filos pseudocelomados ocupan las posiciones más basales en Spiralia y Ecdysozoa lo que sugiere que los ancestros de los celomados protóstomos (ej.- artrópodos, moluscos, anélidos, etc.) fueron pseudocelomados. El ancestro de los platelmintos y nemertinos reducidos pudo haber sido un celomado emparentado con los anélidos. Por otra parte el supergrupo Deuterostomia incluye a los (<a class="mw-redirect" href="/wiki/Cordados" title="Cordados">cordados</a>, <a class="mw-redirect" href="/wiki/Equinodermos" title="Equinodermos">equinodermos</a>, <a class="mw-redirect" href="/wiki/Hemicordados" title="Hemicordados">hemicordados</a> y <a class="mw-redirect" href="/wiki/Xenacelomorfos" title="Xenacelomorfos">xenacelomorfos</a>), de estos filos los equinodermos y hemicordados forman un clado bien soportado por los análisis moleculares y morfológicos, el cual puede estar relacionado con los xenacelomorfos, a su vez los cordados conforman la rama más basal. En la evolución de los animales ha habido un grupo de animales que han sido difíciles de relacionar y que tradicionalmente se consideraron los más primitivos por su carácter simple se trata de los mesozoos (placozoos, <a href="/wiki/Orthonectida" title="Orthonectida">ortonéctidos</a> y <a class="mw-redirect" href="/wiki/Rombozoos" title="Rombozoos">rombozoos</a>), sin embargo se ha demostrado que son formas totalmente reducidas por el <a href="/wiki/Parasitismo" title="Parasitismo">parasitismo</a> y que de hecho no tienen ningún parentesco. Los placozoos serían radiados reducidos, los ortonéctidos anélidos reducidos y los rombozoos descendientes de un linaje espiralio (Spiralia) reducido probablemente emparentado con los anélidos, platelmintos y nemertinos. Algunos <a href="/wiki/Filo" title="Filo">filos</a> fueron clasificados en otros filos como <a href="/wiki/Echiura" title="Echiura">Echiura</a>, <a class="mw-redirect" href="/wiki/Pogonophora" title="Pogonophora">Pogonophora</a>, <a href="/wiki/Sipuncula" title="Sipuncula">Sipuncula</a> y <a href="/wiki/Orthonectida" title="Orthonectida">Orthonectida</a> que si bien no tienen la segmentación y metamerización típica de los anélidos la perdieron durante su evolución. <a href="/wiki/Acanthocephala" title="Acanthocephala">Acanthocephala</a> tradicionalmente considerado un filo separado se clasifican en los rotíferos, ya que evolucionaron de rotíferos según los análisis moleculares y morfológicos. También algunos animales llegaron al punto de ser <a class="mw-redirect" href="/wiki/Unicelular" title="Unicelular">unicelulares</a> como <a href="/wiki/Myxozoa" title="Myxozoa">Myxozoa</a> que anteriormente se clasificaba como <a href="/wiki/Protista" title="Protista">protistas</a>. Los análisis moleculares han demostrado que Myxozoa evolucionó de un ancestro cnidario pluricelular emparentado con <a href="/wiki/Polypodiozoa" title="Polypodiozoa">Polypodiozoa</a>.<sup <p>El siguiente cladograma representa las relaciones filogenéticas entre los diversos filos de animales. Está basada en la segunda edición de Brusca &amp; Brusca (2005);<sup class="reference separada" id="cite_ref-Brusca_19-1"><a href="#cite_note-Brusca-19"><span class="corchete-llamada">[</span>19<span class="corchete-llamada">]</span></a></sup>​ se trata de una hipótesis filogenética "clásica" en la que se reconocen los grandes clados admitidos tradicionalmente (<a class="mw-redirect" href="/wiki/Pseudocelomados" title="Pseudocelomados">pseudocelomados</a>, <a href="/wiki/Articulata_(Protostomia)" title="Articulata (Protostomia)">articulados</a>, etc.) y asume la teoría colonial como la explicación sobre el origen de los metazoos. Investigaciones más recientes ofrecen una visión algo diferente.

<p>Según el punto de vista que se acaba de exponer, los <a href="/wiki/Bilateria" title="Bilateria">bilaterales</a> se subdividen en cuatro grandes linajes:
</p>
<p>Las modernas técnicas de secuenciación de <a href="/wiki/Base_nitrogenada" title="Base nitrogenada">bases</a> del <a class="mw-redirect" href="/wiki/ADN" title="ADN">ADN</a>, <a class="mw-redirect" href="/wiki/ARN" title="ARN">ARN</a> y <a class="mw-redirect" href="/wiki/Prote%C3%ADnas" title="Proteínas">proteínas</a> junto con la metodología de la <a href="/wiki/Clad%C3%ADstica" title="Cladística">cladística</a> han permitido reinterpretar las relaciones filogenéticas de los distintos <a href="/wiki/Filo" title="Filo">filos</a> animales, lo que ha conducido a una revolución en la clasificación de los mismos. Actualmente la mayoría de los zoólogos aceptan las nuevas clasificaciones. Los animales <a href="/wiki/Bilateria" title="Bilateria">bilaterales</a> parecen pertenecer a uno de estos tres linajes:

<p>La <a href="/wiki/Filogenia" title="Filogenia">filogenia</a> de los animales al igual que otras filogenias es un tema no resuelto y todavía no se ha alcanzado un consenso amplio en cuanto a la relación filogenética de los determinados filos debido a la gran cantidad de hipótesis alternativas que se sugieren en los diversos artículos científicos. La <a href="/wiki/Filogen%C3%A9tica_molecular" title="Filogenética molecular">filogenética molecular</a> ha producido una revolución en las relaciones filogenéticas de los animales reemplazando muchas hipótesis tradicionales. Tampoco existe muchos datos morfológicos, fósiles y biogeográficos fiables como para tomar hipótesis alternativas. Especialmente las relaciones filogenéticas de los superfilos <a href="/wiki/Spiralia" title="Spiralia">Spiralia</a><sup class="reference separada" id="cite_ref-38"><a href="#cite_note-38"><span class="corchete-llamada">[</span>38<span class="corchete-llamada">]</span></a></sup>​ y <a href="/wiki/Ecdysozoa" title="Ecdysozoa">Ecdysozoa</a><sup class="reference separada" id="cite_ref-39"><a href="#cite_note-39"><span class="corchete-llamada">[</span>39<span class="corchete-llamada">]</span></a></sup>​ son tan ambiguas que no se puede llegar a un consenso sobre cual es la filogenia correcta.<sup class="reference separada" id="cite_ref-spiralia_40-0"><a href="#cite_note-spiralia-40"><span class="corchete-llamada">[</span>40<span class="corchete-llamada">]</span></a></sup>​ En cuanto a grupos extintos <a href="/wiki/Vendobionta" title="Vendobionta">Vendobionta</a> es un completo enigma evolutivo, tentativamente se le puede considerar en la base de la filogenia animal, aunque solo comparte con las esponjas su simplicidad y hábitat <a class="mw-redirect" href="/wiki/Bent%C3%B3nico" title="Bentónico">bentónico</a>, además las esponjas serían menos antiguas que aquel.<sup class="reference separada" id="cite_ref-41"><a href="#cite_note-41"><span class="corchete-llamada">[</span>41<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Recientemente se ha sugerido que los filos <a href="/wiki/Rotifera" title="Rotifera">Rotifera</a>, <a href="/wiki/Rhombozoa" title="Rhombozoa">Rhombozoa</a>, <a href="/wiki/Ctenophora" title="Ctenophora">Ctenophora</a>, <a href="/wiki/Placozoa" title="Placozoa">Placozoa</a>, <a href="/wiki/Entoprocta" title="Entoprocta">Entoprocta</a>, <a href="/wiki/Platyhelminthes" title="Platyhelminthes">Platyhelminthes</a>, <a href="/wiki/Gastrotricha" title="Gastrotricha">Gastrotricha</a>, <a href="/wiki/Xenacoelomorpha" title="Xenacoelomorpha">Xenacoelomorpha</a>, <a href="/wiki/Chaetognatha" title="Chaetognatha">Chaetognatha</a> y <a href="/wiki/Gnathostomulida" title="Gnathostomulida">Gnathostomulida</a> tienen tasas de evolución genética, lo que impide reconstruir el árbol filogenético y llevan a las <a href="/wiki/Atracci%C3%B3n_de_ramas_largas" title="Atracción de ramas largas">atracciones de ramas largas</a>.<sup class="reference separada" id="cite_ref-spiralia_40-1"><a href="#cite_note-spiralia-40"><span class="corchete-llamada">[</span>40<span class="corchete-llamada">]</span></a></sup>​ Análisis moleculares recientes (2019) utilizando especies de evolución lenta y otros estudios que intentan evitar el error sistemático han dado resultados posiblemente correctos. A diferencia de otros análisis moleculares convencionales que llevan a resultados contradictorios, estos análisis están de acuerdo con los análisis morfológicos y paleontológicos.
