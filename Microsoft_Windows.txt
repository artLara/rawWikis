<p><b>Windows</b> es el nombre de una familia de <a href="/wiki/Distribuci%C3%B3n_de_software" title="Distribución de software">distribuciones</a> de <i><a href="/wiki/Software" title="Software">software</a></i> para <a href="/wiki/Computadora_personal" title="Computadora personal">PC</a>, <a class="mw-redirect" href="/wiki/Servidores" title="Servidores">servidores</a>, <a class="mw-redirect" href="/wiki/Sistemas_empotrados" title="Sistemas empotrados">sistemas empotrados</a> y antiguamente <a href="/wiki/Tel%C3%A9fono_inteligente" title="Teléfono inteligente">teléfonos inteligentes</a> desarrollados y vendidos por <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> y disponibles para múltiples arquitecturas, tales como <a href="/wiki/X86" title="X86">x86</a>, <a href="/wiki/X86-64" title="X86-64">x86-64</a> (<a class="mw-redirect" href="/wiki/X64" title="X64">x64</a>) y <a href="/wiki/Arquitectura_ARM" title="Arquitectura ARM">ARM</a>.
</p>
<p>Desde un punto de vista técnico, no son <a class="mw-redirect" href="/wiki/Sistemas_operativos" title="Sistemas operativos">sistemas operativos</a>, sino que contienen uno (tradicionalmente <a href="/wiki/MS-DOS" title="MS-DOS">MS-DOS</a> o el más actual cuyo núcleo es <a href="/wiki/Windows_NT" title="Windows NT">Windows NT</a>) junto con una amplia variedad de <a href="/wiki/Software" title="Software">software</a>; no obstante, es usual (aunque no necesariamente correcto) denominar al conjunto como <a href="/wiki/Sistema_operativo" title="Sistema operativo">sistema operativo</a> en lugar de <a href="/wiki/Distribuci%C3%B3n_de_software" title="Distribución de software">distribución</a>. Microsoft introdujo un entorno operativo denominado <i>Windows</i> el 20 de noviembre de <a href="/wiki/1985" title="1985">1985</a> como un complemento para <a href="/wiki/MS-DOS" title="MS-DOS">MS-DOS</a> en respuesta al creciente interés en las interfaces gráficas de usuario (<a class="mw-redirect" href="/wiki/GUI" title="GUI">GUI</a>).<sup class="reference separada" id="cite_ref-aboutcomnov_2-0"><a href="#cite_note-aboutcomnov-2"><span class="corchete-llamada">[</span>2<span class="corchete-llamada">]</span></a></sup>​ Microsoft Windows llegó a dominar el mercado mundial de <a class="mw-redirect" href="/wiki/Computador" title="Computador">computadoras</a> personales, con más del 70 % de la cuota de mercado, superando a <a href="/wiki/Mac_OS" title="Mac OS">Mac OS</a>, que había sido introducido en <a href="/wiki/1984" title="1984">1984</a>.
</p>
<p>La versión más reciente de Windows es <a href="/wiki/Windows_11" title="Windows 11">Windows 11</a> para <a href="/wiki/Computadora_de_escritorio" title="Computadora de escritorio">equipos de escritorio</a>, Windows Server 2022 para <a href="/wiki/Windows_NT" title="Windows NT">servidores</a> y <a href="/wiki/Windows_10_Mobile" title="Windows 10 Mobile">Windows 10 Mobile</a> para <a class="mw-redirect" href="/wiki/Dispositivos_m%C3%B3viles" title="Dispositivos móviles">dispositivos móviles</a>. La primera versión en español fue <a href="/wiki/Windows_2.1x" title="Windows 2.1x">Windows 2.1</a>.
</p>
<p>La primera versión se lanzó en <a href="/wiki/1985" title="1985">1985</a> y comenzó a utilizarse de forma generalizada gracias a su <a href="/wiki/Interfaz_gr%C3%A1fica_de_usuario" title="Interfaz gráfica de usuario">interfaz gráfica de usuario</a> (GUI, <i>Graphical User Interface</i>) basada en ventanas. Hasta ese momento (y hasta mucho después como corazón de Windows), el sistema operativo más extendido era <a href="/wiki/MS-DOS" title="MS-DOS">MS-DOS</a> (<i>Microsoft Disk Operating System</i>), que por aquel entonces contaba con una interfaz basada en <a class="mw-redirect" href="/wiki/L%C3%ADnea_de_comandos" title="Línea de comandos">línea de comandos</a>.
</p>
<p><a href="/wiki/Windows_10" title="Windows 10">Windows 10</a> llegó de forma oficial y gratuita a usuarios con licencia genuina de <a href="/wiki/Windows_7" title="Windows 7">Windows 7</a>, <a href="/wiki/Windows_8" title="Windows 8">Windows 8</a> y <a href="/wiki/Windows_8.1" title="Windows 8.1">Windows 8.1</a> el 29 de julio de 2015, es la primera versión que buscó la unificación de dispositivos (escritorio, portátiles, teléfonos inteligentes, tabletas y videoconsolas) bajo una experiencia común. Su reemplazo <a href="/wiki/Windows_11" title="Windows 11">Windows 11</a> fue lanzado oficialmente el 5 de octubre de 2021, como una actualización gratuita a través de <a href="/wiki/Windows_Update" title="Windows Update">Windows Update</a> de Windows 10 para aquellos equipos que cumpla con ciertas especificaciones técnicas compatibles del nuevo sistema. Microsoft promovió que Windows 11 habría mejorado el rendimiento y la facilidad de uso sobre Windows 10, contando con cambios importantes en el Shell de Windows influenciados por el cancelado <a class="mw-redirect" href="/wiki/Windows_10X" title="Windows 10X">Windows 10X</a>, incluido un menú Inicio rediseñado, así como la incompatibilidad con la arquitectura x86 de 32 bits o los sistemas que usan firmware del <a href="/wiki/BIOS" title="BIOS">BIOS</a>.
</p>
<p>.17 %
</p>
<p>La primera versión de Microsoft Windows, versión 1.0, presentada en diciembre de <a href="/wiki/1985" title="1985">1985</a>, compitió con el sistema operativo de <a href="/wiki/Apple" title="Apple">Apple</a>. Carecía de un cierto grado de funcionalidad y logró muy poca popularidad. Windows 1.0 no era un sistema operativo completo; más bien era una extensión gráfica de MS-DOS.
</p>
<p>Windows versión 2.0 fue lanzado en noviembre de 1987 y fue un poco más popular que su predecesor. Windows 2.03 (lanzado en enero de <a href="/wiki/1988" title="1988">1988</a>) incluyó por primera vez ventanas que podían solaparse unas a otras. El resultado de este cambio llevó a <a class="mw-redirect" href="/wiki/Apple_Inc." title="Apple Inc.">Apple</a> a presentar una demanda contra Microsoft, debido a que infringían derechos de autor.
</p>
<p>Windows versión 3.0, lanzado en <a href="/wiki/1990" title="1990">1990</a>, fue la primera versión de Microsoft Windows que consiguió un amplio éxito comercial, vendiendo 2 millones de copias en los primeros seis meses. Presentaba mejoras en la interfaz de usuario y en la multitarea. Recibió un lavado de cara en Windows 3.1, que se hizo disponible para el público en general el 1 de marzo de 1992. El soporte de Windows 3.1 terminó el 31 de diciembre de 2001.
</p>
<p>El 24 de agosto de <a href="/wiki/1995" title="1995">1995</a>, Microsoft lanzó <a href="/wiki/Windows_95" title="Windows 95">Windows 95</a>, una nueva versión del sistema operativo Windows destinada al mercado de consumo pensada para sustituir a <a class="mw-redirect" href="/wiki/Windows_3.1" title="Windows 3.1">Windows 3.1</a> como interfaz gráfica y a <a href="/wiki/MS-DOS" title="MS-DOS">MS-DOS</a> como sistema operativo. En esta edición se introdujeron mejoras muy significativas con respecto a sus antecesores entre las cuales se pueden mencionar los profundos cambios realizados a la interfaz gráfica de usuario de Windows, que era completamente distinta a las de versiones anteriores, y el pasar de usar una arquitectura multitarea cooperativa de 16 bits a usar una arquitectura multitarea apropiativa de 32 bits.
</p>
<p>Windows 95 fue la primera versión de Windows en incluir la barra de tareas y el botón Inicio, los cuales siguieron incluyéndose en versiones posteriores de Windows, además de ser la primera versión en soportar la función de Plug and Play.
</p>
<p>Acompañado por una extensa y millonaria campaña de marketing, Windows 95 fue un gran éxito en el mercado en el momento de su lanzamiento y en breve se convirtió en uno de los sistemas operativos de escritorio más populares.
</p>
<p>El siguiente para la línea del usuario fue lanzado el 25 de junio de <a href="/wiki/1998" title="1998">1998</a>, Microsoft <a href="/wiki/Windows_98" title="Windows 98">Windows 98</a>. Sustancialmente fue criticado por su lentitud y por su falta de fiabilidad en comparación con Windows 95, pero muchos de sus problemas básicos fueron posteriormente rectificados con el lanzamiento de Windows 98 Second Edition en <a href="/wiki/1999" title="1999">1999</a>. El soporte estándar para Windows 98 terminó el 30 de junio de 2002, y el soporte ampliado para Windows 98 terminó el 11 de julio de 2006.
</p>
<p>Como parte de su línea «profesional», Microsoft lanzó <a href="/wiki/Windows_2000" title="Windows 2000">Windows 2000</a> en febrero de <a href="/wiki/2000" title="2000">2000</a>. La versión de consumidor tras Windows 98 fue Windows Me (Windows Millennium Edition). Windows Me implementaba una serie de nuevas tecnologías para Microsoft: en particular fue el «Universal Plug and Play». Durante el 2004 parte del código fuente de Windows 2000 se filtró en internet, esto era malo para Microsoft porque el mismo núcleo utilizado en Windows 2000 se utilizó en Windows XP.
</p>
<p>En octubre de 2001, Microsoft lanzó <a href="/wiki/Windows_XP" title="Windows XP">Windows XP</a>, una versión que se construyó en el kernel de Windows NT que también conserva la usabilidad orientada al consumidor de Windows 95 y sus sucesores. En dos ediciones distintas, «Home» y «Professional», el primero carece en gran medida de la seguridad y características de red de la edición Professional. Además, la primera edición «Media Center» fue lanzada en 2004, con énfasis en el apoyo a la funcionalidad de <a href="/wiki/DVD" title="DVD">DVD</a> y TV, por lo que incluía grabación de TV y un control remoto. A principios de la década de los 2000, Windows se empezaba a posicionar cómo el innovador en el mercado, pero su posición fue en declive.
</p>
<p>El 30 de enero de 2007, Microsoft lanzó <a href="/wiki/Windows_Vista" title="Windows Vista">Windows Vista</a>. Contiene un sinnúmero de características nuevas, a partir de un shell rediseñado, y la interfaz de usuario ofrece importantes cambios técnicos de envergadura, con especial atención a las características de seguridad. Está disponible en seis ediciones diferentes.
</p>
<p>El 22 de octubre de 2009, Microsoft lanzó <a href="/wiki/Windows_7" title="Windows 7">Windows 7</a>. A diferencia de su predecesor, Windows Vista, que introdujo a un gran número de nuevas características, Windows 7 pretendía ser una actualización incremental, enfocada a la línea de Windows, con el objetivo de ser compatible con aplicaciones y hardware con los que Windows Vista no era compatible. Windows 7 tiene soporte multi-touch, un Windows shell rediseñado con una nueva barra de tareas, conocido como Superbar, un sistema red llamado HomeGroup, y mejoras en el rendimiento sobre todo en velocidad y en menor consumo de recursos.
</p>
<p>El 26 de octubre de 2012, Microsoft lanzó <a href="/wiki/Windows_8" title="Windows 8">Windows 8</a>. El mayor cambio introducido es el reemplazo del Menú Inicio por una pantalla de Inicio de tamaño completo, la cual incluye nuevas aplicaciones. Su uso está enfatizado para dispositivos con pantallas táctiles, aunque puede ser utilizado con ratón y teclado. Por primera vez desde Windows 95, el botón de Inicio desaparece de la barra de tareas. El escritorio presenta una nueva interfaz y el explorador de Windows incluye la apariencia "Ribbon" de <a href="/wiki/Microsoft_Office" title="Microsoft Office">Microsoft Office</a>. Una actualización masiva del sistema, Windows 8.1, fue lanzada el 17 de octubre de 2013 con nuevas mejoras de personalización, rendimiento y un botón para la pantalla de inicio, cuya ausencia en Windows 8 fue criticada.
</p>
<p>El 29 de julio de 2015, Microsoft lanzó <a href="/wiki/Windows_10" title="Windows 10">Windows 10</a>. Presenta un conjunto de aplicaciones y una interfaz que permite utilizarse en computadoras personales y dispositivos móviles. Visualmente es parecido a su predecesor, sin embargo, el Menú Inicio regresa en esta versión. Windows 10 introduce un nuevo navegador: <a href="/wiki/Microsoft_Edge" title="Microsoft Edge">Microsoft Edge</a>, y un asistente de voz personal: <a href="/wiki/Microsoft_Cortana" title="Microsoft Cortana">Cortana</a>, junto con una interfaz de sistema minimalista. Se trata de un sistema que a diferencia de sus predecesores, lanzará actualizaciones masivas periódicamente para convertirse en un sistema de servicio. Hasta el 29 de julio de 2016, Microsoft ofreció gratuitamente una actualización a Windows 10 desde una PC con <a href="/wiki/Windows_7" title="Windows 7">Windows 7</a> o <a href="/wiki/Windows_8.1" title="Windows 8.1">Windows 8.1</a>.
</p>
<p>El 24 de junio de 2021, Microsoft anunció y confirmó <a href="/wiki/Windows_11" title="Windows 11">Windows 11</a> el sistema operativo que se lanzó oficialmente el 5 de octubre de 2021. Incorpora una interfaz de usuario basada en <a href="/wiki/Fluent_Design" title="Fluent Design">Fluent Design</a> incluyendo transparencias, sombras, íconos redondeados en todo el sistema. El <a href="/wiki/Men%C3%BA_Inicio" title="Menú Inicio">menú Inicio</a> sufre un rediseño que elimina los <i>tiles</i> del lateral. Los iconos de la <a href="/wiki/Barra_de_tareas" title="Barra de tareas">barra de tareas</a> incluyendo en botón Inicio aparecen centrados en la mitad de la pantalla, un diseño que ha sido comparado al del <i>dock</i> de <a href="/wiki/MacOS" title="MacOS">macOS</a>.
</p>
<p>Las versiones <a class="mw-redirect" href="/wiki/Fases_del_desarrollo_de_software" title="Fases del desarrollo de software">Beta</a>, Alpha o de pruebas, son versiones gratuitas que se crean como base para el próximo sistema operativo de Windows a lanzar. En ellas se colocan algunas funciones que Microsoft ha puesto por defecto y que las lanza al público para demostrarles a los usuarios cómo puede ser la futura versión de Windows. Mientras estas versiones estén activas, los usuarios pueden reportar errores que se encuentren en el sistema operativo, (ya que no tienen soporte para actualizaciones) y si el reporte es solucionado, Microsoft le dará una cantidad de dinero al usuario que ha reportado dicho error.
</p>
<p>Las versiones de prueba son muy diferentes a las originales, puesto que no se comercializan y que tienen nombres distintos. Por ejemplo, la versión que se utilizó para crear Windows 95, fue nominada "Chicago". Estos nombres también se suelen llamar "Codenames". En algunas versiones de prueba (como Windows 8 BETA), no llevan un respectivo codename, y por este motivo llevan el nombre de la próxima versión de Windows de la respectiva versión de pruebas, junto a su indicador "Beta".
</p>
<p>5 años y 10 meses
</p>
<p>Una de las principales críticas que reciben los sistemas operativos Windows es la debilidad del sistema en lo que a <a href="/wiki/Seguridad_inform%C3%A1tica" title="Seguridad informática">seguridad</a> se refiere y el alto índice de <a href="/wiki/Agujero_de_seguridad" title="Agujero de seguridad">vulnerabilidades</a> críticas. El propio <a href="/wiki/Bill_Gates" title="Bill Gates">Bill Gates</a>, fundador de Microsoft, ha asegurado en repetidas ocasiones que la seguridad es objetivo primordial para su empresa.<sup class="reference separada" id="cite_ref-BillGates_17-0"><a href="#cite_note-BillGates-17"><span class="corchete-llamada">[</span>17<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Partiendo de la base de que no existe un sistema completamente libre de <a href="/wiki/Error_de_software" title="Error de software">errores</a>, las críticas se centran en la lentitud con la que la empresa reacciona ante un problema de seguridad que puede llegar a meses<sup class="reference separada" id="cite_ref-18"><a href="#cite_note-18"><span class="corchete-llamada">[</span>18<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-19"><a href="#cite_note-19"><span class="corchete-llamada">[</span>19<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-20"><a href="#cite_note-20"><span class="corchete-llamada">[</span>20<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-21"><a href="#cite_note-21"><span class="corchete-llamada">[</span>21<span class="corchete-llamada">]</span></a></sup>​ o incluso años<sup class="reference separada" id="cite_ref-22"><a href="#cite_note-22"><span class="corchete-llamada">[</span>22<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-23"><a href="#cite_note-23"><span class="corchete-llamada">[</span>23<span class="corchete-llamada">]</span></a></sup>​
de diferencia desde que se avisa de la vulnerabilidad hasta que se publica un <a href="/wiki/Parche_(inform%C3%A1tica)" title="Parche (informática)">parche</a>.
</p>
<p>En algunos casos la falta de respuesta por parte de Microsoft<sup class="reference separada" id="cite_ref-24"><a href="#cite_note-24"><span class="corchete-llamada">[</span>24<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-25"><a href="#cite_note-25"><span class="corchete-llamada">[</span>25<span class="corchete-llamada">]</span></a></sup>​ ha provocado que se desarrollen parches que arreglan problemas de seguridad hechos por terceros.<sup class="reference separada" id="cite_ref-26"><a href="#cite_note-26"><span class="corchete-llamada">[</span>26<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Uno de los pilares en que se basa la seguridad de los productos Windows es la <a class="mw-redirect" href="/wiki/Seguridad_por_ocultaci%C3%B3n" title="Seguridad por ocultación">seguridad por ocultación</a>, en general, un aspecto característico del <a href="/wiki/Software_propietario" title="Software propietario">software propietario</a> que sin embargo parece ser uno de los responsables de la debilidad de este sistema operativo ya que, la propia seguridad por ocultación, constituye una infracción del <a class="mw-redirect" href="/wiki/Principio_de_Kerckhoff" title="Principio de Kerckhoff">principio de Kerckhoff</a>, el cual afirma que la seguridad de un sistema reside en su diseño y no en una supuesta ignorancia del diseño por parte del atacante.<sup class="reference separada" id="cite_ref-27"><a href="#cite_note-27"><span class="corchete-llamada">[</span>27<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El 6 de enero de 2005, Microsoft lanzó una versión Beta de Microsoft AntiSpyware, basado en <a class="new" href="/w/index.php?title=Giant_AntiSpyware&amp;action=edit&amp;redlink=1" title="Giant AntiSpyware (aún no redactado)">Giant AntiSpyware</a> publicado anteriormente. El 14 de febrero de 2006, Microsoft AntiSpyware se convirtió en <a class="mw-redirect" href="/wiki/Windows_Defender" title="Windows Defender">Windows Defender</a> con el lanzamiento de la Beta 2. Windows Defender era un programa diseñado para proteger contra <a href="/wiki/Programa_esp%C3%ADa" title="Programa espía">programas espía</a> y otro software no deseado. Los usuarios de <a href="/wiki/Windows_XP" title="Windows XP">Windows XP</a> y <a href="/wiki/Windows_Server_2003" title="Windows Server 2003">Windows Server 2003</a> que tienen copias originales de Microsoft Windows pueden descargar libremente el programa desde el sitio web de Microsoft y Windows Defender se suministra como parte de <a href="/wiki/Windows_Vista" title="Windows Vista">Windows Vista</a> y <a href="/wiki/Windows_7" title="Windows 7">Windows 7</a>. Windows Defender y <a href="/wiki/Microsoft_Security_Essentials" title="Microsoft Security Essentials">Microsoft Security Essentials</a> se han combinado en un solo programa, denominado Windows Defender. Sus características y la interfaz de usuario se basan en Microsoft Security Essentials. Así mismo, Windows Defender se transformó en un programa antivirus y Spyware. A pesar de que está activada de forma predeterminada, se puede desactivar para utilizar otra solución antivirus.
</p>
<p>En todas las versiones de Windows con NT 3 se han basado en un sistema de permisos de sistema de archivos denominado AGDLP (cuentas, Global, Local, permisos) AGLP que en esencia donde se aplican los permisos de archivo a la carpeta en forma de un grupo local que luego tiene otros 'grupos globales' como miembros. Estos grupos globales mantienen otros grupos o a usuarios según las diferentes versiones de Windows que utiliza. Este sistema varía de otros productos de proveedores tales como Linux y NetWare debido a la 'estática' asignación de permiso se aplica directorio para el archivo o carpeta. Sin embargo con este proceso de AGLP/AGDLP/AGUDLP permite a un pequeño número de permisos estáticos para aplicarse y permite cambios fáciles a los grupos de cuentas sin volver a aplicar los permisos de archivo de los archivos y carpetas.
</p>
<p>Son aplicaciones que trae pre-instaladas el sistema operativo <a href="/wiki/Windows_10" title="Windows 10">Windows 10</a>.
</p>
<p>Microsoft fue demandado por el gobierno de Estados Unidos durante el año 1998 por competencia desleal, y especialmente por la inclusión del navegador <a href="/wiki/Internet_Explorer" title="Internet Explorer">Internet Explorer</a> en su versión <a href="/wiki/Windows_95" title="Windows 95">Windows 95</a>.<sup class="reference separada" id="cite_ref-30"><a href="#cite_note-30"><span class="corchete-llamada">[</span>30<span class="corchete-llamada">]</span></a></sup>​ Tras esta denuncia, la pena le conllevó a Microsoft lanzar una versión de Windows 95 sin su navegador de Internet.
</p>
<p>En 2000 <a class="mw-redirect" href="/wiki/Windows_ME" title="Windows ME">Windows ME</a> se convirtió en un blanco de críticas debido a sus constantes fallos del sistema, que terminaban con un <a href="/wiki/Pantalla_azul_de_la_muerte" title="Pantalla azul de la muerte">pantallazo azul</a> en la mayoría de los casos, además de consumir mayores recursos de lo que se decía. <a href="/wiki/PC_World" title="PC World">PC World</a> incluso posicionó a Windows ME el cuarto "peor producto tecnológico" de la historia en el artículo «Top 25 de los peores productos tecnológicos».
</p>
<p>En 2004 Microsoft lanzó una campaña, llamada «Vayamos a los hechos», en la que mostraba cientos de empresas conocidas que migraron de <a href="/wiki/GNU/Linux" title="GNU/Linux">GNU/Linux</a> a Windows Server y aumentaron su productividad y otro tipo de comparativas,<sup class="reference separada" id="cite_ref-31"><a href="#cite_note-31"><span class="corchete-llamada">[</span>31<span class="corchete-llamada">]</span></a></sup>​ aunque la cerró poco después, en 2007.<sup class="reference separada" id="cite_ref-32"><a href="#cite_note-32"><span class="corchete-llamada">[</span>32<span class="corchete-llamada">]</span></a></sup>​ Los defensores de GNU/Linux desarrollaron su propio estudio argumentando que, en contra de uno de los reclamos de Microsoft, GNU/Linux tiene menores costos administrativos que <a class="new" href="/w/index.php?title=Microsoft_Servers&amp;action=edit&amp;redlink=1" title="Microsoft Servers (aún no redactado)">servidores basados en Windows</a>.<sup class="reference separada" id="cite_ref-33"><a href="#cite_note-33"><span class="corchete-llamada">[</span>33<span class="corchete-llamada">]</span></a></sup>​ Otro estudio realizado por el <i>Yankee Group</i> afirma que la actualización desde una versión de Windows Server a otra plataforma tiene un costo inferior al de cambiar de GNU/Linux a Windows Server.<sup class="reference separada" id="cite_ref-34"><a href="#cite_note-34"><span class="corchete-llamada">[</span>34<span class="corchete-llamada">]</span></a></sup>​
</p>
