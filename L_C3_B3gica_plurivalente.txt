<p>Una <b>lógica plurivalente</b> o <b>lógica polivalente</b> es un <a class="mw-redirect" href="/wiki/Sistema_l%C3%B3gico" title="Sistema lógico">sistema lógico</a> que rechaza el <a href="/wiki/Principio_del_tercero_excluido" title="Principio del tercero excluido">principio del tercero excluido</a> de las <a href="/wiki/L%C3%B3gica_bivalente" title="Lógica bivalente">lógicas bivalentes</a> y admite más valores de verdad que los tradicionales <i>verdadero</i> y <i>falso</i>.<sup class="reference separada" id="cite_ref-1"><a href="#cite_note-1"><span class="corchete-llamada">[</span>1<span class="corchete-llamada">]</span></a></sup>​ Distintas lógicas plurivalentes pueden admitir distintas cantidades de valores de verdad: desde tres, hasta infinito (cualquier número real entre 0 y 1).
</p>
<p>Las lógicas polivalentes se difundieron especialmente a partir de los trabajos de los filósofos polacos <a href="/wiki/Jan_%C5%81ukasiewicz" title="Jan Łukasiewicz">Jan Łukasiewicz</a> y <a class="mw-redirect" href="/wiki/Emil_Post" title="Emil Post">Emil Post</a> y sus relaciones con la <a class="mw-redirect" href="/wiki/F%C3%ADsica_cu%C3%A1ntica" title="Física cuántica">física cuántica</a>, pero fueron expuestas anteriormente, con diferentes enfoques, por <a class="mw-redirect" href="/wiki/Hegel" title="Hegel">Hegel</a>, <a href="/wiki/Hugh_MacColl" title="Hugh MacColl">Hugh MacColl</a>, <a href="/wiki/Charles_Sanders_Peirce" title="Charles Sanders Peirce">Charles Sanders Peirce</a> y <a class="mw-redirect" href="/wiki/Nicolai_A._Vasiliev" title="Nicolai A. Vasiliev">Nicolai A. Vasiliev</a>. <a href="/wiki/Stephen_Kleene" title="Stephen Kleene">Stephen Kleene</a> elaboró las <a class="mw-redirect" href="/wiki/Tablas_de_verdad" title="Tablas de verdad">tablas de verdad</a> para un sistema de lógica trivalente. Un ejemplo para ilustrar la trivalencia en <a href="/wiki/F%C3%ADsica" title="Física">física</a> ha sido la <a href="/wiki/Paradoja" title="Paradoja">paradoja</a> del <a href="/wiki/Gato_de_Schr%C3%B6dinger" title="Gato de Schrödinger">gato de Schrödinger</a>.
</p>
