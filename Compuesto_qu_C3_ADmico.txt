<p>Un <b>compuesto químico</b> es una <a href="/wiki/Sustancia_qu%C3%ADmica" title="Sustancia química">sustancia</a> formada por la combinación química de dos o más <a href="/wiki/Elemento_qu%C3%ADmico" title="Elemento químico">elementos</a> de la <a href="/wiki/Tabla_peri%C3%B3dica_de_los_elementos" title="Tabla periódica de los elementos">tabla periódica</a>.<sup class="reference separada" id="cite_ref-1"><a href="#cite_note-1"><span class="corchete-llamada">[</span>1<span class="corchete-llamada">]</span></a></sup>​ Los compuestos son representados por una <a href="/wiki/F%C3%B3rmula_qu%C3%ADmica" title="Fórmula química">fórmula química</a>. Por ejemplo, el <a href="/wiki/Agua" title="Agua">agua</a> (H<sub>2</sub>O) está constituida por dos <a href="/wiki/%C3%81tomo" title="Átomo">átomos</a> de <a href="/wiki/Hidr%C3%B3geno" title="Hidrógeno">hidrógeno</a> y uno de <a href="/wiki/Ox%C3%ADgeno" title="Oxígeno">oxígeno</a>. Los elementos de un compuesto no se pueden dividir ni separar por procesos físicos (<a href="/wiki/Decantaci%C3%B3n" title="Decantación">decantación</a>, <a href="/wiki/Filtraci%C3%B3n" title="Filtración">filtración</a>, <a href="/wiki/Destilaci%C3%B3n" title="Destilación">destilación</a>), sino solo mediante <a href="/wiki/Reacci%C3%B3n_qu%C3%ADmica" title="Reacción química">procesos químicos</a>. 
</p>
<p>Los compuestos están formados por <a href="/wiki/Mol%C3%A9cula" title="Molécula">moléculas</a> o <a class="mw-redirect" href="/wiki/Iones" title="Iones">iones</a> con <a class="mw-redirect" href="/wiki/Enlace_qu%C3%ADmico" title="Enlace químico">enlaces</a> estables que no obedece a una selección humana arbitraria. Por lo tanto, no son <a href="/wiki/Mezcla" title="Mezcla">mezclas</a>  o <a href="/wiki/Aleaci%C3%B3n" title="Aleación">aleaciones</a> como el <a href="/wiki/Bronce" title="Bronce">bronce</a> o el <a href="/wiki/Chocolate" title="Chocolate">chocolate</a>.<sup class="reference separada" id="cite_ref-2"><a href="#cite_note-2"><span class="corchete-llamada">[</span>2<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-3"><a href="#cite_note-3"><span class="corchete-llamada">[</span>3<span class="corchete-llamada">]</span></a></sup>​ Un <a href="/wiki/Elemento_qu%C3%ADmico" title="Elemento químico">elemento químico</a> unido a un elemento químico idéntico no es un compuesto químico, ya que solo está involucrado un elemento, no dos elementos diferentes.
</p>
<p>Hay cuatro tipos de compuestos, dependiendo de cómo se mantienen unidos los átomos constituyentes:
</p>
<p>Muchos compuestos <a href="/wiki/Qu%C3%ADmica" title="Química">químicos</a> tienen un identificador numérico único asignado por el <a href="/wiki/Chemical_Abstracts_Service" title="Chemical Abstracts Service">Chemical Abstracts Service</a> (CAS): su <a href="/wiki/N%C3%BAmero_de_registro_CAS" title="Número de registro CAS">número CAS</a>.
</p>
<p>En <a href="/wiki/Qu%C3%ADmica_inorg%C3%A1nica" title="Química inorgánica">química inorgánica</a> los <a href="/wiki/Compuesto_inorg%C3%A1nico" title="Compuesto inorgánico">compuestos</a> se representan mediante <a href="/wiki/S%C3%ADmbolo_qu%C3%ADmico" title="Símbolo químico">fórmulas químicas</a>.<sup class="reference separada" id="cite_ref-4"><a href="#cite_note-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​  Una <a href="/wiki/F%C3%B3rmula_qu%C3%ADmica" title="Fórmula química">fórmula química</a> es una forma de expresar información sobre las proporciones de los átomos que constituyen un compuesto químico en particular, utilizando las abreviaturas normalizadas de los elementos químicos y <a href="/wiki/Sub%C3%ADndice_y_super%C3%ADndice" title="Subíndice y superíndice">subíndices</a> para indicar el número de átomos involucrados. Por ejemplo, <a href="/wiki/Agua" title="Agua">el agua</a> se compone de dos <a href="/wiki/%C3%81tomo_de_hidr%C3%B3geno" title="Átomo de hidrógeno">átomos de hidrógeno</a> unidos a uno <a href="/wiki/Ox%C3%ADgeno" title="Oxígeno">de oxígeno</a> átomo: la fórmula química es H<sub>2</sub>O. En el caso de <a href="/wiki/Compuestos_no_estequiom%C3%A9tricos" title="Compuestos no estequiométricos">compuestos no estequiométricos</a>, las proporciones pueden ser reproducibles con respecto a su preparación y dar proporciones fijas de sus elementos componentes, pero proporciones que no son integrales [por ejemplo, para el <a class="new" href="/w/index.php?title=Hidruro_de_paladio&amp;action=edit&amp;redlink=1" title="Hidruro de paladio (aún no redactado)">hidruro de paladio</a>, PdH <sub>x</sub> (0.02 &lt;x &lt;0.58 )].<sup class="reference separada" id="cite_ref-5"><a href="#cite_note-5"><span class="corchete-llamada">[</span>5<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El orden de los elementos en la fórmula de los compuestos inorgánicos comienza por la izquierda con el elemento menos <a href="/wiki/Electronegatividad" title="Electronegatividad">electronegativo</a>, hasta la derecha con el más electronegativo. Por ejemplo en el <a href="/wiki/Cloruro_de_sodio" title="Cloruro de sodio">NaCl</a>, el <a href="/wiki/Cloro" title="Cloro">cloro</a> que es más electronegativo que el <a href="/wiki/Sodio" title="Sodio">sodio</a> va en la parte derecha.<sup class="reference separada" id="cite_ref-6"><a href="#cite_note-6"><span class="corchete-llamada">[</span>6<span class="corchete-llamada">]</span></a></sup>​ Para los compuestos orgánicos existen otras varias reglas y se utilizan <a href="/wiki/F%C3%B3rmula_esqueletal" title="Fórmula esqueletal">fórmulas esqueletales</a> o <a href="/wiki/F%C3%B3rmula_semidesarrollada" title="Fórmula semidesarrollada">semidesarrolladas</a> para su representación.<sup class="reference separada" id="cite_ref-7"><a href="#cite_note-7"><span class="corchete-llamada">[</span>7<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Cualquier sustancia que consista en dos o más tipos diferentes de <a href="/wiki/%C3%81tomo" title="Átomo">átomos</a> (<a href="/wiki/Elemento_qu%C3%ADmico" title="Elemento químico">elementos químicos</a>) en una proporción <a href="/wiki/Estequiometr%C3%ADa" title="Estequiometría">estequiométrica</a> fija puede denominarse <i>compuesto químico.</i> El concepto se entiende mejor cuando se consideran <a href="/wiki/Sustancia_qu%C3%ADmica" title="Sustancia química">sustancias químicas</a> puras.<sup class="reference separada" id="cite_ref-8"><a href="#cite_note-8"><span class="corchete-llamada">[</span>8<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-9"><a href="#cite_note-9"><span class="corchete-llamada">[</span>9<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-10"><a href="#cite_note-10"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​  De la composición de proporciones fijas de dos o más tipos de átomos se desprende que los compuestos químicos se pueden convertir, mediante <a href="/wiki/Reacci%C3%B3n_qu%C3%ADmica" title="Reacción química">una reacción química</a>, en compuestos o sustancias, cada uno con menos átomos.
</p>
<p>Los compuestos químicos tienen una <a href="/wiki/Estructura_qu%C3%ADmica" title="Estructura química">estructura química</a> única y definida que se mantiene unida en una disposición espacial definida por <a class="mw-redirect" href="/wiki/Enlace_qu%C3%ADmico" title="Enlace químico">enlaces químicos</a>. Los compuestos químicos pueden ser compuestos <a href="/wiki/Mol%C3%A9cula" title="Molécula">moleculares</a>, mantenidos juntos por <a href="/wiki/Enlace_covalente" title="Enlace covalente">enlaces covalentes</a>, <a href="/wiki/Sal_(qu%C3%ADmica)" title="Sal (química)">sales</a>mantenidas entre sí por <a href="/wiki/Enlace_i%C3%B3nico" title="Enlace iónico">enlaces iónicos</a>, compuestos intermetálicos mantenidos juntos por <a href="/wiki/Enlace_met%C3%A1lico" title="Enlace metálico">enlaces metálicos</a>, o el subconjunto de <a href="/wiki/Complejo_(qu%C3%ADmica)" title="Complejo (química)">complejos químicos</a> que se mantienen unidos por <a href="/wiki/Enlace_de_coordinaci%C3%B3n" title="Enlace de coordinación">enlaces covalentes coordinados</a> .<sup class="reference separada" id="cite_ref-:0_11-0"><a href="#cite_note-:0-11"><span class="corchete-llamada">[</span>11<span class="corchete-llamada">]</span></a></sup>​ Los <a href="/wiki/Elemento_qu%C3%ADmico" title="Elemento químico">elementos químicos</a> puros generalmente no se consideran compuestos químicos, ya que no cumplen con el requisito de dos o más átomos, aunque a menudo consisten en moléculas compuestas de múltiples átomos (como en la <a href="/wiki/Mol%C3%A9cula_diat%C3%B3mica" title="Molécula diatómica">molécula diatómica</a> H<sub>2</sub>, o la <a href="/wiki/Mol%C3%A9cula" title="Molécula">molécula poliatómica</a> S<sub>8</sub>, etc.)<sup class="reference separada" id="cite_ref-:0_11-1"><a href="#cite_note-:0-11"><span class="corchete-llamada">[</span>11<span class="corchete-llamada">]</span></a></sup>​ Muchos compuestos químicos tienen un identificador numérico único asignado por el <a href="/wiki/Chemical_Abstracts_Service" title="Chemical Abstracts Service">Chemical Abstracts Service</a> (CAS): su <a href="/wiki/N%C3%BAmero_de_registro_CAS" title="Número de registro CAS">número CAS</a>.<sup class="reference separada" id="cite_ref-12"><a href="#cite_note-12"><span class="corchete-llamada">[</span>12<span class="corchete-llamada">]</span></a></sup>​ 
</p>
<p>Hay nomenclatura variable y a veces inconsistente para diferenciar sustancias, que incluyen ejemplos verdaderamente no estequiométricos de los compuestos químicos, que requieren que las proporciones sean fijas. Muchas sustancias químicas sólidas, por ejemplo muchos <a href="/wiki/Silicato" title="Silicato">minerales de silicato</a>, no tienen fórmulas simples que reflejen el enlace químico de los elementos entre sí en proporciones fijas; aun así, estas sustancias <a href="/wiki/Estructura_cristalina" title="Estructura cristalina">cristalinas</a> a menudo se denominan "<a href="/wiki/Compuestos_no_estequiom%C3%A9tricos" title="Compuestos no estequiométricos">compuestos no estequiométricos</a>". Se puede argumentar que están relacionados con dichos productos, en lugar de ser compuestos químicos propiamente dichos, en la medida en que la variabilidad en sus composiciones a menudo se debe a la presencia de elementos extraños atrapados dentro de la estructura cristalina de un compuesto químico <i>verdadero</i>, o debido a perturbaciones en su estructura en relación con el compuesto conocido que surge debido a un exceso o déficit de los elementos constituyentes en lugares de su estructura; tales sustancias no estequiométricas forman la mayor parte de la <a href="/wiki/Corteza_(geolog%C3%ADa)" title="Corteza (geología)">corteza</a> y el <a href="/wiki/Manto_terrestre" title="Manto terrestre">manto</a> de la Tierra. Otros compuestos considerados químicamente idénticos pueden tener cantidades variables de <a href="/wiki/Is%C3%B3topo" title="Isótopo">isótopos</a> pesados o ligeros de los elementos constituyentes, lo que cambia ligeramente la proporción en masa de los elementos.
</p>

</p>
<p>Por su composición, se pueden dividir en dos grandes grupos:<sup class="reference separada" id="cite_ref-13"><a href="#cite_note-13"><span class="corchete-llamada">[</span>13<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Una <a href="/wiki/Mol%C3%A9cula" title="Molécula">molécula</a> es un grupo eléctricamente neutro de dos o más átomos unidos por enlaces químicos.<sup class="reference separada" id="cite_ref-iupac_17-0"><a href="#cite_note-iupac-17"><span class="corchete-llamada">[</span>17<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-18"><a href="#cite_note-18"><span class="corchete-llamada">[</span>18<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-19"><a href="#cite_note-19"><span class="corchete-llamada">[</span>19<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-20"><a href="#cite_note-20"><span class="corchete-llamada">[</span>20<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-21"><a href="#cite_note-21"><span class="corchete-llamada">[</span>21<span class="corchete-llamada">]</span></a></sup>​ Una molécula puede ser homonuclear, es decir, estar formada por átomos de un mismo <a href="/wiki/Elemento_qu%C3%ADmico" title="Elemento químico">elemento químico</a>, como ocurre con dos átomos en la molécula de oxígeno (O<sub>2</sub>); o puede ser heteronuclear, es decir, un compuesto químico compuesto por más de un elemento, como el agua (dos átomos de hidrógeno y un átomo de oxígeno; H<sub>2</sub>O).<sup class="reference separada" id="cite_ref-22"><a href="#cite_note-22"><span class="corchete-llamada">[</span>22<span class="corchete-llamada">]</span></a></sup>​ Los átomos y complejos unidos por <a class="new" href="/w/index.php?title=Enlace_no_covalente&amp;action=edit&amp;redlink=1" title="Enlace no covalente (aún no redactado)">enlaces no covalentes</a> como los <a href="/wiki/Enlace_de_hidr%C3%B3geno" title="Enlace de hidrógeno">enlaces de hidrógeno</a> o los <a href="/wiki/Enlace_i%C3%B3nico" title="Enlace iónico">enlaces iónicos</a> no se suelen considerar como moléculas individuales.
</p>
<p>Un <a href="/wiki/Compuesto_i%C3%B3nico" title="Compuesto iónico">compuesto iónico</a> es un compuesto químico compuesto de <a class="mw-redirect" href="/wiki/Anion" title="Anion">anion</a> que se mantienen unidos por fuerzas electrostáticas denominadas <a href="/wiki/Enlace_i%C3%B3nico" title="Enlace iónico">enlace iónico</a>. El compuesto es neutro en general, pero consta de iones cargados positivamente llamados <a class="mw-redirect" href="/wiki/Cation" title="Cation">cationes</a> e iones cargados negativamente llamados <a class="mw-redirect" href="/wiki/Anion" title="Anion">aniones</a>. Estos pueden ser iones simples como el sodio(Na<sup>+</sup>) y el cloruro (Cl<sup>−</sup>) en el cloruro de sodio, o especies poliatómicas como el amonio (<span class="fquim" style="white-space:nowrap;">NH<span style="display:inline-block; margin-bottom:-0.3em; vertical-align:-0.4em; line-height:1.2em; font-size:85%; text-align:left;">+<br/>4</span></span>) y carbonato (<span class="fquim" style="white-space:nowrap;">CO<span style="display:inline-block; margin-bottom:-0.3em; vertical-align:-0.4em; line-height:1.2em; font-size:85%; text-align:left;">2−<br/>3</span></span>) en el <a href="/wiki/Carbonato_de_amonio" title="Carbonato de amonio">carbonato de amonio</a>.<sup class="reference separada" id="cite_ref-23"><a href="#cite_note-23"><span class="corchete-llamada">[</span>23<span class="corchete-llamada">]</span></a></sup>​ Los iones individuales dentro de un compuesto iónico generalmente tienen múltiples vecinos más cercanos, por lo que no se consideran parte de moléculas, sino parte de una red tridimensional continua, generalmente en una <a href="/wiki/Estructura_cristalina" title="Estructura cristalina">estructura cristalina</a>.<sup class="reference separada" id="cite_ref-24"><a href="#cite_note-24"><span class="corchete-llamada">[</span>24<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Los compuestos iónicos que contienen iones básicos hidróxido (OH<sup>−</sup>) u óxido(O<sup>2−</sup>) se clasifican como <a href="/wiki/Base_(qu%C3%ADmica)" title="Base (química)">bases</a>. Los compuestos iónicos sin estos iones también se conocen como <a href="/wiki/Sal_(qu%C3%ADmica)" title="Sal (química)">sales</a> y pueden formarse mediante <a href="/wiki/Reacci%C3%B3n_%C3%A1cido-base" title="Reacción ácido-base">reacciones ácido-base</a>.<sup class="reference separada" id="cite_ref-25"><a href="#cite_note-25"><span class="corchete-llamada">[</span>25<span class="corchete-llamada">]</span></a></sup>​ Los compuestos iónicos también se pueden producir a partir de sus iones constituyentes por <a href="/wiki/Evaporaci%C3%B3n" title="Evaporación">evaporación</a> de su <a href="/wiki/Disolvente" title="Disolvente">disolvente</a>, <a href="/wiki/Precipitado" title="Precipitado">precipitación</a>, congelación, una reacción en estado sólido o la reacción de <a class="mw-redirect" href="/wiki/Transferencia_de_electrones" title="Transferencia de electrones">transferencia de electrones</a> de metales reactivos con no metales reactivos, como los gases <a class="mw-redirect" href="/wiki/Hal%C3%B3geno" title="Halógeno">halógenos</a>.
</p>
<p>Los compuestos iónicos suelen tener altos puntos de <a href="/wiki/Punto_de_fusi%C3%B3n" title="Punto de fusión">fusión</a> y <a href="/wiki/Punto_de_ebullici%C3%B3n" title="Punto de ebullición">ebullición</a>, y son duros y quebradizos. Como sólidos, casi siempre son <a class="mw-redirect" href="/wiki/Aislante_el%C3%A9ctrico" title="Aislante eléctrico">eléctricamente aislantes</a>, pero cuando se funden o disuelven se vuelven altamente <a href="/wiki/Conductor_el%C3%A9ctrico" title="Conductor eléctrico">conductores</a>, porque se movilizan los iones.<sup class="reference separada" id="cite_ref-26"><a href="#cite_note-26"><span class="corchete-llamada">[</span>26<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Un <a class="new" href="/w/index.php?title=Compuesto_intermet%C3%A1lico&amp;action=edit&amp;redlink=1" title="Compuesto intermetálico (aún no redactado)">compuesto intermetálico</a> es un tipo de <a href="/wiki/Aleaci%C3%B3n" title="Aleación">aleación</a> metálica que forma un compuesto de <a class="mw-redirect" href="/wiki/Estado_s%C3%B3lido" title="Estado sólido">estado sólido</a> ordenado entre dos o más elementos metálicos. Los intermetálicos son generalmente duros y quebradizos, con buenas propiedades mecánicas a altas temperaturas.<sup class="reference separada" id="cite_ref-a_27-0"><a href="#cite_note-a-27"><span class="corchete-llamada">[</span>27<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-28"><a href="#cite_note-28"><span class="corchete-llamada">[</span>28<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-29"><a href="#cite_note-29"><span class="corchete-llamada">[</span>29<span class="corchete-llamada">]</span></a></sup>​ Se pueden clasificar como compuestos intermetálicos estequiométricos o no estequiométricos.<sup class="reference separada" id="cite_ref-a_27-1"><a href="#cite_note-a-27"><span class="corchete-llamada">[</span>27<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Un complejo de coordinación consiste en un átomo o ion central, que generalmente es metálico y se llama centro de coordinación, y una matriz circundante de moléculas o iones unidos, que a su vez se conocen como <a href="/wiki/Ligando" title="Ligando">ligandos</a> o agentes complejantes.<sup class="reference separada" id="cite_ref-30"><a href="#cite_note-30"><span class="corchete-llamada">[</span>30<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-31"><a href="#cite_note-31"><span class="corchete-llamada">[</span>31<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-32"><a href="#cite_note-32"><span class="corchete-llamada">[</span>32<span class="corchete-llamada">]</span></a></sup>​ Muchos compuestos que contienen metales, especialmente los de <a class="mw-redirect" href="/wiki/Metales_de_transici%C3%B3n" title="Metales de transición">metales de transición</a>, son complejos de coordinación.<sup class="reference separada" id="cite_ref-33"><a href="#cite_note-33"><span class="corchete-llamada">[</span>33<span class="corchete-llamada">]</span></a></sup>​ Un complejo de coordinación cuyo centro es un átomo metálico se denomina <a class="mw-redirect" href="/wiki/Complejo_met%C3%A1lico" title="Complejo metálico">complejo metálico</a> o elemento de bloque d.<sup class="reference separada" id="cite_ref-34"><a href="#cite_note-34"><span class="corchete-llamada">[</span>34<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Los compuestos se mantienen unidos por medio de diferentes tipos de enlaces y fuerzas. Las diferencias entre los tipos de enlaces de los compuestos dependen del tipo de elemento presente en el compuesto.
</p>
<p><a href="/wiki/Fuerzas_de_dispersi%C3%B3n_de_London" title="Fuerzas de dispersión de London">Las fuerzas de dispersión de London</a> son las fuerza más débiles entre <a href="/wiki/Fuerza_intermolecular" title="Fuerza intermolecular">las fuerzas intermoleculares</a>. Son fuerzas de atracción temporales que se forman cuando los <a href="/wiki/Electr%C3%B3n" title="Electrón">electrones</a> en dos átomos adyacentes se colocan de manera que crean un <a href="/wiki/Dipolo_el%C3%A9ctrico" title="Dipolo eléctrico">dipolo</a> temporal. Además, estas fuerzas son responsables de la condensación de sustancias <a href="/wiki/Polaridad_(qu%C3%ADmica)" title="Polaridad (química)">no polares</a> en líquidos y posterior congelación a un estado sólido dependiendo de la temperatura del ambiente.<sup class="reference separada" id="cite_ref-35"><a href="#cite_note-35"><span class="corchete-llamada">[</span>35<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Un <a href="/wiki/Enlace_covalente" title="Enlace covalente">enlace covalente</a>, también conocido como enlace molecular, implica el intercambio de electrones entre dos átomos. Principalmente, este tipo de enlace se produce entre elementos que aparecen uno cerca del otro en la <a href="/wiki/Tabla_peri%C3%B3dica_de_los_elementos" title="Tabla periódica de los elementos">tabla periódica de elementos</a>, aunque se observa entre algunos metales y no metales. Esto se debe al mecanismo de este tipo de enlace. Los elementos cercanos en la tabla periódica tienden a tener <a href="/wiki/Electronegatividad" title="Electronegatividad">electronegatividades</a> similares, lo que significa que tienen una afinidad similar por los electrones. Como ninguno de los elementos tiene una afinidad más fuerte para donar o ganar electrones, hace que los elementos compartan electrones de manera que ambos elementos tengan un <a href="/wiki/Regla_del_octeto" title="Regla del octeto">octeto</a> más estable.
</p>
<p>El <a href="/wiki/Enlace_i%C3%B3nico" title="Enlace iónico">enlace iónico</a> se produce cuando <a href="/wiki/Electr%C3%B3n_de_valencia" title="Electrón de valencia">los electrones de valencia</a> se transfieren completamente entre los elementos. Al contrario que el covalente, este enlace químico crea dos iones de carga opuesta. Los metales en enlaces iónicos generalmente pierden sus electrones de valencia, convirtiéndose en <a href="/wiki/Cati%C3%B3n" title="Catión">cationes</a>, cargados positivamente. El no metal ganará los electrones del metal, haciendo que el no metal sea un <a href="/wiki/Ani%C3%B3n" title="Anión">anión</a>, es decir, cargado negativamente. Es decir, los enlaces iónicos se producen entre un donador de electrones, generalmente un metal, y un aceptor de electrones, que tiende a ser un no metal.<sup class="reference separada" id="cite_ref-36"><a href="#cite_note-36"><span class="corchete-llamada">[</span>36<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El <a href="/wiki/Enlace_de_hidr%C3%B3geno" title="Enlace de hidrógeno">enlace de hidrógeno</a> se produce cuando un <a href="/wiki/%C3%81tomo_de_hidr%C3%B3geno" title="Átomo de hidrógeno">átomo de hidrógeno</a> unido a un átomo electronegativo forma una conexión <a href="/wiki/Electrost%C3%A1tica" title="Electrostática">electrostática</a> con otro átomo electronegativo a través de dipolos o cargas que interactúan.<sup class="reference separada" id="cite_ref-37"><a href="#cite_note-37"><span class="corchete-llamada">[</span>37<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-38"><a href="#cite_note-38"><span class="corchete-llamada">[</span>38<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-39"><a href="#cite_note-39"><span class="corchete-llamada">[</span>39<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Un compuesto se puede convertir en una composición química diferente (<a class="mw-redirect mw-disambig" href="/wiki/Productos" title="Productos">productos</a>) mediante la interacción con un segundo compuesto químico (<a class="mw-redirect" href="/wiki/Reactivos" title="Reactivos">reactivos</a>) a través de una <a href="/wiki/Reacci%C3%B3n_qu%C3%ADmica" title="Reacción química">reacción química</a>. En este proceso, los enlaces entre los átomos se rompen en ambos compuestos que interactúan, y luego los enlaces se reforman para que obtener nuevas asociaciones entre los mismos átomos. Esquemáticamente, esta reacción podría describirse como AB + CD → AD + CB, donde A, B, C y D son cada uno átomos únicos; y AB, AD, CD y CB son cada uno compuestos únicos.<sup class="reference separada" id="cite_ref-40"><a href="#cite_note-40"><span class="corchete-llamada">[</span>40<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-41"><a href="#cite_note-41"><span class="corchete-llamada">[</span>41<span class="corchete-llamada">]</span></a></sup>​
