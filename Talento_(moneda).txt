<p>El <b>talento</b> (del <a href="/wiki/Griego_antiguo" title="Griego antiguo">griego</a> τάλαντον, <i>talanton</i> que significa balanza o peso) era una <a href="/wiki/Unidad_de_medida" title="Unidad de medida">unidad de medida</a> monetaria utilizada en la antigüedad. Tiene su origen en <a class="mw-redirect" href="/wiki/Babilonia" title="Babilonia">Babilonia</a> pero se usó ampliamente en todo el mar <a class="mw-redirect" href="/wiki/Mediterr%C3%A1neo" title="Mediterráneo">Mediterráneo</a> durante el <a href="/wiki/Per%C3%ADodo_helen%C3%ADstico" title="Período helenístico">período helenístico</a> y la época de las <a href="/wiki/Guerras_p%C3%BAnicas" title="Guerras púnicas">guerras púnicas</a>. En el <a href="/wiki/Antiguo_Testamento" title="Antiguo Testamento">Antiguo Testamento</a>, equivalía a cerca de <span style="white-space:nowrap">34 kg</span> de plata, y en el <a href="/wiki/Nuevo_Testamento" title="Nuevo Testamento">Nuevo Testamento</a>, a <span style="white-space:nowrap">6000 <a href="/wiki/Dracma_griega_antigua" title="Dracma griega antigua">dracmas</a></span>, o lo que es lo mismo, <span style="white-space:nowrap">21 600 <a href="/wiki/Gramo" title="Gramo">g</a></span> de <a href="/wiki/Plata" title="Plata">plata</a>. 
</p>
<p>Era el peso aproximado del agua necesaria para llenar un <a href="/wiki/%C3%81nfora" title="Ánfora">ánfora</a> (alrededor de un <a href="/wiki/Pie_c%C3%BAbico" title="Pie cúbico">pie cúbico</a>).<sup class="reference separada" id="cite_ref-1"><a href="#cite_note-1"><span class="corchete-llamada">[</span>1<span class="corchete-llamada">]</span></a></sup>​ Un <b>talento griego</b>, o <b>talento ático</b> se correspondía con unos <span style="white-space:nowrap">26 kg</span>,<sup class="reference separada" id="cite_ref-Humphrey_2-0"><a href="#cite_note-Humphrey-2"><span class="corchete-llamada">[</span>2<span class="corchete-llamada">]</span></a></sup>​ un talento romano con <span style="white-space:nowrap">32,3 kg</span>; el talento egipcio con <span style="white-space:nowrap">27 kg</span>,<sup class="reference separada" id="cite_ref-Humphrey_2-1"><a href="#cite_note-Humphrey-2"><span class="corchete-llamada">[</span>2<span class="corchete-llamada">]</span></a></sup>​ y el talento de Babilonia con <span style="white-space:nowrap">30,3 kg</span>.<sup class="reference separada" id="cite_ref-Herodotus_3-0"><a href="#cite_note-Herodotus-3"><span class="corchete-llamada">[</span>3<span class="corchete-llamada">]</span></a></sup>​ En el antiguo Israel se adoptó inicialmente el talento de Babilonia, pero fue modificado posteriormente.<sup class="reference separada" id="cite_ref-JE_4-0"><a href="#cite_note-JE-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​ El <b>talento pesado</b> del Nuevo Testamento eran unos <span style="white-space:nowrap">58,9 kg</span>.<sup class="reference separada" id="cite_ref-JE_4-1"><a href="#cite_note-JE-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>En el <a href="/wiki/Siglo_III_a._C." title="Siglo III a. C.">siglo III a. C.</a> es utilizado expresamente en los tratados entre <a href="/wiki/Antigua_Roma" title="Antigua Roma">Roma</a> y <a href="/wiki/Cartago" title="Cartago">Cartago</a>, sobre todo el talento <a class="mw-redirect" href="/wiki/%C3%81tica" title="Ática">ático</a>, o bien <a href="/wiki/Eubea" title="Eubea">eubeo</a>, equivalente a aproximadamente <span style="white-space:nowrap">27 kg</span>, y dividido en <span style="white-space:nowrap">60 <a href="/wiki/Mina_(unidad)" title="Mina (unidad)">minas</a></span> de <span style="white-space:nowrap">60 <a href="/wiki/Siclo" title="Siclo">siclos</a></span> (o <span style="white-space:nowrap">cien <a href="/wiki/Dracma_griega_antigua" title="Dracma griega antigua">dracmas</a></span>) cada mina. El peso de las monedas y los <a href="/wiki/Quilate" title="Quilate">quilates</a> varían, lo mismo que su división: decimal, duodecimal o sexagesimal.
</p>
<p>Durante un tiempo existieron en las regiones mediterráneas dos tipos de sistemas monetarios básicos: el ático (<span style="white-space:nowrap">6 <a href="/wiki/%C3%93bolo" title="Óbolo">óbolos</a></span> = <span style="white-space:nowrap">1 <a href="/wiki/Dracma_griega_antigua" title="Dracma griega antigua">dracma</a></span>, <span style="white-space:nowrap">100 dracmas</span> = <span style="white-space:nowrap">1 <a href="/wiki/Mina_(unidad)" title="Mina (unidad)">mina</a></span>, <span style="white-space:nowrap">60 minas</span> = <span style="white-space:nowrap">1 talento</span>), que más tarde fue adoptado por Roma, y el <a href="/wiki/Cartago" title="Cartago">cartaginés</a> (<span style="white-space:nowrap">60 shekels</span> = <span style="white-space:nowrap">1 mina</span>, <span style="white-space:nowrap">60 minas</span> = <span style="white-space:nowrap">1 talento</span>), que se mantuvo vigente durante mucho tiempo en el <a class="mw-redirect" href="/wiki/Dinast%C3%ADa_Ptolemaica" title="Dinastía Ptolemaica">periodo ptolemaico</a> de <a href="/wiki/Antiguo_Egipto" title="Antiguo Egipto">Egipto</a>. 
</p>
<p>Esta unidad monetaria del mundo antiguo es una de las que más conocidas por su mención en una de las <a href="/wiki/Par%C3%A1bolas_de_Jes%C3%BAs" title="Parábolas de Jesús">parábolas de Jesús</a>: la <a href="/wiki/Par%C3%A1bola_de_los_talentos" title="Parábola de los talentos">parábola de los talentos</a> (<a href="/wiki/Evangelio_de_Mateo" title="Evangelio de Mateo">Mateo</a> 25:14–30; <a href="/wiki/Evangelio_de_Lucas" title="Evangelio de Lucas">Lucas</a> 19:11–27). De la interpretación de esta parábola deriva «inteligencia», (capacidad de entender), «aptitud» (capacidad para el desempeño o ejercicio de una ocupación) dadas como primeras acepciones por la <a href="/wiki/Real_Academia_Espa%C3%B1ola" title="Real Academia Española">RAE</a> para este término en lengua española.
</p>
