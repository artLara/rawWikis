</p>
<p>El <b>latín</b> (<a href="/wiki/Autoglot%C3%B3nimo" title="Autoglotónimo">autoglotónimo</a>: <i>Lingua Latina</i> o <i>Latina;</i> en <a href="/wiki/Griego_antiguo" title="Griego antiguo">griego clásico</a>: Λατινικὴ ɣλῶττα; en <a href="/wiki/Griego_moderno" title="Griego moderno">neogriego</a>: Λατινική γλώσσα o Λατινικά) es una <a href="/wiki/Lenguas_it%C3%A1licas" title="Lenguas itálicas">lengua itálica</a> perteneciente al subgrupo <a href="/wiki/Lenguas_latino-faliscas" title="Lenguas latino-faliscas">latino-falisco</a>,<sup class="reference separada" id="cite_ref-2"><a href="#cite_note-2"><span class="corchete-llamada">[</span>2<span class="corchete-llamada">]</span></a></sup>​ y a su vez a la familia de las <a href="/wiki/Lenguas_indoeuropeas" title="Lenguas indoeuropeas">lenguas indoeuropeas</a>,<sup class="reference separada" id="cite_ref-3"><a href="#cite_note-3"><span class="corchete-llamada">[</span>3<span class="corchete-llamada">]</span></a></sup>​ que fue hablada en la <a href="/wiki/Antigua_Roma" title="Antigua Roma">Antigua Roma</a> y posteriormente durante la <a href="/wiki/Edad_Media" title="Edad Media">Edad Media</a> y la <a href="/wiki/Edad_Moderna" title="Edad Moderna">Edad Moderna</a>, llegando hasta la <a href="/wiki/Edad_Contempor%C3%A1nea" title="Edad Contemporánea">Edad Contemporánea</a>, pues se mantuvo como lengua científica hasta el siglo <span style="font-variant:small-caps;text-transform:lowercase">XIX</span>. Su nombre deriva de una zona geográfica de la <a href="/wiki/Pen%C3%ADnsula_it%C3%A1lica" title="Península itálica">península itálica</a> donde se desarrolló <a href="/wiki/Roma" title="Roma">Roma</a>, el <a href="/wiki/Lacio" title="Lacio">Lacio</a> (en latín, <i>Latium</i>).
</p>
<p>Adquirió gran importancia con la expansión de Roma,<sup class="reference separada" id="cite_ref-4"><a href="#cite_note-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​ y fue <a href="/wiki/Idioma_oficial" title="Idioma oficial">lengua oficial</a> del <a href="/wiki/Imperio_romano" title="Imperio romano">Imperio romano</a> en gran parte de <a href="/wiki/Europa" title="Europa">Europa</a>, <a href="/wiki/%C3%81frica_del_Norte" title="África del Norte">África septentrional</a> y <a href="/wiki/Oriente_Pr%C3%B3ximo" title="Oriente Próximo">Oriente Próximo</a>, junto con el <a href="/wiki/Griego_antiguo" title="Griego antiguo">griego</a>. Como las demás lenguas indoeuropeas en general, el latín era una <a href="/wiki/Lengua_fusionante" title="Lengua fusionante">lengua flexiva</a> de tipo fusional con un mayor grado de síntesis nominal que las actuales <a href="/wiki/Lenguas_romances" title="Lenguas romances">lenguas romances</a>, en la cual dominaba la <a href="/wiki/Flexi%C3%B3n_(ling%C3%BC%C3%ADstica)" title="Flexión (lingüística)">flexión</a> mediante <a href="/wiki/Sufijo" title="Sufijo">sufijos</a>, combinada en determinadas veces con el uso de las <a href="/wiki/Preposici%C3%B3n" title="Preposición">preposiciones</a>, mientras que en las lenguas modernas derivadas dominan las construcciones analíticas con preposiciones, mientras que se ha reducido la flexión nominal a marcar solo el <a href="/wiki/G%C3%A9nero_gramatical" title="Género gramatical">género</a> y el <a href="/wiki/N%C3%BAmero_gramatical" title="Número gramatical">número</a>, conservando los <a class="mw-redirect" href="/wiki/Caso_gramatical" title="Caso gramatical">casos</a> de <a class="mw-redirect" href="/wiki/Declinaci%C3%B3n_del_lat%C3%ADn" title="Declinación del latín">declinación</a> solo en los <a href="/wiki/Pronombre" title="Pronombre">pronombres</a> personales (estos tienen, además, un orden fijo en los sintagmas verbales).<sup class="reference separada" id="cite_ref-5"><a href="#cite_note-5"><span class="corchete-llamada">[</span>a<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El latín originó un gran número de lenguas europeas, denominadas <a href="/wiki/Lenguas_romances" title="Lenguas romances">lenguas romances</a>, como el <a href="/wiki/Idioma_espa%C3%B1ol" title="Idioma español">español</a>, <a href="/wiki/Idioma_franc%C3%A9s" title="Idioma francés">francés</a>, <a href="/wiki/Idioma_franco-provenzal" title="Idioma franco-provenzal">franco-provenzal</a>, <a href="/wiki/Idioma_friulano" title="Idioma friulano">friulano</a>, <a href="/wiki/Idioma_gallego" title="Idioma gallego">gallego</a>, <a href="/wiki/Idioma_istriano" title="Idioma istriano">istriano</a>, <a href="/wiki/Idioma_istrorrumano" title="Idioma istrorrumano">istrorrumano</a>, <a href="/wiki/Idioma_italiano" title="Idioma italiano">italiano</a>, <a href="/wiki/Idioma_ladino" title="Idioma ladino">ladino</a>, <a href="/wiki/Idioma_ligur" title="Idioma ligur">ligur</a>, <a href="/wiki/Idioma_lombardo" title="Idioma lombardo">lombardo</a>, <a href="/wiki/Idioma_meglenorrumano" title="Idioma meglenorrumano">meglenorrumano</a>, <a href="/wiki/Idioma_napolitano" title="Idioma napolitano">napolitano</a>, <a href="/wiki/Idioma_occitano" title="Idioma occitano">occitano</a>, <a href="/wiki/Idioma_piamont%C3%A9s" title="Idioma piamontés">piamontés</a>, <a href="/wiki/Idioma_portugu%C3%A9s" title="Idioma portugués">portugués</a>, <a href="/wiki/Romanche" title="Romanche">romanche</a>, <a href="/wiki/Idioma_rumano" title="Idioma rumano">rumano</a>, <a href="/wiki/Idioma_sardo" title="Idioma sardo">sardo</a>, <a href="/wiki/Idioma_siciliano" title="Idioma siciliano">siciliano</a>, <a href="/wiki/Idioma_val%C3%B3n" title="Idioma valón">valón</a>, <a href="/wiki/Idioma_v%C3%A9neto" title="Idioma véneto">véneto</a>, <a href="/wiki/Idioma_aragon%C3%A9s" title="Idioma aragonés">aragonés</a>, <a href="/wiki/Idioma_arrumano" title="Idioma arrumano">arrumano</a>, <a href="/wiki/Idioma_asturleon%C3%A9s" title="Idioma asturleonés">asturleonés</a>, <a href="/wiki/Idioma_catal%C3%A1n" title="Idioma catalán">catalán</a>, <a href="/wiki/Idioma_corso" title="Idioma corso">corso</a>, <a href="/wiki/Idioma_emiliano-roma%C3%B1ol" title="Idioma emiliano-romañol">emiliano-romañol</a>, y otros ya extintos, como el <a href="/wiki/Idioma_dalm%C3%A1tico" title="Idioma dalmático">dalmático</a>, <a href="/wiki/Lengua_romance_de_Panonia" title="Lengua romance de Panonia">panonio</a> y el <a class="mw-redirect" href="/wiki/Lengua_moz%C3%A1rabe" title="Lengua mozárabe">mozárabe</a>. También ha influido en las palabras de las lenguas modernas debido a que durante muchos siglos, después de la caída del Imperio romano, continuó usándose en toda Europa como <i><a class="mw-redirect" href="/wiki/Lingua_franca" title="Lingua franca">lingua franca</a></i> para las <a href="/wiki/Ciencia" title="Ciencia">ciencias</a> y la <a href="/wiki/Pol%C3%ADtica" title="Política">política</a>, sin ser seriamente amenazada en esa función por otras lenguas en auge (como el español en el siglo <span style="font-variant:small-caps;text-transform:lowercase">XVII</span> o el francés en el siglo <span style="font-variant:small-caps;text-transform:lowercase">XVIII</span>), hasta prácticamente el siglo <span style="font-variant:small-caps;text-transform:lowercase">XIX</span>.
</p>
<p>La <a href="/wiki/Iglesia_cat%C3%B3lica" title="Iglesia católica">Iglesia católica</a> lo usa como <a href="/wiki/Lengua_lit%C3%BArgica" title="Lengua litúrgica">lengua litúrgica</a> oficial (sea en el <a href="/wiki/Rito_romano" title="Rito romano">rito romano</a> sea en los otros <a href="/wiki/Ritos_latinos" title="Ritos latinos">ritos latinos</a>), aunque desde el <a href="/wiki/Concilio_Vaticano_II" title="Concilio Vaticano II">Concilio Vaticano II</a> se permiten además las lenguas vernáculas.<sup class="reference separada" id="cite_ref-6"><a href="#cite_note-6"><span class="corchete-llamada">[</span>5<span class="corchete-llamada">]</span></a></sup>​ También se usa para los nombres binarios de la <a class="mw-redirect" href="/wiki/Clasificaci%C3%B3n_cient%C3%ADfica" title="Clasificación científica">clasificación científica</a> de los reinos animal y vegetal, para denominar figuras o instituciones del mundo del Derecho, como lengua de redacción del <i>Corpus Inscriptionum Latinarum</i>, y en artículos de revistas científicas publicadas total o parcialmente en esta lengua.
</p>
<p>El <a href="/wiki/Alfabeto_latino" title="Alfabeto latino">alfabeto latino</a>, derivado del <a href="/wiki/Alfabeto_griego" title="Alfabeto griego">alfabeto griego</a> (en sí derivado del <a href="/wiki/Alfabeto_fenicio" title="Alfabeto fenicio">alfabeto fenicio</a>), es ampliamente el <a href="/wiki/Alfabeto" title="Alfabeto">alfabeto</a> más usado del mundo con diversas variantes de unas lenguas a otras. El estudio del latín, junto con el del <a class="mw-redirect" href="/wiki/Griego_cl%C3%A1sico" title="Griego clásico">griego clásico</a>, es parte de los llamados <i>estudios clásicos</i>, y aproximadamente hasta los <a href="/wiki/A%C3%B1os_1970" title="Años 1970">años 1970</a> fue estudio casi imprescindible en las <a href="/wiki/Humanidades" title="Humanidades">humanidades</a>. Hasta el día de hoy, en países como <a href="/wiki/Alemania" title="Alemania">Alemania</a>, en los <i><a href="/wiki/Gymnasium" title="Gymnasium">Gymnasien</a></i> se enseña latín o griego junto a lenguas modernas.
</p>
<p>La historia del latín comienza en el siglo <span style="font-variant:small-caps;text-transform:lowercase">VIII</span> a. C. y llega, por lo menos, hasta la <a href="/wiki/Edad_Media" title="Edad Media">Edad Media</a>; se pueden distinguir los siguientes períodos:
</p>
<p>El latín aparece hacia el año 1000 a. C. en el <a href="/wiki/Italia_central" title="Italia central">centro</a> de <a href="/wiki/Italia" title="Italia">Italia</a>, al sur del río <a href="/wiki/T%C3%ADber" title="Tíber">Tíber</a>, con los <a class="mw-redirect" href="/wiki/Montes_Apeninos" title="Montes Apeninos">Apeninos</a> al este y el <a href="/wiki/Mar_Tirreno" title="Mar Tirreno">mar Tirreno</a> al oeste, en una región llamada <i><a href="/wiki/Latium" title="Latium">Latium</a></i> (<a href="/wiki/Lacio" title="Lacio">Lacio</a>), de donde proviene el nombre de la lengua y el de sus primeros habitantes, los <a href="/wiki/Latinos" title="Latinos">latinos</a>; sin embargo, los primeros testimonios escritos datan del siglo <span style="font-variant:small-caps;text-transform:lowercase">VI</span> a. C., como la <a href="/wiki/Inscripci%C3%B3n_Duenos" title="Inscripción Duenos">inscripción de Duenos</a> y otras similares.
</p>
<p>En los primeros siglos de <a href="/wiki/Roma" title="Roma">Roma</a>, desde su fundación hasta el siglo <span style="font-variant:small-caps;text-transform:lowercase">IV</span> a. C., el latín tenía una extensión territorial limitada: Roma y algunas partes de Italia, y una población escasa. Era una lengua de campesinos y pastores.
</p>
<p>Así lo demuestran las <a href="/wiki/Etimolog%C3%ADa" title="Etimología">etimologías</a> de muchos términos del <a href="/wiki/Religi%C3%B3n" title="Religión">culto religioso</a>, del <a href="/wiki/Derecho" title="Derecho">derecho</a> o de la <a href="/wiki/Ej%C3%A9rcito_romano" title="Ejército romano">vida militar</a>. Destacamos los términos <i>stipulare</i> ('estipular'), derivado de <i>stipa</i> ('paja'), o <i>emolumentum</i> ('emolumento'), derivado de <i>emolere</i> ('moler el grano'), en el lenguaje del derecho.
</p>
<p>En este sentido, los <a href="/wiki/Latinos" title="Latinos">latinos</a>, desde época clásica al menos, hablaban de un <i>sermo rusticus</i> ('habla del campo'), opuesto al <i>sermo urbanus</i>, tomando conciencia de esta variedad dialectal del latín. «En el campo latino se dice <i>edus</i> ('cabrito') lo que en la ciudad <i>haedus</i> con una a añadida como en muchas palabras».<sup class="reference separada" id="cite_ref-7"><a href="#cite_note-7"><span class="corchete-llamada">[</span>6<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Después del periodo de dominación etrusca y la invasión de los <a href="/wiki/Galia" title="Galia">galos</a> (390 a. C.), la ciudad se fue extendiendo, en forma de <a href="/wiki/Rep%C3%BAblica_romana" title="República romana">República</a>, por el resto de <a href="/wiki/Italia_(%C3%A9poca_romana)" title="Italia (época romana)">Italia</a>. A finales del siglo <span style="font-variant:small-caps;text-transform:lowercase">IV</span> a. C., Roma se había impuesto a sus vecinos <a href="/wiki/Pueblos_antiguos_de_Italia" title="Pueblos antiguos de Italia">itálicos</a>. Los <a href="/wiki/Etruscos" title="Etruscos">etruscos</a> dejaron su impronta en la lengua y la cultura de Roma, pero los <a href="/wiki/Italiotas" title="Italiotas">italiotas</a> presentes en la <a href="/wiki/Magna_Grecia" title="Magna Grecia">Magna Grecia</a> influyeron más en el latín, dotándolo de un rico <a href="/wiki/L%C3%A9xico" title="Léxico">léxico</a>.
</p>
<p>El latín de la ciudad de Roma se impuso a otras variedades de otros lugares del Lacio, de las que apenas quedaron algunos retazos en el <a class="new" href="/w/index.php?title=Lat%C3%ADn_literario&amp;action=edit&amp;redlink=1" title="Latín literario (aún no redactado)">latín literario</a>. Esto hizo del latín una lengua con muy pocas diferencias <a href="/wiki/Dialecto" title="Dialecto">dialectales</a>, al contrario de lo que pasó en griego. Podemos calificar, pues, al latín de lengua unitaria.
</p>
<p>Después, la conquista de nuevos territorios fuera de Italia, llamados <a href="/wiki/Anexo:Provincias_romanas" title="Anexo:Provincias romanas">provincias</a>, empezando con las <a class="mw-redirect" href="/wiki/Galias" title="Galias">Galias</a> por parte de <a href="/wiki/Julio_C%C3%A9sar" title="Julio César">César</a>, hasta la <a href="/wiki/Dacia_(provincia_romana)" title="Dacia (provincia romana)">Dacia</a> (Rumania) por parte de <a href="/wiki/Trajano" title="Trajano">Trajano</a>, supuso la expansión del latín en un inmenso territorio y la incorporación de una ingente cantidad de nuevos hablantes.
</p>
<p>Paralelamente a la expansión territorial de Roma, el latín se desarrolló como lengua literaria y como <i>lingua franca</i> a la vez que el <a href="/wiki/Idioma_griego" title="Idioma griego">griego</a>, que había tenido estos papeles antes. Desde el siglo <span style="font-variant:small-caps;text-transform:lowercase">II</span> a. C., con <a href="/wiki/Plauto" title="Plauto">Plauto</a> y <a href="/wiki/Terencio" title="Terencio">Terencio</a>, hasta el <a href="/wiki/200" title="200">año 200 d. C.</a> con <a href="/wiki/Apuleyo" title="Apuleyo">Apuleyo</a> tenemos una forma de latín que no tiene ninguna variación sustancial.<sup class="reference separada" id="cite_ref-8"><a href="#cite_note-8"><span class="corchete-llamada">[</span>7<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El latín era una <a href="/wiki/Lenguas_it%C3%A1licas" title="Lenguas itálicas">lengua itálica</a> del <a href="/wiki/Lenguas_latino-faliscas" title="Lenguas latino-faliscas">subgrupo latino-falisco</a>, lo que significa que la mayoría de sus <a href="/wiki/Gram%C3%A1tica" title="Gramática">elementos gramaticales</a> y la mayor parte de su <a href="/wiki/L%C3%A9xico" title="Léxico">léxico</a> provienen, por evolución natural, del <a href="/wiki/Idioma_protoit%C3%A1lico" title="Idioma protoitálico">protoitálico</a>, el supuesto ancestro de las lenguas itálicas.
</p>
<p>El idioma original de los grupos latinos, al desarrollarse en la <a href="/wiki/Pen%C3%ADnsula_it%C3%A1lica" title="Península itálica">península itálica</a>, se vio influido por el contacto con hablantes de otros grupos lingüísticos presentes en la Italia antigua, tanto indoeuropeos hablantes de otros subgrupos de lenguas itálicas, principalmente de la <a href="/wiki/Lenguas_osco-umbras" title="Lenguas osco-umbras">rama osco-umbra</a> (<a href="/wiki/Oscos" title="Oscos">oscos</a>, <a href="/wiki/Umbros" title="Umbros">umbros</a>, <a href="/wiki/Samnitas" title="Samnitas">samnitas</a>, <a href="/wiki/Picenos" title="Picenos">picenos</a>…), como hablantes de lenguas indoeuropeas no itálicas (<a href="/wiki/Mesapios" title="Mesapios">mesapios</a>, <a href="/wiki/Antigua_Grecia" title="Antigua Grecia">griegos</a>, <a href="/wiki/Celta" title="Celta">celtas</a>…), 
así como hablantes de <a href="/wiki/Lenguas_preindoeuropeas" title="Lenguas preindoeuropeas">lenguas preindoeuropeas</a> (<a href="/wiki/Etruscos" title="Etruscos">etruscos</a>, <a href="/wiki/Ligures" title="Ligures">ligures</a>, <a href="/wiki/Cultura_nur%C3%A1gica" title="Cultura nurágica">nuragicos</a>…). Suelen distinguirse tres tipos de influencia sociolingüística:
</p>
<p>Esta distinción, sin embargo, puede no resultar del todo operativa; por ejemplo, el etrusco pudo haber sido a la vez substrato, adstrato y superestrato en diferentes épocas.
</p>
<p>Los habitantes de las regiones de la <a class="mw-redirect" href="/wiki/Antigua_Italia" title="Antigua Italia">antigua Italia</a> en las que posteriormente se difundió el latín eran hablantes nativos de otras lenguas, tanto indoeuropeas del grupo itálico (como el latín), como de otros grupos de lenguas indoeuropeas y preindoeuropeas, que al ser asimilados finalmente a la cultura latina ejercieron cierta influencia lingüística de <a href="/wiki/Sustrato_ling%C3%BC%C3%ADstico" title="Sustrato lingüístico">sustrato</a>. A veces, para indicar estas lenguas, se habla de <b>sustrato mediterráneo,</b> que proporcionó al latín el nombre de algunas plantas y animales que los indoeuropeos conocieron al llegar. Son lenguas muy poco conocidas, pues quedan solo unos pocos restos escritos, algunos aún sin descifrar. Un sustrato del <a href="/wiki/Evoluci%C3%B3n_hist%C3%B3rica_del_lat%C3%ADn#Del_latín_arcaico_al_latín_clásico" title="Evolución histórica del latín">latín arcaico</a> en la ciudad de Roma y alrededores fue claramente la <a href="/wiki/Idioma_etrusco" title="Idioma etrusco">lengua etrusca</a>.
</p>
<p>En cuanto a la influencia del <a href="/wiki/Lenguas_osco-umbras" title="Lenguas osco-umbras">sustrato indoeuropeo osco-umbro</a>, resulta interesante el hecho de que prefigura algunas de las características fonéticas y fonológicas que más tarde aparecerían en las lenguas romances (ciertas <a href="/wiki/Palatalizaci%C3%B3n" title="Palatalización">palatalizaciones</a> y <a href="/wiki/Monoptongaci%C3%B3n" title="Monoptongación">monoptongaciones</a>), pues muchos hablantes de <a href="/wiki/Lenguas_it%C3%A1licas" title="Lenguas itálicas">lenguas itálicas</a> al <a href="/wiki/Romanizaci%C3%B3n_(transliteraci%C3%B3n)" title="Romanización (transliteración)">romanizarse</a> conservaron ciertos rasgos fonéticos propios, incluso (marginalmente) dentro de las lenguas románicas.
</p>
<p>Fenómenos de este tipo son la influencia <a href="/wiki/Celta" title="Celta">céltica</a> a la que se atribuye la lenición de las consonantes intervocálicas o la [y] francesa, el <a href="/wiki/Euskera" title="Euskera">vasco</a> (o alguna lengua parecida), al que se atribuye la aspiración de la /f/ española en /h/, o el influjo <a href="/wiki/Lenguas_eslavas" title="Lenguas eslavas">eslavo</a>, responsable de la centralización de las vocales <a href="/wiki/Idioma_rumano" title="Idioma rumano">rumanas</a>.
</p>
<p><b>Sustrato etrusco</b>: La influencia del etrusco en la <a href="/wiki/Fonolog%C3%ADa" title="Fonología">fonología</a> latina se refleja en el hecho de desarrollar algunas aspiradas (<i>pul<b>ch</b>er</i>, 'hermoso') y la tendencia a cerrar <i>-o</i> en <i>-u.</i> Las inscripciones etruscas muestran una tendencia a realizar como <a href="/wiki/Aspiraci%C3%B3n_(fon%C3%A9tica)" title="Aspiración (fonética)">aspiradas</a> <a class="mw-redirect" href="/wiki/Oclusiva" title="Oclusiva">oclusivas</a> <a class="mw-redirect" href="/wiki/Consonante_sorda" title="Consonante sorda">sordas</a> previamente no-aspiradas, y poseía un <a class="mw-redirect" href="/wiki/Sistema_fonol%C3%B3gico" title="Sistema fonológico">sistema fonológico</a> de solo cuatro timbres vocálicos /a, e, i, u/, teniendo este último una cualidad entre [o] y [u] que habría influido en la tendencia del latín a cerrar algunas /*o/ en [u].
</p>
<p>Además los numerales latinos <i>duodeviginti</i> ('18') y <i>undeviginti</i> ('19') son claramente <a href="/wiki/Calco_sem%C3%A1ntico" title="Calco semántico">calcos</a> lingüísticos formados a partir de las formas etruscas <i>esl-em zathrum</i> ('18') <i>thu-nem zathrum</i>, '19' (donde <i>zathrum</i> es la forma etrusca para '20', <i>esl-</i> '2' y <i>thun-</i> '1'). También es un hecho de sustrato del etrusco en latín el sufijo <i>-na</i> en palabras como <i>persona,</i> etc.
</p>
<p>Durante un tiempo, Roma tuvo importantes contingentes de población de origen etrusco, por lo que el etrusco fue tanto una lengua substrato como una lengua superestrato, al menos durante el período que abarca la monarquía romana y, en menor medida, la república romana. La influencia del etrusco es particularmente notoria en ciertas áreas del léxico, como la relacionada con el <a href="/wiki/Teatro" title="Teatro">teatro</a> y la adivinación. Roma también sufrió invasiones de los <a class="mw-redirect" href="/wiki/Galia_Cisalpina" title="Galia Cisalpina">galos cisalpinos</a>, aunque no parecen existir importantes indicios de influencia <a href="/wiki/Lenguas_celtas" title="Lenguas celtas">celta</a> en el latín. Sí existen algunas evidencias en el vocabulario de préstamos léxicos directos de lenguas osco-umbras, que constituyen la principal influencia de tipo substrato en el latín clásico.
</p>
<p>Por otra parte, si bien desde antiguo los romanos tenían contactos con <a href="/wiki/Pueblos_germ%C3%A1nicos" title="Pueblos germánicos">pueblos germánicos</a> no existen fenómenos de influencia léxica en latín clásico. A diferencia de lo que sucede con las lenguas románicas occidentales que, entre los siglos <span style="font-variant:small-caps;text-transform:lowercase">V</span> y <span style="font-variant:small-caps;text-transform:lowercase">VIII</span>, recibieron numerosos préstamos léxicos del germánico occidental y del germánico oriental. Esto contrasta con la profunda influencia que el latín ejerció en el predecesor del <a href="/wiki/Alto_alem%C3%A1n_antiguo" title="Alto alemán antiguo">alto alemán antiguo</a>. Igualmente, existen abundantes rastros de la administración romana en la <a href="/wiki/Toponimia" title="Toponimia">toponimia</a> de regiones que hoy son de habla germánica, como por ejemplo <a href="/wiki/Colonia_(Alemania)" title="Colonia (Alemania)">Colonia</a>. Los elementos germánicos en la Romania occidental proceden del período del Bajo Imperio, y constituyen el principal superestrato en <a href="/wiki/Lat%C3%ADn_tard%C3%ADo" title="Latín tardío">latín tardío</a>. El flujo no se interrumpió en la formación de las lenguas románicas. Las influencias de los pueblos godo, alemánico, borgoñés, franco y lombardo en las lenguas románicas se da mayoritariamente en el campo de la toponimia y la <a href="/wiki/Antroponimia" title="Antroponimia">antroponimia</a>. Aparte de estos, el número de <a href="/wiki/Pr%C3%A9stamo_ling%C3%BC%C3%ADstico" title="Préstamo lingüístico">préstamos</a> es bastante reducido.
</p>
<p>Es la debida al contacto con pueblos que convivieron con los latinos sin tenerlos dominados ni depender de ellos. Este tipo de influencia se nota más en el estilo y el léxico adquiridos que en los cambios fónicos de la lengua. Los adstratos <a class="mw-redirect" href="/wiki/Osco" title="Osco">osco</a>, <a href="/wiki/Idioma_umbro" title="Idioma umbro">umbro</a> y <a href="/wiki/Griego_antiguo" title="Griego antiguo">griego</a>, son responsables del <a href="/wiki/Alfabeto" title="Alfabeto">alfabeto</a> y sobre lo relacionado con la <a href="/wiki/Mitolog%C3%ADa" title="Mitología">mitología</a>, pues los romanos tomaron <i>prestados</i> los dioses helenos, aunque con nombres latinos.
</p>
<p><b><a href="/wiki/Adstrato" title="Adstrato">Adstrato</a> griego</b>: la entrada masiva de préstamos y calcos <a class="mw-redirect" href="/wiki/%C3%81tico_(dialecto)" title="Ático (dialecto)">áticos</a> y <a class="mw-redirect" href="/wiki/J%C3%B3nico_(dialecto)" title="Jónico (dialecto)">jónicos</a> puso en guardia a los latinos desde tiempos muy tempranos, encabezados por <a href="/wiki/Cat%C3%B3n_el_Viejo" title="Catón el Viejo">Catón el Viejo</a> en el siglo <span style="font-variant:small-caps;text-transform:lowercase">III</span> a. C. Pero en la Edad de Oro de la literatura latina los romanos se rindieron ante la evidente superioridad del idioma griego antiguo. Bien pueden resumir este sentimiento los famosos versos de <a href="/wiki/Horacio" title="Horacio">Horacio</a>: «<i>Graecia capta ferum victorem cepit et artis / intulit agresti Latio</i>» («La Grecia conquistada conquistó a su fiero vencedor e introdujo las artes en el rústico Lacio»).<sup class="reference separada" id="cite_ref-9"><a href="#cite_note-9"><span class="corchete-llamada">[</span>8<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Esta entrada masiva de <a class="mw-redirect" href="/wiki/Helenismo" title="Helenismo">helenismos</a> no se limitó a la literatura, las ciencias o las artes. Afectó a todos los ámbitos de la <a href="/wiki/Idioma" title="Idioma">lengua</a>, léxico, gramatical y estilístico, de modo que podemos encontrar el origen griego en muchas palabras comunes de las <a href="/wiki/Lenguas_romances" title="Lenguas romances">lenguas románicas.</a>
</p>
<p>Después de la Edad Clásica, el cristianismo fue uno de los factores más potentes para introducir en la lengua latina hablada una serie de elementos griegos nuevos. Ej: παραβολή &gt; parábola. Encontramos esta palabra dentro de la terminología retórica, pero sale de ella cuando se usa por los cristianos y adquiere el sentido de <a href="/wiki/Par%C3%A1bolas_de_Jes%C3%BAs" title="Parábolas de Jesús">parábola</a>, es decir, predicación de la vida de Jesús. Poco a poco va adquiriendo el sentido más general de «palabra», que sustituye en toda la Romanía al elemento que significaba «palabra» (<i>verbum</i>). El verbo que deriva de <i>parabole</i> (<i>parabolare, parolare</i>) sustituye en gran parte de la Romanía al verbo que significaba «hablar» (<i>loquor</i>).
</p>
<p>El cuerpo de libros escritos en latín, retiene un legado duradero de cultura de la <a href="/wiki/Roma" title="Roma">Antigua Roma.</a> Los romanos produjeron una extensa cantidad de libros de <a href="/wiki/Poes%C3%ADa" title="Poesía">poesía</a>, <a href="/wiki/Comedia" title="Comedia">comedia</a>, <a href="/wiki/Tragedia" title="Tragedia">tragedia</a>, <a href="/wiki/S%C3%A1tira" title="Sátira">sátira</a>, <a href="/wiki/Historia" title="Historia">historia</a> y <a href="/wiki/Ret%C3%B3rica" title="Retórica">retórica</a>, trazando arduamente al modo de otras culturas, particularmente al estilo de la más madura <a href="/wiki/Literatura_griega" title="Literatura griega">literatura griega</a>. Un tiempo después de que el Imperio romano de occidente cayese, la lengua latina continuaba jugando un papel muy importante en la cultura europea occidental.
</p>
<p>La <a href="/wiki/Literatura_en_lat%C3%ADn" title="Literatura en latín">literatura latina</a> normalmente se divide en distintos períodos. En lo que respecta a la primera, la literatura primitiva, solo restan unas pocas obras sobrevivientes, los libros de Plauto y Terencio; se han conservado dentro de los más populares autores de todos los períodos. Muchas otras, incluyendo la mayoría de los autores prominentes del latín clásico, han desaparecido, aunque bien algunas han sido redescubiertas siglos después.
</p>
<p>El periodo del latín clásico, cuando la literatura latina es ampliamente considerada en su cumbre, se divide en la Edad Dorada, que cubre aproximadamente el periodo del inicio de siglo <span style="font-variant:small-caps;text-transform:lowercase">I</span> a. C. hasta la mitad del siglo <span style="font-variant:small-caps;text-transform:lowercase">I</span> d. C.; y la Edad de Plata, que se extiende hasta el siglo <span style="font-variant:small-caps;text-transform:lowercase">II</span> d. C. La literatura escrita después de la mitad del siglo <span style="font-variant:small-caps;text-transform:lowercase">II</span> es comúnmente denigrada e ignorada.
</p>
<p>En el Renacimiento muchos autores clásicos fueron redescubiertos y su estilo fue conscientemente imitado. Pero sobre todo, se imitó a Cicerón, y su estilo se ha apreciado como el perfecto culmen del latín. El latín medieval fue frecuentemente despreciado como <a href="/wiki/Lat%C3%ADn_macarr%C3%B3nico" title="Latín macarrónico">latín macarrónico</a>; en cualquier caso, muchas grandes obras de la literatura latina fueron producidas entre la antigüedad y la <a href="/wiki/Edad_Media" title="Edad Media">Edad Media</a>, aunque no sea de los antiguos romanos.
</p>
<p>La <a href="/wiki/Literatura_romana" title="Literatura romana">literatura latina romana</a> abarca dos partes: la literatura indígena y la imitada.
</p>
<p>El <b>latín culto</b> fue la lengua literaria, que utilizaban en sus obras autores como Plauto, Terencio, Cicerón, Tito Livio, Virgilio, Horacio, Ovidio, Séneca y Plinio el Viejo entre los siglos <small>III</small> <a href="/wiki/Antes_de_Cristo" title="Antes de Cristo">a. C.</a> y <small>II</small> <a href="/wiki/Anno_Domini" title="Anno Domini">d. C.</a>, aunque se hablaba muy poco. Era prácticamente incomprensible para la mayor parte del pueblo romano que, en su lugar, hablaba la lengua popular, el <b><a href="/wiki/Lat%C3%ADn_vulgar" title="Latín vulgar">latín vulgar</a></b> (<i>vulgus</i> significa ‘pueblo’), que se escribía bastante poco.
</p>
<p>El <i>plebeius sermo</i> tenía un vocabulario muy reducido y tomó palabras originarias de distintas provincias dominadas por los romanos, desconocidas por el <i>sermo nobilis.</i>  De igual manera, en el latín vulgar se dieron fenómenos tales como la reducción del significado de muchas palabras, la ampliación del significado de otras, el cambio de significado de términos usuales en el latín literario o el cambio de significado en otros casos.
</p>
<p>Asimismo, en la <i>lingua romana rustica</i> se fueron dando diferencias en la pronunciación de las palabras y en la duración de las vocales.  El latín clásico era sintético, es decir, que expresaba varias ideas en una sola palabra, mientras que el latín vulgar explicaba los conceptos a través de muchos términos; se volvió una lengua analítica.  El <i>sermo vulgaris</i> hace uso del artículo demostrativo para precisar el género y número del sustantivo al que acompañan: <i>ille vir</i> (el hombre) - <i>illa femina</i> (la mujer).  En el latín vulgar se pierden las terminaciones que indicaban los seis casos del singular y del plural, conservando el acusativo.  El género neutro se vuelve masculino, por ejemplo: <i>baculum</i> (báculo), <i>corpus</i> (cuerpo), <i>damnum</i> (daño), <i>frigus</i> (frío), <i>cor</i> (corazón).
</p>
<p>Tras la caída del <a href="/wiki/Imperio_romano" title="Imperio romano">Imperio romano</a>, el latín todavía fue usado durante varios siglos como la única lengua escrita en el mundo posterior al estado romano. En la cancillería del rey, en la liturgia de la <a href="/wiki/Iglesia_cat%C3%B3lica" title="Iglesia católica">Iglesia católica</a> o en los libros escritos en los monasterios, la única lengua usada era el latín. Un latín muy cuidado, aunque poco a poco se vio influido por su expresión hablada. Ya en el siglo <span style="font-variant:small-caps;text-transform:lowercase">VII</span>, el <a href="/wiki/Lat%C3%ADn_vulgar" title="Latín vulgar">latín vulgar</a> había comenzado a diferenciarse originando el <a href="/wiki/Protorromance" title="Protorromance">protorromance</a> y después las primeras fases de las actuales <a href="/wiki/Lenguas_romances" title="Lenguas romances">lenguas romances</a>.
</p>
<p>Con el renacimiento <a href="/wiki/Carlomagno" title="Carlomagno">carolingio</a> del siglo <span style="font-variant:small-caps;text-transform:lowercase">IX</span>, los mayores pensadores de la época, como el lombardo <a href="/wiki/Pablo_el_Di%C3%A1cono" title="Pablo el Diácono">Pablo el Diácono</a> o el inglés <a href="/wiki/Alcuino_de_York" title="Alcuino de York">Alcuino de York</a>, se ocuparon de reorganizar la cultura y la enseñanza en su imperio. En lo que se refiere al latín, las reformas se dirigieron a la recuperación más correcta de forma escrita, lo que le separó definitivamente de la evolución que siguieron las lenguas romances.
</p>
<p>Luego, con el surgimiento de las primeras y pocas <a href="/wiki/Universidad" title="Universidad">universidades</a>, las enseñanzas dadas por personas que provenían de toda Europa eran rigurosamente en latín. Pero un cierto latín, el que no podía decirse que fuera la lengua de <a class="mw-redirect" href="/wiki/Marco_Tulio_Cicer%C3%B3n" title="Marco Tulio Cicerón">Cicerón</a> u <a href="/wiki/Horacio" title="Horacio">Horacio</a>. Los doctos de las universidades elaboraron un latín particular, escolástico, adaptado a expresar los conceptos abstractos y ricos en elaborados matices de la filosofía de la época. El latín ya no era la lengua de comunicación que fue en el mundo romano; todavía era una lengua viva y vital, todo menos estática.
</p>
<p>En el siglo <span style="font-variant:small-caps;text-transform:lowercase">XIV</span>, en <a href="/wiki/Italia" title="Italia">Italia</a>, surgió un movimiento cultural que favoreció un renovado interés por el latín antiguo: el <a href="/wiki/Humanismo_renacentista" title="Humanismo renacentista">Humanismo</a>. Comenzado ya por <a href="/wiki/Petrarca" title="Petrarca">Petrarca</a>, sus mayores exponentes fueron <a href="/wiki/Poggio_Bracciolini" title="Poggio Bracciolini">Poggio Bracciolini</a>, <a href="/wiki/Lorenzo_Valla" title="Lorenzo Valla">Lorenzo Valla</a>, <a href="/wiki/Marsilio_Ficino" title="Marsilio Ficino">Marsilio Ficino</a> y <a href="/wiki/Coluccio_Salutati" title="Coluccio Salutati">Coluccio Salutati</a>. Aquí la lengua clásica empezó a ser objeto de estudios profundos que marcaron el nacimiento, de hecho, de la <a href="/wiki/Filolog%C3%ADa_cl%C3%A1sica" title="Filología clásica">filología clásica</a>.
</p>
<p>En la <a href="/wiki/Edad_Moderna" title="Edad Moderna">Edad Moderna</a>, el latín aún se usa como lengua de la cultura y de la <a href="/wiki/Ciencia" title="Ciencia">ciencia</a>, pero va siendo sustituido paulatinamente por los idiomas locales. En latín escribieron, por ejemplo, <a href="/wiki/Nicol%C3%A1s_Cop%C3%A9rnico" title="Nicolás Copérnico">Nicolás Copérnico</a> e <a href="/wiki/Isaac_Newton" title="Isaac Newton">Isaac Newton</a>. <a class="mw-redirect" href="/wiki/Galileo" title="Galileo">Galileo</a> fue de los primeros científicos en escribir en un idioma distinto del latín (en italiano, hacia 1600), y <a href="/wiki/Hans_Christian_%C3%98rsted" title="Hans Christian Ørsted">Oersted</a> de los últimos en escribir en latín, en la primera mitad del siglo <span style="font-variant:small-caps;text-transform:lowercase">XIX</span>.
</p>
<p>Al conjunto de formas que puede tomar una misma palabra según su caso se le denomina paradigma de flexión. Los paradigmas de flexión de sustantivos y adjetivos se denominan en gramática latina <a href="/wiki/Declinaci%C3%B3n_(gram%C3%A1tica)" title="Declinación (gramática)">declinaciones</a>, mientras que los paradigmas de flexión de los verbos se llaman <a href="/wiki/Conjugaci%C3%B3n" title="Conjugación">conjugaciones</a>. En latín el paradigma de flexión varía de acuerdo con el <a href="/wiki/Tema_(flexi%C3%B3n)" title="Tema (flexión)">tema</a> al que está adscrita la palabra. Los nombres y adjetivos se agrupan en cinco declinaciones, mientras que los verbos se agrupan dentro de cuatro tipos básicos de conjugaciones.
</p>
<p>En latín, el sustantivo, el adjetivo (flexión nominal) y el pronombre (flexión pronominal) adoptan diversas formas de acuerdo con su función sintáctica en la oración, formas conocidas como <a href="/wiki/Caso_(gram%C3%A1tica)" title="Caso (gramática)">casos gramaticales</a>. Existen en latín clásico seis formas que pueden tomar cada sustantivo, adjetivo o pronombre («casos»):
</p>
<p>Existen además restos de un caso adicional <a class="mw-redirect" href="/wiki/Idioma_proto-indoeuropeo" title="Idioma proto-indoeuropeo">indoeuropeo</a>: el <a href="/wiki/Caso_locativo" title="Caso locativo">locativo</a> (indicando localización, bien en el espacio, bien en el tiempo): <i>rurī</i> 'en el campo', <i>domī</i> 'en (la) casa'.
</p>
<p>El adjetivo también tiene formas flexivas, dado que concuerda necesariamente con un sustantivo en caso, género y número.
</p>
<p>A grandes rasgos hay dos temas dentro de la conjugación del verbo latino, <i>infectum</i> y <i>perfectum</i>: en el infectum están los tiempos que no indican un fin, una terminación, como el presente, el imperfecto y el futuro; son tiempos que no señalan el acto acabado, sino que, sea que está ocurriendo en el <a href="/wiki/Presente_(gram%C3%A1tica)" title="Presente (gramática)">presente</a>, <a href="/wiki/Pret%C3%A9rito_imperfecto" title="Pretérito imperfecto">ocurría con repetición</a> en el pasado (sin indicar cuando acabó), o bien un acto <a href="/wiki/Futuro" title="Futuro">futuro</a>. En este tema del verbo la raíz no cambia, al contrario que con el perfectum, que tiene su propia terminación irregular (<i>capere: pf. cepi — scribere: pf. scripsi — ferre pf. tuli — esse pf. fui — dicere pf. dixi</i>).
</p>
<p>El perfecto (del latín <i>perfectum</i>, de <i>perficere</i> 'terminar', 'completar') en cambio indica tiempos ya ocurridos, terminados, que son el <a class="new" href="/w/index.php?title=Pret%C3%A9rito_perfecto&amp;action=edit&amp;redlink=1" title="Pretérito perfecto (aún no redactado)">pretérito</a>, el <a class="mw-redirect" href="/wiki/Pluscuamperfecto" title="Pluscuamperfecto">pluscuamperfecto</a> y el <a href="/wiki/Futuro_perfecto" title="Futuro perfecto">futuro perfecto</a>.
</p>
<p>Ambos cuentan con los siguientes <a href="/wiki/Modo_gramatical" title="Modo gramatical">modos gramaticales</a> (a excepción del imperativo, que no existe en perfectum): el <a href="/wiki/Modo_indicativo_(gram%C3%A1tica)" title="Modo indicativo (gramática)">indicativo</a>, que expresa la realidad, certeza, la verdad objetiva; el <a class="mw-redirect" href="/wiki/Subjuntivo" title="Subjuntivo">subjuntivo</a> expresa irrealidad, subordinación, duda, hechos no constatados, a veces usado como <a class="mw-redirect" href="/wiki/Optativo" title="Optativo">optativo</a>; el <a href="/wiki/Modo_imperativo" title="Modo imperativo">imperativo</a>, que denota mandato, ruego, exhortación, y el <a href="/wiki/Infinitivo" title="Infinitivo">infinitivo</a>, una forma impersonal del verbo, usada como subordinado ante otro, o dando una idea en abstracto. Con seis personas en cada tiempo —primera, segunda y tercera, cada una en singular y plural— y dos <a href="/wiki/Voz_gramatical" title="Voz gramatical">voces</a> —<a href="/wiki/Voz_activa" title="Voz activa">activa</a> cuando el sujeto es el agente y <a href="/wiki/Voz_pasiva" title="Voz pasiva">pasiva</a> cuando el sujeto padece una acción no ejecutada por él—, más los restos de una voz <a href="/wiki/Voz_gramatical#Voz_media" title="Voz gramatical">media</a>, un verbo no <a href="/wiki/Verbo_deponente" title="Verbo deponente">deponente</a> normalmente posee unas 130 desinencias.
</p>
<p>Los <a href="/wiki/Verbo" title="Verbo">verbos</a> en latín usualmente se identifican por cinco diferentes temas de conjugaciones (los grupos de verbos con formas flexivas similares): el tema en -a larga (-ā-), el tema en -e larga (-ē-), tema en consonante, tema en -i larga (-ī-) y, por último, el tema en -i breve (-i-). Básicamente solo hay un modo de la conjugación latina de los verbos, pero vienen influidos por cierta <a href="/wiki/Vocal" title="Vocal">vocal</a> que provoca algunos cambios en sus <a class="mw-redirect" href="/wiki/Desinencia" title="Desinencia">desinencias</a>. Por ejemplo, en su terminación de futuro: mientras lo común era indicarlo mediante un tiempo proveniente del <a class="mw-redirect" href="/wiki/Subjuntivo" title="Subjuntivo">subjuntivo</a>, en los verbos influidos por <i>E</i> o <i>A</i> larga, el futuro sonaría exactamente igual que el presente, por lo que tuvieron que cambiar sus desinencias.
</p>
<p>El objeto de la <a href="/wiki/Sintaxis" title="Sintaxis">sintaxis</a> es organizar las partes del discurso de acuerdo con las normas de la lengua para expresar correctamente el mensaje. La <a href="/wiki/Concordancia_gramatical" title="Concordancia gramatical">concordancia</a>, que es un sistema de reglas de los accidentes gramaticales, en latín afecta a <a href="/wiki/G%C3%A9nero_gramatical" title="Género gramatical">género</a>, <a href="/wiki/N%C3%BAmero_gramatical" title="Número gramatical">número</a>, <a class="mw-redirect" href="/wiki/Caso_gramatical" title="Caso gramatical">caso</a> y <a href="/wiki/Persona_gramatical" title="Persona gramatical">persona</a>. Esta jerarquiza las categorías gramaticales, de tal manera que el verbo y el <a href="/wiki/Adjetivo" title="Adjetivo">adjetivo</a> adecúan sus rasgos a los del nombre con el que conciertan. Las concordancias son adjetivo/sustantivo o de verbo/sustantivo. Obsérvese el ejemplo: «<i>Anim<b>us</b> aequ<b>us</b> optim<b>um</b> est aerumnae condiment<b>um</b></i>» («Un ánimo equitativamente bueno es el condimento de la miseria»).<sup class="reference separada" id="cite_ref-10"><a href="#cite_note-10"><span class="corchete-llamada">[</span>9<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Mediante la construcción se sitúan los <a href="/wiki/Sintagma" title="Sintagma">sintagmas</a> en el discurso. En latín el orden de la frase es <a class="mw-redirect" href="/wiki/SOV" title="SOV">S-O-V</a>, o sea, primero va el sujeto, el objeto, y al final el verbo. Esta idea de construcción supone que las palabras tienen ese orden natural; no es tan fácil de establecer en rigor. Un ejemplo de orden natural sería «<i>Omnia mutantur, nihil interit</i>» («Todo cambia, nada perece»).<sup class="reference separada" id="cite_ref-11"><a href="#cite_note-11"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​ Por oposición, al orden que incluye desviaciones de la norma, por razones éticas o estéticas, se le da el nombre de figurado, inverso u oblicuo, como en «<i>Vim Demostenes habuit</i>», donde <i>Demostenes</i> ha sido desplazado de su primer lugar propio (el primer lugar en la frase podía usarse para señalar la palabra más importante: puesta en relieve).
</p>
<p>para una explicación de los símbolos usados)
</p>
<p>El latín se pronunciaba de forma diferente en los tiempos antiguos, en los tiempos clásicos y en los posclásicos; también era diferente el latín culto de los diversos dialectos de latín vulgar. Al ser el latín una lengua muerta, no se sabe con exactitud la <a href="/wiki/Pronunciaci%C3%B3n" title="Pronunciación">pronunciación</a> de la grafía latina: históricamente se han propuesto diversas formas. Las más conocidas son la eclesiástica (o italiana) que se acerca más a la pronunciación del italiano que a la del latín clásico<sup class="reference separada" id="cite_ref-:3_12-0"><a href="#cite_note-:3-12"><span class="corchete-llamada">[</span>11<span class="corchete-llamada">]</span></a></sup>​, la <i><a class="new" href="/w/index.php?title=Pronuntiatio_restituta&amp;action=edit&amp;redlink=1" title="Pronuntiatio restituta (aún no redactado)">pronuntiatio restituta</a></i> (pronunciación reconstruida), que es el intento de reconstruir la <a href="/wiki/Fon%C3%A9tica" title="Fonética">fonética</a> original, y la erasmista. La comparación con otras lenguas indoeuropeas también es importante para determinar el probable valor fonético de ciertas letras.
</p>
<p>No hay un acuerdo entre los estudiosos. Pero parece ser que el latín, a lo largo de su historia, pasó por períodos en los que el acento era musical y por otros en los que el acento era de intensidad. Lo que está claro es que el acento tónico depende de la cantidad de las sílabas según el siguiente esquema:
</p>
<p>El latín clásico tenía cinco vocales breves /a, e, i, o, u/ y cinco vocales largas /ā, ē, ī, ō, ū/ con valor de distinción fonológica.
</p>
<p>El sistema fonológico del vocalismo latino estaba conformada por la oposición dos tipos de cantidad o duración: las vocales de mayor duración, denominadas largas, y las de menor duración, denominadas breves. En la actualidad el símbolo (˘) lo usamos para designar las vocales breves y el símbolo (¯) los empleamos para designar las vocales largas.
</p>
<p>/ă/ breve /ĕ/ breve /ĭ/ breve /ŏ/ breve /ŭ/ breve
</p>
<p>/ā/ larga /ē/ larga /ī/ larga /ō/ larga /ū/ larga
</p>
<p>Los siguientes son algunos ejemplos de pares contrastivos donde la diferencia es solo la duración de una vocal.
</p>
<p>La <i>y (i græca)</i> originalmente no formaba parte del sistema vocálico latino y solo aparecía en préstamos cultos griegos. Su pronunciación en el griego clásico correspondía aproximadamente a la de la <i>u</i> francesa o <i>ü</i> alemana [y]. En latín generalmente se pronunciaba como una <i>i</i>, pues para la población poco educada resultó difícil pronunciar la /y/ griega. Otras evidencias a favor de la existencia del sonido /y/ en latín es que era una de las tres <a href="/wiki/Letras_claudias" title="Letras claudias">letras claudias</a>, concretamente la llamada <i>sonus medius</i> (escrito como: Ⱶ) se creó para representar un sonido intermedio entre [i] y [u], muy probablemente [y] (o tal vez [ɨ]) que aparecía estar detrás de ciertas vacilaciones como <small>OPTUMUS / OPTIMUS</small> 'óptimo', <small>LACRUMA / LACRIMA</small> 'lágrima'.
</p>
<p>Tanto unas como otras podían darse en cualquier posición; es decir, no tenían ninguna relevancia fonológica el acento ni la intensidad.
</p>
<p>El tratamiento de las vocales del latín clásico varía según el tipo de sílaba en que se encuentran. Estaban muy influenciadas por el acento. El acento original de la lengua “mater” del latín era musical y libre, pero ese sistema desapareció y ya no estaba reflejado en el latín clásico, en el que el acento carga sobre la penúltima sílaba si esta es larga y sobre la antepenúltima si la penúltima es breve. Sin embargo los estudiosos están divididos en lo que respecta a sus opiniones sobre la naturaleza del acento latino, aunque la opinión de la mayoría de los lingüistas cree que el acento tonal o musical es la que se mantuvo hasta el siglo <span style="font-variant:small-caps;text-transform:lowercase">IV</span> d. C.
</p>
<p>Las vocales del sistema fonológico latín clásico eran a, e, i, o, u, que podían ser largas o breves, y las combinaciones en diptongo de las tres primeras con las semivocales o sonantes i, u, r, l, m, n. El tratamiento de estos sonidos heredados en latín varía según el tipo de sílaba en que aparecen. Pueden dividirse en sílaba inicial, sílaba media y sílaba final. En el latín más antiguo estas vocales estaban acentuadas, y por ello se mantienen con regular constancia.
</p>

<p><b>Sílabas no iniciales<sup class="reference separada" id="cite_ref-:2_15-1"><a href="#cite_note-:2-15"><span class="corchete-llamada">[</span>14<span class="corchete-llamada">]</span></a></sup>​</b>
</p>
<p>En sílabas no iniciales, como hemos visto más arriba, las vocales breves y diptongos breves experimentaron alteraciones que diferían según la sílaba terminase en vocal o consonante. Todo esto lo podemos resumir bajo los epígrafes de las sílabas abiertas y las cerradas.
</p>
<p><b>Las sonantes</b>
</p>
<p>Ciertos tipos de sonidos, según el contexto fonético en que se hallen, funcionan como vocales o consonantes, es decir, como centro silábico o no.
</p>
<p><b>Consonantismo</b><sup class="reference separada" id="cite_ref-:0_13-1"><a href="#cite_note-:0-13"><span class="corchete-llamada">[</span>12<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-:2_15-2"><a href="#cite_note-:2-15"><span class="corchete-llamada">[</span>14<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las consonantes <small>F, K, L, M, N, P, S</small> se pronunciaban como en español. La <small>B, D, G</small> eran siempre oclusivas sonoras. La <small>C</small> representaba los sonidos [k] y [g] en latín arcaico, aunque en latín clásico se reservó solo para el sonido [k] al crearse la letra <small>G</small>. El <a href="/wiki/D%C3%ADgrafo" title="Dígrafo">dígrafo</a> <small>QU</small> correspondía en latín tardío a [kw] (en latín arcaico seguramente era una <a href="/wiki/Consonante" title="Consonante">labiovelar</a> [k<sup>w</sup>]). La pronunciación de <small>R</small> sencilla no está clara. Podría haber sido como la del español (que según la posición es [ɾ] <a class="mw-redirect" href="/wiki/Vibrante_simple" title="Vibrante simple">vibrante simple</a> o [r] <a class="mw-redirect" href="/wiki/Vibrante_m%C3%BAltiple" title="Vibrante múltiple">vibrante múltiple</a>) o tal vez como la del italiano (que muchas veces es [ɾ] incluso en inicio de palabra); entre dos vocales podría haber sido igual a la "rr" del español (por lo que <small>CARŌ</small> sonaría con la misma de "rr" de "carro") o tal vez una geminada [ɾː]. La letra <small>V</small> representaba según el contexto la semiconsonante /w/ o las vocales /ŭ, ū/. En latín tardío <small>V</small> pasó a [β], reforzándose en [b] inicial en algunos dialectos occidentales y fricativa dándose [v] en la mayor parte de la Romania. La <i>x</i> tenía el sonido [ks], como en <i>éxito</i>. La <small>Z</small> originalmente no formaba parte del alfabeto latino y aparecía solamente en algunos préstamos griegos y correspondía, al principio, al sonido [dz] como en la palabra italiana <i>gazza</i>, luego terminó fricativizándose en [z].
</p>
<p>No se sabe con certeza la pronunciación exacta de la <i>s</i> latina. Teniendo en cuenta que era la única sibilante en el sistema consonántico latino, y que en el desarrollo del francés podría haber sido la causa del desarrollo de la vocal [a] del francés medieval a [ɑ] antes de ella (ej. <i>casse</i>, del latín <small>CAPSA</small>, pronunciada originalmente [kasə] y luego [kɑsə]), muchos lingüistas consideran que tenía un sonido de realización apicoalveolar o predorsodental de /s/, parecido al del español del medio y norte de España. Algunos han propuesto que en muchas lenguas con una única sibilante el alófono principal de /s/ es apicoalveolar, ya que no existe la necesidad de distinguirlo de otro fonema que sería la [ʃ]. Aunque por otra parte, sí existen lenguas con una sibilante donde la /s/ no es apicoalveolar, por ejemplo el <a href="/wiki/Espa%C3%B1ol_de_Am%C3%A9rica" title="Español de América">español de América</a>. Quizás este hecho sea el origen del <a href="/wiki/Rotacismo" title="Rotacismo">rotacismo</a> intervocálico latino en palabras como <i><small>FLOS</small> &gt; <small>FLŌRĒS</small></i> (&lt; *<i>floses)</i>.
</p>
<p>El sistema consonántico del latín clásico estaba formado por cuatro subsistemas: el de las consonantes nasales, el de las líquidas, el de las semivocales y el de las orales no líquidas:
</p>
<p>Los fonemas consonánticos comprendían una riza variedad de oclusivas, sordas (p, t, k, q y qʷ), sonoras (b, d, g, y gʷ), con los correspondientes sonidos aspirados. La única fricativa era la s. El latín no distingue entre la serie palatal y la velar ni entre aspiradas sordas y sonoras. Del sistema mencionado, el latín conservó generalmente p, t, k, (q), qʷ y b, d, g, (g), afectando los cambios importantes a las labiovelares sonoras y a las oclusivas aspiradas.
</p>
<p><a href="/wiki/Lat%C3%ADn_vulgar" title="Latín vulgar">Latín vulgar</a> (en latín, <i>sermo vulgaris</i>) (o latín tardío) es un término que se emplea para referirse a los dialectos <a class="mw-redirect" href="/wiki/Vern%C3%A1cula" title="Vernácula">vernáculos</a> del latín hablado en las <a class="mw-redirect" href="/wiki/Provincia_romana" title="Provincia romana">provincias</a> del <a href="/wiki/Imperio_romano" title="Imperio romano">Imperio romano</a>. En particular, el término se refiere al período tardío, que abarca hasta que esos dialectos se diferenciaron los unos de los otros lo suficiente como para que se les considerase el período temprano de las <a href="/wiki/Lenguas_romances" title="Lenguas romances">lenguas romances</a>. La diferenciación que se suele asignar al siglo <span style="font-variant:small-caps;text-transform:lowercase">IX</span> aproximadamente.
</p>
<p>Ya en el ámbito de la gramática, habría que destacar los siguientes fenómenos: en el sistema verbal, la creación de formas compuestas (normalmente mediante la combinación de <i>habere</i> con el participio pasado de otro verbo) paralelas al paradigma sintético ya existente; y la construcción de la pasiva con el auxiliar ser y el participio del verbo que se conjuga (el francés y el italiano también emplean ser como auxiliar en los tiempos compuestos de verbos de «estado» y «movimiento»).
</p>
<p>Los seis casos de la declinación latina se redujeron y posteriormente se reemplazaron con frases prepositivas (el rumano moderno mantiene un sistema de tres casos, tal vez por influencia eslava; hasta el siglo <span style="font-variant:small-caps;text-transform:lowercase">XVIII</span> también algunas variantes romanches de Suiza tenían caso). Si en latín no había artículos, los romances los desarrollaron a partir de los determinantes; son siempre <a class="mw-redirect" href="/wiki/Procl%C3%ADtico" title="Proclítico">proclíticos</a>, menos en rumano, lengua en la que van pospuestos al sustantivo.
</p>
<p>En cuanto a los demostrativos, la mayoría de las lenguas románicas cuenta con tres deícticos que expresan «cercanía» (<i>este</i>), «distancia media» (<i>ese</i>) y «lejanía» (<i>aquel</i>). Sin embargo, el francés, el rumano y el extinto romance andalusí distinguen solo dos términos (uno para «proximidad» y otro para «lejanía»). El género neutro desapareció en todas partes menos en Rumania, Galicia y Asturias, en la que existen algunos sustantivos no contables con terminación en neutro (-o) y una terminación propia igual en el adjetivo cuando concuerda con sustantivos no contables o "de materia", ya acaben en -a, -o, -u o consonante. El orden sintáctico responde a la libre disposición de los elementos en la oración propia del latín. Aun así domina ordenación sintagmática de <b>sujeto + verbo + objeto</b> (aunque las lenguas del sureste permiten mayor flexibilidad en la ubicación del sujeto).
</p>
<p>El latín tardío o latín vulgar cambió muchos de los sonidos del latín culto o clásico (<a class="external text" href="http://recursos.cnice.mec.es/latingriego/Palladium/latin/esl132ca2.php" rel="nofollow">1</a>).
</p>
<p>Los más importantes procesos fonológicos que afectaron al consonantismo fueron: la edición de intervocálicas (las sordas se sonorizan y las sonoras desaparecen) y la palatalización de consonantes velares y dentales, a menudo con una africación posterior (<i>lactuca</i> &gt; gallego, <i>leituga</i>; español, <i>lechuga</i>; catalán, <i>lletuga</i>). Ambos procesos tuvieron mayor incidencia en el Oeste (de las lenguas occidentales, el sardo fue la única que no palatalizó). Otra característica es la reducción de las geminadas latinas, que solamente preservó el italiano.
</p>
<p>Aquí también se podrían agregar algunos otros cambios fonéticos, como la pérdida de la /d/ intervocálica en <a href="/wiki/Idioma_espa%C3%B1ol" title="Idioma español">español</a> o la pérdida de la /n/ y /l/ en <a href="/wiki/Idioma_portugu%C3%A9s" title="Idioma portugués">portugués</a>, <a href="/wiki/Idioma_gallego" title="Idioma gallego">gallego</a>, <a href="/wiki/Idioma_catal%C3%A1n" title="Idioma catalán">catalán</a> y <a class="mw-redirect" href="/wiki/Occitano" title="Occitano">occitano</a>.
</p>
<p>El latín de ser una marcada <a href="/wiki/Lengua_sint%C3%A9tica" title="Lengua sintética">lengua sintética</a> pasó a ser poco a poco una <a class="mw-redirect" href="/wiki/Lengua_anal%C3%ADtica" title="Lengua analítica">lengua analítica</a>, en la que el orden de las palabras es un elemento de <a href="/wiki/Sintaxis" title="Sintaxis">sintaxis</a> necesario. Ya en el latín arcaico empezó a constatarse la desestima de este modelo y se advierte su reemplazo por un sistema de preposiciones. Este sistema no se propició de forma definitiva hasta que ocurrieron los cambios fonéticos del latín vulgar. Esto provocó que el sistema de casos fuera difícil de mantener, perdiéndolos paulatinamente en un lapso relativamente rápido.
</p>
<p>Algunos dialectos conservaron una parte de este tipo de flexiones: el francés antiguo logró mantener un sistema de casos con un <a href="/wiki/Caso_nominativo" title="Caso nominativo">nominativo</a> y uno <a href="/wiki/Caso_oblicuo" title="Caso oblicuo">oblicuo</a> hasta entrado el siglo <span style="font-variant:small-caps;text-transform:lowercase">XII</span>. El <a class="mw-redirect" href="/wiki/Occitano" title="Occitano">occitano</a> antiguo también conservó un sistema parecido, así como el <a class="mw-redirect" href="/wiki/Retorromano" title="Retorromano">retorromano</a>, que lo perdió hace unos 100 años. El <a href="/wiki/Idioma_rumano" title="Idioma rumano">rumano</a> aún preserva un separado genitivo-dativo con vestigios de un <a class="mw-redirect" href="/wiki/Vocativo" title="Vocativo">vocativo</a> en las voces femeninas.
</p>
<p>La distinción entre el <a href="/wiki/N%C3%BAmero_gramatical" title="Número gramatical">singular</a> y el plural se marcaba con dos formas diferentes en las lenguas romances. En el norte y en el oeste de la <a class="new" href="/w/index.php?title=L%C3%ADnea_Spezia-Rimini&amp;action=edit&amp;redlink=1" title="Línea Spezia-Rimini (aún no redactado)">línea Spezia-Rimini</a>, al norte de Italia, el singular usualmente se distingue del plural por una /s/ final, que se presenta en el antiguo plural <a href="/wiki/Caso_acusativo" title="Caso acusativo">acusativo</a>. Al sur y al este de esta misma línea, se produce una alternancia vocálica final, proveniente del nominativo plural de la primera y la segunda declinación.
</p>
<p>La influencia del lenguaje coloquial, que prestaba mucha importancia al elemento <a href="/wiki/Deixis" title="Deixis">deíctico</a> o señalador, originó un profuso empleo de los <a class="new" href="/w/index.php?title=Art%C3%ADculos_demostrativos&amp;action=edit&amp;redlink=1" title="Artículos demostrativos (aún no redactado)">demostrativos</a>. Aumentó muy significativamente el número de demostrativos que acompañaban al sustantivo, sobre todo haciendo referencia a un elemento nombrado antes. En este empleo <a href="/wiki/An%C3%A1fora_(gram%C3%A1tica)" title="Anáfora (gramática)">anafórico</a>, el valor demostrativo de <i>ille</i> (o de <i>ipse</i>, en algunas regiones) fue desdibujándose para aplicarse también a todo <a href="/wiki/Sustantivo" title="Sustantivo">sustantivo</a> que se refiriese a seres u objetos consabidos. De este modo, surgieron los <a class="mw-redirect" href="/wiki/Art%C3%ADculo_definido" title="Artículo definido">artículos definidos</a> (el, la, los, las, lo) inexistentes en latín clásico y presentes en todas las lenguas romances. A su vez, el numeral <i>unus</i>, empleado con el valor indefinido de alguno, cierto, extendió sus usos acompañando al sustantivo que designaba entes no mencionados antes, cuya entrada en el discurso suponía la introducción de información nueva. Con ese nuevo empleo de <i>unus</i>, surgieron los artículos definidos (un, una, unos, unas) que tampoco existían en latín clásico.
</p>
<p>En latín clásico los determinantes solían quedar en el interior de la frase. Sin embargo, el latín vulgar propendía a una colocación en que las palabras se sucedieran con arreglo a una progresiva determinación, al tiempo que el período sintáctico se hacía menos extenso. Al final de la época imperial este nuevo orden se abría paso incluso en la lengua escrita, aunque permanecían restos del antiguo, sobre todo en las oraciones subordinadas.
</p>
<p>Las <a href="/wiki/Preposici%C3%B3n" title="Preposición">preposiciones</a> existentes hasta ese momento eran insuficientes para las nuevas necesidades gramaticales y el latín vulgar tuvo que generar nuevas. Así, se crearon muchas preposiciones nuevas, fusionando muchas veces dos o tres que ya existiesen previamente, como es el caso de <i>detrás</i> (de + trans), <i>dentro</i> (de + intro), <i>desde</i> (de + ex + de), <i>hacia</i> (facie + ad), <i>adelante</i> (adenante + ad + de + in + ante).
</p>
<p>Hoy en día, el latín sigue siendo utilizado como <a href="/wiki/Lengua_lit%C3%BArgica" title="Lengua litúrgica">lengua litúrgica</a> oficial de la Iglesia católica de <a class="mw-redirect" href="/wiki/Rito_latino" title="Rito latino">rito latino</a>. Su estatus de <a href="/wiki/Lengua_muerta" title="Lengua muerta">lengua muerta</a> (no sujeta a evolución) le confiere particular utilidad para usos <a href="/wiki/Liturgia" title="Liturgia">litúrgicos</a> y <a href="/wiki/Teolog%C3%ADa" title="Teología">teológicos</a>, ya que es necesario que los significados de las palabras se mantengan estables. Así, los textos que se manejan en esas disciplinas conservarán su significado y su sentido para lectores de distintos siglos. Además, esta lengua se usa en medios radiofónicos y de prensa de la <a href="/wiki/Santa_Sede" title="Santa Sede">Santa Sede</a>. El <a href="/wiki/Papa" title="Papa">papa</a> entrega sus mensajes escritos en este idioma; las publicaciones oficiales de la Santa Sede son en latín, a partir de las cuales se traducen a otros idiomas. En noviembre de 2012 fue fundada la <a href="/wiki/Pontificia_Academia_de_Latinidad" title="Pontificia Academia de Latinidad">Pontificia Academia de Latinidad</a> por el papa Benedicto XVI para potenciar el latín en todo el mundo. En la <a class="mw-redirect" href="/wiki/Iglesia_anglicana" title="Iglesia anglicana">Iglesia anglicana</a>, después de la publicación del <i><a href="/wiki/Libro_de_Oraci%C3%B3n_Com%C3%BAn" title="Libro de Oración Común">Libro de Oración Común</a></i> anglicano de 1559, una edición en latín fue publicada en 1560 para usarse en universidades; como en la de Oxford, donde la liturgia se celebra aún en latín.<sup class="reference separada" id="cite_ref-16"><a href="#cite_note-16"><span class="corchete-llamada">[</span>15<span class="corchete-llamada">]</span></a></sup>​ Más recientemente apareció una edición en latín del <i>Libro de Oración Común</i> de los Estados Unidos de 1979.<sup class="reference separada" id="cite_ref-17"><a href="#cite_note-17"><span class="corchete-llamada">[</span>16<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Por otra parte, la <a class="mw-redirect" href="/wiki/Nomenclatura_(biolog%C3%ADa)" title="Nomenclatura (biología)">nomenclatura</a> de <a href="/wiki/Especie" title="Especie">especies</a> y <a href="/wiki/Tax%C3%B3n" title="Taxón">grupos</a> de la clasificación biológica sigue haciéndose con términos en latín o latinizados. Además de la terminología de la <a href="/wiki/Filosof%C3%ADa" title="Filosofía">filosofía</a>, <a href="/wiki/Derecho" title="Derecho">derecho</a> y <a href="/wiki/Medicina" title="Medicina">medicina</a>, donde se preservan muchos términos, <a class="mw-redirect" href="/wiki/Locuciones_latinas" title="Locuciones latinas">locuciones</a> y abreviaciones latinas. En la cultura popular aún puede verse escrito en los lemas de universidades u otras organizaciones y también puede oírse en diálogos de películas que se desarrollan en época romana como <i><a href="/wiki/Sebastiane" title="Sebastiane">Sebastiane</a></i> y <i><a class="mw-redirect" href="/wiki/La_Pasi%C3%B3n_de_Cristo_(pel%C3%ADcula)" title="La Pasión de Cristo (película)">La Pasión de Cristo</a></i>.
