<p>La <b>leche</b> (del <a class="mw-redirect" href="/wiki/Idioma_lat%C3%ADn" title="Idioma latín">latín</a>: <i>lac</i>) es una <a href="/wiki/Secreci%C3%B3n" title="Secreción">secreción</a> <a href="/wiki/Nutrimento" title="Nutrimento">nutritiva</a> de color blanquecino opaco producida por las <a href="/wiki/C%C3%A9lula" title="Célula">células</a> secretoras de las <a class="mw-redirect" href="/wiki/Gl%C3%A1ndulas_mamarias" title="Glándulas mamarias">glándulas mamarias</a> de los <a href="/wiki/Mammalia" title="Mammalia">mamíferos</a>, incluidos los <a href="/wiki/Monotremata" title="Monotremata">monotremas</a>.<sup class="reference separada" id="cite_ref-lechq_1-0"><a href="#cite_note-lechq-1"><span class="corchete-llamada">[</span>1<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-delaval_2-0"><a href="#cite_note-delaval-2"><span class="corchete-llamada">[</span>2<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-crlnd_3-0"><a href="#cite_note-crlnd-3"><span class="corchete-llamada">[</span>3<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-4"><a href="#cite_note-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​ Su principal función es la de nutrir a las <a href="/wiki/Descendencia" title="Descendencia">crías</a> hasta que sean capaces de digerir otros <a href="/wiki/Alimento" title="Alimento">alimentos</a>, además de proteger su <a href="/wiki/Tracto_gastrointestinal" title="Tracto gastrointestinal">tracto gastrointestinal</a> contra <a class="mw-redirect" href="/wiki/Pat%C3%B3geno" title="Patógeno">patógenos</a>, <a href="/wiki/Toxina" title="Toxina">toxinas</a> e <a href="/wiki/Inflamaci%C3%B3n" title="Inflamación">inflamación</a> y contribuir a su salud metabólica regulando los procesos de obtención de energía, en especial el <a href="/wiki/Metabolismo" title="Metabolismo">metabolismo</a> de la <a href="/wiki/Glucosa" title="Glucosa">glucosa</a> y la <a href="/wiki/Insulina" title="Insulina">insulina</a>.<sup class="reference separada" id="cite_ref-5"><a href="#cite_note-5"><span class="corchete-llamada">[</span>5<span class="corchete-llamada">]</span></a></sup>​ Esta capacidad es una de las características que definen a los <a class="mw-redirect" href="/wiki/Mam%C3%ADferos" title="Mamíferos">mamíferos</a>. Es el único fluido que ingieren las crías de los mamíferos (niño de <a href="/wiki/Pecho" title="Pecho">pecho</a> en el caso de los seres humanos) hasta el <a href="/wiki/Destete" title="Destete">destete</a>. La secreción láctea de una hembra en los días anteriores y posteriores al <a href="/wiki/Parto" title="Parto">parto</a> se llama <a href="/wiki/Calostro" title="Calostro">calostro</a>.
</p>
<p>Las leches de algunos de los mamíferos <a class="mw-redirect" href="/wiki/Animales_dom%C3%A9sticos" title="Animales domésticos">domésticos</a> (de <a class="mw-redirect" href="/wiki/Vaca" title="Vaca">vaca</a>, principalmente, pero también de <a class="mw-disambig" href="/wiki/B%C3%BAfalo" title="Búfalo">búfala</a>, <a class="mw-redirect" href="/wiki/Oveja" title="Oveja">oveja</a>, <a class="mw-redirect" href="/wiki/Cabra" title="Cabra">cabra</a>, <a class="mw-redirect" href="/wiki/Yegua" title="Yegua">yegua</a>, <a class="mw-redirect" href="/wiki/Camello" title="Camello">camella</a>, <a href="/wiki/Alces_alces" title="Alces alces">alce</a>, <a class="mw-redirect" href="/wiki/Cerdo" title="Cerdo">cerda</a> y otros) forman parte de la <a href="/wiki/Alimentaci%C3%B3n_humana" title="Alimentación humana">alimentación humana</a> corriente en algunas <a href="/wiki/Cultura" title="Cultura">culturas</a>, base de numerosos <a class="mw-redirect" href="/wiki/Productos_l%C3%A1cteos" title="Productos lácteos">productos lácteos</a>, como la <a href="/wiki/Mantequilla" title="Mantequilla">mantequilla</a>, el <a href="/wiki/Queso" title="Queso">queso</a> y el <a href="/wiki/Yogur" title="Yogur">yogur</a>, entre otros.<sup class="reference separada" id="cite_ref-6"><a href="#cite_note-6"><span class="corchete-llamada">[</span>6<span class="corchete-llamada">]</span></a></sup>​ Es muy frecuente el empleo de derivados de la leche en las <a href="/wiki/Industria_alimentaria" title="Industria alimentaria">industrias agroalimentarias</a>, <a href="/wiki/Industria_qu%C3%ADmica" title="Industria química">químicas</a> y <a href="/wiki/Industria_farmac%C3%A9utica" title="Industria farmacéutica">farmacéuticas</a>, como la <a href="/wiki/Leche_condensada" title="Leche condensada">leche condensada</a>, la <a href="/wiki/Leche_en_polvo" title="Leche en polvo">leche en polvo</a>, la <a href="/wiki/Case%C3%ADna" title="Caseína">caseína</a> o la <a href="/wiki/Lactosa" title="Lactosa">lactosa</a>.<sup class="reference separada" id="cite_ref-7"><a href="#cite_note-7"><span class="corchete-llamada">[</span>7<span class="corchete-llamada">]</span></a></sup>​ La leche de vaca se utiliza también en la alimentación animal. Está compuesta principalmente por <a href="/wiki/Agua" title="Agua">agua</a>, <a href="/wiki/Ion" title="Ion">iones</a> (<a class="mw-redirect" href="/wiki/Sal" title="Sal">sal</a>, minerales y <a href="/wiki/Calcio" title="Calcio">calcio</a>), <a class="mw-redirect" href="/wiki/Gl%C3%BAcidos" title="Glúcidos">glúcidos</a> (<a href="/wiki/Lactosa" title="Lactosa">lactosa</a>), materia <a href="/wiki/Grasa" title="Grasa">grasa</a> y <a class="mw-redirect" href="/wiki/Prote%C3%ADnas" title="Proteínas">proteínas</a>.<sup class="reference separada" id="cite_ref-delaval_2-1"><a href="#cite_note-delaval-2"><span class="corchete-llamada">[</span>2<span class="corchete-llamada">]</span></a></sup>​ Hay evidencias de que, además, en la leche de casi todos los mamíferos (incluidos los humanos) se pueden formar por rotura de las caseínas péptidos bioactivos denominados <a class="new" href="/w/index.php?title=Casomorfina&amp;action=edit&amp;redlink=1" title="Casomorfina (aún no redactado)">casomorfinas</a>, que actúan como agonistas de los receptores de opioides, mimetizando el efecto biológico de la morfina. La suposición de que una de ellas, la β-casomorfina-7 está implicada en el desarrollo de autismo o enfermedades cardiovasculares carece de evidencias científicas.<sup class="reference separada" id="cite_ref-8"><a href="#cite_note-8"><span class="corchete-llamada">[</span>8<span class="corchete-llamada">]</span></a></sup>​ La leche de los <a href="/wiki/Mam%C3%ADferos_marinos" title="Mamíferos marinos">mamíferos marinos</a>, como las <a href="/wiki/Balaenidae" title="Balaenidae">ballenas</a> (por ejemplo), es mucho más rica en grasas y <a class="mw-redirect" href="/wiki/Nutriente" title="Nutriente">nutrientes</a> que la de los mamíferos terrestres.<sup class="reference separada" id="cite_ref-9"><a href="#cite_note-9"><span class="corchete-llamada">[</span>9<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Una parte de la población presenta <a href="/wiki/Intolerancia_a_la_lactosa" title="Intolerancia a la lactosa">intolerancia al azúcar de la leche</a> (la <a href="/wiki/Lactosa" title="Lactosa">lactosa</a>). Puede ser de origen genético (<a href="/wiki/Intolerancia_a_la_lactosa#Deficiencia_primaria_de_lactasa" title="Intolerancia a la lactosa">intolerancia a la lactosa primaria</a>) o debida a enfermedades que dañan el intestino delgado (<a href="/wiki/Intolerancia_a_la_lactosa#Deficiencia_secundaria_de_lactasa" title="Intolerancia a la lactosa">intolerancia a la lactosa secundaria o adquirida</a>). Cualquier persona con intolerancia genética cuyo intestino está sano es capaz de consumir al menos 12 g de lactosa en cada comida (la cantidad contenida en una taza de leche) sin experimentar ningún síntoma o solo síntomas leves.<sup class="reference separada" id="cite_ref-NIHconsensus2010_10-0"><a href="#cite_note-NIHconsensus2010-10"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-Szilagyi2015_11-0"><a href="#cite_note-Szilagyi2015-11"><span class="corchete-llamada">[</span>11<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-SilanikoveLeitner2015_12-0"><a href="#cite_note-SilanikoveLeitner2015-12"><span class="corchete-llamada">[</span>12<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-ByersSavaiano2005_13-0"><a href="#cite_note-ByersSavaiano2005-13"><span class="corchete-llamada">[</span>13<span class="corchete-llamada">]</span></a></sup>​ El consumo de productos lácteos por parte de personas con intolerancia a la lactosa no produce daños en el <a href="/wiki/Tracto_gastrointestinal" title="Tracto gastrointestinal">tracto gastrointestinal</a>, sino que se limita a molestias digestivas transitorias.<sup class="reference separada" id="cite_ref-Heyman_14-0"><a href="#cite_note-Heyman-14"><span class="corchete-llamada">[</span>14<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-NIHconsensus2010_10-1"><a href="#cite_note-NIHconsensus2010-10"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-Szilagyi2015_11-1"><a href="#cite_note-Szilagyi2015-11"><span class="corchete-llamada">[</span>11<span class="corchete-llamada">]</span></a></sup>​ Las reacciones a cantidades más pequeñas de lácteos no se explican por una intolerancia de tipo genético, sino que indican la existencia de una enfermedad intestinal no diagnosticada (principalmente la <a href="/wiki/Celiaqu%C3%ADa" title="Celiaquía">enfermedad celíaca</a> y la <a href="/wiki/Sensibilidad_al_gluten_no_cel%C3%ADaca" title="Sensibilidad al gluten no celíaca">sensibilidad al gluten no celíaca</a>) o alergia a las proteínas de la leche.<sup class="reference separada" id="cite_ref-NIHconsensus2010_10-2"><a href="#cite_note-NIHconsensus2010-10"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-AzizHadjivassiliou2015_15-0"><a href="#cite_note-AzizHadjivassiliou2015-15"><span class="corchete-llamada">[</span>15<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-PalWoodford2015_16-0"><a href="#cite_note-PalWoodford2015-16"><span class="corchete-llamada">[</span>16<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-CrittendenBennett2005_17-0"><a href="#cite_note-CrittendenBennett2005-17"><span class="corchete-llamada">[</span>17<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-NwaruHickstein2014_18-0"><a href="#cite_note-NwaruHickstein2014-18"><span class="corchete-llamada">[</span>18<span class="corchete-llamada">]</span></a></sup>​ Etiquetar a una persona simplemente con intolerancia a la lactosa sin realizar un exhaustivo estudio incluyendo todas las pruebas médicas necesarias, provoca con frecuencia largos retrasos en el diagnóstico de enfermedades subyacentes graves, causantes de la malabsorción de lactosa, la más frecuente la enfermedad celíaca.<sup class="reference separada" id="cite_ref-GrimhedenAnderlid_19-0"><a href="#cite_note-GrimhedenAnderlid-19"><span class="corchete-llamada">[</span>19<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El consumo humano de la leche de origen animal comenzó hace unos once mil años con la domesticación del ganado durante el llamado <a href="/wiki/%C3%93ptimo_clim%C3%A1tico" title="Óptimo climático">óptimo climático</a>. Este proceso se dio en especial en <a href="/wiki/Oriente_Medio" title="Oriente Medio">Oriente Medio</a>, impulsando la <a href="/wiki/Revoluci%C3%B3n_neol%C3%ADtica" title="Revolución neolítica">revolución neolítica</a>.<sup class="reference separada" id="cite_ref-orign_20-0"><a href="#cite_note-orign-20"><span class="corchete-llamada">[</span>20<span class="corchete-llamada">]</span></a></sup>​ El primer animal que se domesticó fue la <a class="mw-redirect" href="/wiki/Vaca" title="Vaca">vaca</a>, a partir del <a href="/wiki/Bos_primigenius" title="Bos primigenius">uro</a>, después la <a class="mw-redirect" href="/wiki/Cabra" title="Cabra">cabra</a>, aproximadamente en las mismas fechas, y finalmente la <a href="/wiki/Ovis_orientalis_aries" title="Ovis orientalis aries">oveja</a>, entre 9000 y 8000 a. C. Existen hipótesis, como la del <a class="new" href="/w/index.php?title=Hip%C3%B3tesis_del_genotipo_ahorrador&amp;action=edit&amp;redlink=1" title="Hipótesis del genotipo ahorrador (aún no redactado)">genotipo ahorrador</a>, que afirman que este supuso un cambio fundamental en los hábitos alimentarios de las poblaciones cazadoras-recolectoras, que pasaron de alimentarse con ingestas abundantes, pero esporádicas a recibir aportes diarios de <a class="mw-redirect" href="/wiki/Hidratos_de_carbono" title="Hidratos de carbono">carbohidratos</a>. Según esta teoría, este cambio hizo que las poblaciones euro-asiáticas se volviesen más resistentes a la <a class="mw-redirect" href="/wiki/Diabetes" title="Diabetes">diabetes tipo 2</a> y más tolerantes a la <a href="/wiki/Lactosa" title="Lactosa">lactosa</a> en comparación con otras poblaciones humanas que solo más recientemente conocieron los productos derivados de la ganadería. Sin embargo, esta hipótesis no ha podido ser verificada e incluso su propio autor, <a class="new" href="/w/index.php?title=James_V._Neel&amp;action=edit&amp;redlink=1" title="James V. Neel (aún no redactado)">James V. Neel</a> la ha refutado, alegando que las diferencias observadas en poblaciones humanas podrían deberse a otros factores ambientales.<sup class="reference separada" id="cite_ref-21"><a href="#cite_note-21"><span class="corchete-llamada">[</span>21<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-22"><a href="#cite_note-22"><span class="corchete-llamada">[</span>22<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Respecto a la capacidad de los adultos para tolerar los productos lácteos sin fermentar, en especial la leche, se han esgrimido varias hipótesis. Una de ellas es que el gen responsable de la <i>lactasa </i>(enzima que hidroliza la <a href="/wiki/Lactosa" title="Lactosa">lactosa</a>), un gen raro y poco frecuente en las poblaciones europeas del Neolítico, posiblemente se ha conservado como consecuencia de incluir los productos lácteos en la alimentación humana.<sup class="reference separada" id="cite_ref-23"><a href="#cite_note-23"><span class="corchete-llamada">[</span>23<span class="corchete-llamada">]</span></a></sup>​ Habría aparecido hace siete mil quinientos años en una zona centrada alrededor de la actual <a href="/wiki/Hungr%C3%ADa" title="Hungría">Hungría</a>, y aunque este gen compensaría la deficiente síntesis de <a href="/wiki/Vitamina_D" title="Vitamina D">vitamina D</a> en <a href="/wiki/Latitud" title="Latitud">latitudes</a> septentrionales, este no parece un factor imprescindible para su aparición.<sup class="reference separada" id="cite_ref-24"><a href="#cite_note-24"><span class="corchete-llamada">[</span>24<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Durante la <a href="/wiki/Edad_Antigua" title="Edad Antigua">Edad Antigua</a> y la <a href="/wiki/Edad_Media" title="Edad Media">Edad Media</a>, la leche era muy difícil de conservar y, por esta razón, se consumía fresca o en forma de <a href="/wiki/Queso" title="Queso">quesos</a>. Con el tiempo se fueron añadiendo otros productos lácteos como la <a href="/wiki/Mantequilla" title="Mantequilla">mantequilla</a>. La <a class="mw-redirect" href="/wiki/Revoluci%C3%B3n_industrial" title="Revolución industrial">revolución industrial</a> en <a href="/wiki/Europa" title="Europa">Europa</a>, alrededor de <a href="/wiki/1830" title="1830">1830</a>, trajo la posibilidad de transportar la leche fresca desde las zonas rurales a las grandes ciudades gracias a las mejoras en los <a href="/wiki/Transporte" title="Transporte">transportes</a>. Asimismo, han ido apareciendo nuevos instrumentos en la industria de procesado de la leche. Uno de los más conocidos es el de la <a href="/wiki/Pasteurizaci%C3%B3n" title="Pasteurización">pasteurización</a>, sugerida para la leche por primera vez en <a href="/wiki/1886" title="1886">1886</a> por el químico agrícola alemán <a href="/wiki/Franz_von_Soxhlet" title="Franz von Soxhlet">Franz von Soxhlet</a>. Estas innovaciones han conseguido que la leche tenga un aspecto más saludable, unos tiempos de conservación más predecibles y un procesado más higiénico.
</p>
<p>La producción de leche para nutrir a las crías pudo ser un rasgo evolutivo asociado a la <a href="/wiki/Hormona" title="Hormona">hormona</a> <a href="/wiki/Prolactina" title="Prolactina">prolactina</a>. Se sabe que algunas especies de peces del género <i><a href="/wiki/Symphysodon" title="Symphysodon">Symphysodon</a></i> nutren a sus crías con un fluido semejante a la leche.
</p>
<p>La llamada «<a href="/wiki/Leche_de_buche" title="Leche de buche">leche de buche</a>» está presente en diversos grupos de <a href="/wiki/Aves" title="Aves">aves</a>, como las <a href="/wiki/Columbidae" title="Columbidae">palomas</a>, los <a href="/wiki/Phoenicopterus" title="Phoenicopterus">flamencos</a> o los <a href="/wiki/Spheniscidae" title="Spheniscidae">pingüinos</a>. Desde el punto de vista biológico se trata de una verdadera leche, secretada por glándulas especializadas.<sup class="reference separada" id="cite_ref-25"><a href="#cite_note-25"><span class="corchete-llamada">[</span>25<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Sin embargo, es en los <a href="/wiki/Mammalia" title="Mammalia">mamíferos</a> donde esta <a href="/wiki/Adaptaci%C3%B3n_biol%C3%B3gica" title="Adaptación biológica">adaptación evolutiva</a> se hace característica. Se cree que estos proceden de un grupo cercano a los <a class="mw-redirect" href="/wiki/Trithelodontidae" title="Trithelodontidae">tritelodóntidos</a> de finales del periodo <a href="/wiki/Tri%C3%A1sico" title="Triásico">triásico</a>. Estas mismas fuentes creen que ya mostraban signos de <a href="/wiki/Lactancia" title="Lactancia">lactancia</a>.
</p>
<p>Entre las muchas teorías existentes, se ha propuesto que la producción de leche surgió porque los antepasados <a href="/wiki/Synapsida" title="Synapsida">sinápsidos</a> de los mamíferos tenían huevos con cáscara blanda, como los actuales monotremas, lo cual provocaba su rápida desecación. La leche sería de ese modo una modificación de la secreción de las <a href="/wiki/Gl%C3%A1ndula_sudor%C3%ADpara" title="Glándula sudorípara">glándulas sudoríparas</a> destinada a transferir agua a los huevos.<sup class="reference separada" id="cite_ref-Oftedal_26-0"><a href="#cite_note-Oftedal-26"><span class="corchete-llamada">[</span>26<span class="corchete-llamada">]</span></a></sup>​ Otros autores, en una teoría que puede ser complementaria de la anterior, opinan que las glándulas mamarias proceden del <a href="/wiki/Sistema_inmunitario" title="Sistema inmunitario">sistema inmunitario</a> innato y que la lactación sería, en parte, una respuesta inflamatoria al daño <a href="/wiki/Tejido_(biolog%C3%ADa)" title="Tejido (biología)">tisular</a> y la infección.<sup class="reference separada" id="cite_ref-inmunitario_27-0"><a href="#cite_note-inmunitario-27"><span class="corchete-llamada">[</span>27<span class="corchete-llamada">]</span></a></sup>​ Aunque existen dificultades, varios enfoques aproximan la fecha de aparición en la historia evolutiva:
</p>
<p>El fósil más antiguo de los mamíferos <a href="/wiki/Eutheria" title="Eutheria">placentarios</a> descubierto hasta la fecha es <i><a href="/wiki/Eomaia" title="Eomaia">Eomaia scansoria</a></i>, un pequeño animal que exteriormente se asemejaba a los <a class="mw-redirect" href="/wiki/Roedor" title="Roedor">roedores</a> actuales y vivió hace ciento veinticinco millones de años durante el periodo <a href="/wiki/Cret%C3%A1cico" title="Cretácico">Cretácico</a>. Es casi seguro que este animal produjo leche como los mamíferos placentarios actuales.<sup class="reference separada" id="cite_ref-eomaia_30-0"><a href="#cite_note-eomaia-30"><span class="corchete-llamada">[</span>30<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La <a href="/wiki/Gen%C3%A9tica" title="Genética">genética</a> de la leche trata, por una parte, de describir los <a href="/wiki/Gen" title="Gen">genes</a> implicados en su biosíntesis, así como su <a class="mw-redirect" href="/wiki/Regulaci%C3%B3n_gen%C3%A9tica" title="Regulación genética">regulación</a> y, por otra, de la selección de razas o individuos o su <a class="mw-redirect" href="/wiki/Organismo_modificado_gen%C3%A9ticamente" title="Organismo modificado genéticamente">modificación genética</a> para aumentar la producción, su calidad o utilidades. De esto último también se ocupa la <a href="/wiki/Zootecnia" title="Zootecnia">zootecnia</a>.
</p>
<p>La producción de leche está regulada por hormonas lactogénicas (insulina, prolactina y glucocorticoides), citoquinas y factores de crecimiento y por sustrato. Estas activan factores de transcripción, tales como Stat5 (activado por prolactina). Se han identificado varias secuencias diana de estos factores, como el anterior y también para BLGe-1, OCT-1, C/EBP, Gr, Ets-1, YY1, Factor 5, Ying Yang 1 y la proteína de unión al promotor CCAAT.<sup class="reference separada" id="cite_ref-Promotores_31-0"><a href="#cite_note-Promotores-31"><span class="corchete-llamada">[</span>31<span class="corchete-llamada">]</span></a></sup>​ Estos elementos se suelen situar a una distancia variable, según especies (en las caseínas sensibles al calcio humanas es una de las más distantes al origen de la transcripción, a –4700/ –4550 nucleótidos) y se reúnen en grupos (clusters) que contienen tanto elementos negativos como positivos, regulándose por combinaciones de factores, de ahí la gran variabilidad en la regulación de cada proteína. Por ejemplo, las caseínas parecen regularse independientemente unas de otras. (Fox y McSweeney, 2003)
Los transcritos (mRNA) de las proteínas de la leche llegan a constituir el 60-80 % de todo el ARN presente en una célula epitelial durante la lactancia.
</p>
<p>Las redes de regulación génica en la producción de leche no se comprenden bien todavía. De un estudio realizado mediante microarrays, localización celular, interacciones interproteicas y minería de datos génicos en la literatura se han podido extraer algunas conclusiones generales:<sup class="reference separada" id="cite_ref-32"><a href="#cite_note-32"><span class="corchete-llamada">[</span>32<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las células epiteliales secretoras de leche separan activamente los materiales procedentes de los <a href="/wiki/Vaso_sangu%C3%ADneo" title="Vaso sanguíneo">vasos sanguíneos</a> circundantes, en lo que se ha llamado “barrera mamaria” (en analogía a la <a href="/wiki/Barrera_hematoencef%C3%A1lica" title="Barrera hematoencefálica">barrera hematoencefálica</a>). Una vez franqueada la barrera, las células obtienen los precursores que necesitan para la fabricación de leche a través de su <a class="mw-redirect" href="/wiki/Membrana_celular" title="Membrana celular">membrana</a> basal y basolateral, que serían: iones, <a href="/wiki/Glucosa" title="Glucosa">glucosa</a>, <a href="/wiki/%C3%81cido_graso" title="Ácido graso">ácidos grasos</a> y aminoácidos. En rumiantes también se utiliza el <a href="/wiki/%C3%81cido_ac%C3%A9tico" title="Ácido acético">acetato</a> y el β-hidroxibutirato como precursores. Algunas proteínas, en especial las <a class="mw-redirect" href="/wiki/Inmunoglobulina" title="Inmunoglobulina">inmunoglobulinas</a> también pueden traspasar esta barrera.<sup class="reference separada" id="cite_ref-33"><a href="#cite_note-33"><span class="corchete-llamada">[</span>33<span class="corchete-llamada">]</span></a></sup>​ La leche es expulsada por la membrana apical. Los lípidos de la leche se sintetizan en el <a class="mw-redirect" href="/wiki/Ret%C3%ADculo_endopl%C3%A1smico" title="Retículo endoplásmico">retículo endoplásmico</a> liso, en tanto que la <a href="/wiki/Case%C3%ADna" title="Caseína">caseína</a> debe madurar en el <a href="/wiki/Aparato_de_Golgi" title="Aparato de Golgi">aparato de Golgi</a>, donde también tiene lugar la biosíntesis de la <a href="/wiki/Lactosa" title="Lactosa">lactosa</a>.
</p>
<p>Desde el punto de vista <a href="/wiki/Histolog%C3%ADa" title="Histología">histológico</a>, la leche se produce en las <a href="/wiki/Gl%C3%A1ndula_mamaria" title="Glándula mamaria">glándulas mamarias</a>, que son una evolución por <a href="/wiki/Hipertrofia" title="Hipertrofia">hipertrofia</a> de las <a href="/wiki/Gl%C3%A1ndula_sudor%C3%ADpara" title="Glándula sudorípara">glándulas sudoríparas apocrinas</a> asociadas al <a href="/wiki/Pelo" title="Pelo">pelo</a>, lo cual aún se evidencia en los <a href="/wiki/Ornithorhynchus_anatinus" title="Ornithorhynchus anatinus">ornitorrincos</a>.<sup class="reference separada" id="cite_ref-ornitorrinco_34-0"><a href="#cite_note-ornitorrinco-34"><span class="corchete-llamada">[</span>34<span class="corchete-llamada">]</span></a></sup>​ La glándula mamaria activa está compuesta por lóbulos, cada uno de los cuales posee numerosos lobulillos y estos a su vez pequeños alvéolos con <a href="/wiki/Epitelio" title="Epitelio">células epiteliales cilíndricas</a> altas o bajas, dependiendo del ciclo de actividad, que son las encargadas de producir la leche. Entre estas y la <a href="/wiki/L%C3%A1mina_basal" title="Lámina basal">lámina basal</a> del alvéolo se encuentran algunas células mioepiteliales estrelladas. El epitelio de los conductos entre los lobulillos es un ejemplo destacado de epitelio biestratificado cúbico (Bloom-Fawcet, 1999).
</p>

</p>
<p>Actualmente, la leche que más se utiliza en la producción de derivados <a href="/wiki/L%C3%A1cteo" title="Lácteo">lácteos</a> es la de vaca (debido a las propiedades que posee, a la cantidad que se obtiene, agradable sabor, fácil digestión, así como la gran cantidad de derivados obtenidos). Sin embargo, no es la única que se explota. También están la leche de <a href="/wiki/Capra_aegagrus_hircus" title="Capra aegagrus hircus">cabra</a>, <a href="/wiki/Equus_africanus_asinus" title="Equus africanus asinus">burra</a>, <a href="/wiki/Equus_ferus_caballus" title="Equus ferus caballus">yegua</a>, <a href="/wiki/Camelus" title="Camelus">camella</a>, entre otras. El consumo de determinados tipos de leche depende de la región y el tipo de animales disponibles. La <a href="/wiki/Leche_de_cabra" title="Leche de cabra">leche de cabra</a> es ideal para elaborar <a href="/wiki/Dulce_de_leche" title="Dulce de leche">dulce de leche</a> (también llamado <i>cajeta</i>) y en las regiones árticas se emplea la leche de ballena. La leche de burra y de yegua son las que contienen menos materia grasa, mientras que la de foca contiene más de un 50 % de aquella.
</p>
<p>La leche de origen humano no se produce ni se distribuye a escala industrial. Sin embargo, puede obtenerse mediante donaciones. Existen <a href="/wiki/Banco_de_leche" title="Banco de leche">bancos de leche</a> que se encargan de recogerla para proporcionársela a niños <a class="mw-redirect" href="/wiki/Nacimiento_prematuro" title="Nacimiento prematuro">prematuros</a> o <a href="/wiki/Alergia_a_alimentos" title="Alergia a alimentos">alérgicos</a> que no pueden recibirla de otro modo. A nivel mundial, hay varias especies de animales de las que se puede obtener leche: la <a href="/wiki/Ovis_orientalis_aries" title="Ovis orientalis aries">oveja</a>, la <a href="/wiki/Capra_aegagrus_hircus" title="Capra aegagrus hircus">cabra</a>, la <a href="/wiki/Equus_ferus_caballus" title="Equus ferus caballus">yegua</a>, la <a class="mw-redirect" href="/wiki/Equus_asinus" title="Equus asinus">burra</a>, la <a href="/wiki/Camelus" title="Camelus">camella</a> (y otras camélidas, como la <a href="/wiki/Lama_glama" title="Lama glama">llama</a> o la <a href="/wiki/Vicugna_pacos" title="Vicugna pacos">alpaca</a>), la <a class="mw-redirect" href="/wiki/Bos_grunniens" title="Bos grunniens">yak</a>, la <a href="/wiki/Bubalus_bubalis" title="Bubalus bubalis">búfala</a>, la hembra del <a href="/wiki/Rangifer_tarandus" title="Rangifer tarandus">reno</a> y la <a href="/wiki/Alces_alces" title="Alces alces">alce</a>.
</p>

<p>No todas las leches de los mamíferos poseen las mismas propiedades. Por regla general puede decirse que la leche es un líquido de color blanco mate y ligeramente <a href="/wiki/Viscosidad" title="Viscosidad">viscoso</a>, cuya composición y características físico-químicas varían sensiblemente según las especies animales, e incluso según las diferentes razas. Estas características también varían en el curso del período de lactación, así como en el curso de su tratamiento.
</p>
<p>La leche aporta a los seres humanos <a href="/wiki/Calcio" title="Calcio">calcio</a>, <a href="/wiki/Vitamina_A" title="Vitamina A">vitaminas A</a> y <a href="/wiki/Vitamina_D" title="Vitamina D">D</a>, <a href="/wiki/%C3%81cido_graso" title="Ácido graso">ácidos grasos</a> y <a href="/wiki/Prote%C3%ADna" title="Proteína">proteínas</a>.<sup class="reference separada" id="cite_ref-51"><a href="#cite_note-51"><span class="corchete-llamada">[</span>51<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La leche de vaca tiene una <a href="/wiki/Densidad" title="Densidad">densidad</a> media de 1,032 g/ml. Es una <a href="/wiki/Mezcla" title="Mezcla">mezcla</a> compleja y <a class="mw-redirect" href="/wiki/Heterog%C3%A9nea" title="Heterogénea">heterogénea</a> compuesta por un sistema <a href="/wiki/Coloide" title="Coloide">coloidal</a> de tres fases:
</p>
<p>Contiene una proporción importante de <a href="/wiki/Agua" title="Agua">agua</a> (cerca del 87 %). El resto constituye el <a class="mw-redirect" href="/wiki/Extracto_seco" title="Extracto seco">extracto seco</a> que representa 130 <a href="/wiki/Gramo" title="Gramo">gramos</a> (g) por <a href="/wiki/Litro" title="Litro">l</a> y en el que hay de 35 a 45 g de materia grasa.
</p>
<p>Otros componentes principales son los <a class="mw-redirect" href="/wiki/Gl%C3%BAcidos" title="Glúcidos">glúcidos</a> <a href="/wiki/Lactosa" title="Lactosa">lactosa</a>, las <a class="mw-redirect" href="/wiki/Prote%C3%ADnas" title="Proteínas">proteínas</a> y los <a href="/wiki/L%C3%ADpidos" title="Lípidos">lípidos</a>. Los componentes orgánicos (glúcidos, lípidos, proteínas, <a class="mw-redirect" href="/wiki/Vitaminas" title="Vitaminas">vitaminas</a>), y los componentes minerales (<a href="/wiki/Calcio" title="Calcio">Ca</a>, <a href="/wiki/Sodio" title="Sodio">Na</a>, <a href="/wiki/Potasio" title="Potasio">K</a>, <a href="/wiki/Magnesio" title="Magnesio">Mg</a>, <a href="/wiki/Cloro" title="Cloro">Cl</a>). La leche contiene diferentes grupos de nutrientes. Las sustancias orgánicas (glúcidos, lípidos, proteínas) están presentes en cantidades más o menos iguales y constituyen la principal fuente de energía. Estos nutrientes se reparten en elementos constructores, las proteínas, y en compuestos energéticos, los glúcidos y los lípidos.
</p>
<p>El <a href="/wiki/PH" title="PH">pH</a> de la leche es ligeramente ácido (pH comprendido entre 6,6 y 6,8). Otra propiedad química importante es la acidez, o cantidad de <a href="/wiki/%C3%81cido_l%C3%A1ctico" title="Ácido láctico">ácido láctico</a> que contiene, que suele estar en torno al 0,15-0,16 %.
</p>
<p>Las sustancias proteicas de la leche son las más importantes en el aspecto químico. Se clasifican en dos grupos: proteínas (la caseína se presenta en 80 % del total proteínica, mientras que las proteínas del <a href="/wiki/Suero_de_leche" title="Suero de leche">suero</a> lo hacen en un 20 %), y las <a class="mw-redirect" href="/wiki/Enzimas" title="Enzimas">enzimas</a>.<sup class="reference separada" id="cite_ref-52"><a href="#cite_note-52"><span class="corchete-llamada">[</span>52<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La actividad enzimática depende de dos factores: la <a href="/wiki/Temperatura" title="Temperatura">temperatura</a> y el pH; y está presente en todo el sistema de diversas formas. La <a href="/wiki/Fosfatasa" title="Fosfatasa">fosfatasa</a> es un inhibidor a temperaturas de <a href="/wiki/Pasteurizaci%C3%B3n" title="Pasteurización">pasteurización</a> e indica que se realizó bien la pasteurización. La <a href="/wiki/Reductasa" title="Reductasa">reductasa</a> es producida por microorganismos ajenos a la leche y su presencia indica que está contaminada. La <a class="new" href="/w/index.php?title=Xantoxidasa&amp;action=edit&amp;redlink=1" title="Xantoxidasa (aún no redactado)">xantoxidasa</a> en combinación con <a href="/wiki/Nitrato_de_potasio" title="Nitrato de potasio">nitrato de potasio</a> (KNO<sub>3</sub>) inhibe el crecimiento de bacterias butíricas. La <a href="/wiki/Lipasa" title="Lipasa">lipasa</a> oxida las grasas y da olor rancio a los productos y se inhibe con pasteurización. La <a href="/wiki/Catalasa" title="Catalasa">catalasa</a> se incrementa con la <a href="/wiki/Mastitis" title="Mastitis">mastitis</a> y, si bien no deteriora el alimento, se usa como <a href="/wiki/Indicador_microbiol%C3%B3gico" title="Indicador microbiológico">indicador microbiológico</a>.
</p>
<p>Inmediatamente después del <a href="/wiki/Parto" title="Parto">parto</a>, la hembra del <a class="mw-redirect" href="/wiki/Mam%C3%ADfero" title="Mamífero">mamífero</a> comienza a producir secreciones mamarias; durante los dos o tres primeros días produce el <a href="/wiki/Calostro" title="Calostro">calostro</a>. Pasado este período, el animal sintetiza propiamente la leche durante todo el periodo de lactancia, que varía de 180 a 300 días (dependiendo de muchos factores), con una producción media diaria muy fluctuante que va desde 3 hasta 25 litros. La leche se sintetiza fundamentalmente en la glándula mamaria, pero una gran parte de sus constituyentes provienen del suero de la <a href="/wiki/Sangre" title="Sangre">sangre</a>.<sup class="reference separada" id="cite_ref-53"><a href="#cite_note-53"><span class="corchete-llamada">[</span>53<span class="corchete-llamada">]</span></a></sup>​ Su composición química es muy compleja y completa, lo que refleja su gran importancia en la alimentación de las crías. La composición de la leche depende de las necesidades de la especie durante el periodo de crianza.<sup class="reference separada" id="cite_ref-gresti_54-0"><a href="#cite_note-gresti-54"><span class="corchete-llamada">[</span>54<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La lactosa es un disacárido presente únicamente en leches y sus derivados, representando el principal y único glúcido.<sup class="reference separada" id="cite_ref-55"><a href="#cite_note-55"><span class="corchete-llamada">[</span>55<span class="corchete-llamada">]</span></a></sup>​ Sin embargo, se han identificado pequeñas cantidades de <a href="/wiki/Glucosa" title="Glucosa">glucosa</a>, <a href="/wiki/Galactosa" title="Galactosa">galactosa</a>, <a href="/wiki/Sacarosa" title="Sacarosa">sacarosa</a>, <a href="/wiki/Cerebr%C3%B3sido" title="Cerebrósido">cerebrósidos</a> y <a class="new" href="/w/index.php?title=Aminoaz%C3%BAcar&amp;action=edit&amp;redlink=1" title="Aminoazúcar (aún no redactado)">aminoazúcares</a> derivados de la <a href="/wiki/Hexosamina" title="Hexosamina">hexosamina</a>.
</p>
<p>La lactosa se sintetiza en la <a href="/wiki/Gl%C3%A1ndula_mamaria" title="Glándula mamaria">glándula mamaria</a> por un sistema enzimático en el que interviene la α-lactoalbúmina para después segregarse en la leche. Es un 15 % menos <a href="/wiki/Sabor_dulce" title="Sabor dulce">edulcorante</a> que la <a href="/wiki/Sacarosa" title="Sacarosa">sacarosa</a> y contribuye, junto con las <a href="/wiki/Sabor_salado" title="Sabor salado">sales</a>, al sabor global del alimento. La <a href="/wiki/Enzima" title="Enzima">enzima</a> <a href="/wiki/Lactasa" title="Lactasa">lactasa</a> <a href="/wiki/Hidr%C3%B3lisis" title="Hidrólisis">hidroliza</a> el <a href="/wiki/Enlace_glucos%C3%ADdico" title="Enlace glucosídico">enlace glucosídico</a> y separa el azúcar en <a href="/wiki/Glucosa" title="Glucosa">glucosa</a> y <a href="/wiki/Galactosa" title="Galactosa">galactosa</a>, pero su nivel varía entre las diferentes poblaciones humanas. En los seres humanos, varias mutaciones genéticas han permitido seguir tolerando la lactosa durante la edad adulta (persistencia de lactosa), con mayor o menor prevalencia según la zona geográfica.<sup class="reference separada" id="cite_ref-DengMisselwitz_56-0"><a href="#cite_note-DengMisselwitz-56"><span class="corchete-llamada">[</span>56<span class="corchete-llamada">]</span></a></sup>​ Las poblaciones que no poseen esta mutación (que son principalmente las <a href="/wiki/Asia" title="Asia">asiáticas</a> y <a href="/wiki/%C3%81frica" title="África">africanas</a>) presentan una <a href="/wiki/Intolerancia_a_la_lactosa#Deficiencia_primaria_de_lactasa" title="Intolerancia a la lactosa">deficiencia primaria o permanente de lactasa</a>.<sup class="reference separada" id="cite_ref-DengMisselwitz_56-1"><a href="#cite_note-DengMisselwitz-56"><span class="corchete-llamada">[</span>56<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las personas sanas con deficiencia primaria o permanente de lactasa son capaces de consumir al menos 12 g de lactosa por comida (la cantidad contenida en una taza de leche) sin experimentar ningún síntoma o solo síntomas leves.<sup class="reference separada" id="cite_ref-NIHconsensus2010_10-3"><a href="#cite_note-NIHconsensus2010-10"><span class="corchete-llamada">[</span>10<span comidas, eligiendo leche baja en lactosa, sustituyendo la leche por yogur o quesos curados, o tomando suplementos de lactasa.<sup class="reference separada" id="cite_ref-Szilagyi2015_11-3"><a href="#cite_note-Szilagyi2015-11"><span class="corchete-llamada">[</span>11<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-SilanikoveLeitner2015_12-2"><a de alimentos lácteos por parte de personas con deficiencia primaria de lactasa puede permitir una adaptación favorable de las bacterias del <a href="/wiki/Colon" title="Colon">colon</a>, que pueden ayudar a la descomposición de la lactosa, permitiendo una tolerancia progresiva y mantenida a la lactosa.<sup class="reference separada" id="cite_ref-Szilagyi2015_11-4"><a </p>

class="corchete-llamada">[</span>58<span class="corchete-llamada">]</span></a></sup>​ No obstante, el consumo de productos lácteos por parte de personas con intolerancia a la lactosa no produce daños en el <a href="/wiki/Tracto_gastrointestinal" title="Tracto gastrointestinal">tracto gastrointestinal</a>, sino que se limita a estos síntomas transitorios.<sup class="reference separada" id="cite_ref-Heyman_14-1"><a href="#cite_note-Heyman-14"><span class="corchete-llamada">[</span>14<span class="corchete-llamada">]</span></a></sup>​ Una gran parte de las persona que creen tener intolerancia a la lactosa no presentan en realidad malabsorción de lactosa, sino que sus síntomas se deben a la presencia de enfermedades no diagnosticadas, tales como la <a href="/wiki/Celiaqu%C3%ADa" title="Celiaquía">enfermedad celíaca</a>, la <a href="/wiki/Enfermedad_inflamatoria_intestinal" title="Enfermedad inflamatoria intestinal">enfermedad inflamatoria intestinal</a> o el <a class="mw-redirect" href="/wiki/Sobrecrecimiento_bacteriano" title="Sobrecrecimiento bacteriano">sobrecrecimiento bacteriano</a>.<sup class="reference separada" id="cite_ref-NIHconsensus2010_10-4"><a href="#cite_note-NIHconsensus2010-10"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​ Asimismo, la intolerancia a la lactosa con frecuencia es confunida con una <a href="/wiki/Alergia_a_alimentos" title="Alergia a alimentos">alergia</a> a la leche, especialmente difícil de diagnosticar cuando es no mediada por IgE.<sup class="reference separada" id="cite_ref-CrittendenBennett2005_17-1"><a href="#cite_note-CrittendenBennett2005-17"><span class="corchete-llamada">[</span>17<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las propiedades de la leche son el reflejo de los ácidos grasos que contiene. Así tenemos varios grupos de <a href="/wiki/L%C3%ADpido" title="Lípido">lípidos</a> presentes en la leche: <a class="mw-redirect" href="/wiki/Triacilglic%C3%A9rido" title="Triacilglicérido">triacilglicéridos</a>, <a class="mw-redirect" href="/wiki/Diacilglic%C3%A9rido" title="Diacilglicérido">diacilglicéridos</a>, <a class="new" href="/w/index.php?title=Monoacilglic%C3%A9rido&amp;action=edit&amp;redlink=1" title="Monoacilglicérido (aún no redactado)">monoacilglicéridos</a>, <a href="/wiki/Fosfol%C3%ADpido" title="Fosfolípido">fosfolípidos</a>, ácidos grasos libres, <a href="/wiki/Esterol" title="Esterol">esteroles</a> y sus <a href="/wiki/%C3%89ster" title="Éster">ésteres</a>, y algunos <a class="mw-redirect" href="/wiki/Gl%C3%BAcidos" title="Glúcidos">glúcidos</a>.
</p>
<p>Los <a class="mw-redirect" href="/wiki/Triacilglic%C3%A9rido" title="Triacilglicérido">triacilglicéridos</a> se encuentran como pequeñas partículas llamadas glóbulos. Contienen una gran cantidad de ácidos grasos, identificándose hasta 400 tipos diferentes en la leche de vaca (los <a href="/wiki/Aceite_vegetal" title="Aceite vegetal">aceites</a> tiene entre 8 y 10). La leche es el alimento que tiene la composición lipídica más compleja.<sup class="reference separada" id="cite_ref-gresti_54-1"><a href="#cite_note-gresti-54"><span class="corchete-llamada">[</span>54<span class="corchete-llamada">]</span></a></sup>​ Sin embargo, el 96 % del total lo conforman solo 14 <a href="/wiki/%C3%81cido_graso" title="Ácido graso">ácidos grasos</a>, siendo los más importantes el <a href="/wiki/%C3%81cido_mir%C3%ADstico" title="Ácido mirístico">ácido mirístico</a>, el <a href="/wiki/%C3%81cido_palm%C3%ADtico" title="Ácido palmítico">ácido palmítico</a> y el <a href="/wiki/%C3%81cido_oleico" title="Ácido oleico">ácido oleico</a>. La gran cantidad de grasas se debe a la alimentación del bovino y a la intensa actividad del <a href="/wiki/Rumen" title="Rumen">rumen</a>.<sup class="reference separada" id="cite_ref-60"><a href="#cite_note-60"><span class="corchete-llamada">[</span>60<span class="corchete-llamada">]</span></a></sup>​ En el caso de las focas, el exceso de contenido graso se debe a la dieta basada en <a href="/wiki/Pescado" title="Pescado">peces</a> y es parte de una adaptación natural para que la cría soporte el frío extremo. En el caso de la leche humana, el contenido graso depende de la nutrición equilibrada de la mujer durante el <a class="mw-redirect" href="/wiki/Embarazo" title="Embarazo">embarazo</a> y la <a href="/wiki/Lactancia" title="Lactancia">lactancia</a>; de ahí que una dieta plenamente <a href="/wiki/Omn%C3%ADvoro" title="Omnívoro">omnívora</a> beneficie al contenido graso exacto de la leche.<sup class="reference separada" id="cite_ref-61"><a href="#cite_note-61"><span class="corchete-llamada">[</span>61<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>De todas las proteínas presentes en la leche, las más comunes y representativas son tres, y todas son <a href="/wiki/Case%C3%ADna" title="Caseína">caseínas</a>:<sup class="reference separada" id="cite_ref-62"><a href="#cite_note-62"><span class="corchete-llamada">[</span>62<span class="corchete-llamada">]</span></a></sup>​ la caseína-α<sub>s1</sub>, la caseína-β y la caseína-κ. En la industria láctea, es muy importante la caseína-κ, que posee, entre otras, las siguientes características:<sup class="reference separada" id="cite_ref-63"><a href="#cite_note-63"><span class="corchete-llamada">[</span>63<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La caseína-κ es útil principalmente para la elaboración de quesos<sup class="reference separada" id="cite_ref-64"><a href="#cite_note-64"><span class="corchete-llamada">[</span>64<span class="corchete-llamada">]</span></a></sup>​ (la más rica en este tipo de caseína es la leche de vaca, mientras que la más pobre proviene de la leche humana) debido a que al ser hidrolizada por la <a href="/wiki/Renina" title="Renina">renina</a> es posible que se precipite en paracaseína-κ, la cual al reaccionar con el <a href="/wiki/Calcio" title="Calcio">calcio</a> genera paracaseinato de calcio.
</p>
<p>Las caseínas interaccionan entre sí formando una dispersión coloidal que consiste en partículas esféricas llamadas <a href="/wiki/Micela" title="Micela">micelas</a> con un diámetro que suele variar entre 60 a 450 <a href="/wiki/Nan%C3%B3metro" title="Nanómetro">nm</a> poseyendo un promedio de 130 nm. A pesar de la abundante literatura científica sobre la posible estructura de una micela, no hay consenso sobre el tema. Existe un modelo propuesto que considera que la micela se encuentra a su vez constituida por subunidades de la misma forma, con un diámetro de entre 10 y 20 nm.
</p>
<p>El modelo arriba ilustrado permite observar cómo las subunidades se enlazan entre sí gracias a los iones de calcio. Se sugiere que el <a href="/wiki/Fosfato_de_calcio" title="Fosfato de calcio">fosfato de calcio</a> se une a los grupos NH<sub>2</sub><sup>-</sup> de la lisina; el calcio interacciona con el <a class="mw-redirect" href="/wiki/Grupo_carboxilo" title="Grupo carboxilo">grupo carboxilo</a> ionizado (COO<sup>-</sup>). Las submicelas se constituyen a partir de la interacción constante entre las caseínas α, β y κ. Hay que resaltar la función estabilizadora de la caseína κ contra la precipitación de calcio de otras fracciones proteínicas. La gran cantidad de modelos fisicoquímicos (por citar algunos: Rose, Garnier y Ribadeau, Morr, Schmidt, Slattery, Waugh, Noble, etc.), todos concuerdan en que las unidades hidrófobas entre las moléculas de proteínas aseguran la estabilidad de la micela.<sup class="reference separada" id="cite_ref-65"><a href="#cite_note-65"><span class="corchete-llamada">[</span>65<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-66"><a href="#cite_note-66"><span class="corchete-llamada">[</span>66<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>A partir de 10 litros de leche de vaca se puede producir de 1 a 2 kg de <a href="/wiki/Queso" title="Queso">queso</a> (es decir, en su mayor parte de caseína) y un promedio de 8 a 9 kg de <a href="/wiki/Suero_de_leche" title="Suero de leche">suero de leche</a>. El suero es el conjunto de todos los componentes de la leche que no se integran en la <a href="/wiki/Coagulaci%C3%B3n" title="Coagulación">coagulación</a> de la caseína, y de acuerdo con el tipo de leche (es decir, de la especie de la que proviene) se pueden tener dos tipos de sueros, clasificados por su sabor:
</p>
<p>Los lactatos y los fosfatos (sales muy comunes en el suero) ayudan a guardar el <a class="mw-redirect" href="/wiki/Equilibrio_%C3%A1cido-base" title="Equilibrio ácido-base">equilibrio ácido-base</a> e influyen mucho en las propiedades del suero (estabilidad y <a class="mw-redirect" href="/wiki/Precipitaci%C3%B3n_(qu%C3%ADmica)" title="Precipitación (química)">precipitación</a> térmica).<sup class="reference separada" id="cite_ref-67"><a href="#cite_note-67"><span class="corchete-llamada">[</span>67<span class="corchete-llamada">]</span></a></sup>​
El suero tiene una proporción baja de proteínas, sin embargo poseen más calidad nutritiva que las caseínas del queso. La excesiva producción de suero al elaborar queso ha sido siempre una preocupación y se han ideado muchas formas de aprovecharlo. Una de las más sencillas, de tipo casero, es calentarlo para precipitar las proteínas y luego prensarlo o <a href="/wiki/Filtraci%C3%B3n" title="Filtración">filtrarlo</a>. En muchas poblaciones de <a href="/wiki/M%C3%A9xico" title="México">México</a> suele comerse inmediatamente después de salarlo (y recibe el nombre de <a href="/wiki/Reques%C3%B3n" title="Requesón">requesón</a>).
Sus aplicaciones industriales suelen venir una vez que se le deshidrata, cuando es poco soluble. Durante la evaporación (para eliminar el agua) y la aspersión (para secarlo) puede perder sus propiedades nutricionales por lo que el <a href="/wiki/PH" title="PH">pH</a> y la temperatura de estos dos procesos deben vigilarse con esmero durante el secado del extracto.<sup class="reference separada" id="cite_ref-68"><a href="#cite_note-68"><span class="corchete-llamada">[</span>68<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-69"><a href="#cite_note-69"><span class="corchete-llamada">[</span>69<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las proteínas del suero son compactas, globulares, con un peso molecular que varía entre 14 000 y 1 000 000 <a class="mw-redirect" href="/wiki/Dalton_(unidad)" title="Dalton (unidad)">daltons</a>, y son <a href="/wiki/Solubilidad" title="Solubilidad">solubles</a> en un amplio intervalo de pH (se mantienen intactas cuando la leche se corta de manera natural, ya que no ha habido presencia de calor que desnaturalice las proteínas). En estado natural no se asocian con las caseínas, pero en la leches tratadas térmicamente y <a href="/wiki/Homogeneizaci%C3%B3n" title="Homogeneización">homogeneizadas</a>, una parte de estas proteínas sí lo hace.<sup class="reference separada" id="cite_ref-dargal_70-0"><a href="#cite_note-dargal-70"><span class="corchete-llamada">[</span>70<span class="corchete-llamada">]</span></a></sup>​ Las proteínas del suero constan por lo menos de 8 fracciones diferentes, todas sensibles a temperaturas altas (procesos térmicos) y por ello son las primeras en degradarse con procesos como la <a href="/wiki/Pasteurizaci%C3%B3n" title="Pasteurización">pasteurización</a> o la <a class="mw-redirect" href="/wiki/UHT" title="UHT">UHT</a>. La razón por la que la leche no se descompone estando fuera de refrigeración una vez tratada térmicamente es porque las proteínas del suero, al desnaturalizarse, liberan un grupo <a class="mw-redirect" href="/wiki/Sulfhidrilo" title="Sulfhidrilo">sulfhidrilo</a> que reduce la actividad de la oxidación de manera parcial.<sup class="reference separada" id="cite_ref-dargal_70-1"><a href="#cite_note-dargal-70"><span class="corchete-llamada">[</span>70<span class="corchete-llamada">]</span></a></sup>​

</p>
<p>La leche recién obtenida es un sustrato ideal para un gran número de géneros bacterianos, algunos beneficiosos y otros perjudiciales, que provocan alteraciones diversas del alimento y sus propiedades:<sup class="reference separada" id="cite_ref-77"><a href="#cite_note-77"><span class="corchete-llamada">[</span>77<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Aunque en el esterilizado se eliminan la mayor cantidad de este tipo de gérmenes, estos dejan una huella enzimática (<a class="mw-redirect" href="/wiki/Proteasa" title="Proteasa">proteasa</a>) que resiste las altas temperaturas provocando en las leches un amargor característico cumplido el 50 % del tiempo de su caducidad.
En la industria láctea, este tipo de bacterias (Familia <a href="/wiki/Pseudomonas" title="Pseudomonas">pseudomonas</a>) son responsables de conferir un <a href="/wiki/Sabor_amargo" title="Sabor amargo">sabor amargo</a> a cremas y leches blancas.
</p>
<p>Como control de calidad, la leche cruda o leche bronca (sin pasteurizar) se analiza antes de determinar el destino como producto terminado, si el recuento de gérmenes es mayor que 100 000 UFC (Unidades Formadoras de Colonias) es una leche de inferior calidad que una cuyo recuento sea menor a ese número. También se determina la potencialidad de <a href="/wiki/Brucelosis" title="Brucelosis">brucelosis</a> que pudiera presentar.
</p>
<p>Su diversificada composición, en la que entran <a class="mw-redirect" href="/wiki/Grasas" title="Grasas">grasas</a> (donde los <a class="mw-redirect" href="/wiki/Triglic%C3%A9ridos" title="Triglicéridos">triglicéridos</a> son la fracción mayoritaria con el 98 % del total lipídico y cuyos <a class="mw-redirect" href="/wiki/%C3%81cidos_grasos" title="Ácidos grasos">ácidos grasos</a> que los forman son mayormente saturados), <a class="mw-redirect" href="/wiki/Prote%C3%ADnas" title="Proteínas">proteínas</a>, (<a href="/wiki/Case%C3%ADna" title="Caseína">caseína</a>, <a href="/wiki/Alb%C3%BAmina" title="Albúmina">albúmina</a> y proteínas del suero) y <a class="mw-redirect" href="/wiki/Gl%C3%BAcidos" title="Glúcidos">glúcidos</a> (<a href="/wiki/Lactosa" title="Lactosa">lactosa</a>, <a href="/wiki/Az%C3%BAcar" title="Azúcar">azúcar</a> específica de la leche), la convierten en un alimento completo. Además, la leche entera de vaca es una importante fuente de <a href="/wiki/Vitamina" title="Vitamina">vitaminas</a> (vitaminas <a href="/wiki/Vitamina_A" title="Vitamina A">A</a>, <a class="mw-redirect" href="/wiki/Vitamina_B" title="Vitamina B">B</a>, <a href="/wiki/Vitamina_D" title="Vitamina D">D3</a>, <a href="/wiki/Vitamina_E" title="Vitamina E">E</a>). La vitamina D es la que fija el <a class="mw-redirect" href="/wiki/Fosfato_c%C3%A1lcico" title="Fosfato cálcico">fosfato de calcio</a> a <a href="/wiki/Diente" title="Diente">dientes</a> y <a href="/wiki/Hueso" title="Hueso">huesos</a>, por lo que es especialmente recomendable para niños.<sup class="reference separada" id="cite_ref-79"><a href="#cite_note-79"><span class="corchete-llamada">[</span>79<span class="corchete-llamada">]</span></a></sup>​ El calostro es un líquido de color amarillento, rico en <a class="mw-redirect" href="/wiki/Prote%C3%ADnas" title="Proteínas">proteínas</a> y <a class="mw-redirect" href="/wiki/Anticuerpos" title="Anticuerpos">anticuerpos</a>, indispensables para la <a href="/wiki/Inmunizaci%C3%B3n" title="Inmunización">inmunización</a> del recién nacido. A pesar de ello, no tiene aplicación industrial.
</p>
<p>La leche cruda o leche bronca no sería apta para su <a href="/wiki/Comercio" title="Comercio">comercialización</a> y consumo sin ser sometida a ciertos <a href="/wiki/Industria" title="Industria">procesos industriales</a> que aseguraran que la carga microbiológica está dentro de unos límites seguros.<sup class="reference separada" id="cite_ref-80"><a href="#cite_note-80"><span class="corchete-llamada">[</span>80<span class="corchete-llamada">]</span></a></sup>​ Por eso, una leche con garantías de salubridad debe haber sido ordeñada con métodos modernos e higiénicos de succión en los cuales no hay contacto físico con la leche. Después de su <a class="mw-redirect" href="/wiki/Orde%C3%B1ar" title="Ordeñar">ordeño</a>, ha de enfriarse y almacenarse en un <a href="/wiki/Tanque_de_leche" title="Tanque de leche">tanque de leche</a> en agitación y ser transportada en cisternas isotermas hasta las plantas de procesado.
</p>
<p>En dichas plantas, ha de <a href="/wiki/Qu%C3%ADmica_anal%C3%ADtica" title="Química analítica">analizarse</a> la leche antes de su descarga para ver que cumple con unas características óptimas para el consumo.
</p>
<p>Entre los análisis, están los <a href="/wiki/Fisicoqu%C3%ADmica" title="Fisicoquímica">fisicoquímicos</a> para ver su composición en <a href="/wiki/Grasa" title="Grasa">grasa</a> y <a class="mw-redirect" href="/wiki/Extracto_seco" title="Extracto seco">extracto seco</a>, entre otros parámetros, para detectar posibles fraudes por aguado, los <a class="mw-redirect" href="/wiki/Cualidad_organol%C3%A9ptica" title="Cualidad organoléptica">organolépticos</a>, para detectar sabores extraños y los bacteriológicos, que detectan la presencia de <a href="/wiki/Bacteria" title="Bacteria">bacterias patógenas</a> y de <a class="new" href="/w/index.php?title=Inhibidores_en_la_leche&amp;action=edit&amp;redlink=1" title="Inhibidores en la leche (aún no redactado)">antibióticos</a>. Estos pasan a la leche procedentes de la vaca en tratamiento <a class="mw-redirect" href="/wiki/Veterinario" title="Veterinario">veterinario</a> y a su vez pasan al consumidor. La leche que no cumple con los requisitos de calidad, debe ser rechazada.
</p>
<p>Una vez comprobado su estado óptimo, es almacenada en cisternas de gran capacidad y dispuesta para su envasado comercial.
</p>
<p>La leche, según la aplicación comercial que se le vaya a dar puede pasar por una gran cantidad de procesos, conocidos como procesos de depuración. Estos aseguran la calidad sanitaria de la leche, y se listan a continuación:<sup class="reference separada" id="cite_ref-81"><a href="#cite_note-81"><span class="corchete-llamada">[</span>81<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Una vez que ya se realizó la depuración, la leche puede ser tratada para el consumo humano mediante la aplicación de calor para la eliminación parcial o total de bacterias.
</p>
<p>De acuerdo con el objetivo requerido, se empleará la termización, la pasteurización, la ultrapasteurización o la esterilización.<sup class="reference separada" id="cite_ref-84"><a href="#cite_note-84"><span class="corchete-llamada">[</span>84<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-85"><a href="#cite_note-85"><span class="corchete-llamada">[</span>85<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La esterilización puede ocurrir en unas autoclaves en línea denominadas <i>Barriquands</i>. Las leches blancas tratadas de este modo se embalan en <a class="mw-redirect" href="/wiki/Tetrabrik" title="Tetrabrik">tetrabrik</a> o cajas de cartón especial higienizadas y recubiertas internamente con un filme satinado.
</p>
<p>Después de un tratamiento térmico la refrigeración puede ser prescindible debido a que no es necesario bajar la temperatura en todos los casos, solamente cuando la leche aún posee microorganismos.
</p>
<p>De acuerdo con la calidad microbiana saliente se considera la refrigeración; de ahí que la termización tenga refrigeración obligada y la esterilizada no. Si no existen bacterias o actividad enzimática la leche no se alterará a temperatura ambiente; si dejamos cualquier leche en un vaso y sin tapar entonces el oxígeno hará lo propio como agente oxidante, mas no debido a actividades internas de la leche.<sup class="reference separada" id="cite_ref-86"><a href="#cite_note-86"><span class="corchete-llamada">[</span>86<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La presentación de la leche en el mercado es variable, ya que se acepta por regla general la alteración de sus propiedades para satisfacer las preferencias de los consumidores. Una alteración muy frecuente es <a href="/wiki/Liofilizaci%C3%B3n" title="Liofilización">deshidratarla</a> (<i>liofilización</i>) como <a href="/wiki/Leche_en_polvo" title="Leche en polvo">leche en polvo</a> para facilitar su transporte y almacenaje tras su ordeñado. También es usual reducir el contenido de grasa, aumentar el de calcio y agregar sabores.
</p>
<p>Los requisitos que debe cumplir un producto para ubicarse en las diferentes categorías varían mucho de acuerdo a la definición de cada país:
</p>
<p>Se han popularizado las llamadas <a href="/wiki/Leche_vegetal" title="Leche vegetal">leches vegetales</a> como alternativas a la leche de origen animal elaborables a partir de diversas materias primas, tales como arroz, coco, cebada, almendra, <a href="/wiki/Bebida_de_avena" title="Bebida de avena">avena</a>, soja, avellana, mijo o maní.<sup class="reference separada" id="cite_ref-95"><a href="#cite_note-95"><span class="corchete-llamada">[</span>95<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-96"><a href="#cite_note-96"><span class="corchete-llamada">[</span>96<span class="corchete-llamada">]</span></a></sup>​ Suelen ser consumidas por personas <a href="/wiki/Veganismo" title="Veganismo">veganas</a>, quienes se abstienen de ingerir todo producto de origen animal. Sin embargo en la mayoría de los países no es legal llamar "leche" a productos de origen vegetal; muchas veces se les denomina «jugos» o «bebidas», y el término «leche» es considerado solamente aplicable a líquidos provenientes de glándulas de mamíferos.<sup class="reference separada" id="cite_ref-SethiTyagi2016_97-0"><a href="#cite_note-SethiTyagi2016-97"><span class="corchete-llamada">[</span>97<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-UE1308/2013_98-0"><a href="#cite_note-UE1308/2013-98"><span class="corchete-llamada">[</span>98<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Los consumidores interpretan erróneamente que las "leches" vegetales son un sustituto directo de la <a class="mw-redirect" href="/wiki/Leche_de_vaca" title="Leche de vaca">leche de vaca</a>, pero la mayoría de estas bebidas carecen del equilibrio nutricional de la leche de origen animal, son bajas en proteínas, grasas, calorías y hierro, y algunas tienen contenidos de proteínas y calcio extremadamente bajos.<sup class="reference separada" id="cite_ref-VangaRaghavan2018_99-0"><a href="#cite_note-VangaRaghavan2018-99"><span class="corchete-llamada">[</span>99<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-MakinenWanhalinna2016_100-0"><a href="#cite_note-MakinenWanhalinna2016-100"><span class="corchete-llamada">[</span>100<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-SethiTyagi2016_97-1"><a href="#cite_note-SethiTyagi2016-97"><span class="corchete-llamada">[</span>97<span class="corchete-llamada">]</span></a></sup>​ No son un adecuado sustituto de la leche materna, de las <a href="/wiki/F%C3%B3rmula_para_lactantes" title="Fórmula para lactantes">fórmulas infantiles</a> ni de la leche de vaca en los primeros dos años de vida.<sup class="reference separada" id="cite_ref-SethiTyagi2016_97-2"><a href="#cite_note-SethiTyagi2016-97"><span class="corchete-llamada">[</span>97<span class="corchete-llamada">]</span></a></sup>​ En el caso de niños mayores de dos años que por razones médicas no pueden consumir leche, la recomendación es elegir bebidas fortificadas y que contenga al menos 6 g. de proteína por cada 250 ml.<sup class="reference separada" id="cite_ref-SethiTyagi2016_97-3"><a href="#cite_note-SethiTyagi2016-97"><span class="corchete-llamada">[</span>97<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Con el objetivo de no confundir al consumidor, en los países de la <a href="/wiki/Uni%C3%B3n_Europea" title="Unión Europea">Unión Europea</a> desde 2013 la legislación prohíbe el uso de la palabra "leche" para designar las bebidas vegetales.<sup class="reference separada" id="cite_ref-SethiTyagi2016_97-4"><a href="#cite_note-SethiTyagi2016-97"><span class="corchete-llamada">[</span>97<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-UE1308/2013_98-1"><a href="#cite_note-UE1308/2013-98"><span class="corchete-llamada">[</span>98<span class="corchete-llamada">]</span></a></sup>​
<style data-mw-deduplicate="TemplateStyles:r149592375">.mw-parser-output .flexquote{display:flex;flex-direction:column;background-color:#F9F9F9;border-left:3px solid #c8ccd1;font-size:90%;margin:1em 4em;padding:.4em .8em}.mw-parser-output .flexquote>.flex{display:flex;flex-direction:row}.mw-parser-output .flexquote>.flex>.quote{width:100%}.mw-parser-output .flexquote>.flex>.separator{border-left:1px solid #c8ccd1;border-top:1px solid #c8ccd1;margin:.4em .8em}.mw-parser-output .flexquote>.cite{text-align:right}@media all and (max-width:600px){.mw-parser-output .flexquote>.flex{flex-direction:column}}</style>
</p>
<p>La Norma Oficial Mexicana NOM-155-SCFI-2012 considera «leche» únicamente al «producto obtenido de la secreción de las glándulas mamarias de las vacas» (desestima literal y legalmente considerar «leche» las secreciones mamarias del resto de otras especies mamíferas).<sup class="reference separada" id="cite_ref-vegetalmexico_101-0"><a href="#cite_note-vegetalmexico-101"><span class="corchete-llamada">[</span>101<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El Ministerio de Agricultura del Perú tampoco incluye en la definición de leche ningún producto de origen vegetal.<sup class="reference separada" id="cite_ref-vegetalperu_102-0"><a href="#cite_note-vegetalperu-102"><span class="corchete-llamada">[</span>102<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-vegetalperu2_103-0"><a href="#cite_note-vegetalperu2-103"><span class="corchete-llamada">[</span>103<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-vegetalperu3_104-0"><a href="#cite_note-vegetalperu3-104"><span class="corchete-llamada">[</span>104<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Durante tiempo se especuló con que el <a href="/wiki/Raquitismo" title="Raquitismo">raquitismo</a> podría ser causado por una deficiencia en la dieta, que podría estar relacionada con un bajo consumo de leche. Pero se observó que los niños de zonas urbanas y clima templado frecuentemente desarrollaban el raquitismo, por lo que comenzó a sospecharse que la falta de exposición al sol podría ser el factor desencadenante. Finalmente en 1919 se observó por parte de Mellamby y colaboradores que tanto la administración de aceite de hígado de bacalao como fuente principal de <a href="/wiki/Vitamina_D" title="Vitamina D">vitamina D</a> como la exposición al sol curaban el raquitismo.<sup class="reference separada" id="cite_ref-Glisson_105-0"><a href="#cite_note-Glisson-105"><span class="corchete-llamada">[</span>105<span class="corchete-llamada">]</span></a></sup>​ Esta proposición fue realizada en 1912 por <a href="/wiki/Casimir_Funk" title="Casimir Funk">Casimir Funk</a>, al descubrir y acuñar el término <i>vitamina</i>.<sup class="reference separada" id="cite_ref-106"><a href="#cite_note-106"><span class="corchete-llamada">[</span>106<span class="corchete-llamada">]</span></a></sup>​ En nuestros días el consumo de leche ha llevado a ciertas empresas a crear una variedad de productos que posean similares características a las de la leche, publicitando que su consumo ayuda a evitar <a href="/wiki/Artritis" title="Artritis">artritis</a>, <a href="/wiki/Osteoporosis" title="Osteoporosis">osteoporosis</a> y otros padecimientos relacionados con la desmineralización de los huesos; a la vez que ciertos nutricionistas recomendaban su consumo diario en los años 1980.<sup class="reference separada" id="cite_ref-107"><a href="#cite_note-107"><span class="corchete-llamada">[</span>107<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Estudios posteriores indican que no es tanto la cantidad de calcio que ingerimos lo que importa sino la cantidad que perdemos diariamente en la orina, debido a nuestro estilo de vida. A mayor ingesta de proteína, sobre todo de origen animal (incluso de leche y quesos) es mayor la cantidad de calcio que se pierde en la orina.<sup class="reference separada" id="cite_ref-108"><a href="#cite_note-108"><span class="corchete-llamada">[</span>108<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>En cuanto a la <a href="/wiki/Pirosis" title="Pirosis">pirosis</a> (conocida comúnmente como “acidez”), sensación de quemazón en el <a href="/wiki/Es%C3%B3fago" title="Esófago">esófago</a> causada por la regurgitación de ácido gástrico, se creyó durante mucho tiempo que la leche era un tratamiento eficaz para eliminarla. Si bien puede contrarrestar este síntoma por ser una sustancia ligeramente ácida ( cercana a la neutralidad 6,5 a 6,8), al mismo tiempo el calcio y la caseína estimulan la secreción de jugos gástricos causando un “rebote” que puede incrementar el ácido.<sup class="reference separada" id="cite_ref-109"><a href="#cite_note-109"><span class="corchete-llamada">[</span>109<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La producción de la leche comienza a partir de la inseminación artificial de vacas para preñarlas y que así empiecen a producir las hormonas necesarias para que sus glándulas mamarias comiencen a generarla, naturalmente para luego amamantar a su cría. Una vez generado este efecto, la cría es apartada total o parcialmente de su ejemplar progenitor para dar lugar a la extracción de leche para consumo humano.
</p>
<p>Debido a que tiene un periodo de caducidad corto (sobre todo si se conserva fresca) debe ser distribuida tan pronto como sea posible tras su ordeñado. En varios países la leche suele ser repartida a los hogares diariamente, pero las presiones económicas han hecho que este servicio sea cada vez menos popular. En algunas zonas, además, la dispersión hace prácticamente imposible la realización del reparto de leche. En estos casos las personas optan por comprar la leche en establecimientos como supermercados, vaquerías, tiendas de autoservicio o tiendas de barrio. Antes de la popularización de los <a href="/wiki/Envase" title="Envase">envases</a> <a href="/wiki/Pl%C3%A1stico" title="Plástico">plásticos</a> o <a class="mw-redirect" href="/wiki/Tetra_brik" title="Tetra brik">tetra briks</a>, (que inicialmente se creó especialmente para conservar mejor las propiedades de los lácteos)<sup class="reference separada" id="cite_ref-111"><a href="#cite_note-111"><span class="corchete-llamada">[</span>111<span class="corchete-llamada">]</span></a></sup>​ la leche se vendía en envolturas de papel y <a href="/wiki/Botella" title="Botella">botellas</a> de vidrio.
</p>
<p>En algunos países como en el <a href="/wiki/Reino_Unido" title="Reino Unido">Reino Unido</a> se tiene la costumbre de que un lechero reparta por el vecindario la leche durante la mañana. La leche se entrega en botellas de vidrio con tapas de aluminio frente la puerta de casa. Las tapas de color plateado significan que la leche está homogeneizada, rojo plateado indica semi-descremada, azul plateado indica que la leche está descremada y la dorada indica que proviene del canal insular.
</p>
<p>Las botellas vacías son <a href="/wiki/Reciclaje" title="Reciclaje">recicladas</a>. El lechero regresa al día siguiente a dejar una nueva caja rellena y llevarse las botellas vacías para que sean rellenadas y vueltas a entregar al día siguiente. Actualmente algunas franquicias se oponen al reparto diario y optan por intervalos mayores. Esta forma de reparto es frecuente también en <a class="mw-redirect" href="/wiki/EE._UU." title="EE. UU.">EE. UU.</a>
</p>
<p>Hoy en día, la mejora de los envases y recipientes que contienen la leche ha permitido que sea posible su consumo con bajos requerimientos de conservación en casi todo el mundo.
</p>
<p>El sabor que proporciona la leche es ligeramente dulce (debido a la <a href="/wiki/Lactosa" title="Lactosa">lactosa</a>), las <a class="mw-redirect" href="/wiki/Cocer" title="Cocer">cocciones</a> prolongadas de la leche provocan la <a href="/wiki/Reacci%C3%B3n_de_Maillard" title="Reacción de Maillard">reacción de Maillard</a> entre la lactosa y las proteínas de la leche dando lugar a unos colores tostados. Muchas de las propiedades de la leche desaparecen cuando se mezclan en los platos, uno de los usos fundamentales es proporcionar humedad a algunas preparaciones, llegando a contribuir de forma tímida en los sabores y en las texturas. Es importante mencionar que gran parte de los lácteos son empleados en algunas cocinas de todo el mundo, en algunas de ellas como la <a href="/wiki/Gastronom%C3%ADa_de_Turqu%C3%ADa" title="Gastronomía de Turquía">cocina turca</a>,<sup class="reference separada" id="cite_ref-112"><a href="#cite_note-112"><span class="corchete-llamada">[</span>112<span class="corchete-llamada">]</span></a></sup>​ la <a href="/wiki/Gastronom%C3%ADa_de_la_India" title="Gastronomía de la India">India</a><sup class="reference separada" id="cite_ref-113"><a href="#cite_note-113"><span class="corchete-llamada">[</span>113<span class="corchete-llamada">]</span></a></sup>​ o la <a href="/wiki/Gastronom%C3%ADa_de_M%C3%A9xico" title="Gastronomía de México">cocina mexicana</a><sup class="reference separada" id="cite_ref-114"><a href="#cite_note-114"><span class="corchete-llamada">[</span>114<span class="corchete-llamada">]</span></a></sup>​ son conocidas por su variedad y oferta de recetas diversas.
</p>
<p>La leche es ingrediente de algunas <a href="/wiki/Sopa" title="Sopa">sopas</a> (principalmente las de tipo crema), en las que se añade para reforzar ciertos sabores. También se usa a veces al <a href="/wiki/Revuelto_de_huevo" title="Revuelto de huevo">revolver huevos</a> para que tarden más en cuajar, en el <a href="/wiki/Pur%C3%A9_de_papas" title="Puré de papas">puré de papas</a>, en la elaboración de <a href="/wiki/Salsa_(gastronom%C3%ADa)" title="Salsa (gastronomía)">salsas</a> como la <a href="/wiki/Bechamel" title="Bechamel">bechamel</a> y en postres como el <a href="/wiki/Arroz_con_leche" title="Arroz con leche">arroz con leche</a>, los <a href="/wiki/Flan" title="Flan">flanes</a> (<a class="mw-redirect" href="/wiki/Pudin" title="Pudin">pudines</a>), el típico <a href="/wiki/Tres_leches" title="Tres leches">pastel tres leches</a>, la <a class="new" href="/w/index.php?title=S%C3%A9mola_con_leche&amp;action=edit&amp;redlink=1" title="Sémola con leche (aún no redactado)">sémola con leche</a> (Chile), <a href="/wiki/Helado" title="Helado">helados</a>, etc. Es muy empleado en bebidas tales como el <a href="/wiki/Caf%C3%A9_con_leche" title="Café con leche">café con leche</a> (expresado en el <a href="/wiki/Arte_del_latte" title="Arte del latte">arte del latte</a>), <a href="/wiki/Batido" title="Batido">batidos</a>, <a href="/wiki/Chocolate_caliente" title="Chocolate caliente">chocolate caliente</a>, merengadas, <a href="/wiki/Chicha_criolla" title="Chicha criolla">chicha</a> (Venezuela), etc. Incluso, forma parte de algunas bebidas alcohólicas como el <a class="mw-redirect" href="/wiki/Ponche_Crema" title="Ponche Crema">Ponche Crema</a> venezolano, el <a href="/wiki/Rompope" title="Rompope">rompope</a>, etc.; en algunos <a href="/wiki/C%C3%B3ctel" title="Cóctel">cócteles</a> o simplemente tomarse mezclada con bebidas como el <a class="mw-redirect" href="/wiki/An%C3%ADs_(licor)" title="Anís (licor)">anís</a>.
</p>
<p>La leche no solo ha sido valorada como alimento por los humanos, sino que es la base de interpretaciones simbólicas. Ejemplo de esto es la denominación de la <a href="/wiki/V%C3%ADa_L%C3%A1ctea" title="Vía Láctea">Vía Láctea</a><sup class="reference separada" id="cite_ref-megasitio_116-0"><a href="#cite_note-megasitio-116"><span class="corchete-llamada">[</span>116<span class="corchete-llamada">]</span></a></sup>​ (<a href="/wiki/Galaxia" title="Galaxia">galaxia</a> en la que se encuentra nuestro <a href="/wiki/Sistema_solar" title="Sistema solar">sistema solar</a>). Los grandes cinturones de estrellas que se pueden ver entre las constelaciones de <a href="/wiki/Perseo_(constelaci%C3%B3n)" title="Perseo (constelación)">Perseo</a>, <a href="/wiki/Casiopea_(constelaci%C3%B3n)" title="Casiopea (constelación)">Casiopea</a> y <a class="new" href="/w/index.php?title=Cerfeous_(constelaci%C3%B3n)&amp;action=edit&amp;redlink=1" title="Cerfeous (constelación) (aún no redactado)">Cefeo</a>, fueron bautizados como la vía láctea para recordar la historia en la que <a href="/wiki/Hera" title="Hera">Hera</a> quería amamantar a <a href="/wiki/Heracles" title="Heracles">Heracles</a> (Hércules, en la mitología romana), y este la mordió tan fuerte que un chorro de leche salió disparada hacia el cosmos.<sup class="reference separada" id="cite_ref-megasitio_116-1"><a href="#cite_note-megasitio-116"><span class="corchete-llamada">[</span>116<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La leche y sus derivados han dado lugar a <a href="/wiki/Mito" title="Mito">mitos</a> desde la <a href="/wiki/India" title="India">India</a> hasta <a href="/wiki/Escandinavia" title="Escandinavia">Escandinavia</a>. En el <a href="/wiki/Antiguo_Testamento" title="Antiguo Testamento">Antiguo Testamento</a> aparecen como símbolos de abundancia y creación. Era tomada como un ofrecimiento a los dioses<sup class="reference separada" id="cite_ref-117"><a href="#cite_note-117"><span class="corchete-llamada">[</span>117<span class="corchete-llamada">]</span></a></sup>​ y por lo tanto era tenida como un acto divino relacionado con la vida.
</p>
<p>La leche también puede simbolizar la belleza y la estética femenina.<sup class="reference separada" id="cite_ref-cleo_118-0"><a href="#cite_note-cleo-118"><span class="corchete-llamada">[</span>118<span class="corchete-llamada">]</span></a></sup>​ <a class="mw-redirect" href="/wiki/Cleopatra_VII" title="Cleopatra VII">Cleopatra</a> utilizaba los baños de leche para realzar su belleza y aprovechar las <a href="#Características_generales">propiedades</a> hacia la piel que ofrece la misma.<sup class="reference separada" id="cite_ref-cleo_118-1"><a href="#cite_note-cleo-118"><span class="corchete-llamada">[</span>118<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Por su color siempre constante (con tonalidades de blanco amarillento en todas las especies) ha recibido connotaciones religiosas, como la <a href="/wiki/Virginidad" title="Virginidad">pureza</a>.<sup class="reference separada" id="cite_ref-119"><a href="#cite_note-119"><span class="corchete-llamada">[</span>119<span class="corchete-llamada">]</span></a></sup>​ En la obra <i>La lechera</i> del pintor neerlandés <a href="/wiki/Johannes_Vermeer" title="Johannes Vermeer">Vermeer</a>, la blancura de la leche alude a la pureza y virtudes de la joven.<sup class="reference separada" id="cite_ref-lalechera_115-1"><a href="#cite_note-lalechera-115"><span class="corchete-llamada">[</span>115<span class="corchete-llamada">]</span></a></sup>​ Algunos de los quesos más famosos se asocian al principal país productor. Así, el <a class="mw-redirect" href="/wiki/Queso_Roquefort" title="Queso Roquefort">queso Roquefort</a> es típico de Francia, el <a href="/wiki/Mozzarella" title="Mozzarella">mozzarella</a> y el queso azul <a href="/wiki/Gorgonzola" title="Gorgonzola">gorgonzola</a> a los platillos de <a href="/wiki/Italia" title="Italia">Italia</a>, o el <a class="mw-redirect" href="/wiki/Queso_Zamorano" title="Queso Zamorano">Zamorano</a> de la región de <a href="/wiki/Castilla_y_Le%C3%B3n" title="Castilla y León">Castilla y León</a>, en <a href="/wiki/Espa%C3%B1a" title="España">España</a>.<sup class="reference separada" id="cite_ref-120"><a href="#cite_note-120"><span class="corchete-llamada">[</span>120<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>En botánica, el blanco de la leche fue comparado con el color de las <a class="mw-redirect" href="/wiki/Gallina" title="Gallina">gallinas</a> para bautizar a la planta <i><a href="/wiki/Ornithogalum_umbellatum" title="Ornithogalum umbellatum">Ornithogalum umbellatum</a></i> como <i>Leche de gallina</i>, planta nativa del <a class="mw-redirect" href="/wiki/Mediterr%C3%A1neo" title="Mediterráneo">Mediterráneo</a> de la familia de las <i><a class="mw-redirect" href="/wiki/Lili%C3%A1ceas" title="Liliáceas">Liliáceas</a></i>.<sup class="reference separada" id="cite_ref-121"><a href="#cite_note-121"><span class="corchete-llamada">[</span>121<span class="corchete-llamada">]</span></a></sup>​
</p>
