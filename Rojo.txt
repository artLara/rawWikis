<p>El <b>rojo</b> o <b>colorado</b><sup class="reference separada" id="cite_ref-1"><a href="#cite_note-1"><span class="corchete-llamada">[</span>1<span class="corchete-llamada">]</span></a></sup>​, también poéticamente <b>roso</b><sup class="reference separada" id="cite_ref-2"><a href="#cite_note-2"><span class="corchete-llamada">[</span>2<span class="corchete-llamada">]</span></a></sup>​, es un <a href="/wiki/Color" title="Color">color</a> que se percibe ante la <a href="/wiki/Fotorreceptor" title="Fotorreceptor">fotorrecepción</a> de una <a href="/wiki/Luz" title="Luz">luz</a> cuya <a href="/wiki/Longitud_de_onda" title="Longitud de onda">longitud de onda</a> dominante mide entre 619 y 780 <a href="/wiki/Nan%C3%B3metro" title="Nanómetro">nm</a>. Se asemeja a la coloración de la <a href="/wiki/Sangre" title="Sangre">sangre</a> <a href="/wiki/Arteria" title="Arteria">arterial</a> de los <a href="/wiki/Vertebrata" title="Vertebrata">vertebrados</a>.<sup class="reference separada" id="cite_ref-DicColor_3-0"><a href="#cite_note-DicColor-3"><span class="corchete-llamada">[</span>3<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Se considera que el rojo es un <a class="mw-redirect" href="/wiki/Color_c%C3%A1lido" title="Color cálido">color cálido</a>, con una relación con el fuego y la incandescencia. El rojo estándar puede verse en el recuadro de la derecha, arriba; los valores que se dan debajo corresponden al mismo.
</p>
<p>La denominación de color rojo abarca al conjunto de coloraciones similares al estándar: <b>rojizas</b>.<sup class="reference separada" id="cite_ref-GuiaCol_4-0"><a href="#cite_note-GuiaCol-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La palabra <b>rojo</b> deriva del <a href="/wiki/Lat%C3%ADn" title="Latín">latín</a> <i>rŭssum</i> o <i>rŭsseŭs</i>, ‘rojo subido’,<sup class="reference separada" id="cite_ref-Corominas_5-0"><a href="#cite_note-Corominas-5"><span class="corchete-llamada">[</span>5<span class="corchete-llamada">]</span></a></sup>​ relacionado con <i>ruber</i>, ‘rojo’, del <a href="/wiki/Idioma_protoindoeuropeo" title="Idioma protoindoeuropeo">protoindoeuropeo</a> <i>*reudh–</i>, ‘rojo’.<sup class="reference separada" id="cite_ref-6"><a href="#cite_note-6"><span class="corchete-llamada">[</span>6<span class="corchete-llamada">]</span></a></sup>​ El término «rojo» comenzó a usarse regularmente en <a href="/wiki/Idioma_espa%C3%B1ol" title="Idioma español">idioma castellano</a> durante el siglo <span style="font-variant:small-caps;text-transform:lowercase">XV</span>, en la <a href="/wiki/Edad_Media" title="Edad Media">Edad Media</a> ya existía, pero era de uso poco común. De acuerdo con el <a href="/wiki/Filolog%C3%ADa" title="Filología">filólogo</a> <a class="mw-redirect" href="/wiki/Joan_Corominas" title="Joan Corominas">Joan Corominas</a>, «rojo» significaba antiguamente un color rojizo, mientras que las denominaciones tradicionales del color de la sangre eran «<a href="/wiki/Bermejo_(color)" title="Bermejo (color)">bermejo</a>», «colorado» y «encarnado».<sup class="reference separada" id="cite_ref-Corominas_5-1"><a href="#cite_note-Corominas-5"><span class="corchete-llamada">[</span>5<span class="corchete-llamada">]</span></a></sup>​
El rojo es el único color para cuyo nombre se ha hallado una raíz protoindoeuropea común y también es el color de la sangre.<sup class="reference separada" id="cite_ref-7"><a href="#cite_note-7"><span class="corchete-llamada">[</span>7<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Un sinónimo de rojo es <b>colorado</b>, del latín <i>colorātus</i>, de <i>colorāre</i>, ‘colorar’.<sup class="reference separada" id="cite_ref-9"><a href="#cite_note-9"><span class="corchete-llamada">[</span>9<span class="corchete-llamada">]</span></a></sup>​ <b>Encarnado</b> también es un sinónimo, aunque se aplica tanto al color rojo como al color <a class="mw-redirect" href="/wiki/Carnaci%C3%B3n" title="Carnación">carne o carnación</a>.<sup class="reference separada" id="cite_ref-10"><a href="#cite_note-10"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​ Al color rojo vivo se le llama en ocasiones <b>punzó</b>, esto a partir del <a href="/wiki/Galicismo" title="Galicismo">galicismo</a> muy frecuente en la primera mitad del siglo <span style="font-variant:small-caps;text-transform:lowercase">XIX</span> <i>ponceau</i>, nombre francés de la <a class="mw-redirect" href="/wiki/Amapola" title="Amapola">amapola</a> silvestre con pétalos rojos.<sup class="reference separada" id="cite_ref-11"><a href="#cite_note-11"><span class="corchete-llamada">[</span>11<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El rojo es uno de los cuatro <a href="/wiki/Psicolog%C3%ADa_del_color" title="Psicología del color">colores psicológicos</a> <a href="/wiki/Color_primario" title="Color primario">primarios</a>, junto con el <a href="/wiki/Amarillo" title="Amarillo">amarillo</a>, el <a href="/wiki/Verde" title="Verde">verde</a> y el <a href="/wiki/Azul" title="Azul">azul</a>. Además, es considerado un color cálido, junto con el <a href="/wiki/Naranja_(color)" title="Naranja (color)">naranja</a>, el amarillo y todas las coloraciones que tienden a estos.<sup class="reference separada" id="cite_ref-DicColor_3-1"><a href="#cite_note-DicColor-3"><span class="corchete-llamada">[</span>3<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>En el sistema de síntesis <a href="/wiki/S%C3%ADntesis_sustractiva_de_color" title="Síntesis sustractiva de color">sustractiva</a> de color, donde los colores se crean mezclando <a href="/wiki/Pigmento" title="Pigmento">pigmentos</a> o <a href="/wiki/Tinte" title="Tinte">tintes</a> (pinturas, colorantes, tintas), los <a class="mw-redirect" href="/wiki/Colores_primarios" title="Colores primarios">colores primarios</a> son el <a class="mw-redirect" href="/wiki/Cian_(color)" title="Cian (color)">cian</a>, el <a href="/wiki/Magenta" title="Magenta">magenta</a> y el <a href="/wiki/Amarillo" title="Amarillo">amarillo</a>. El rojo, junto con el <a href="/wiki/Verde" title="Verde">verde</a> y el <a href="/wiki/Azul" title="Azul">azul</a>, son los <a class="mw-redirect" href="/wiki/Colores_secundarios" title="Colores secundarios">colores secundarios</a> en este sistema, es decir que cuando se trabaja con pigmentos de cualquier clase, para obtener rojo se deben mezclar dos de los colores primarios sustractivos (específicamente, <a href="/wiki/Amarillo" title="Amarillo">amarillo</a> y <a href="/wiki/Magenta" title="Magenta">magenta</a>).
</p>
<p>El procedimiento de impresión por <a class="mw-redirect" href="/wiki/Cuatricrom%C3%ADa" title="Cuatricromía">cuatricromía</a> (que se usa para imprimir, por ejemplo, libros y revistas en color) emplea los colores primarios sustractivos con el agregado de negro. De allí que un color para cuatricromía se describa mediante el porcentaje de <a href="/wiki/Tinta" title="Tinta">tinta</a> de cada uno de estos cuatro colores que entra en su composición. Como el rojo se forma con la adición de magenta y amarillo, un área impresa en color rojo intenso estará compuesta por C=0 (0 % de cian), M=100 (100 % de magenta), Y=100 (100 % de amarillo) y K=0 (0 % de negro). <i>Véase <a class="mw-redirect" href="/wiki/CMYK" title="CMYK">CMYK</a>.</i>
</p>
<p>En este sistema de cromosíntesis, el <a href="/wiki/Colores_complementarios" title="Colores complementarios">color complementario</a> del rojo es el <a class="mw-redirect" href="/wiki/Cian_(color)" title="Cian (color)">cian</a>.
</p>
<p>Durante mucho tiempo el rojo tuvo estatus de color primario sustractivo. De hecho, ha formado parte de casi todos grupos de colores básicos, principales o primarios que diferentes estudiosos del tema propusieron a lo largo de la historia.<sup class="reference separada" id="cite_ref-12"><a href="#cite_note-12"><span class="corchete-llamada">[</span>12<span class="corchete-llamada">]</span></a></sup>​ En el siglo <span style="font-variant:small-caps;text-transform:lowercase">XIX</span> por ejemplo, el tipo de rojo más usado como primario fue el <a href="/wiki/Carm%C3%ADn" title="Carmín">carmín</a> o <a href="/wiki/Carmes%C3%AD" title="Carmesí">carmesí</a>.<sup class="reference separada" id="cite_ref-13"><a href="#cite_note-13"><span class="corchete-llamada">[</span>13<span class="corchete-llamada">]</span></a></sup>​ Los estudios de <a href="/wiki/Isaac_Newton" title="Isaac Newton">Newton</a> sobre la naturaleza de la luz y del color suscitaron numerosas teorías sobre el tema, no siempre correctas. A principios del siglo <span style="font-variant:small-caps;text-transform:lowercase">XVIII</span> algunos tratados de pintura habían adaptado el círculo de colores creado por Newton a las necesidades del arte pictórico y señalaban que los tres colores primarios eran el rojo, el amarillo y el azul del círculo newtoniano. La práctica de utilizar esta tríada de primarios en pintura continúa hasta el día de hoy.<sup class="reference separada" id="cite_ref-Handprint_14-0"><a href="#cite_note-Handprint-14"><span class="corchete-llamada">[</span>14<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Actualmente, tras varios siglos de desarrollo de pigmentos y estudios sobre el color, se considera que los tres colores primarios sustractivos óptimos son el cian, el magenta y el amarillo, pero en el ámbito del arte, por razones técnicas y tradicionales, generalmente se prefiere continuar usando los primarios «antiguos».<sup class="reference separada" id="cite_ref-Handprint_14-1"><a href="#cite_note-Handprint-14"><span class="corchete-llamada">[</span>14<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>En este sistema de cromosíntesis, el <a href="/wiki/Colores_complementarios" title="Colores complementarios">color complementario</a> del rojo es el <a href="/wiki/Verde" title="Verde">verde</a>.
</p>
<p>En el <a href="/wiki/S%C3%ADntesis_aditiva_de_color" title="Síntesis aditiva de color">sistema aditivo de síntesis de color</a>, en el cual los colores se obtienen mezclando luz de color en lugar de pigmentos, el rojo es un <a class="mw-redirect" href="/wiki/Colores_secundarios" title="Colores secundarios">color primario</a>, junto con el <a href="/wiki/Verde" title="Verde">verde</a> y el <a href="/wiki/Azul" title="Azul">azul</a>. Esto significa que cuando se trabaja con luz de color, basta con mezclar esos tres colores en diferentes proporciones para obtener todos los demás. Para crear las tonalidades claras y oscuras, se reduce o aumenta la luminosidad.
</p>
<p>Este sistema aditivo de colores luz es el que utilizan los monitores y televisores para producir colores. En este sistema, un color se describe con valores numéricos para cada uno de sus componentes (rojo, verde y azul), indicando al rojo con «R», al verde con «G» y al azul con «B». En una escala de valores de 0 a 255, el rojo aditivo puro se expresa como R=255 (rojo al valor máximo), G=0 (nada de verde) y B=0 (nada de azul). <i>Véase <a class="mw-redirect" href="/wiki/Modelo_de_color_RGB" title="Modelo de color RGB">RGB</a>.</i>
</p>
<p>Este rojo fue uno de los primeros colores que pudieron reproducir los <a href="/wiki/Computadora_personal" title="Computadora personal">ordenadores personales</a> al abandonar la monocromía, a principios de los años <a href="/wiki/1980" title="1980">1980</a>.
</p>
<p>En este sistema de cromosíntesis, el <a href="/wiki/Colores_complementarios" title="Colores complementarios">color complementario</a> del rojo es el <a class="mw-redirect" href="/wiki/Cian_(color)" title="Cian (color)">cian</a>.
</p>
<p><b>Rojo espectral</b> es la región de <a href="/wiki/Luz" title="Luz">luz</a> de color rojo del <a href="/wiki/Espectro_electromagn%C3%A9tico" title="Espectro electromagnético">espectro electromagnético</a> que el ojo humano es capaz de percibir. La <a href="/wiki/Longitud_de_onda" title="Longitud de onda">longitud de onda</a> de la luz roja es de alrededor de 645-700 <a href="/wiki/Nan%C3%B3metro" title="Nanómetro">nm</a>; las <a href="/wiki/Frecuencia" title="Frecuencia">frecuencias</a> más bajas que el rojo —y, por lo tanto, de longitudes de onda más largas— se denominan <a class="mw-redirect" href="/wiki/Infrarroja" title="Infrarroja">infrarrojas</a> y no son visibles. Así, el color rojo corresponde a la frecuencia más baja de luz discernible por el <a href="/wiki/Ojo" title="Ojo">ojo</a> humano.
</p>
<p>En Occidente, la interpretación tradicional del cromatismo del <a href="/wiki/Arco%C3%ADris" title="Arcoíris">arcoíris</a> sostiene que este contiene siete colores, que corresponden a los siete colores en que <a href="/wiki/Isaac_Newton" title="Isaac Newton">Newton</a> dividió el espectro de luz visible. En este contexto, el rojo es considerado el primer color visible, tanto del <a href="/wiki/Isaac_Newton#Trabajos_sobre_la_luz" title="Isaac Newton">espectro newtoniano</a> como del arcoíris.<sup class="reference separada" id="cite_ref-DicColor_3-2"><a href="#cite_note-DicColor-3"><span class="corchete-llamada">[</span>3<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>De acuerdo con la longitud de onda, el espectro que corresponde al rojo puede dividirse en cuatro zonas:
</p>
<p>Las personas con dificultades para distinguir el color rojo se denominan protanómalas. La <a href="/wiki/Protanomal%C3%ADa" title="Protanomalía">protanomalía</a> puede consistir en una capacidad parcial de percepción del color rojo o en una incapacidad total, dependiendo de la condición de los <a href="/wiki/Cono_(c%C3%A9lula)" title="Cono (célula)">conos L</a> del individuo protanómalo. Los conos L son las células de la <a href="/wiki/Retina" title="Retina">retina</a> que perciben las longitudes de onda lumínicas largas; si son defectuosos, producen una ceguera parcial al rojo, y si faltan del todo producen una ceguera total a este color, o <a class="new" href="/w/index.php?title=Protanop%C3%ADa&amp;action=edit&amp;redlink=1" title="Protanopía (aún no redactado)">protanopía</a>.<sup class="reference separada" id="cite_ref-17"><a href="#cite_note-17"><span class="corchete-llamada">[</span>17<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>En la naturaleza, el color rojo cumple un papel importante en la coloración de advertencia, también llamada aposemática, de los animales. Ciertas especies utilizan este color —generalmente en combinación con <a href="/wiki/Negro_(color)" title="Negro (color)">negro</a> u otros colores contrastantes— para advertir a los depredadores de su toxicidad o mal sabor.<sup class="reference separada" id="cite_ref-18"><a href="#cite_note-18"><span class="corchete-llamada">[</span>18<span class="corchete-llamada">]</span></a></sup>​ El rojo usado en estos casos es intenso, apropiado para distinguirse en el entorno natural, donde son más frecuentes los verdes, azules y marrones.
</p>
<p>La sangre oxigenada es roja debido a la presencia de hemoglobina oxigenada.
</p>
<p>Cuando se utiliza para describir la coloración animal natural, «rojo» generalmente se refiere a un color <a href="/wiki/Marr%C3%B3n" title="Marrón">marrón</a>, rojizo o jengibre. En este sentido se utiliza para describir los colores de la capa de perros y vacas de color marrón rojizo y en los nombres de varias especies de animales o razas como el zorro, la ardilla roja, ciervos rojos, el petirrojo, <i>Calidris canutus</i>, el colirrojo real, el zorzal alirrojo, el <i>setter</i> rojo, ganado rojo Devon, etc. 
</p>
<p>Cuando se utiliza para las flores, «rojo» se refiere a menudo al purpúreo (trébol rojo, heleborina roja) o colores rosados (valeriana roja).
</p>

<p>En la liturgia católica, el rojo simboliza la sangre derramada por el Evangelio y al <a href="/wiki/Esp%C3%ADritu_Santo" title="Espíritu Santo">Espíritu Santo</a>. Con relación a la sangre, se utiliza durante las fiestas de los mártires, apóstoles (salvo S. Juan que no murió mártir), las relacionadas con la <a href="/wiki/Pasi%C3%B3n_de_Cristo" title="Pasión de Cristo">Pasión de Cristo</a> (<a href="/wiki/Domingo_de_Ramos" title="Domingo de Ramos">Domingo de Ramos</a>, <a href="/wiki/Viernes_Santo" title="Viernes Santo">Viernes Santo</a>, <a href="/wiki/Vera_Cruz_(cristianismo)" title="Vera Cruz (cristianismo)">Santa Cruz</a>) y con la <a href="/wiki/Eucarist%C3%ADa" title="Eucaristía">Eucaristía</a> en algunas tradiciones, como la <a href="/wiki/Rito_ambrosiano" title="Rito ambrosiano">ambrosiana</a>. En relación con el Espíritu Santo: <a href="/wiki/Pentecost%C3%A9s" title="Pentecostés">Pentecostés</a> y durante la celebración del sacramento de la <a href="/wiki/Confirmaci%C3%B3n" title="Confirmación">confirmación</a>, aunque en este último caso también se puede usar blanco. Actualmente también se usa en las exequias papales, como reminiscencia de que el Papa antiguamente no revestía nunca casulla morada (ni verde).
</p>

Como se ve, coincide con el rojo aditivo puro, y en programación se lo invoca con el nombre <i>red</i> (rojo). Más abajo figuran los demás rojos web X11.
