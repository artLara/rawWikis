<p>El <b>carbono</b> (del <a class="mw-redirect" href="/wiki/Idioma_lat%C3%ADn" title="Idioma latín">latín</a>, <i>carbo</i>, 'carbón') es un <a href="/wiki/Elemento_qu%C3%ADmico" title="Elemento químico">elemento químico</a> con <a href="/wiki/S%C3%ADmbolo_qu%C3%ADmico" title="Símbolo químico">símbolo</a> <b>C</b>, <a href="/wiki/N%C3%BAmero_at%C3%B3mico" title="Número atómico">número atómico</a> <b>6</b> y <a href="/wiki/Masa_at%C3%B3mica" title="Masa atómica">masa atómica</a> <b>12,01</b>. Es un <a href="/wiki/No_metal" title="No metal">no metal</a> y <a class="mw-redirect" href="/wiki/Tetravalente" title="Tetravalente">tetravalente</a>, disponiendo de cuatro electrones para formar enlaces químicos <a href="/wiki/Enlace_covalente" title="Enlace covalente">covalentes</a>. Tres <a class="mw-redirect" href="/wiki/Is%C3%B3topos_del_carbono" title="Isótopos del carbono">isótopos del carbono</a> se producen de forma natural, los estables <a href="/wiki/Carbono-12" title="Carbono-12"><sup>12</sup>C</a> y <a href="/wiki/Carbono-13" title="Carbono-13"><sup>13</sup>C</a> y el <a href="/wiki/Radiois%C3%B3topo" title="Radioisótopo">isótopo radiactivo</a> <a href="/wiki/Carbono-14" title="Carbono-14"><sup>14</sup>C</a>, que decae con una <a href="/wiki/Periodo_de_semidesintegraci%C3%B3n" title="Periodo de semidesintegración">vida media</a> de unos 5730 años.<sup class="reference separada" id="cite_ref-isotopes_1-0"><a href="#cite_note-isotopes-1"><span class="corchete-llamada">[</span>1<span class="corchete-llamada">]</span></a></sup>​ El carbono es uno de los <a href="/wiki/Descubrimiento_de_los_elementos_qu%C3%ADmicos" title="Descubrimiento de los elementos químicos">pocos elementos conocidos desde la antigüedad</a>,<sup class="reference separada" id="cite_ref-D2_2-0"><a href="#cite_note-D2-2"><span class="corchete-llamada">[</span>2<span class="corchete-llamada">]</span></a></sup>​ y es el pilar básico de la <a href="/wiki/Qu%C3%ADmica_org%C3%A1nica" title="Química orgánica">química orgánica</a>. Está presente en la Tierra en estado de cuerpo simple (<a href="/wiki/Carb%C3%B3n" title="Carbón">carbón</a> y <a href="/wiki/Diamante" title="Diamante">diamantes</a>), de <a href="/wiki/Compuesto_inorg%C3%A1nico" title="Compuesto inorgánico">compuestos inorgánicos</a> (CO<sub>2</sub> y CaCO<sub>3</sub>) y de <a href="/wiki/Compuesto_org%C3%A1nico" title="Compuesto orgánico">compuestos orgánicos</a> (<a href="/wiki/Biomasa_(ecolog%C3%ADa)" title="Biomasa (ecología)">biomasa</a>, <a href="/wiki/Petr%C3%B3leo" title="Petróleo">petróleo</a> y <a href="/wiki/Gas_natural" title="Gas natural">gas natural</a>). También se han sintetizado muchas nuevas estructuras basadas en el carbono: <a href="/wiki/Carb%C3%B3n_activado" title="Carbón activado">carbón activado</a>, <a href="/wiki/Negro_de_humo" title="Negro de humo">negro de humo</a>, <a href="/wiki/Fibra_de_carbono" title="Fibra de carbono">fibras</a>, <a class="mw-redirect" href="/wiki/Nanotubos" title="Nanotubos">nanotubos</a>, <a class="mw-redirect" href="/wiki/Fullerenos" title="Fullerenos">fullerenos</a> y <a href="/wiki/Grafeno" title="Grafeno">grafeno</a>.
</p>
<p>El carbono es el 15.º elemento más abundante en la corteza terrestre,<sup class="reference separada" id="cite_ref-3"><a href="#cite_note-3"><span class="corchete-llamada">[</span>3<span class="corchete-llamada">]</span></a></sup>​ y el cuarto elemento más <a href="/wiki/Abundancia_de_los_elementos_qu%C3%ADmicos" title="Abundancia de los elementos químicos">abundante en el universo en masa</a> después del <a href="/wiki/Hidr%C3%B3geno" title="Hidrógeno">hidrógeno</a>, el <a href="/wiki/Helio" title="Helio">helio</a> y el <a href="/wiki/Ox%C3%ADgeno" title="Oxígeno">oxígeno</a>. La abundancia del carbono, su diversidad única de <a href="/wiki/Compuesto_org%C3%A1nico" title="Compuesto orgánico">compuestos orgánicos</a> y su inusual capacidad para formar <a href="/wiki/Pol%C3%ADmero" title="Polímero">polímeros</a> a las temperaturas comúnmente encontradas en la <a href="/wiki/Tierra" title="Tierra">Tierra</a>, permite que este elemento sirva como componente común de <a href="/wiki/Vida" title="Vida">toda la vida conocida</a>. Es el segundo elemento más abundante en el <a href="/wiki/Cuerpo_humano" title="Cuerpo humano">cuerpo humano</a> en masa (aproximadamente el 18,5%) después del oxígeno.<sup class="reference separada" id="cite_ref-4"><a href="#cite_note-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Los átomos de carbono pueden unirse de diferentes maneras, denominadas <a href="/wiki/Al%C3%B3tropos_del_carbono" title="Alótropos del carbono">alótropos del carbono</a>, reflejo de las condiciones de formación. Los más conocidos que ocurren naturalmente son el <a href="/wiki/Grafito" title="Grafito">grafito</a>, el <a href="/wiki/Diamante" title="Diamante">diamante</a> y el <a href="/wiki/Carbono_amorfo" title="Carbono amorfo">carbono amorfo</a>.<sup class="reference separada" id="cite_ref-therm_prop_5-0"><a href="#cite_note-therm_prop-5"><span class="corchete-llamada">[</span>5<span class="corchete-llamada">]</span></a></sup>​ Las <a class="mw-redirect" href="/wiki/Propiedades_f%C3%ADsicas" title="Propiedades físicas">propiedades físicas</a> del carbono varían ampliamente con la forma alotrópica. Por ejemplo, el grafito es <a href="/wiki/Opacidad" title="Opacidad">opaco</a> y negro, mientras que el diamante es altamente <a class="mw-redirect" href="/wiki/Transparencia" title="Transparencia">transparente</a>. El grafito es lo suficientemente blando como para formar una raya en el papel (de ahí su nombre, del verbo griego "γράφειν" que significa 'escribir'), mientras que el diamante es el material natural más duro conocido. El grafito es un buen <a href="/wiki/Conductor_el%C3%A9ctrico" title="Conductor eléctrico">conductor eléctrico</a> mientras que el diamante tiene una baja <a href="/wiki/Conductividad_el%C3%A9ctrica" title="Conductividad eléctrica">conductividad eléctrica</a>. En condiciones normales, el diamante, los <a class="mw-redirect" href="/wiki/Nanotubo_de_carbono" title="Nanotubo de carbono">nanotubos de carbono</a> y el <a href="/wiki/Grafeno" title="Grafeno">grafeno</a> tienen las <a href="/wiki/Conductividad_t%C3%A9rmica" title="Conductividad térmica">conductividades térmicas</a> más altas de todos los materiales conocidos. Todos los alótropos del carbono son <a href="/wiki/S%C3%B3lido" title="Sólido">sólidos</a> en condiciones normales, siendo el grafito la forma <a href="/wiki/Equilibrio_termodin%C3%A1mico" title="Equilibrio termodinámico">termodinámicamente estable</a>. Son químicamente resistentes y requieren altas temperaturas para reaccionar incluso con oxígeno.
</p>
<p>El <a href="/wiki/Estado_de_oxidaci%C3%B3n" title="Estado de oxidación">estado de oxidación</a> más común del carbono en los <a class="mw-redirect" href="/wiki/Compuestos_inorg%C3%A1nicos" title="Compuestos inorgánicos">compuestos inorgánicos</a> es +4, mientras que +2 se encuentra en el <a href="/wiki/Mon%C3%B3xido_de_carbono" title="Monóxido de carbono">monóxido de carbono</a> y en complejos <a href="/wiki/Carbonilo_de_metal" title="Carbonilo de metal">carbonilos</a> de <a class="mw-redirect" href="/wiki/Metales_de_transici%C3%B3n" title="Metales de transición">metales de transición</a>. Las mayores fuentes de carbono inorgánico son las <a href="/wiki/Caliza" title="Caliza">calizas</a>, <a href="/wiki/Dolomita" title="Dolomita">dolomitas</a> y <a href="/wiki/Di%C3%B3xido_de_carbono" title="Dióxido de carbono">dióxido de carbono</a>, pero cantidades significativas se producen en depósitos orgánicos de <a href="/wiki/Carb%C3%B3n" title="Carbón">carbón</a>, <a href="/wiki/Turba" title="Turba">turba</a>, <a href="/wiki/Petr%C3%B3leo" title="Petróleo">petróleo</a> y <a class="mw-redirect" href="/wiki/Clatrato_de_metano" title="Clatrato de metano">clatratos de metano</a>. El carbono forma un gran número de compuestos, más que cualquier otro elemento, con casi diez millones de compuestos descritos hasta la fecha<sup class="reference separada" id="cite_ref-lanl_6-0"><a href="#cite_note-lanl-6"><span class="corchete-llamada">[</span>6<span class="corchete-llamada">]</span></a></sup>​ (con 500.000 compuestos nuevos por año), siendo sin embargo ese número solo una fracción del número de compuestos teóricamente posibles bajo condiciones estándar. Por esta razón, a menudo el carbono se ha descrito como el «rey de los elementos».<sup class="reference separada" id="cite_ref-7"><a href="#cite_note-7"><span class="corchete-llamada">[</span>7<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La combustión del carbono en todas sus formas ha sido la base del desarrollo tecnológico desde tiempos prehistóricos. Los materiales basados en el carbono tienen aplicaciones en numerosas áreas de vanguardia tecnológica: <a href="/wiki/Material_compuesto" title="Material compuesto">materiales compuestos</a>, <a href="/wiki/Bater%C3%ADa_de_ion_de_litio" title="Batería de ion de litio">baterías de iones de litio</a>, <a class="mw-redirect" href="/wiki/Carb%C3%B3n_activo" title="Carbón activo">descontaminación del aire y del agua</a>, <a href="/wiki/Electrodo" title="Electrodo">electrodos</a> para <a href="/wiki/Horno_de_arco_el%C3%A9ctrico" title="Horno de arco eléctrico">hornos de arco</a>, en la <a href="/wiki/Aluminio#Producción" title="Aluminio">síntesis de aluminio</a>, etc.
</p>
<p>El carbono es un elemento notable por varias razones. Sus formas <a class="mw-redirect" href="/wiki/Alotr%C3%B3picas" title="Alotrópicas">alotrópicas</a> incluyen, una de las sustancias más blandas (el grafito) y una de las más duras (el diamante) y, desde el punto de vista económico, es de los materiales más baratos (carbón) y uno de los más caros (diamante). Más aún, presenta una gran afinidad para enlazarse químicamente con otros átomos pequeños, incluyendo otros átomos de carbono con los que puede formar largas cadenas, y su pequeño <a href="/wiki/Radio_at%C3%B3mico" title="Radio atómico">radio atómico</a> le permite formar enlaces múltiples. Así, con el <a href="/wiki/Ox%C3%ADgeno" title="Oxígeno">oxígeno</a> forma el <a href="/wiki/Di%C3%B3xido_de_carbono" title="Dióxido de carbono">dióxido de carbono</a>, vital para el crecimiento de las <a href="/wiki/Vegetal" title="Vegetal">plantas</a> (ver <a href="/wiki/Ciclo_del_carbono" title="Ciclo del carbono">ciclo del carbono</a>); con el <a href="/wiki/Hidr%C3%B3geno" title="Hidrógeno">hidrógeno</a> forma numerosos compuestos denominados genéricamente <a href="/wiki/Hidrocarburo" title="Hidrocarburo">hidrocarburos</a>, esenciales para la industria y el transporte en la forma de <a href="/wiki/Combustible_f%C3%B3sil" title="Combustible fósil">combustibles fósiles</a>; y combinado con <a href="/wiki/Ox%C3%ADgeno" title="Oxígeno">oxígeno</a> e hidrógeno forma gran variedad de compuestos como, por ejemplo, los <a href="/wiki/%C3%81cido_graso" title="Ácido graso">ácidos grasos</a>, esenciales para la vida, y los <a href="/wiki/%C3%89ster" title="Éster">ésteres</a> que dan sabor a las frutas; además es vector, a través del <a href="/wiki/Ciclo_CNO" title="Ciclo CNO">ciclo carbono-nitrógeno</a>, de parte de la <a href="/wiki/Energ%C3%ADa" title="Energía">energía</a> producida por el <a href="/wiki/Sol" title="Sol">Sol</a>.<sup class="reference separada" id="cite_ref-8"><a href="#cite_note-8"><span class="corchete-llamada">[</span>8<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Se conocen cinco formas alotrópicas del carbono, además del amorfo: <a href="/wiki/Grafito" title="Grafito">grafito</a>, <a href="/wiki/Diamante" title="Diamante">diamante</a>, <a href="/wiki/Fullereno" title="Fullereno">fullereno</a>, <a href="/wiki/Grafeno" title="Grafeno">grafeno</a> y <a href="/wiki/Carbino" title="Carbino">carbino</a>.<sup class="reference separada" id="cite_ref-9"><a href="#cite_note-9"><span class="corchete-llamada">[</span>9<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Una de las formas en las cuales se encuentra el carbono es el grafito, caracterizado por tener sus átomos "en los vértices de hexágonos que tapizan un plano",<sup class="reference separada" id="cite_ref-Gasque_1_10-0"><a href="#cite_note-Gasque_1-10"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​ es de color negro, opaco y blando, y es el material del cual está hecha la parte interior de los lápices de madera. El grafito tiene exactamente los mismos átomos del diamante, pero por estar dispuestos en diferente forma tienen distintas propiedades físicas y químicas. Los diamantes naturales se forman en lugares donde el carbono ha sido sometido a grandes presiones y altas temperaturas. Su estructura es tetraédrica, que da como resultado una red tridimensional y a diferencia del grafito tiene un grado de dureza alto: 10 Mohs. Los diamantes se pueden crear artificialmente, sometiendo el grafito a temperaturas y presiones muy altas. El precio del grafito es menor al de los diamantes naturales, pero si se han elaborado adecuadamente tienen la misma dureza, color y transparencia.
</p>
<p>La forma amorfa es esencialmente grafito, pero no llega a adoptar una estructura cristalina macroscópica. Esta es la forma presente en la mayoría de los <a href="/wiki/Carb%C3%B3n" title="Carbón">carbones</a> y en el hollín.
</p>
<p>A presión normal, el carbono adopta la forma del <a href="/wiki/Grafito" title="Grafito">grafito</a>, en la que cada átomo está unido a otros tres en un plano compuesto de celdas hexagonales; este estado se puede describir como tres <a href="/wiki/Electr%C3%B3n" title="Electrón">electrones</a> de valencia en <a href="/wiki/Orbital_at%C3%B3mico" title="Orbital atómico">orbitales</a> híbridos planos sp² y el cuarto en el orbital <i>p</i>.
</p>
<p>Las dos formas de grafito conocidas alfa (hexagonal) y beta (romboédrica) tienen propiedades físicas idénticas. Los grafitos naturales contienen más del 30 % de la forma beta, mientras que el grafito sintético contiene únicamente la forma alfa. La forma alfa puede transformarse en beta mediante procedimientos mecánicos, y esta recristalizar en forma alfa al calentarse por encima de 1000 <a href="/wiki/Grado_Celsius" title="Grado Celsius">°C</a>.
</p>
<p>Debido a la deslocalización de los <a href="/wiki/Electr%C3%B3n" title="Electrón">electrones</a> del <a href="/wiki/Orbital_at%C3%B3mico" title="Orbital atómico">orbital pi</a>, el grafito es <a href="/wiki/Conductor_el%C3%A9ctrico" title="Conductor eléctrico">conductor</a> de la electricidad, propiedad que permite su uso en procesos de <a href="/wiki/Electroerosi%C3%B3n" title="Electroerosión">electroerosión</a>. El material es blando y las diferentes capas, a menudo separadas por átomos intercalados, se encuentran unidas por <a class="mw-redirect" href="/wiki/Fuerzas_de_van_der_Waals" title="Fuerzas de van der Waals">enlaces de Van de Waals</a>, siendo relativamente fácil que unas deslicen respecto de otras, lo que le da utilidad como <a href="/wiki/Lubricante" title="Lubricante">lubricante</a>.
</p>
<p>A muy altas presiones, el carbono adopta la forma del <a href="/wiki/Diamante" title="Diamante">diamante</a>, en el cual cada átomo está unido a otros cuatro átomos de carbono, encontrándose los 4 electrones en orbitales sp³, como en los hidrocarburos. El diamante presenta la misma <a href="/wiki/Redes_de_Bravais" title="Redes de Bravais">estructura</a> cúbica que el <a href="/wiki/Silicio" title="Silicio">silicio</a> y el <a href="/wiki/Germanio" title="Germanio">germanio</a> y, gracias a la resistencia del <a class="mw-redirect" href="/wiki/Enlace_qu%C3%ADmico" title="Enlace químico">enlace químico</a> carbono-carbono, es, junto con el <a href="/wiki/Nitruro_de_boro" title="Nitruro de boro">nitruro de boro</a>, la sustancia más <a href="/wiki/Dureza" title="Dureza">dura</a> conocida. La transición a grafito a temperatura ambiente es tan lenta que es indetectable. Bajo ciertas condiciones, el carbono cristaliza como <a href="/wiki/Lonsdale%C3%ADta" title="Lonsdaleíta">lonsdaleíta</a>, una forma similar al diamante pero hexagonal.
</p>
<p>El orbital híbrido sp<sup>1</sup> que forma <a href="/wiki/Enlace_covalente" title="Enlace covalente">enlaces covalentes</a> solo es de interés en <a href="/wiki/Qu%C3%ADmica" title="Química">química</a>, manifestándose en algunos compuestos, como por ejemplo el <a href="/wiki/Acetileno" title="Acetileno">acetileno</a>.
</p>
<p>Los <a href="/wiki/Fullereno" title="Fullereno">fullerenos</a> fueron descubiertos hace 15 años<sup class="reference separada" id="cite_ref-Gasque_1_10-1"><a href="#cite_note-Gasque_1-10"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​ tienen una estructura similar al grafito, pero el empaquetamiento hexagonal se combina con pentágonos (y en ciertos casos, heptágonos), lo que curva los planos y permite la aparición de estructuras de forma <a href="/wiki/Esfera" title="Esfera">esférica</a>, <a href="/wiki/Elipsoide" title="Elipsoide">elipsoidal</a> o <a class="mw-redirect" href="/wiki/Cilindro_(geometr%C3%ADa)" title="Cilindro (geometría)">cilíndrica</a>. El constituido por 60 átomos de carbono, que presenta una estructura tridimensional y geometría similar a un balón de fútbol, es especialmente estable. Los fullerenos en general, y los derivados del C<sub>60</sub> en particular, son objeto de intensa investigación en química desde su descubrimiento a mediados de los 1980.
</p>
<p>A esta familia pertenecen también los <a href="/wiki/Nanotubo" title="Nanotubo">nanotubos</a> de carbono, que pueden describirse como capas de grafito enrolladas en forma cilíndrica y rematadas en sus extremos por hemiesferas (fulerenos), y que constituyen uno de los primeros productos industriales de la <a href="/wiki/Nanotecnolog%C3%ADa" title="Nanotecnología">nanotecnología</a>.
</p>
<p>El principal uso industrial del carbono es como un componente de hidrocarburos, especialmente los combustibles fósiles (<a href="/wiki/Petr%C3%B3leo" title="Petróleo">petróleo</a> y <a href="/wiki/Gas_natural" title="Gas natural">gas natural</a>). Del primero se obtienen, por <a href="/wiki/Destilaci%C3%B3n" title="Destilación">destilación</a> en las <a href="/wiki/Refiner%C3%ADa" title="Refinería">refinerías</a>, <a href="/wiki/Gasolina" title="Gasolina">gasolinas</a>, <a href="/wiki/Queroseno" title="Queroseno">queroseno</a> y <a href="/wiki/Aceite" title="Aceite">aceites</a>, siendo además la materia prima empleada en la obtención de <a href="/wiki/Pl%C3%A1stico" title="Plástico">plásticos</a>. El segundo se está imponiendo como <a href="/wiki/Fuente_de_energ%C3%ADa" title="Fuente de energía">fuente de energía</a> por su combustión más limpia. Otros usos son:
</p>
<p>El carbono fue descubierto en la prehistoria y ya era conocido en la antigüedad, a pesar de que en esta la manufacturaban mediante la <a class="mw-redirect" href="/wiki/Combusti%C3%B3n_incompleta" title="Combustión incompleta">combustión incompleta</a> de materiales orgánicos. Los últimos alótropos conocidos, los fullerenos (C<sub>60</sub>), fueron descubiertos como subproducto en experimentos realizados con gases moleculares en la década de <a href="/wiki/A%C3%B1os_1980" title="Años 1980">los 80</a>. Se asemejan a un balón de fútbol, por lo que coloquialmente se les llama futbolenos.
</p>
<p><a href="/wiki/Isaac_Newton" title="Isaac Newton">Newton</a>, en 1704, intuyó que el diamante podía ser combustible, pero no se consiguió quemar un diamante hasta 1772 en que <a href="/wiki/Antoine_Lavoisier" title="Antoine Lavoisier">Lavoisier</a> demostró que en la reacción de combustión se producía CO<sub>2</sub>.
</p>
<p><a href="/wiki/Smithson_Tennant" title="Smithson Tennant">Tennant</a> demostró que el diamante era carbono puro en 1797. El isótopo más común del carbono es el <sup>12</sup>C; en 1961 este isótopo se eligió para reemplazar al isótopo oxígeno-16 como base de los pesos atómicos, y se le asignó un peso atómico de 12.
</p>
<p>Los primeros compuestos de carbono se identificaron en la materia viva a principios del siglo <span style="font-variant:small-caps;text-transform:lowercase">XIX</span>, y por ello el estudio de los compuestos de carbono se llamó química orgánica.
</p>
<p>El carbono no se creó durante el <i><a href="/wiki/Big_Bang" title="Big Bang">Big Bang</a></i> porque hubiera necesitado la <a class="mw-redirect" href="/wiki/Proceso_triple_alfa" title="Proceso triple alfa">triple colisión</a> de <a href="/wiki/Part%C3%ADcula_alfa" title="Partícula alfa">partículas alfa</a> (<a href="/wiki/N%C3%BAcleo_at%C3%B3mico" title="Núcleo atómico">núcleos atómicos</a> de <a href="/wiki/Helio" title="Helio">helio</a>) y el Universo se expandió y enfrió demasiado rápido para que la probabilidad de que ello aconteciera fuera significativa. Donde sí ocurre este proceso es en el interior de las <a href="/wiki/Estrella" title="Estrella">estrellas</a> (en la fase <a class="mw-redirect" href="/wiki/Diagrama_HR" title="Diagrama HR">RH (Rama horizontal)</a>) donde este elemento es abundante, encontrándose además en otros cuerpos celestes como los cometas y en las atmósferas de los planetas. Algunos <a href="/wiki/Meteorito" title="Meteorito">meteoritos</a> contienen diamantes microscópicos que se formaron cuando el <a class="mw-redirect" href="/wiki/Sistema_Solar" title="Sistema Solar">Sistema Solar</a> era aún un <a href="/wiki/Disco_protoplanetario" title="Disco protoplanetario">disco protoplanetario</a>.
</p>
<p>En combinaciones con otros elementos, el carbono se encuentra en la atmósfera terrestre y disuelto en el agua, y acompañado de menores cantidades de <a href="/wiki/Calcio" title="Calcio">calcio</a>, <a href="/wiki/Magnesio" title="Magnesio">magnesio</a> y <a href="/wiki/Hierro" title="Hierro">hierro</a> forma enormes masas <a href="/wiki/Roca" title="Roca">rocosas</a> (<a href="/wiki/Caliza" title="Caliza">caliza</a>, <a href="/wiki/Dolomita" title="Dolomita">dolomita</a>, <a href="/wiki/M%C3%A1rmol" title="Mármol">mármol</a>, etc).
</p>
<p>El grafito se encuentra en grandes cantidades en <a href="/wiki/Rusia" title="Rusia">Rusia</a>, <a href="/wiki/Estados_Unidos" title="Estados Unidos">Estados Unidos</a>, <a href="/wiki/M%C3%A9xico" title="México">México</a>, <a href="/wiki/Groenlandia" title="Groenlandia">Groenlandia</a> y la <a href="/wiki/India" title="India">India</a>.
</p>
<p>Los diamantes naturales se encuentran asociados a <a href="/wiki/Roca_volc%C3%A1nica" title="Roca volcánica">rocas volcánicas</a> (<a href="/wiki/Kimberlita" title="Kimberlita">kimberlita</a> y <a class="mw-redirect" href="/wiki/Lamproita" title="Lamproita">lamproita</a>). Los mayores depósitos de diamantes se encuentran en el <a href="/wiki/%C3%81frica" title="África">África</a> (<a href="/wiki/Sud%C3%A1frica" title="Sudáfrica">Sudáfrica</a>, <a href="/wiki/Namibia" title="Namibia">Namibia</a>, <a href="/wiki/Botsuana" title="Botsuana">Botsuana</a>, <a href="/wiki/Rep%C3%BAblica_del_Congo" title="República del Congo">República del Congo</a> y <a href="/wiki/Sierra_Leona" title="Sierra Leona">Sierra Leona</a>).<sup class="reference separada" id="cite_ref-11"><a href="#cite_note-11"><span class="corchete-llamada">[</span>11<span class="corchete-llamada">]</span></a></sup>​ Existen además depósitos importantes en <a href="/wiki/Canad%C3%A1" title="Canadá">Canadá</a>, <a href="/wiki/Rusia" title="Rusia">Rusia</a>, <a href="/wiki/Brasil" title="Brasil">Brasil</a> y <a href="/wiki/Australia" title="Australia">Australia</a>.<sup>[<i><a href="/wiki/Wikipedia:Verificabilidad" title="Wikipedia:Verificabilidad">cita requerida</a></i>]</sup>
</p>
<p>El más importante <a href="/wiki/%C3%93xido" title="Óxido">óxido</a> de carbono es el <a href="/wiki/Di%C3%B3xido_de_carbono" title="Dióxido de carbono">dióxido de carbono</a> (CO<sub>2</sub>), un componente minoritario de la atmósfera terrestre (del orden del 0,04 % en peso) producido y usado por los seres vivos (ver <a href="/wiki/Ciclo_del_carbono" title="Ciclo del carbono">ciclo del carbono</a>). En el agua forma trazas de <a href="/wiki/%C3%81cido_carb%C3%B3nico" title="Ácido carbónico">ácido carbónico</a> (H<sub>2</sub>CO<sub>3</sub>) —las burbujas de muchos refrescos— pero, al igual que otros compuestos similares, es inestable, aunque a través de él pueden producirse <a href="/wiki/Ion" title="Ion">iones</a> carbonato estables por <a href="/wiki/Resonancia_(qu%C3%ADmica)" title="Resonancia (química)">resonancia</a>. Algunos <a href="/wiki/Mineral" title="Mineral">minerales</a> importantes, como la <a href="/wiki/Calcita" title="Calcita">calcita</a>, son carbonatos.
</p>
<p>Los otros óxidos son el <a href="/wiki/Mon%C3%B3xido_de_carbono" title="Monóxido de carbono">monóxido de carbono</a> (CO) y el más raro <a class="mw-redirect" href="/wiki/Sub%C3%B3xido_de_carbono" title="Subóxido de carbono">subóxido de carbono</a> (C<sub>3</sub>O<sub>2</sub>). El monóxido se forma durante la combustión incompleta de materias orgánicas y es incoloro e inodoro. Dado que la molécula de CO contiene un enlace triple, es muy polar, por lo que manifiesta una acusada tendencia a unirse a la <a href="/wiki/Hemoglobina" title="Hemoglobina">hemoglobina</a>, formando un nuevo compuesto muy peligroso denominado Carboxihemoglobina, impidiéndoselo al oxígeno, por lo que se dice que es un <a class="new" href="/w/index.php?title=Asfixiante&amp;action=edit&amp;redlink=1" title="Asfixiante (aún no redactado)">asfixiante</a> de sustitución. El ion <a href="/wiki/Cianuro" title="Cianuro">cianuro</a> (CN<sup>−</sup>), tiene una estructura similar y se comporta como los iones <a class="mw-redirect" href="/wiki/Haluro" title="Haluro">haluro</a>.
</p>
<p>Con <a href="/wiki/Metal" title="Metal">metales</a>, el carbono forma tanto <a href="/wiki/Carburo" title="Carburo">carburos</a> como <a href="/wiki/Acetiluro" title="Acetiluro">acetiluros</a>, ambos muy <a href="/wiki/%C3%81cido" title="Ácido">ácidos</a>. A pesar de tener una <a href="/wiki/Electronegatividad" title="Electronegatividad">electronegatividad</a> alta, el carbono puede formar carburos covalentes como es el caso de <a href="/wiki/Carburo_de_silicio" title="Carburo de silicio">carburo de silicio</a> (SiC) cuyas propiedades se asemejan a las del <a href="/wiki/Diamante" title="Diamante">diamante</a>.
</p>
<p>En <a href="/wiki/1961" title="1961">1961</a> la <a class="mw-redirect" href="/wiki/IUPAC" title="IUPAC">IUPAC</a> adoptó el <a href="/wiki/Is%C3%B3topo" title="Isótopo">isótopo</a> <a href="/wiki/Carbono-12" title="Carbono-12"><sup>12</sup>C</a> como la base para la <a href="/wiki/Masa_at%C3%B3mica" title="Masa atómica">masa atómica</a> de los elementos químicos.
</p>
<p>El <a href="/wiki/Carbono-14" title="Carbono-14">carbono-14</a> es un <a href="/wiki/Radiois%C3%B3topo" title="Radioisótopo">radioisótopo</a> con un <a href="/wiki/Periodo_de_semidesintegraci%C3%B3n" title="Periodo de semidesintegración">periodo de semidesintegración</a> de 5730 años que se emplea de forma extensiva en la datación de especímenes orgánicos.
</p>
<p>Los isótopos naturales y estables del carbono son el <sup>12</sup>C (98,89 %) y el <a href="/wiki/Carbono-13" title="Carbono-13"><sup>13</sup>C</a> (1,11 %). Las proporciones de estos isótopos en un ser vivo  se expresan en variación (±‰) respecto de la referencia VPDB (<i>Vienna Pee Dee Belemnite</i>, fósiles <a class="mw-redirect" href="/wiki/Cret%C3%A1ceo" title="Cretáceo">cretácicos</a> de <a class="mw-redirect" href="/wiki/Belemnites" title="Belemnites">belemnites</a>, en <a href="/wiki/Carolina_del_Sur" title="Carolina del Sur">Carolina del Sur</a>). El δC-13 del CO<sub>2</sub> de la <a href="/wiki/Atm%C3%B3sfera_terrestre" title="Atmósfera terrestre">atmósfera terrestre</a> es −7‰. El carbono fijado por <a href="/wiki/Fotos%C3%ADntesis" title="Fotosíntesis">fotosíntesis</a> en los tejidos de las <a href="/wiki/Vegetal" title="Vegetal">plantas</a> es significativamente más pobre en <sup>13</sup>C que el CO<sub>2</sub> de la atmósfera.
</p>
<p>La mayoría de las plantas presentan valores de δC-13 entre −24 y −34‰. Otras plantas acuáticas, de desierto, de marismas saladas y hierbas tropicales, presentan valores de δC-13 entre −6 y −19‰ debido a diferencias en la reacción de fotosíntesis. Un tercer grupo intermedio constituido por las <a href="/wiki/Alga" title="Alga">algas</a> y <a href="/wiki/Liquen" title="Liquen">líquenes</a> presentan valores entre −12 y −23‰. El estudio comparativo de los valores de δC-13 en plantas y organismos puede proporcionar información valiosa relativa a la cadena alimenticia de los seres vivos.
</p>
<p>Los compuestos de carbono tienen un amplio rango de <a class="mw-redirect" href="/wiki/T%C3%B3xico" title="Tóxico">toxicidad</a>. El <a href="/wiki/Mon%C3%B3xido_de_carbono" title="Monóxido de carbono">monóxido de carbono</a>, presente en los gases de escape de los motores de combustión y el <a href="/wiki/Cianuro" title="Cianuro">cianuro</a> (CN) son <b>extremadamente tóxicos</b> para los mamíferos, entre ellos las personas. Los <a href="/wiki/Gas" title="Gas">gases</a> orgánicos <a class="mw-redirect" href="/wiki/Eteno" title="Eteno">eteno</a>, <a class="mw-redirect" href="/wiki/Etino" title="Etino">etino</a> y <a href="/wiki/Metano" title="Metano">metano</a> son <b>explosivos e inflamables</b> en presencia de aire. Por el contrario, muchos otros compuestos no son tóxicos sino esenciales para la vida.
</p>
<p>El carbono puro tiene una <a href="/wiki/Toxicidad" title="Toxicidad">toxicidad</a> extremadamente baja para los humanos y puede ser manejado e incluso ingerido en forma segura en la forma de <a href="/wiki/Grafito" title="Grafito">grafito</a> o <a class="mw-redirect" href="/wiki/Carboncillo_(dibujo)" title="Carboncillo (dibujo)">carboncillo</a>. Es resistente a la disolución y ataque químico, incluso en los contenidos acidificados del <a class="mw-redirect" href="/wiki/Tracto_digestivo" title="Tracto digestivo">tracto digestivo</a>. Esto resulta en que una vez que entra a los tejidos corporales lo más probable es que permanezcan allí en forma indefinida. El <a href="/wiki/Negro_de_carb%C3%B3n" title="Negro de carbón">negro de carbón</a> fue probablemente el primer <a href="/wiki/Pigmento" title="Pigmento">pigmento</a> en ser usado para hacer <a href="/wiki/Tatuaje" title="Tatuaje">tatuajes</a> y se encontró que <a href="/wiki/%C3%96tzi" title="Ötzi">Ötzi el hombre del hielo</a> tenía tatuajes hechos con carbón que sobrevivieron durante su vida y 5200 años después de su muerte.<sup class="reference separada" id="cite_ref-12"><a href="#cite_note-12"><span class="corchete-llamada">[</span>12<span class="corchete-llamada">]</span></a></sup>​ Sin embargo, la inhalación en grandes cantidades del polvo de carbón u hollín (negro de carbón) puede ser peligroso, al irritar los tejidos del <a class="mw-redirect" href="/wiki/Pulmones" title="Pulmones">pulmón</a> y causar una enfermedad conocida como <a href="/wiki/Neumoconiosis_de_los_mineros_del_carb%C3%B3n" title="Neumoconiosis de los mineros del carbón">neumoconiosis de los mineros del carbón</a>. De forma similar el polvo de diamante usado como un abrasivo puede ser dañino si se ingiere o inhala. También las micropartículas de carbón producidas por los gases de escape de los <a href="/wiki/Motor_di%C3%A9sel" title="Motor diésel">motores diésel</a> se pueden acumular en los pulmones al ser inhaladas.<sup class="reference separada" id="cite_ref-13"><a href="#cite_note-13"><span class="corchete-llamada">[</span>13<span class="corchete-llamada">]</span></a></sup>​ En estos ejemplos, los efectos dañinos pueden resultar de la contaminación de las partículas de carbón con elementos químicos orgánicos o de <a href="/wiki/Metal_pesado" title="Metal pesado">metales pesados</a> más que del carbón en sí mismo.
</p>
<p>Generalmente el carbono tiene baja toxicidad para casi toda la <a href="/wiki/Ser_vivo" title="Ser vivo">vida en la Tierra</a>, sin embargo, para algunas criaturas es tóxico - por ejemplo, las <a href="/wiki/Nanopart%C3%ADcula" title="Nanopartícula">nanopartículas</a> de carbón son <a href="/wiki/Toxina" title="Toxina">toxinas</a> mortales para la <i><a href="/wiki/Drosophila" title="Drosophila">Drosophila</a></i>.<sup class="reference separada" id="cite_ref-14"><a href="#cite_note-14"><span class="corchete-llamada">[</span>14<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>También el carbono se puede quemar vigorosa y brillantemente en la presencia de aire a alta temperatura, como en el caso del <a href="/wiki/Incendio_de_Windscale" title="Incendio de Windscale">Incendio de Windscale</a>, el que fue causado por la repentina liberación de <a class="new" href="/w/index.php?title=Efecto_Wigner&amp;action=edit&amp;redlink=1" title="Efecto Wigner (aún no redactado)">energía Wigner</a> acumulada en el núcleo de grafito. Grandes acumulaciones de carbón, que han permanecido inertes por centenares de millones de años en la ausencia de <a href="/wiki/Ox%C3%ADgeno" title="Oxígeno">oxígeno</a>, pueden <a href="/wiki/Combusti%C3%B3n" title="Combustión">incendiarse espontáneamente</a> cuando son expuestas al aire, como por ejemplo en los desechos de las <a href="/wiki/Miner%C3%ADa" title="Minería">minas de carbón</a>.
</p>
<p>Entre la gran variedad de compuestos de carbono se pueden incluir <a href="/wiki/Veneno" title="Veneno">venenos</a> letales tales como la <a class="mw-redirect" href="/wiki/Tetradotoxina" title="Tetradotoxina">tetradotoxina</a>, la <a href="/wiki/Ricina" title="Ricina">ricina</a> <a href="/wiki/Lectina" title="Lectina">lectina</a> obtenida de las semillas de <a class="mw-redirect" href="/wiki/Ricino" title="Ricino">ricino</a> (<i>Ricinus communis</i>), el <a href="/wiki/Cianuro" title="Cianuro">cianuro</a> (CN<sup>−</sup>) y el <a href="/wiki/Intoxicaci%C3%B3n_por_mon%C3%B3xido_de_carbono" title="Intoxicación por monóxido de carbono">envenenamiento por monóxido de carbono</a>.
</p>
