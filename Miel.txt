<p>La <b>miel</b> es un fluido muy <a href="/wiki/Sabor_dulce" title="Sabor dulce">dulce</a> y <a class="mw-redirect" href="/wiki/Viscoso" title="Viscoso">viscoso</a> producido por <a href="/wiki/Anthophila" title="Anthophila">abejas</a> del género <i><a href="/wiki/Apis_(g%C3%A9nero)" title="Apis (género)">Apis</a></i>, principalmente la <a href="/wiki/Apis_mellifera" title="Apis mellifera">abeja doméstica</a>, a partir del <a href="/wiki/N%C3%A9ctar_(bot%C3%A1nica)" title="Néctar (botánica)">néctar</a> de las <a href="/wiki/Flor" title="Flor">flores</a> o de secreciones de partes vivas de <a class="mw-redirect" href="/wiki/Planta" title="Planta">plantas</a> o de excreciones de <a class="mw-redirect" href="/wiki/Insecto" title="Insecto">insectos</a> chupadores de plantas (<a href="/wiki/Aphidoidea" title="Aphidoidea">áfidos</a>). Estas sustancias son recogidas por las abejas, luego transformadas al combinarlas con sustancias propias, depositadas, deshidratadas y almacenadas en los <a class="mw-redirect" href="/wiki/Panales_de_cera" title="Panales de cera">panales</a> para su maduración.<sup class="reference separada" id="cite_ref-Soares2017_1-0"><a href="#cite_note-Soares2017-1"><span class="corchete-llamada">[</span>1<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-Miguel2017_2-0"><a href="#cite_note-Miguel2017-2"><span class="corchete-llamada">[</span>2<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-Samarghandian2017_3-0"><a href="#cite_note-Samarghandian2017-3"><span class="corchete-llamada">[</span>3<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-Ajibola2015_4-0"><a href="#cite_note-Ajibola2015-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​ La intervención del hombre en el proceso de explotación de los panales de la <a href="/wiki/Colmena" title="Colmena">colmena</a> es conocida como <a href="/wiki/Apicultura" title="Apicultura">apicultura</a>.<sup class="reference separada" id="cite_ref-Pasupuleti2017_5-0"><a href="#cite_note-Pasupuleti2017-5"><span class="corchete-llamada">[</span>5<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La miel es utilizada desde hace miles de años en todo el mundo, ya sea como saborizante de brebajes, <a href="/wiki/Alimento" title="Alimento">alimento</a> o <a href="/wiki/Medicamento" title="Medicamento">medicamento</a>. Su composición es variable sin embargo su principal componente, son <a class="mw-redirect" href="/wiki/Carbohidrato" title="Carbohidrato">carbohidratos</a> en la forma de <a href="/wiki/Monosac%C3%A1rido" title="Monosacárido">monosacáridos</a> como la <a href="/wiki/Fructosa" title="Fructosa">fructosa</a> y la <a href="/wiki/Glucosa" title="Glucosa">glucosa</a>, así como <a href="/wiki/Disac%C3%A1rido" title="Disacárido">disacáridos</a> tales como <a href="/wiki/Maltosa" title="Maltosa">maltosa</a>, <a href="/wiki/Isomaltosa" title="Isomaltosa">isomaltosa</a>, <a class="new" href="/w/index.php?title=Maltulosa&amp;action=edit&amp;redlink=1" title="Maltulosa (aún no redactado)">maltulosa</a>, <a class="mw-redirect" href="/wiki/Sucrosa" title="Sucrosa">sucrosa</a>, <a href="/wiki/Turanosa" title="Turanosa">turanosa</a> y <a href="/wiki/Nigerosa" title="Nigerosa">nigerosa</a>. Estos ingredientes son los responsables del intenso <a href="/wiki/Sabor_dulce" title="Sabor dulce">dulzor</a> de la miel. Además, contiene <a href="/wiki/Oligosac%C3%A1rido" title="Oligosacárido">oligosacáridos</a> como la <a href="/wiki/Panosa" title="Panosa">panosa</a>; <a href="/wiki/Enzima" title="Enzima">enzimas</a> como la <a href="/wiki/Amilasa" title="Amilasa">amilasa</a>, <a href="/wiki/Peroxidasa" title="Peroxidasa">peróxido oxidasas</a>, <a href="/wiki/Catalasa" title="Catalasa">catalasa</a> y <a href="/wiki/Fosforilasa" title="Fosforilasa">fosforilasa ácida</a>; además contiene <a href="/wiki/Amino%C3%A1cido" title="Aminoácido">aminoácidos</a>, algunas <a class="mw-redirect" href="/wiki/Vitamina_B" title="Vitamina B">vitaminas B</a>, <a href="/wiki/Vitamina_C" title="Vitamina C">C</a>, <a class="mw-redirect" href="/wiki/Niacina" title="Niacina">niacina</a>, <a href="/wiki/%C3%81cido_f%C3%B3lico" title="Ácido fólico">ácido fólico</a>, minerales como <a href="/wiki/Hierro" title="Hierro">hierro</a> y <a href="/wiki/Zinc" title="Zinc">zinc</a>, y <a href="/wiki/Antioxidante" title="Antioxidante">antioxidantes</a>.<sup class="reference separada" id="cite_ref-Meo2017_6-0"><a href="#cite_note-Meo2017-6"><span class="corchete-llamada">[</span>6<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-Al-Waili2012_7-0"><a href="#cite_note-Al-Waili2012-7"><span class="corchete-llamada">[</span>7<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La producción mundial media de miel es aproximadamente de 1 200 000 <a href="/wiki/T" title="T">t</a>. Los principales países productores son <a class="mw-redirect" href="/wiki/China" title="China">China</a>, <a href="/wiki/Turqu%C3%ADa" title="Turquía">Turquía</a>, <a href="/wiki/Argentina" title="Argentina">Argentina</a>, <a href="/wiki/Ucrania" title="Ucrania">Ucrania</a>, <a href="/wiki/M%C3%A9xico" title="México">México</a> y <a href="/wiki/Estados_Unidos" title="Estados Unidos">Estados Unidos</a>. Los principales <a class="mw-redirect" href="/wiki/Importador" title="Importador">importadores</a> a nivel mundial son la <a href="/wiki/Uni%C3%B3n_Europea" title="Unión Europea">Unión Europea</a> y los <a href="/wiki/Estados_Unidos" title="Estados Unidos">Estados Unidos</a>.<sup class="reference separada" id="cite_ref-Meo2017_6-1"><a href="#cite_note-Meo2017-6"><span class="corchete-llamada">[</span>6<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La miel es producida por las abejas que recolectan <a href="/wiki/N%C3%A9ctar_(bot%C3%A1nica)" title="Néctar (botánica)">néctar</a> y <a href="/wiki/Roc%C3%ADo_de_miel" title="Rocío de miel">rocío de miel</a> para su uso como azúcares, consumidos para apoyar el <a href="/wiki/Metabolismo" title="Metabolismo">metabolismo</a> de la actividad muscular durante la búsqueda de alimento, o almacenada a largo plazo.<sup class="reference separada" id="cite_ref-pmid8901631_8-0"><a href="#cite_note-pmid8901631-8"><span class="corchete-llamada">[</span>8<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-NHB_Bee_facts_9-0"><a href="#cite_note-NHB_Bee_facts-9"><span class="corchete-llamada">[</span>9<span class="corchete-llamada">]</span></a></sup>​como suministros, Las abejas acceden a parte del néctar recolectado para apoyar la actividad de los músculos del vuelo, destina a la regurgitación, digestión .<sup class="reference separada" id="cite_ref-pmid8901631_8-1"><a href="#cite_note-pmid8901631-8"><span class="corchete-llamada">[</span>8<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-cd_10-0"><a href="#cite_note-cd-10"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​ En climas fríos o cuando escasean otras fuentes de alimentos, las abejas adultas y larvales consumen la miel almacenada.<sup class="reference separada" id="cite_ref-NHB_Bee_facts_9-1"><a href="#cite_note-NHB_Bee_facts-9"><span class="corchete-llamada">[</span>9<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Al idear enjambres de <a class="mw-redirect" href="/wiki/Abejas_mel%C3%ADferas" title="Abejas melíferas">abejas melíferas</a> para anidar en colmenas artificiales, las personas han podido semidomesticar los insectos y cosechar el exceso de miel. En la colmena o en un nido salvaje, los tres tipos de abejas son:
</p>
<p>Al salir de la colmena, una abeja recolecta néctar de flores rico en azúcar, lo succiona a través de su <a href="/wiki/Prob%C3%B3scide" title="Probóscide">probóscide</a> y lo coloca en su <a href="/wiki/Proventr%C3%ADculo" title="Proventrículo">proventrículo</a>, que se encuentra justo dorsal a su estómago de alimento. En las <i><a href="/wiki/Apis_mellifera" title="Apis mellifera">apis mellifera</a></i>, el estómago de miel contiene alrededor de <span style="white-space:nowrap">40 mg</span> de néctar, o aproximadamente el 50% del peso descargado de la abeja, que puede requerir más de mil flores y más de una hora para llenarse. El néctar generalmente presenta un contenido de agua del 70 al 80 %.<sup class="reference separada" id="cite_ref-12"><a href="#cite_note-12"><span class="corchete-llamada">[</span>12<span class="corchete-llamada">]</span></a></sup>​ Las enzimas salivales y las proteínas de la <a href="/wiki/Piezas_bucales_de_los_insectos#Hipofaringe" title="Piezas bucales de los insectos">glándula hipofaríngea</a> de la abeja se agregan al néctar para comenzar a descomponer los azúcares, aumentando ligeramente el contenido de agua. Las abejas recolectoras luego regresan a la colmena, donde regurgitan y transfieren el néctar a las abejas de la colmena. Las abejas de la colmena luego usan sus estómagos de miel para ingerir y regurgitar el néctar, formando burbujas entre sus mandíbulas repetidamente hasta que se digiere parcialmente. Las burbujas crean una gran superficie por volumen y una parte del agua se elimina por evaporación.<sup class="reference separada" id="cite_ref-pmid8901631_8-2"><a href="#cite_note-pmid8901631-8"><span class="corchete-llamada">[</span>8<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-cd_10-1"><a href="#cite_note-cd-10"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-13"><a href="#cite_note-13"><span class="corchete-llamada">[</span>13<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-14"><a href="#cite_note-14"><span class="corchete-llamada">[</span>14<span class="corchete-llamada">]</span></a></sup>​ Las <a href="/wiki/Enzima_digestiva" title="Enzima digestiva">enzimas digestivas</a> de la abeja <a href="/wiki/Hidr%C3%B3lisis" title="Hidrólisis">hidrolizan</a>, convierten la sacarosa en una mezcla de glucosa y fructosa y descomponen otros almidones y proteínas, aumentando la acidez.<sup class="reference separada" id="cite_ref-pmid8901631_8-3"><a href="#cite_note-pmid8901631-8"><span class="corchete-llamada">[</span>8<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-cd_10-2"><a href="#cite_note-cd-10"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-15"><a href="#cite_note-15"><span class="corchete-llamada">[</span>15<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las abejas trabajan juntas como un grupo con la regurgitación y la digestión durante 20 minutos, pasando el néctar de una abeja a otra, hasta que el producto llega a los panales en calidad de almacenamiento.<sup class="reference separada" id="cite_ref-cd_10-3"><a href="#cite_note-cd-10"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​ Luego se coloca en celdas de panal y se deja sin sellar mientras todavía tiene un alto contenido de agua (alrededor de 50 a 70 %) y levaduras naturales que, sin control, harían que los azúcares en la miel recién formada fermenten.<sup class="reference separada" id="cite_ref-NHB_Bee_facts_9-2"><a href="#cite_note-NHB_Bee_facts-9"><span class="corchete-llamada">[</span>9<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-16"><a href="#cite_note-16"><span class="corchete-llamada">[</span>16<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-17"><a href="#cite_note-17"><span class="corchete-llamada">[</span>17<span class="corchete-llamada">]</span></a></sup>​ Las abejas se encuentran entre los pocos insectos que pueden generar grandes cantidades de calor corporal, y las de la colmena regulan constantemente la temperatura de esta, ya sea calentando con sus cuerpos o enfriando con la evaporación del agua, para mantener una temperatura bastante constante de aproximadamente <span style="white-space:nowrap">35 °C</span> en las áreas de almacenamiento de miel. El proceso continúa mientras las abejas de la colmena aletean constantemente para hacer circular el aire y evaporar el agua de la miel a un contenido de alrededor del 18 %, elevando la concentración de azúcar más allá del <a href="/wiki/Solubilidad" title="Solubilidad">punto de saturación</a> y evitando la <a href="/wiki/Fermentaci%C3%B3n" title="Fermentación">fermentación</a>.<sup class="reference separada" id="cite_ref-NHB_Bee_facts_9-3"><a href="#cite_note-NHB_Bee_facts-9"><span class="corchete-llamada">[</span>9<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-cd_10-4"><a href="#cite_note-cd-10"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​ Luego, las abejas tapan las celdas con cera para sellarlas.<sup class="reference separada" id="cite_ref-NHB_Bee_facts_9-4"><a href="#cite_note-NHB_Bee_facts-9"><span class="corchete-llamada">[</span>9<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-cd_10-5"><a href="#cite_note-cd-10"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​ Tal como la saca de la colmena un apicultor, la miel tiene una vida útil prolongada y no fermentará si se sella correctamente.<sup class="reference separada" id="cite_ref-NHB_Bee_facts_9-5"><a href="#cite_note-NHB_Bee_facts-9"><span class="corchete-llamada">[</span>9<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La miel tiene cualidades reconocidas por los seres humanos desde tiempos remotos, como alimento y para endulzar, con un poder mayor a la caña de azúcar.<sup class="reference separada" id="cite_ref-Meo2017_6-2"><a href="#cite_note-Meo2017-6"><span class="corchete-llamada">[</span>6<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-Al-Waili2012_7-1"><a href="#cite_note-Al-Waili2012-7"><span class="corchete-llamada">[</span>7<span class="corchete-llamada">]</span></a></sup>​
Las primeras evidencias de su consumo aparecen en <a class="mw-redirect" href="/wiki/Pinturas_rupestres" title="Pinturas rupestres">pinturas rupestres</a> del <a href="/wiki/Mesol%C3%ADtico" title="Mesolítico">Mesolítico</a>, hacia el <a href="/wiki/VI_milenio_a._C." title="VI milenio a. C.">6000 a. C.</a>. 
</p>
<p>Existen registros prehistóricos en pinturas rupestres de la utilización de la miel.<sup class="reference separada" id="cite_ref-19"><a href="#cite_note-19"><span class="corchete-llamada">[</span>19<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Existen diversas referencias históricas a esta sustancia. Hacia el <a href="/wiki/Siglo_XXVI_a._C." title="Siglo XXVI a. C.">2500 a. C.</a>, los <a class="mw-redirect" href="/wiki/Sumerio" title="Sumerio">sumerios</a> ya conocían sus propiedades medicinales.<sup class="reference separada" id="cite_ref-Scielo_20-0"><a href="#cite_note-Scielo-20"><span class="corchete-llamada">[</span>20<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Además de las citas bíblicas (compuestas en los últimos siglos del I milenio a. C.), muchos otros pueblos, como los antiguos egipcios o los griegos, por ejemplo, se referían a la miel como un producto sagrado, llegando a servir como forma de pagar los impuestos.
</p>
<p>En excavaciones egipcias con más de 2000 años fueron encontradas muestras de miel perfectamente conservadas en vasijas ligeramente tapadas que aún eran comestibles y solamente tenían que calentarse.
</p>
<p>Existen cerca de 320 variedades de miel de distintos orígenes florales. El sabor, color y olor de la miel depende de las distintas fuentes de flores y plantas visitadas por las abejas.<sup class="reference separada" id="cite_ref-Meo2017_6-3"><a href="#cite_note-Meo2017-6"><span class="corchete-llamada">[</span>6<span class="corchete-llamada">]</span></a></sup>​

<p>La miel de flores es transparente y se solidifica con el tiempo dependiendo de su procedencia vegetal y de la temperatura. Por debajo de 14 °C se acelera el proceso de solidificación. Las mieles de brezo se endurecen muy pronto y las de castaño tardan mucho.
</p>
<p>El estudio del <a href="/wiki/Polen" title="Polen">polen</a> en la miel virgen (<a href="/wiki/Melisopalinolog%C3%ADa" title="Melisopalinología">melisopalinología</a>) permite determinar su origen floral. Dado que las partículas de polen están electrostáticamente cargadas y atraen otras partículas, las técnicas usadas en la melisopalinología pueden usarse en estudios medioambientales de partículas <a href="/wiki/Radiactividad" title="Radiactividad">radiactivas</a>, <a href="/wiki/Polvo" title="Polvo">polvo</a> o <a href="/wiki/Contaminaci%C3%B3n" title="Contaminación">contaminación</a>.
</p>
<p>La miel contiene:<sup class="reference separada" id="cite_ref-Pasupuleti2017_5-1"><a href="#cite_note-Pasupuleti2017-5"><span class="corchete-llamada">[</span>5<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La falta de humedad es una condición fundamental para la conservación de la miel. Mientras el porcentaje de humedad permanezca por debajo de 18 % nada podrá crecer en ella. Por encima de ese valor pueden aparecer procesos fermentativos.
</p>
<p>El contenido en minerales de la miel es muy bajo. Los más frecuentes son <a href="/wiki/Calcio" title="Calcio">calcio</a>, <a href="/wiki/Cobre" title="Cobre">cobre</a>, <a href="/wiki/Hierro" title="Hierro">hierro</a>, <a href="/wiki/Magnesio" title="Magnesio">magnesio</a>, <a href="/wiki/Manganeso" title="Manganeso">manganeso</a>, <a href="/wiki/Zinc" title="Zinc">zinc</a>, <a href="/wiki/F%C3%B3sforo" title="Fósforo">fósforo</a> y <a href="/wiki/Potasio" title="Potasio">potasio</a>. Están presentes también alrededor de la mitad de los aminoácidos existentes, ácidos orgánicos (ácido acético, ácido cítrico, entre otros) y vitaminas del complejo B, vitamina C, D y E. La miel posee también una variedad considerable de <a href="/wiki/Antioxidante" title="Antioxidante">antioxidantes</a> (<a href="/wiki/Flavonoide" title="Flavonoide">flavonoides</a> y fenólicos).
</p>
<p>La miel se usa principalmente en la <a href="/wiki/Gastronom%C3%ADa" title="Gastronomía">gastronomía</a>, tanto en <a href="/wiki/Receta_culinaria" title="Receta culinaria">recetas</a> dulces como saladas, es habitual como acompañamiento de <a href="/wiki/Pan" title="Pan">pan</a> o <a href="/wiki/Tostada" title="Tostada">tostadas</a> (en <a href="/wiki/Desayuno" title="Desayuno">desayunos</a> y <a href="/wiki/Merienda" title="Merienda">meriendas</a>) y como <a href="/wiki/Edulcorante" title="Edulcorante">edulcorante</a> de diversas bebidas. Al ser rica en <a href="/wiki/Az%C3%BAcares" title="Azúcares">azúcares</a> como la <a href="/wiki/Fructosa" title="Fructosa">fructosa</a>, la miel es <a href="/wiki/Higroscopia" title="Higroscopia">higroscópica</a> (absorbe humedad del aire), por lo que el añadir una pequeña cantidad a panes y pasteles hace que estos endurezcan más lentamente. 
La miel es el ingrediente principal del <a href="/wiki/Hidromiel" title="Hidromiel">hidromiel</a>, producido a partir de miel y agua. También se conoce como «vino de miel».
</p>
<p>Los beneficios de la miel a nivel medicinal ya era conocidos hacia el <a href="/wiki/Siglo_XXVI_a._C." title="Siglo XXVI a. C.">2500 a. C.</a> por los <a class="mw-redirect" href="/wiki/Sumerios" title="Sumerios">sumerios</a> en <a href="/wiki/Mesopotamia" title="Mesopotamia">Mesopotamia</a>. Los <a class="mw-redirect" href="/wiki/Antiguos_egipcios" title="Antiguos egipcios">antiguos egipcios</a>, los <a class="mw-redirect" href="/wiki/Asirio" title="Asirio">asirios</a>, los <a href="/wiki/Historia_de_China" title="Historia de China">chinos</a>, los <a href="/wiki/Antigua_Grecia" title="Antigua Grecia">griegos</a> y los <a href="/wiki/Antigua_Roma" title="Antigua Roma">romanos</a> utilizaban la miel para tratar heridas. Fue luego "redescubierta" por la medicina moderna debido a sus importantes propiedades bactericidas en heridas infectadas con bacterias multirresistentes a los antibióticos.<sup class="reference separada" id="cite_ref-Scielo_20-1"><a href="#cite_note-Scielo-20"><span class="corchete-llamada">[</span>20<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Se puede usar externamente debido a sus propiedades <a class="mw-redirect" href="/wiki/Antimicrobianas" title="Antimicrobianas">antimicrobianas</a> y <a class="mw-redirect" href="/wiki/Antis%C3%A9pticas" title="Antisépticas">antisépticas</a>, ayudando a cicatrizar y a prevenir infecciones en heridas o quemaduras superficiales.<sup class="reference separada" id="cite_ref-Scielo_20-2"><a href="#cite_note-Scielo-20"><span class="corchete-llamada">[</span>20<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las abejas añaden además una <a href="/wiki/Enzima" title="Enzima">enzima</a> llamada <a href="/wiki/Glucosa_oxidasa" title="Glucosa oxidasa">glucosa oxidasa</a>. Cuando la miel se aplica sobre las heridas, esta <a href="/wiki/Enzima" title="Enzima">enzima</a> produce la liberación local de <a href="/wiki/Per%C3%B3xido_de_hidr%C3%B3geno" title="Peróxido de hidrógeno">peróxido de hidrógeno</a>.
</p>
<p>Las mieles más oscuras tienen mayor poder <a href="/wiki/Antioxidante" title="Antioxidante">antioxidante</a> por ser más ricas en compuestos <a class="mw-redirect" href="/wiki/Fen%C3%B3licos" title="Fenólicos">fenólicos</a> como <a class="mw-redirect" href="/wiki/Flavonoides" title="Flavonoides">flavonoides</a> y <a class="mw-redirect" href="/wiki/Taninos" title="Taninos">taninos</a>.<sup class="reference separada" id="cite_ref-Scielo_20-3"><a href="#cite_note-Scielo-20"><span class="corchete-llamada">[</span>20<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Debido a su contenido de azúcares simples, de asimilación rápida, la miel es altamente calórica (cerca de 3,4 kcal/g), por lo que es útil como fuente de energía rápida.
</p>
<p>La miel virgen también contiene <a href="/wiki/Enzima" title="Enzima">enzimas</a> que ayudan a su digestión, así como diversas <a href="/wiki/Vitamina" title="Vitamina">vitaminas</a> y <a href="/wiki/Antioxidante" title="Antioxidante">antioxidantes</a>. Por esto suele recomendarse el consumo de la miel a temperaturas no superiores a 60 <a class="mw-redirect" href="/wiki/Grado_celsius" title="Grado celsius">°C</a>, pues a mayor temperatura empieza a perder propiedades beneficiosas al volatilizarse algunos de estos elementos.
</p>
<p>Es usada para el <a href="/wiki/Tratamientos_alternativos_usados_para_el_resfriado_com%C3%BAn" title="Tratamientos alternativos usados para el resfriado común">alivio sintomático del resfriado</a>.<sup class="reference separada" id="cite_ref-21"><a href="#cite_note-21"><span class="corchete-llamada">[</span>21<span class="corchete-llamada">]</span></a></sup>​ Estudios en personas de entre 2 y 18 años con infecciones en las vías respiratorias demostraron que es capaz de aliviar las membranas irritadas en la parte posterior de la garganta y que tiene efectos antioxidantes y antivirales. Además, un informe de la <a class="mw-redirect" href="/wiki/OMS" title="OMS">OMS</a> la considera segura, fuera del período de la lactancia, para aliviar la <a href="/wiki/Tos" title="Tos">tos</a>.<sup class="reference separada" id="cite_ref-ABUELA_22-0"><a href="#cite_note-ABUELA-22"><span class="corchete-llamada">[</span>22<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La dulzura y textura de jarabe calman el dolor de garganta, pero también influye su contenido antioxidante y su efecto antimicrobiano. Para niños menores de un año no es recomendable porque existe el peligro del desarrollo del <a href="/wiki/Botulismo" title="Botulismo">botulismo</a>. Este último riesgo se hace ínfimo en niños de más edad.<sup class="reference separada" id="cite_ref-ABUELA_22-1"><a href="#cite_note-ABUELA-22"><span class="corchete-llamada">[</span>22<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Los catarros se combaten popularmente en algunos países endulzando con miel el <a class="mw-redirect" href="/wiki/Zumo_de_lim%C3%B3n" title="Zumo de limón">zumo de limón</a> o el <a href="/wiki/T%C3%A9" title="Té">té</a> de <a class="mw-redirect" href="/wiki/Cebolla" title="Cebolla">cebolla</a>.<sup class="reference separada" id="cite_ref-23"><a href="#cite_note-23"><span class="corchete-llamada">[</span>23<span class="corchete-llamada">]</span></a></sup>​ La <a href="/wiki/Organizaci%C3%B3n_Mundial_de_la_Salud" title="Organización Mundial de la Salud">Organización Mundial de la Salud</a> recomienda el uso de miel para el alivio de la <a href="/wiki/Tos" title="Tos">tos</a> en niños mayores de un año.<sup class="reference separada" id="cite_ref-24"><a href="#cite_note-24"><span class="corchete-llamada">[</span>24<span class="corchete-llamada">]</span></a></sup>​ Sin embargo, la miel no muestra mayores beneficios que otros medicamentos como el <a href="/wiki/Dextrometorfano" title="Dextrometorfano">dextrometorfano</a>, pero sin los efectos adversos asociados.<sup class="reference separada" id="cite_ref-25"><a href="#cite_note-25"><span class="corchete-llamada">[</span>25<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Es utilizada en cosmética (cremas, <a href="/wiki/Mascarilla_facial" title="Mascarilla facial">mascarillas de limpieza facial</a>, tónicos, etc.) debido a sus cualidades <a class="mw-redirect" href="/wiki/Astringente" title="Astringente">astringentes</a> y suavizantes.<sup class="reference separada" id="cite_ref-Scielo_20-4"><a href="#cite_note-Scielo-20"><span class="corchete-llamada">[</span>20<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Es un excelente conservante natural. Sin embargo, no siempre es saludable. Debido a que procede de flores silvestres, hay algunos momentos y lugares en los que la miel producida por las abejas es altamente tóxica. Los <a href="/wiki/Rhododendron" title="Rhododendron">rododendros</a> y <a href="/wiki/Azalea" title="Azalea">azaleas</a> producen un néctar altamente venenoso para los humanos, aunque inofensivo para las abejas, que producen así una miel mortífera. En algunas regiones del mundo las colmenas se vacían inmediatamente después de la temporada de flores, eliminando cualquier residuo para evitar envenenamientos accidentales. Existen historias del uso de miel venenosa como arma de guerra en la antigüedad, pero no son corroborables. Dicha miel venenosa es muy difícil de encontrar. La forma de la flor de azalea hace que a las abejas le resulte difícil acceder al néctar, y en la época en la que florecen hay casi siempre otras flores más atractivas para las abejas.
</p>
<p>Es altamente perdurable, no caduca. Gracias a su alta concentración de azúcar, mata a las bacterias por <a href="/wiki/Lisis" title="Lisis">lisis</a> <a href="/wiki/%C3%93smosis" title="Ósmosis">osmótica</a>. Las <a href="/wiki/Levadura" title="Levadura">levaduras</a> aerotransportadas no pueden prosperar en la miel debido a la baja humedad que contiene. Los traslados de cuerpos humanos en la antigüedad se hacían sumergidos en miel; por ejemplo <a href="/wiki/Alejandro_Magno" title="Alejandro Magno">Alejandro Magno</a> fue trasladado desde <a class="mw-redirect" href="/wiki/Babilonia" title="Babilonia">Babilonia</a> hasta <a href="/wiki/Alejandr%C3%ADa" title="Alejandría">Alejandría</a> en <a href="/wiki/Egipto" title="Egipto">Egipto</a> en el <a href="/wiki/323_a._C." title="323 a. C.">323 a. C.</a> y el de <a href="/wiki/Agesilao_II" title="Agesilao II">Agesilao II</a>, rey de <a href="/wiki/Esparta" title="Esparta">Esparta</a>, desde Egipto hasta su ciudad natal en el <a href="/wiki/360_a._C." title="360 a. C.">360 a. C.</a>, utilizándose miel para evitar la descomposición. El efecto preservante de la miel se debe a su baja concentración de agua y es idéntico al que permite la prolongada conservación de los dulces y de las frutas en almíbar, donde el alto contenido en azúcar disminuye el contenido relativo de agua.
</p>
<p>Los contenidos altos de <a href="/wiki/Glucosa" title="Glucosa">glucosa</a> y <a href="/wiki/Fructosa" title="Fructosa">fructosa</a> en la dieta (más del 20 % de la energía total)<sup class="reference separada" id="cite_ref-howard_26-0"><a href="#cite_note-howard-26"><span class="corchete-llamada">[</span>26<span class="corchete-llamada">]</span></a></sup>​ están relacionados con un aumento de la <a class="mw-redirect" href="/wiki/Presi%C3%B3n_arterial" title="Presión arterial">presión arterial</a><sup class="reference separada" id="cite_ref-27"><a href="#cite_note-27"><span class="corchete-llamada">[</span>27<span class="corchete-llamada">]</span></a></sup>​
y con un aumento de la <a class="mw-redirect" href="/wiki/Grasa_corporal" title="Grasa corporal">grasa corporal</a><sup class="reference separada" id="cite_ref-28"><a href="#cite_note-28"><span class="corchete-llamada">[</span>28<span class="corchete-llamada">]</span></a></sup>​
y en la <a class="new" href="/w/index.php?title=Grasa_hep%C3%A1tica&amp;action=edit&amp;redlink=1" title="Grasa hepática (aún no redactado)">grasa hepática</a>,<sup class="reference separada" id="cite_ref-29"><a href="#cite_note-29"><span class="corchete-llamada">[</span>29<span class="corchete-llamada">]</span></a></sup>​ en el <a href="/wiki/Colesterol" title="Colesterol">colesterol</a>, en los problemas cardíacos<sup class="reference separada" id="cite_ref-howard_26-1"><a href="#cite_note-howard-26"><span class="corchete-llamada">[</span>26<span class="corchete-llamada">]</span></a></sup>​ y en la <a class="mw-redirect" href="/wiki/Diabetes_tipo_2" title="Diabetes tipo 2">diabetes tipo 2</a> (nunca en la <a class="mw-redirect" href="/wiki/Diabetes_tipo_1" title="Diabetes tipo 1">diabetes tipo 1</a>).
</p>
<p>El consumo de miel está totalmente desaconsejado para niños menores de 12 meses. A diferencia de los adultos y niños más mayores, los jugos del <a class="mw-redirect" href="/wiki/Sistema_digestivo" title="Sistema digestivo">sistema digestivo</a> de los bebés aún no han desarrollado la suficiente acidez, por lo que los compuestos dulces de la miel crean el ambiente adecuado para el crecimiento de las <a class="mw-redirect" href="/wiki/Esporas" title="Esporas">esporas</a> de <i><a href="/wiki/Clostridium_botulinum" title="Clostridium botulinum">Clostridium botulinum</a></i>, que produce el <a href="/wiki/Botulismo" title="Botulismo">botulismo</a>. Estas toxinas, inofensivas para el <a href="/wiki/Est%C3%B3mago" title="Estómago">estómago</a> de los adultos, pueden ser mortales en niños pequeños.<sup class="reference separada" id="cite_ref-30"><a href="#cite_note-30"><span class="corchete-llamada">[</span>30<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Determinadas mieles procedentes de algunas especies de plantas de la familia de las <a class="mw-redirect" href="/wiki/Eric%C3%A1ceas" title="Ericáceas">ericáceas</a> como <i><a href="/wiki/Rhododendron_ponticum" title="Rhododendron ponticum">Rhododendron ponticum</a></i> o <a class="mw-redirect" href="/wiki/Solan%C3%A1ceas" title="Solanáceas">solanáceas</a> como <i><a href="/wiki/Datura_stramonium" title="Datura stramonium">Datura stramonium</a></i> e <i><a href="/wiki/Hyoscyamus_niger" title="Hyoscyamus niger">Hyoscyamus niger</a></i> son tóxicas. En muchos casos la sustancia tóxica es la <a href="/wiki/Grayanotoxina" title="Grayanotoxina">grayanotoxina</a>.<sup class="reference separada" id="cite_ref-31"><a href="#cite_note-31"><span class="corchete-llamada">[</span>31<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Junto a la miel las abejas producen otros importantes productos: la <a href="/wiki/Cera" title="Cera">cera</a>, la <a href="/wiki/Jalea_real" title="Jalea real">jalea real</a> y el <a href="/wiki/Prop%C3%B3leo" title="Propóleo">propóleo</a>.
</p>
<p>Entre los productos equivalentes o similares en aspecto o dulzor podemos encontrar, las <a class="new" href="/w/index.php?title=Mieles_de_savia&amp;action=edit&amp;redlink=1" title="Mieles de savia (aún no redactado)">mieles de savia</a>, tales cómo la <a href="/wiki/Miel_de_palma" title="Miel de palma">miel de palma</a>, la <a href="/wiki/Miel_de_d%C3%A1til" title="Miel de dátil">miel de dátil</a> y el <a href="/wiki/Jarabe_de_arce" title="Jarabe de arce">jarabe de arce</a>; e igualmente el <a href="/wiki/Arrope" title="Arrope">Arrope</a> y  la pulpa de las vainas del <a href="/wiki/Gleditsia_triacanthos" title="Gleditsia triacanthos">algarrobo de miel</a> entre otros.
</p>
