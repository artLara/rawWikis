<p>El <b>idioma japonés</b> o <b>idioma nipón</b><span style="font-weight: normal"> (<span class="t_nihongo_kanji" lang="ja">日本語</span><span class="t_nihongo_comma" style="display:none">,</span> <span class="t_nihongo_romaji"><i><span class="unicode"><a href="/wiki/Archivo:Ja-nihongo.ogg" title="Acerca de este sonido"><img alt="Acerca de este sonido" data-file-height="20" data-file-width="20" decoding="async" height="11" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/8a/Loudspeaker.svg/11px-Loudspeaker.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/8a/Loudspeaker.svg/17px-Loudspeaker.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/8a/Loudspeaker.svg/22px-Loudspeaker.svg.png 2x" width="11"/></a> <a class="internal" href="//upload.wikimedia.org/wikipedia/commons/9/99/Ja-nihongo.ogg" title="Ja-nihongo.ogg">Nihongo</a> <small class="metadata audiolinkinfo" style="cursor:help;">(<a href="/wiki/Ayuda:Multimedia" title="Ayuda:Multimedia"><span style="cursor:help;" title="¿Problemas al reproducir este archivo?">?</span></a>·<a href="/wiki/Archivo:Ja-nihongo.ogg" title="Archivo:Ja-nihongo.ogg"><span style="cursor:help;" title="Acerca de este sonido">i</span></a>)</small></span></i></span><span class="t_nihongo_help"><sup><a href="/wiki/Ayuda:Idioma_japon%C3%A9s" title="Ayuda:Idioma japonés"><span class="t_nihongo_icon" style="color:DarkBlue;font:bold 80% sans-serif;text-decoration:none;padding:0 .1em;">?</span></a></sup></span>)</span> es un idioma de <a class="mw-redirect" href="/wiki/Asia_oriental" title="Asia oriental">Asia Oriental</a> hablado por alrededor de 128 millones de personas, principalmente en <a href="/wiki/Jap%C3%B3n" title="Japón">Japón</a>, donde es la <a href="/wiki/Lengua_nacional" title="Lengua nacional">lengua nacional</a>. El japonés es el principal idioma de las <a href="/wiki/Lenguas_jap%C3%B3nicas" title="Lenguas japónicas">lenguas japónicas</a> que no están relacionadas genéticamente con otras lenguas, es decir son <a href="/wiki/Lengua_aislada" title="Lengua aislada">lenguas aisladas</a>.
</p>
<p>Poco se sabe acerca del origen del idioma. Hay textos chinos del <a href="/wiki/Siglo_III" title="Siglo III">siglo <span style="font-variant:small-caps;text-transform:lowercase">III</span></a> que documentan algunas palabras japonesas, pero los primeros textos extensos no surgen hasta el <a href="/wiki/Siglo_VIII" title="Siglo VIII">siglo <span style="font-variant:small-caps;text-transform:lowercase">VIII</span></a>. Durante el <a href="/wiki/Per%C3%ADodo_Heian" title="Período Heian">período Heian</a> (794-1185), el <a href="/wiki/Idioma_chino" title="Idioma chino">chino</a> ejerció una profunda influencia en el vocabulario y fonología del <a href="/wiki/Japon%C3%A9s_antiguo" title="Japonés antiguo">japonés antiguo</a> y del <a href="/wiki/Japon%C3%A9s_medio_temprano" title="Japonés medio temprano">medio temprano</a>. El <a href="/wiki/Japon%C3%A9s_medio_tard%C3%ADo" title="Japonés medio tardío">japonés medio tardío</a> (1185-1600) se acerca sustancialmente al japonés contemporáneo. En este período se incorporan también los primeros <a href="/wiki/Pr%C3%A9stamo_ling%C3%BC%C3%ADstico" title="Préstamo lingüístico">préstamos</a> europeos. Tras el fin del aislamiento en 1853, la cantidad de préstamos de lenguas europeas aumentó considerablemente, especialmente del <a href="/wiki/Idioma_ingl%C3%A9s" title="Idioma inglés">inglés</a>.
</p>
<p>Aunque la lengua no tiene relación genética con el chino, la <a href="/wiki/Escritura_japonesa" title="Escritura japonesa">escritura japonesa</a> utiliza caracteres chinos llamados <i><a href="/wiki/Kanji" title="Kanji">kanji</a></i> (漢字) y gran parte del vocabulario proviene del chino. Junto a los <i>kanji</i>, el japonés utiliza dos <a href="/wiki/Silabario_(sistema_de_escritura)" title="Silabario (sistema de escritura)">silabarios</a>: el <i><a href="/wiki/Hiragana" title="Hiragana">hiragana</a></i> (ひらがな) y el <i><a href="/wiki/Katakana" title="Katakana">katakana</a></i> (カタカナ). El <a href="/wiki/Alfabeto_latino" title="Alfabeto latino">alfabeto latino</a> (en japonés <i>romaji</i>) se utiliza para escribir <a class="mw-redirect" href="/wiki/Acr%C3%B3nimos" title="Acrónimos">acrónimos</a> y el idioma emplea tanto los <a class="mw-redirect" href="/wiki/N%C3%BAmeros_ar%C3%A1bigos" title="Números arábigos">números arábigos</a> como los <a href="/wiki/Numeraci%C3%B3n_china" title="Numeración china">chinos</a>.
</p>
<p>Aunque originario de Asia nororiental, su parentesco filogenético es incierto. Comúnmente se ha clasificado como una <a href="/wiki/Lengua_aislada" title="Lengua aislada">lengua aislada</a>,<sup class="reference separada" id="cite_ref-Shibatani_3-0"><a href="#cite_note-Shibatani-3"><span class="corchete-llamada">[</span>3<span class="corchete-llamada">]</span></a></sup>​ al no haberse podido establecer parentesco con otros idiomas, es decir, un idioma sin relación con ningún otro. Sin embargo, según algunos lingüistas, el japonés estándar moderno no es una lengua aislada sino que es parte de la <a href="/wiki/Lenguas_jap%C3%B3nicas" title="Lenguas japónicas">familia japónica</a> junto a <a href="/wiki/Lenguas_ryukyuenses" title="Lenguas ryukyuenses">varias lenguas</a> de las <a href="/wiki/Islas_Ry%C5%ABky%C5%AB" title="Islas Ryūkyū">islas Ryūkyū</a> (también consideradas <a href="/wiki/Dialecto" title="Dialecto">dialectos</a> japoneses) todas ellas derivadas del <a href="/wiki/Idioma_protojap%C3%B3nico" title="Idioma protojapónico">protojapónico</a>. Aunque aparte de sus descendientes modernos no ha podido demostrarse un parentesco filogenético inequívoco entre el protojapónico (o sus descendientes modernos) y ninguna otra lengua de Asia. Sin embargo, aunque no se ha podido establecer firmemente ningún parentesco claro, no faltan las hipótesis que señalan algunas coincidencias con el <a href="/wiki/Idioma_coreano" title="Idioma coreano">coreano</a> y con las <a href="/wiki/Lenguas_altaicas" title="Lenguas altaicas">lenguas altaicas</a> o las <a href="/wiki/Lenguas_austronesias" title="Lenguas austronesias">lenguas austronesias</a>. La antigua lengua del <a href="/wiki/Goguryeo" title="Goguryeo">reino de Goguryeo</a>, llamada <a href="/wiki/Idioma_goguryeo" title="Idioma goguryeo">goguryano</a>, muestra similitudes que podrían deberse a una relación filogenética, lamentablemente esta lengua está mal documentada. Además cabe señalar que el japonés no está relacionado ni filogenéticamente ni tipológicamente con el <a href="/wiki/Idioma_chino" title="Idioma chino">chino</a>, si bien parte importante del vocabulario del japonés moderno son préstamos y cultismos tomados del <a href="/wiki/Chino_cl%C3%A1sico" title="Chino clásico">chino clásico</a>. Tampoco parece existir ninguna relación con la lengua <a href="/wiki/Idioma_ainu" title="Idioma ainu">ainu</a> (con la que comparte rasgos tipológicos, pero no elementos que sugieran origen filogenético común).
Es un hecho comprobado que existen correlaciones sistemáticas entre los fonemas de las lenguas primitivas coreanas y del japonés antiguo.<sup class="reference separada" id="cite_ref-4"><a href="#cite_note-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-5"><a href="#cite_note-5"><span class="corchete-llamada">[</span>5<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-6"><a href="#cite_note-6"><span class="corchete-llamada">[</span>6<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-7"><a href="#cite_note-7"><span class="corchete-llamada">[</span>7<span class="corchete-llamada">]</span></a></sup>​ No obstante, aún no está claro si esas correlaciones se deben a un origen común o a préstamos léxicos masivos a lo largo de los siglos, producto del intercambio cultural.<sup class="reference separada" id="cite_ref-autogenerated1_8-0"><a href="#cite_note-autogenerated1-8"><span class="corchete-llamada">[</span>8<span class="corchete-llamada">]</span></a></sup>​ Una teoría alternativa adscribe este idioma a la macrofamilia de las <a href="/wiki/Lenguas_austronesias" title="Lenguas austronesias">lenguas austronesias</a>. Según esta hipótesis, la lengua japonesa conforma el extremo norte de un grupo del que forman parte las lenguas aborígenes de <a class="mw-redirect" href="/wiki/Taiw%C3%A1n" title="Taiwán">Taiwán</a>, el <a href="/wiki/Idioma_tagalo" title="Idioma tagalo">tagalo</a> y otros idiomas de <a href="/wiki/Filipinas" title="Filipinas">Filipinas</a> y el <a href="/wiki/Idioma_indonesio" title="Idioma indonesio">malayo-indonesio</a> en todas sus variantes. En general la investigación contemporánea oscila entre ambas hipótesis:<sup class="reference separada" id="cite_ref-autogenerated1_8-1"><a href="#cite_note-autogenerated1-8"><span class="corchete-llamada">[</span>8<span class="corchete-llamada">]</span></a></sup>​ reconoce una fuerte influencia continental, posiblemente ligado al <a href="/wiki/Idioma_coreano" title="Idioma coreano">coreano</a>, y, al mismo tiempo, considera la posibilidad de la existencia de un sustrato austronésico. 
</p>
<p>Desde el 250 a. C. los <a href="/wiki/Per%C3%ADodo_Yayoi" title="Período Yayoi">pueblos yayoi</a> llegados del continente asiático comenzaron a poblar las islas del archipiélago japonés, donde se inició el desarrollo de una lengua arcaica (<i>Yamato kotoba</i> - 倭言葉) de estructura polisilábica, así como una cultura propia. No sería hasta el siglo <span style="font-variant:small-caps;text-transform:lowercase">III</span> d. C. cuando intelectuales coreanos introducen la cultura china en las islas niponas. Esta invasión cultural duró aproximadamente cuatro siglos, durante los cuales se introdujeron ciencias, artes, y religión, así como el sistema de escritura chino. 
</p>
<p>Los japoneses comenzaron a usar los caracteres chinos (kanji - <i>漢字</i> significa caracteres Han) conservando el sonido original chino, (si bien adaptándolo a su propio sistema fonético) y añadiendo además la pronunciación nativa a esos símbolos.<sup class="reference separada" id="cite_ref-9"><a href="#cite_note-9"><span class="corchete-llamada">[</span>9<span class="corchete-llamada">]</span></a></sup>​ Por ello, hoy en día, al estudiar el sistema de kanji es necesario aprender ambas lecturas, la <a href="/wiki/On%27yomi" title="On'yomi">lectura china</a> (onyomi <i>音読み</i>) y la lectura japonesa (kunyomi <i>訓読み</i>), si bien dichos adjetivos no deben prestarse a errores: Ambas pronunciaciones son propias del japonés, y son diferentes a las del <a href="/wiki/Idioma_chino" title="Idioma chino">chino</a> moderno, aun así el sonido del onyomi es la aproximación japonesa al sonido chino del entonces y dependía también de la variante hablada que estaba en el poder. Además de los kanji, en el japonés existen dos silabarios para representar todos sus sonidos, creados a partir de la simplificación de ciertos kanji. Los silabarios se denominan <a href="/wiki/Hiragana" title="Hiragana">hiragana</a> y <a href="/wiki/Katakana" title="Katakana">katakana</a> y son un sistema de escritura único del japonés, ausente en el <a href="/wiki/Idioma_chino" title="Idioma chino">chino</a> y el <a href="/wiki/Idioma_coreano" title="Idioma coreano">coreano</a>. El japonés moderno utiliza los tres sistemas de escritura, <a href="/wiki/Kanji" title="Kanji">kanji</a>, hiragana y katakana, circunscribiendo el uso de cada uno para diferentes funciones, si bien hay ocasiones en que dos de ellos pueden usarse indistintamente.
</p>
<p>Debido a la particular <a href="/wiki/Historia_de_Jap%C3%B3n" title="Historia de Japón">historia de Japón</a>, el idioma japonés incluye elementos ausentes en las lenguas indoeuropeas, siendo uno de los más conocidos, un rico sistema de <i>honoríficos</i> (keigo <i>敬語</i>) que resultan en formas verbales y construcciones gramaticales específicas para indicar la jerarquía relativa entre el que habla y el que escucha, así como el nivel de respeto hacia el interlocutor.
</p>
<p>El japonés está difundido en su mayor parte, como es lógico, en <a href="/wiki/Jap%C3%B3n" title="Japón">Japón</a>, donde es hablado por la mayor parte de la población. Hay comunidades de inmigrantes japoneses en <a href="/wiki/Haw%C3%A1i" title="Hawái">Hawái</a> que también utilizan el idioma (más de 250 000, el 30% de la población), en <a href="/wiki/California" title="California">California</a> (<a href="/wiki/Estados_Unidos" title="Estados Unidos">Estados Unidos</a>) unas 300 000; en <a href="/wiki/Brasil" title="Brasil">Brasil</a> unas 400 000 y un número importante en <a href="/wiki/Argentina" title="Argentina">Argentina</a> y en la costa de <a href="/wiki/Per%C3%BA" title="Perú">Perú</a>, así como otras partes del mundo. En las antiguas colonias japonesas como <a href="/wiki/Corea" title="Corea">Corea</a>, <a href="/wiki/Manchuria" title="Manchuria">Manchuria</a> (<a href="/wiki/Rep%C3%BAblica_Popular_China" title="República Popular China">China</a>), <a href="/wiki/Guam" title="Guam">Guam</a>, <a class="mw-redirect" href="/wiki/Taiw%C3%A1n" title="Taiwán">Taiwán</a>, <a href="/wiki/Filipinas" title="Filipinas">Filipinas</a>, <a href="/wiki/Islas_Marshall" title="Islas Marshall">Islas Marshall</a> y <a href="/wiki/Palaos" title="Palaos">Palaos</a> es conocido también por las personas de edad avanzada que recibieron instrucción escolar en este idioma. No obstante, la mayor parte de estas personas prefiere no utilizarlo. 
</p>
<p>Se hablan docenas de dialectos en Japón. La profusión se debe a muchos factores, incluido el período de tiempo que el archipiélago japonés ha estado habitado, el terreno montañoso de la isla y la larga historia de aislamiento externo e interno de Japón. Los dialectos típicamente difieren en términos de acento tonal, morfología inflexional, vocabulario y uso de partículas. Algunos incluso difieren en los inventarios de vocales y consonantes, aunque esto es poco común.
</p>
<p>La principal distinción en los acentos japoneses es entre los tipo <a href="/wiki/Tokio" title="Tokio">Tokio</a> (東京 式, Tōkyō-shiki) y los tipo <a href="/wiki/Kioto" title="Kioto">Kioto</a>-<a href="/wiki/Osaka" title="Osaka">Osaka</a> (京阪 式, Keihan-shiki). Dentro de cada tipo hay varias subdivisiones. Los dialectos de tipo Kyoto-Osaka se encuentran en la región central, aproximadamente formados por las regiones de <a href="/wiki/Regi%C3%B3n_de_Kansai" title="Región de Kansai">Kansai</a>, <a href="/wiki/Regi%C3%B3n_de_Shikoku" title="Región de Shikoku">Shikoku</a> y <a href="/wiki/Regi%C3%B3n_de_Hokuriku" title="Región de Hokuriku">Hokuriku</a> occidental.
</p>
<p>Los dialectos de regiones periféricas, como <a href="/wiki/Regi%C3%B3n_de_T%C5%8Dhoku" title="Región de Tōhoku">Tōhoku</a> o <a href="/wiki/Prefectura_de_Kagoshima" title="Prefectura de Kagoshima">Kagoshima</a>, pueden ser ininteligibles para los hablantes de otras partes del país. Hay algunas islas lingüísticas en pueblos de montaña o islas aisladas, como la isla Hachijō-jima, cuyos dialectos descienden del dialecto oriental del japonés antiguo. Los dialectos de la región de Kansai son hablados o conocidos por muchos japoneses, y el dialecto de Osaka en particular está asociado con la comedia. Los dialectos de Tōhoku y el norte de Kantō están asociados con agricultores típicos regionales.
</p>
<p>Respecto a los <a href="/wiki/Islas_Ry%C5%ABky%C5%AB" title="Islas Ryūkyū">idiomas ryukyuenses</a>, que se hablan en <a href="/wiki/Okinawa_(Okinawa)" title="Okinawa (Okinawa)">Okinawa</a> y las <a href="/wiki/Islas_Amami" title="Islas Amami">Islas Amami</a> (políticamente parte de Kagoshima), el gobierno japonés mantiene la política de que todas las lenguas ryukyuenses son meramente "dialectos del japonés", a pesar de que son lo suficientemente distintas como para ser consideradas una rama separada de la familia japonesa; no solo cada idioma es ininteligible para los hablantes de japonés, sino que la mayoría son ininteligibles para quienes hablan otros idiomas ryukyuenses. Sin embargo, en contraste con los lingüistas, muchos japoneses comunes tienden a considerar los idiomas ryukyuenses como dialectos del japonés. La <a href="/wiki/Corte_Imperial_de_Kioto" title="Corte Imperial de Kioto">corte imperial</a> también parece haber hablado una variante inusual de japonés de la época.<sup class="reference separada" id="cite_ref-10"><a href="#cite_note-10"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​ Lo más probable es que se trate de la forma hablada del idioma <a href="/wiki/Japon%C3%A9s_cl%C3%A1sico" title="Japonés clásico">japonés clásico</a>, un estilo de escritura que prevaleció durante el período <a href="/wiki/Per%C3%ADodo_Heian" title="Período Heian">Heian</a>, pero comenzó a declinar a fines de la era <a href="/wiki/Era_Meiji" title="Era Meiji">Meiji</a>.<sup class="reference separada" id="cite_ref-11"><a href="#cite_note-11"><span class="corchete-llamada">[</span>11<span class="corchete-llamada">]</span></a></sup>​ Los idiomas ryukyuenses son hablados por un número cada vez menor de personas mayores, por lo que la <a class="mw-redirect" href="/wiki/UNESCO" title="UNESCO">UNESCO</a> los clasificó como en peligro de extinción, ya que podrían extinguirse en 2050. Los jóvenes utilizan principalmente el japonés y no pueden entender los idiomas ryukyuenses. El japonés de Okinawa (no confundir con el <a href="/wiki/Idioma_okinawense" title="Idioma okinawense">idioma okinawense</a>) es una variante del japonés estándar influenciado por los idiomas ryukyuenses. Es el dialecto principal que se habla entre los jóvenes en las islas Ryūkyū.<sup class="reference separada" id="cite_ref-12"><a href="#cite_note-12"><span class="corchete-llamada">[</span>12<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El japonés moderno se ha generalizado en todo el país (incluidas las islas Ryūkyū) debido a la educación, los medios de comunicación y el aumento de la movilidad dentro de Japón, así como a la integración económica.
</p>
<p>Las variantes del idioma han sido confirmadas desde el <a href="/wiki/Antiguo_Jap%C3%B3n" title="Antiguo Japón">antiguo Japón</a>, a través del <a href="/wiki/Man%27y%C5%8Dsh%C5%AB" title="Man'yōshū">Man'yōshū</a>. Los escritos japoneses más antiguos conocidos que incluyen dialectos orientales cuyas características no fueron heredadas por los dialectos modernos a excepción de algunas partes como la isla de Hachijo.<sup>[<i><a href="/wiki/Wikipedia:Verificabilidad" title="Wikipedia:Verificabilidad">cita requerida</a></i>]</sup>Con la modernización del japonés en la parte tardía del <a href="/wiki/Siglo_XIX" title="Siglo XIX">siglo <span style="font-variant:small-caps;text-transform:lowercase">XIX</span></a>, el gobierno ha promovido el uso del japonés estándar lo cual ha hecho que este dialecto no solo sea altamente conocido solo en Japón sino también en todo el mundo. El idioma ha ido cambiando al fusionarse diversos dialectos y recibir influencia de otras lenguas.
</p>
<p>La historia de la lengua japonesa se suele dividir en cuatro periodos diferentes.<sup class="reference separada" id="cite_ref-13"><a href="#cite_note-13"><span class="corchete-llamada">[</span>13<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Algunas de las similitudes léxicas entre las <a href="/wiki/Lenguas_austronesias" title="Lenguas austronesias">lenguas austronesias</a> y el japonés podrían deberse a la influencia adstrato de algunas lenguas, aunque la evidencia en favor de dicha influencia prehistórica no es concluyente. A partir del siglo <span style="font-variant:small-caps;text-transform:lowercase">VII</span> sí es notoria la influencia de la <a class="mw-redirect" href="/wiki/Cultura_china" title="Cultura china">cultura china</a> en Japón y la adopción en esta lengua de numerosos préstamos léxicos procedentes del idioma chino para designar conceptos técnicos y culturales asociados a la influencia china. El chino clásico es al japonés, algo similar a lo que las raíces de origen griego son a las <a class="mw-redirect" href="/wiki/Lenguas_europeas" title="Lenguas europeas">lenguas europeas</a>: una fuente de elementos léxicos para formar <a href="/wiki/Neologismo" title="Neologismo">neologismos</a>. El propio sistema de escritura japonesa es en sí mismo una muestra de la influencia cultural china en el japonés.
</p>
<p>A partir del <a href="/wiki/Siglo_XVI" title="Siglo XVI">siglo <span style="font-variant:small-caps;text-transform:lowercase">XVI</span></a> el japonés adoptará algunos términos procedentes del <a href="/wiki/Idioma_portugu%C3%A9s" title="Idioma portugués">portugués</a>, del <a href="/wiki/Idioma_espa%C3%B1ol" title="Idioma español">español</a>, del <a href="/wiki/Idioma_neerland%C3%A9s" title="Idioma neerlandés">neerlandés</a> y de otras lenguas de colonización europea. Y a partir del siglo <span style="font-variant:small-caps;text-transform:lowercase">XX</span> la influencia del <a href="/wiki/Idioma_ingl%C3%A9s" title="Idioma inglés">inglés</a> como fuente de nuevos préstamos al japonés es hegemónica.
</p>
<p>Tiene cinco <a href="/wiki/Vocal" title="Vocal">vocales</a> y dieciséis <a href="/wiki/Consonante" title="Consonante">consonantes</a><sup class="reference separada" id="cite_ref-Shibatani_3-1"><a href="#cite_note-Shibatani-3"><span class="corchete-llamada">[</span>3<span class="corchete-llamada">]</span></a></sup>​ y es muy restrictivo en la formación de <a href="/wiki/S%C3%ADlaba" title="Sílaba">sílabas</a>. El acento es <a class="mw-redirect" href="/wiki/Acento_mel%C3%B3dico" title="Acento melódico">tonal</a>, con dos tonos diferentes: alto y bajo.
</p>
<p>El <a class="new" href="/w/index.php?title=Fonolog%C3%ADa_del_japon%C3%A9s&amp;action=edit&amp;redlink=1" title="Fonología del japonés (aún no redactado)">sistema fonológico japonés</a> consta de cinco vocales, que escritas en caracteres latinos son: <i>a, i, u, e, o</i>, según el orden tradicional. Se pronuncian igual que en castellano /a, e, i, o/ salvo la <i>u</i> /ɯ/, que se pronuncia con los labios extendidos, esto es, se trata de una <a class="mw-redirect" href="/wiki/Vocal_no_redondeada" title="Vocal no redondeada">vocal no redondeada</a>. 
Las vocales pueden ser normales o largas, en cuyo caso poseen una duración doble de la normal y se consideran como sílabas separadas. 
</p>
<p>Los fonemas consonánticos son dieciséis o quince, dependiendo de si se considera o no que el <i><a href="/wiki/Sokuon" title="Sokuon">sokuon</a></i> corresponde a una consonante geminada o al <a class="mw-redirect" href="/wiki/Archifonema" title="Archifonema">archifonema</a> /Q/, representado en la escritura por el símbolo <i><a href="/wiki/Sokuon" title="Sokuon">sokuon</a></i> que adopta el mismo sonido de la consonante que le sigue o en ocasiones se pronuncia como una <a class="mw-redirect" href="/wiki/Oclusi%C3%B3n_glotal" title="Oclusión glotal">oclusión glotal</a>. La cuenta de sonidos es mucho más alta si se cuentan los <a href="/wiki/Al%C3%B3fono" title="Alófono">alófonos</a> de aparición consistente, representados entre corchetes en la siguiente tabla. Debe considerarse además que los <a href="/wiki/Pr%C3%A9stamo_ling%C3%BC%C3%ADstico" title="Préstamo lingüístico">préstamos</a> tomados de otras lenguas a partir del siglo <span style="font-variant:small-caps;text-transform:lowercase">XX</span>, particularmente del inglés, pueden conservar fonemas ajenos al inventario tradicional.
</p>
<p>El japonés es una lengua de estructura <a href="/wiki/Lengua_aglutinante" title="Lengua aglutinante">aglutinante</a> que combina diversos elementos lingüísticos en palabras simples. Cada uno de estos elementos tiene una significación fija y apta para existir separadamente. El japonés es casi exclusivamente sufijante, con muy pocos prefijos, como por ejemplo, los honoríficos o-(お), go- (ご), por lo que los únicos procesos para la formación de palabras son la <a href="/wiki/Composici%C3%B3n_(ling%C3%BC%C3%ADstica)" title="Composición (lingüística)">composición</a> y la <a href="/wiki/Derivaci%C3%B3n_(ling%C3%BC%C3%ADstica)" title="Derivación (lingüística)">derivación</a> mediante <a href="/wiki/Sufijo" title="Sufijo">sufijos</a>.
