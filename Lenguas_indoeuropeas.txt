<p>Con el nombre de <b>lenguas indoeuropeas</b> se conoce a la mayor <a href="/wiki/Familia_de_lenguas" title="Familia de lenguas">familia de lenguas</a> del mundo en número de hablantes. La familia indoeuropea, a la que pertenecen la mayoría de las <a href="/wiki/Idioma" title="Idioma">lenguas</a> de <a href="/wiki/Europa" title="Europa">Europa</a>, <a href="/wiki/Gran_Ir%C3%A1n" title="Gran Irán">Gran Irán</a> y <a class="mw-redirect" href="/wiki/Asia_meridional" title="Asia meridional">Asia meridional</a>, incluye más de 150 idiomas hablados por alrededor de 3200 millones de personas (aproximadamente un 45 % de la población mundial).<sup class="reference separada" id="cite_ref-Crystal_1-1"><a href="#cite_note-Crystal-1"><span class="corchete-llamada">[</span>1<span class="corchete-llamada">]</span></a></sup>​ De estas, unos 1200 millones corresponden a hablantes de las <a href="/wiki/Lenguas_indoiranias" title="Lenguas indoiranias">lenguas indoiranias</a>, unos 950 millones de hablantes de las <a href="/wiki/Lenguas_romances" title="Lenguas romances">lenguas romances</a> y unos 820 millones de hablantes de las <a href="/wiki/Lenguas_germ%C3%A1nicas" title="Lenguas germánicas">lenguas germánicas</a>.
</p>
<p>El jesuita francés <a class="new" href="/w/index.php?title=Gaston-Laurent_Coeurdoux&amp;action=edit&amp;redlink=1" title="Gaston-Laurent Coeurdoux (aún no redactado)">Gaston-Laurent Coeurdoux</a> fue el primero en notar las similitudes entre el <a href="/wiki/S%C3%A1nscrito" title="Sánscrito">sánscrito</a>, el <a href="/wiki/Lat%C3%ADn" title="Latín">latín</a> y el <a class="mw-redirect" href="/wiki/Idioma_griego_antiguo" title="Idioma griego antiguo">griego</a>, e incluso entre el <a href="/wiki/Idioma_alem%C3%A1n" title="Idioma alemán">alemán</a> y el <a href="/wiki/Idioma_ruso" title="Idioma ruso">ruso</a>, en una memoria enviada a la <a class="mw-redirect" href="/wiki/Acad%C3%A9mie_des_inscriptions_et_belles-lettres" title="Académie des inscriptions et belles-lettres">Académie des inscriptions et belles-lettres</a> de <a href="/wiki/Francia" title="Francia">Francia</a> en 1767. A veces se atribuye erróneamente al <a class="mw-redirect" href="/wiki/Fil%C3%B3logo" title="Filólogo">filólogo</a> británico sir <a href="/wiki/William_Jones_(fil%C3%B3logo)" title="William Jones (filólogo)">William Jones</a> haber sido el primero en notar las semejanzas entre el <a href="/wiki/S%C3%A1nscrito" title="Sánscrito">sánscrito</a>, el <a href="/wiki/Lat%C3%ADn" title="Latín">latín</a> y el <a href="/wiki/Idioma_griego" title="Idioma griego">griego</a>, pues en <i><b>The Sanskrit Language</b></i> (1786) supone que esas tres lenguas tienen una raíz común y que, además, pueden estar ligadas al <a href="/wiki/Idioma_g%C3%B3tico" title="Idioma gótico">gótico</a>, a las <a href="/wiki/Lenguas_celtas" title="Lenguas celtas">lenguas celtas</a> y al <a href="/wiki/Idioma_persa" title="Idioma persa">persa</a>. <a href="/wiki/Franz_Bopp" title="Franz Bopp">Franz Bopp</a> apoyó esta hipótesis al comparar sistemáticamente estas lenguas con otras y encontrar múltiples <a href="/wiki/Cognado" title="Cognado">cognados</a>. Desde el siglo <span style="font-variant:small-caps;text-transform:lowercase">XIX</span>, los estudiosos llamaron a esta familia <b>lenguas indogermánicas</b>. Posteriormente pasó a emplearse el término <b>indoeuropeo</b> (excepto en alemán). Un buen ejemplo de la conexión indoeuropea es la enorme similitud descubierta entre el <a href="/wiki/S%C3%A1nscrito" title="Sánscrito">sánscrito</a> y dialectos antiguos del <a href="/wiki/Idioma_lituano" title="Idioma lituano">lituano</a>.
</p>
<p>El idioma común ancestral se conoce como <b><a href="/wiki/Idioma_protoindoeuropeo" title="Idioma protoindoeuropeo">protoindoeuropeo</a></b>. Existe desacuerdo en torno al punto geográfico en el que se originó <i>(<a href="/wiki/Urheimat_protoindoeuropeo" title="Urheimat protoindoeuropeo">urheimat</a>)</i>. Los principales lugares propuestos son el sur de <a href="/wiki/Rusia" title="Rusia">Rusia</a>, el <a href="/wiki/Hip%C3%B3tesis_de_los_kurganes" title="Hipótesis de los kurganes">óblast de Kurgán</a>, el sudeste de <a href="/wiki/Ucrania" title="Ucrania">Ucrania</a>, <a href="/wiki/Armenia" title="Armenia">Armenia</a> o <a href="/wiki/Ir%C3%A1n" title="Irán">Irán</a>.
</p>
<p>Esta familia está formada por las siguientes subfamilias: <a href="/wiki/Idioma_alban%C3%A9s" title="Idioma albanés">albanesa</a>, <a href="/wiki/Idioma_armenio" title="Idioma armenio">armenia</a>, <a href="/wiki/Lenguas_b%C3%A1lticas" title="Lenguas bálticas">báltica</a>, <a href="/wiki/Lenguas_celtas" title="Lenguas celtas">céltica</a>, <a href="/wiki/Lenguas_eslavas" title="Lenguas eslavas">eslava</a>, <a href="/wiki/Lenguas_germ%C3%A1nicas" title="Lenguas germánicas">germánica</a>, <a href="/wiki/Lenguas_griegas" title="Lenguas griegas">griega</a>, <a href="/wiki/Lenguas_indoiranias" title="Lenguas indoiranias">indoirania</a> (que incluye las <a class="mw-redirect" href="/wiki/Lenguas_%C3%ADndicas" title="Lenguas índicas">lenguas indoarias</a> y las <a href="/wiki/Lenguas_iranias" title="Lenguas iranias">iranias</a>) e <a href="/wiki/Lenguas_it%C3%A1licas" title="Lenguas itálicas">itálica</a> (que incluye el <a href="/wiki/Lat%C3%ADn" title="Latín">latín</a> y las <a class="mw-redirect" href="/wiki/Lengua_romance" title="Lengua romance">lenguas romances</a>). A ellas se suman dos subfamilias hoy desaparecidas: la <a href="/wiki/Lenguas_anatolias" title="Lenguas anatolias">anatolia</a> (que incluye la lengua de los <a class="mw-redirect" href="/wiki/Hititas" title="Hititas">hititas</a>) y la <a class="mw-redirect" href="/wiki/Idioma_tocario" title="Idioma tocario">tocaria</a>. Desde la segunda mitad del siglo XVIII, y durante todo el <a href="/wiki/Siglo_XIX" title="Siglo XIX">siglo XIX</a>, la <a href="/wiki/Ling%C3%BC%C3%ADstica_hist%C3%B3rica" title="Lingüística histórica">lingüística histórica</a> y la <a class="mw-redirect" href="/wiki/Neogram%C3%A1tica" title="Neogramática">neogramática</a> intentaron reunir datos suficientes para demostrar que este conjunto de lenguas, aparentemente diversas, formaban parte de una única familia.
</p>
<p>Los documentos del <a href="/wiki/S%C3%A1nscrito" title="Sánscrito">sánscrito</a> y del <a href="/wiki/Idioma_griego" title="Idioma griego">griego clásico</a> (los más antiguos de las lenguas indoeuropeas si exceptuamos los hititas, que por entonces no estaban descifrados) presentan las formas características propias de las lenguas indoeuropeas, lo que demuestra la existencia de una lengua madre común. Las relaciones entre el <a href="/wiki/S%C3%A1nscrito" title="Sánscrito">sánscrito</a>, el <a href="/wiki/Idioma_griego" title="Idioma griego">griego clásico</a> y el <a href="/wiki/Lat%C3%ADn" title="Latín">latín</a> se habían comprobado ya hacia principios del <a href="/wiki/Siglo_XIX" title="Siglo XIX">siglo XIX</a>.
</p>
<p>Por otro lado, los <a href="/wiki/Gram%C3%A1tica" title="Gramática">gramáticos</a> de la <a href="/wiki/India" title="India">India</a> elaboraron una clasificación sistemática de los elementos que constituyeron antiguamente el sánscrito. El estudio realizado en la <a href="/wiki/India" title="India">India</a> se completa con otro estudio sistemático y comparativo de los sistemas <a href="/wiki/Fon%C3%A9tica" title="Fonética">fonéticos</a> y <a href="/wiki/Gram%C3%A1tica" title="Gramática">gramaticales</a> de las lenguas europeas.
</p>
<p>La conclusión de este esfuerzo conjunto fue el establecimiento de la existencia del <a href="/wiki/Idioma_protoindoeuropeo" title="Idioma protoindoeuropeo">protoindoeuropeo</a>, lengua madre común a los idiomas estudiados, efectuándose una reconstrucción de los rasgos <a href="/wiki/Fon%C3%A9tica" title="Fonética">fonéticos</a> y <a href="/wiki/Gram%C3%A1tica" title="Gramática">gramaticales</a> que este debía tener. El indoeuropeo es, pues, una lengua reconstruida y fechada hacia el <a href="/wiki/Siglo_XXXI_a._C." title="Siglo XXXI a. C.">3000 a. C.</a>, puesto que hacia el <a href="/wiki/Siglo_XXI_a._C." title="Siglo XXI a. C.">2000 a. C.</a> ya se encuentran rasgos de diferenciación notables entre las lenguas nacidas del mismo.
</p>
<p>En general, las lenguas indoeuropeas, muestran cierta pérdida progresiva de la <a href="/wiki/Flexi%C3%B3n_(ling%C3%BC%C3%ADstica)" title="Flexión (lingüística)">flexión</a>. Por lo que se supone, el <a class="mw-redirect" href="/wiki/Idioma_proto-indoeuropeo" title="Idioma proto-indoeuropeo">protoindoeuropeo</a> fue una lengua muy flexiva, como lo demuestran otras lenguas clásicas, como el sánscrito, el <a class="mw-redirect" href="/wiki/Av%C3%A9stico" title="Avéstico">avéstico</a> y el griego. Frente a esto, algunas lenguas modernas, tras un largo proceso evolutivo, están orientadas hacia una vía analítica, como por ejemplo el <a href="/wiki/Idioma_ingl%C3%A9s" title="Idioma inglés">inglés</a>, el <a href="/wiki/Idioma_franc%C3%A9s" title="Idioma francés">francés</a> y el <a href="/wiki/Idioma_persa" title="Idioma persa">persa</a>, usando complementos con preposición y verbos auxiliares en lugar de la declinación nominal y la conjugación verbal; otras, como el <a href="/wiki/Idioma_espa%C3%B1ol" title="Idioma español">español</a> o el <a href="/wiki/Idioma_italiano" title="Idioma italiano">italiano</a> siguen manteniendo una gran flexión en sus conjugaciones, tiempos y modos verbales aunque también han perdido la mayor parte de la flexividad en sustantivos y adjetivos que se daba en las <a class="mw-disambig" href="/wiki/Declinaci%C3%B3n" title="Declinación">declinaciones</a> latinas; mientras que lenguas como el <a href="/wiki/Idioma_alem%C3%A1n" title="Idioma alemán">alemán</a> o el <a href="/wiki/Idioma_ruso" title="Idioma ruso">ruso</a>, en cambio, mantienen un sistema de declinaciones.
</p>
<p>En gran parte, la pérdida de los elementos flexivos ha sido el resultado de un largo proceso que ha conducido a la pérdida de las sílabas finales de las palabras; así, muchas de las indoeuropeas eran más breves que las correspondientes protoindoeuropeas. Además, en otras lenguas ha tenido lugar el desarrollo de nuevos procedimientos gramaticales y ha habido numerosos cambios de significado en algunas palabras concretas.
</p>
<p>El <a href="/wiki/Idioma_protoindoeuropeo" title="Idioma protoindoeuropeo">protoindoeuropeo</a> presenta muchos rasgos que han desaparecido de la mayoría de las lenguas indoeuropeas modernas. De hecho, entre las lenguas indoeuropeas se encuentran tipologías gramaticales que las hacen muy diferentes entre sí, no siendo cierto que todas las lenguas indoeuropeas conserven actualmente "parecido" entre sí, y su relación <a href="/wiki/Filogenia" title="Filogenia">filogenética</a> muchas veces solo es accesible mediante el estudio comparado profundo de las mismas y no por su aspecto superficial o las características gramaticales más evidentes. Esto se debe a que estas lenguas han seguido evoluciones marcadamente diferentes en cada región donde se hablan. Sin embargo, se reconocen algunas características casi universales en todas ellas:
</p>
<p>El <a href="/wiki/S%C3%A1nscrito" title="Sánscrito">sánscrito</a>, el <a href="/wiki/Lat%C3%ADn" title="Latín">latín</a> y el <a href="/wiki/Griego_antiguo" title="Griego antiguo">griego clásico</a> distinguían entre tres <a href="/wiki/G%C3%A9nero_gramatical" title="Género gramatical">géneros gramaticales</a>: masculino, femenino y neutro. Aunque muchas lenguas indoeuropeas más modernas han perdido alguno de estos tres géneros, en las <a href="/wiki/Lenguas_romances" title="Lenguas romances">lenguas romances</a> (con la excepción del <a class="mw-redirect" href="/wiki/Asturleon%C3%A9s" title="Asturleonés">asturleonés</a>), las <a href="/wiki/Lenguas_celtas" title="Lenguas celtas">lenguas celtas</a> modernas y las <a href="/wiki/Lenguas_b%C3%A1lticas" title="Lenguas bálticas">lenguas bálticas</a>, el género neutro se ha asimilado al masculino o al femenino. En <a href="/wiki/Idioma_neerland%C3%A9s" title="Idioma neerlandés">neerlandés</a> y algunas de las lenguas escandinavas, el femenino ha desaparecido manteniéndose la oposición entre masculino y neutro. En inglés, la distinción de género solo existe en los pronombres de tercera <a href="/wiki/Persona_gramatical" title="Persona gramatical">persona</a> de singular (marginalmente cuando el referente es un vehículo o un país puede usarse <i>she</i> para referirse a ellos), aunque en <a href="/wiki/Idioma_anglosaj%C3%B3n" title="Idioma anglosajón">inglés antiguo</a> el género también existía en los demostrativos y el artículo. Algunas lenguas modernas, como el <a href="/wiki/Idioma_armenio" title="Idioma armenio">armenio</a>, han perdido completamente la distinción de género tanto en el nombre como en el pronombre.<sup class="reference separada" id="cite_ref-2"><a href="#cite_note-2"><span class="corchete-llamada">[</span>2<span class="corchete-llamada">]</span></a></sup>​ En muchas lenguas iranias modernas existen solo dos géneros: en <a href="/wiki/Idioma_persa" title="Idioma persa">persa moderno</a> solo existe distinción entre género humano y no humano y en <a href="/wiki/Idioma_past%C3%BAn" title="Idioma pastún">pastún</a> entre masculino y femenino.<sup class="reference separada" id="cite_ref-3"><a href="#cite_note-3"><span class="corchete-llamada">[</span>3<span class="corchete-llamada">]</span></a></sup>​ También muchas lenguas índicas han perdido alguno de los tres géneros presentes en sánscrito, el <a class="mw-redirect" href="/wiki/Hindi" title="Hindi">hindi</a>-<a href="/wiki/Urdu" title="Urdu">urdu</a><sup class="reference separada" id="cite_ref-4"><a href="#cite_note-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​ solo diferencia entre masculino y femenino, habiéndose perdido el neutro. En <a href="/wiki/Idioma_bengal%C3%AD" title="Idioma bengalí">bengalí</a> la pérdida ha ido más allá y la distinción de género ya no existe, o más exactamente no es morfológicamente productiva, aunque hay residuos en el léxico.
</p>
<p>El número de géneros en el indoeuropeo más antiguo reconstruible es dudoso, ya que parece que las <a href="/wiki/Lenguas_anatolias" title="Lenguas anatolias">lenguas anatolias</a> más antiguas solo reflejan una distinción entre género animado y género inanimado en el adjetivo. Rodríguez Adrados ha propuesto que esta es la distinción más antigua y secundariamente apareció en el resto de las ramas también el género femenino.<sup class="reference separada" id="cite_ref-5"><a href="#cite_note-5"><span class="corchete-llamada">[</span>5<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>En el estadio más antiguo de las lenguas indoiranias, griegas, eslavas y celtas existían tres posibilidades para el número: <a href="/wiki/Singular" title="Singular">singular</a>, <a href="/wiki/N%C3%BAmero_dual" title="Número dual">dual</a> y <a href="/wiki/Plural" title="Plural">plural</a>. En las otras ramas de la familia se registran solo dos números: singular y plural (marginalmente en <a href="/wiki/Lat%C3%ADn" title="Latín">latín</a> se tiene <i>vīgintī</i> '20' con terminación de dual). Actualmente el dual ha desaparecido de todas las ramas de la familia indoeuropea, excepto entre las <a href="/wiki/Lenguas_eslavas" title="Lenguas eslavas">lenguas eslavas</a>.
</p>
<p>Se ha reconstruido al antecesor de todo el indoeuropeo no anatolio (pIE-II) como una lengua en la que habrían existido tres números, como en las ramas indoirania y griega. Sin embargo, el anatolio solo testimonia dos números, por lo que probablemente el protoindoeuropeo común (pIE-I) habría sido una lengua con solo dos números, siendo la creación del dual una innovación posterior del indoeuropeo no anatolio.
</p>
<p>Las lenguas indoeuropeas más antiguas de todas las ramas de la familia (<a href="/wiki/Griego_mic%C3%A9nico" title="Griego micénico">griego micénico</a>, <a href="/wiki/Idioma_hitita" title="Idioma hitita">hitita</a>, <a href="/wiki/S%C3%A1nscrito" title="Sánscrito">sánscrito</a>, <a href="/wiki/Lat%C3%ADn" title="Latín">latín</a>, antiguo irlandés, <a href="/wiki/Antiguo_eslavo_eclesi%C3%A1stico" title="Antiguo eslavo eclesiástico">eslavo eclesiástico</a>, ...) son <a class="mw-redirect" href="/wiki/Lenguas_flexivas" title="Lenguas flexivas">lenguas flexivas</a> con un sistema de 5 a 8 <a href="/wiki/Caso_(gram%C3%A1tica)" title="Caso (gramática)">casos morfológicos</a>. El número de casos del <a href="/wiki/Idioma_protoindoeuropeo" title="Idioma protoindoeuropeo">protoindoeuropeo</a> es materia de debate porque no está claro que el sistema máximo de casos con caso <a href="/wiki/Caso_nominativo" title="Caso nominativo">nominativo</a>, <a href="/wiki/Caso_vocativo" title="Caso vocativo">vocativo</a>, <a href="/wiki/Caso_acusativo" title="Caso acusativo">acusativo</a>, <a href="/wiki/Caso_genitivo" title="Caso genitivo">genitivo</a>, <a href="/wiki/Caso_dativo" title="Caso dativo">dativo</a>, <a class="mw-redirect" href="/wiki/Caso_ablativo" title="Caso ablativo">ablativo</a>, <a href="/wiki/Caso_locativo" title="Caso locativo">locativo</a> e <a href="/wiki/Caso_instrumental" title="Caso instrumental">instrumental</a> que encontramos en sánscrito se remonte enteramente al estadio más antiguo reconstruible. De hecho, algunos autores argumentan que existen residuos de un pre-protoindoeuropeo no flexivo anterior al protoindoeuropeo común.<sup class="reference separada" id="cite_ref-6"><a href="#cite_note-6"><span class="corchete-llamada">[</span>6<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Muchas lenguas indoeuropeas modernas, sin embargo, han perdido gran parte del sistema de casos y la conjugación que caracterizaban a las lenguas indoeuropeas más antiguas. Así, entre las lenguas romances, derivadas del latín, solo el <a href="/wiki/Idioma_rumano" title="Idioma rumano">rumano</a> conserva un sistema reducido de casos. Las <a href="/wiki/Lenguas_germ%C3%A1nicas" title="Lenguas germánicas">lenguas germánicas</a> igualmente han reducido el número de casos con formas distintivas, habiendo desaparecido en inglés por completo las marcas de caso específicas en el sustantivo. Las <a href="/wiki/Lenguas_indoiranias" title="Lenguas indoiranias">lenguas indoiranias</a> han sufrido así mismo una acusada disminución del número de casos. El <a href="/wiki/Idioma_indost%C3%A1nico" title="Idioma indostánico">hindi-urdu</a> posee un sistema de solo tres casos: directo o nominativo, vocativo y <a href="/wiki/Caso_oblicuo" title="Caso oblicuo">oblicuo</a> o <a href="/wiki/Caso_preposicional" title="Caso preposicional">preposicional</a>. Una situación similar se da en muchas lenguas iranias, como el <a href="/wiki/Idioma_past%C3%BAn" title="Idioma pastún">pastún</a> de Afganistán. El <a href="/wiki/Idioma_griego" title="Idioma griego">griego moderno</a> también ha reducido el número de casos respecto al <a class="mw-redirect" href="/wiki/Griego_cl%C3%A1sico" title="Griego clásico">griego clásico</a>, pero junto con las <a href="/wiki/Lenguas_eslavas" title="Lenguas eslavas">lenguas eslavas</a> y el <a href="/wiki/Idioma_lituano" title="Idioma lituano">lituano</a> forma parte de las lenguas indoeuropeas con una flexión nominal con mayor número de casos distintos.
</p>
<p>El sistema verbal de la mayoría de las ramas del indoeuropeo parece haber sufrido más cambios que el sistema de <a href="/wiki/Flexi%C3%B3n_(ling%C3%BC%C3%ADstica)" title="Flexión (lingüística)">flexión nominal</a>. Por esa razón, la reconstrucción se ha basado más en las terminaciones y las <a href="/wiki/Alomorfo" title="Alomorfo">marcas morfológicas</a> que en las categorías representadas.
</p>
<p>Antes del descubrimiento de las <a href="/wiki/Lenguas_anatolias" title="Lenguas anatolias">lenguas anatolias</a> y su parentesco con las lenguas indoeuropeas, el sistema verbal reconstruido para el protoindoeuropeo se basaba ampliamente en el <a href="/wiki/Idioma_griego" title="Idioma griego">griego</a> y en el <a href="/wiki/S%C3%A1nscrito" title="Sánscrito">sánscrito</a>. Este sistema reconstruido constaría de:
</p>
<p>Este sistema máximo, llamado modelo grecoario o indogriego, fue considerado el resultado de innovaciones tardías cuando se conoció mejor el sistema verbal del anatolio. El sistema verbal del indoeuropeo más antiguo resulta, sin embargo, de difícil reconstrucción, ya que el anatolio presenta un sistema verbal mucho más simple y es, por tanto, imposible distinguir hasta qué punto se debe a pérdida de modos o tiempos o hasta qué punto el sistema de las lenguas con una conjugación más amplia es el resultado de innovaciones.
</p>
<p>En las lenguas modernas, especialmente en las europeas, han aparecido numerosas formas verbales basadas en <a href="/wiki/Verbo_auxiliar" title="Verbo auxiliar">verbos auxiliares</a> y <a href="/wiki/Per%C3%ADfrasis_verbal" title="Perífrasis verbal">perífrasis</a>. Así, las lenguas romances y las germánicas, como el inglés o el alemán, han perdido las formas sintéticas de la voz pasiva y las formas de perfecto, presentes en lenguas antiguas como el <a href="/wiki/Lat%C3%ADn" title="Latín">latín</a> o el <a href="/wiki/Idioma_g%C3%B3tico" title="Idioma gótico">gótico</a>, habiendo sido substituidas con formas perifrásticas con los verbos 'ser' y 'haber'.
</p>
<p>El léxico común heredado es la evidencia más clara del parentesco genético entre las lenguas indoeuropeas. <p>Tradicionalmente, el indoeuropeo se clasificaba en dos grupos: <a href="/wiki/Satem" title="Satem">lenguas satem</a> y <a href="/wiki/Centum" title="Centum">lenguas centum</a>, que recibieron esos nombres dependiendo de si la serie de fonemas velares protoindoeuropeos /*k, *g, *g<sup>h</sup>/ se palatalizaba (<i>*kntom</i> '[el número] 100' es en avéstico <i>satem</i>) o no (<i>*kntom</i> dio en latín <i>centum</i>). Sin embargo, hoy en día no se concede apenas importancia al cambio (uno entre tantos), ni se cree que sea un criterio sólido que clasifique adecuadamente a las lenguas indoeuropeas. De hecho, varias clasificaciones que tratan de reconstruir el <a href="/wiki/Clad%C3%ADstica" title="Cladística">árbol cladístico</a> de las lenguas indoeuropeas ni siquiera consideran a las lenguas <i>satem</i> como una rama propiamente.<sup class="reference separada" id="cite_ref-8"><a href="#cite_note-8"><span class="corchete-llamada">[</span>8<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-9"><a href="#cite_note-9"><span class="corchete-llamada">[</span>9<span class="corchete-llamada">]</span></a></sup>​ Eso sugiere que la palatalización se extendió entre lo que eran diferentes ramas de la familia.
</p>
<p>La primera rama que se separó del tronco común fue la <a href="/wiki/Lenguas_anatolias" title="Lenguas anatolias">rama anatolia</a> y algo más tarde la <a class="mw-redirect" href="/wiki/Idioma_tocario" title="Idioma tocario">rama tocaria</a>, ambas ramas están actualmente extintas (sin descendientes hablados actualmente). Estas dos ramas no presentan la típica palatalización de las lenguas <i>satem</i>, incluso cuando el tocario es una rama "oriental". De hecho es la única lengua oriental que no palataliza, lo cual sugiere que la palatalización en la que se basa la división <i>centum</i> / <i>satem</i> es relativamente tardía. Las subdivisiones restantes son algo más discutidas, aunque parece claro que el griego, el armenio y probablemente otras <a href="/wiki/Lenguas_paleobalc%C3%A1nicas" title="Lenguas paleobalcánicas">lenguas paleobalcánicas</a> formarían juntas una subdivisión. Esta rama greco-armenia incluye tanto lenguas <i>satem</i> como lenguas <i>centum</i>, razón por la cual se considera que la división <i>satem</i>-<i>centum</i> no es adecuada, dado que ninguna de las dos subdivisiones serían <a href="/wiki/Clado" title="Clado">ramas cladísticas</a> propiamente dichas.
</p>
<p>Las relaciones internas entre estos grupos de último nivel son algo más complicadas y polémicas y todavía existen discrepancias menores. Por ejemplo, aunque universalmente se reconoce una especial relación entre el <a href="/wiki/Lenguas_griegas" title="Lenguas griegas">grupo griego</a> y algunas <a href="/wiki/Lenguas_paleobalc%C3%A1nicas" title="Lenguas paleobalcánicas">lenguas paleobalcánicas</a> (en particular el <a href="/wiki/Idioma_armenio" title="Idioma armenio">armenio</a>), no está claro por ejemplo si las <a href="/wiki/Lenguas_indoarias" title="Lenguas indoarias">lenguas indoarias</a> se formaron por una división de un hipotético proto-indoeslavo o, por el contrario, deben considerarse el resultado de la escisión de un hipotético proto-indogriego. Y problemas similares se encuentran con las lenguas germánicas, las lenguas eslavas, las lenguas celtas o las lenguas itálicas.
</p>
<p>Recientemente han aparecido tres intentos de clasificación interna agrupando los subgrupos básicos anteriores: El <a href="/wiki/%C3%81rbol_filogen%C3%A9tico" title="Árbol filogenético">árbol filogenético</a> de Gray-Atkinson (<i>The 'New Zealand' family tree</i>, 2003), el árbol filogenético de Ringe-Warnow-Taylor (<i>The 'Pennsylvania' family tree</i>, 2002) y el árbol filogenético de Gamallo-Pichel-Alegria (<i>Measuring Language Distance of Indoeuropean Languages</i>, 2020). El primero se basa estrictamente en léxico compartido y substituido, el segundo se basa en <a href="/wiki/Isoglosa" title="Isoglosa">isoglosas</a> fonológicas y morfológicas, el tercero en isoglosas y léxico derivado del protoindoeuropeo. Aunque ambas clasificaciones presentan algunos puntos comunes, también difieren de manera importante en otros detalles. <p>Existen dudas de que las lenguas satem constituyan una unidad filogenética válida dentro del indoeuropeo, como una vez se pensó. Si los árboles filogenéticos propuestos por Ringe y Warnow son correctos, las lenguas satem constituirían una unidad filogenética (a diferencia de las lenguas centum, que serían el resto de lenguas que no formarían un grupo filogenético válido).
</p>
<p>Las lenguas centum no constituyen una <a class="mw-redirect" href="/wiki/Filog%C3%A9nesis" title="Filogénesis">unidad filogenética</a> válida dentro del indoeuropeo, como una vez se pensó. Ya que entre estas lenguas se encuentra tanto el grupo anatolio que fue la primera rama en diferenciarse del indoeuropeo como lenguas que solo más tarde se diferenciarían entre sí. Dicho de otra manera, no sería posible reconstruir un "protocentum" que por diversificación hubiera dado lugar a las lenguas centum que fuera diferente del protoindoeuropeo común. <p>Actualmente no existe una evidencia incontrovertible de que las lenguas indoeuropeas muestren un parentesco claro con lenguas de otras familias, aunque existe un cierto número de propuestas tentativas que sugieren que es posible reconocer el parentesco lejano de las lenguas indoeuropeas con otras familias lingüísticas de <a href="/wiki/Eurasia" title="Eurasia">Eurasia</a>. La <a href="/wiki/Macrofamilia_nostr%C3%A1tica" title="Macrofamilia nostrática">hipótesis nostrática</a> (Pedersen, <a href="/wiki/Vladislav_%C3%8Dlich-Sv%C3%ADtych" title="Vladislav Ílich-Svítych">Ílich-Svítych</a> y Dolgopolski) y la hipótesis euroasiática de Greenberg sostienen que las <a href="/wiki/Lenguas_ur%C3%A1licas" title="Lenguas urálicas">lenguas urálicas</a>, las <a href="/wiki/Lenguas_afroasi%C3%A1ticas" title="Lenguas afroasiáticas">lenguas afroasiáticas</a> y otras muestran un parentesco reconocible con el <a class="mw-redirect" href="/wiki/Idioma_proto-indoeuropeo" title="Idioma proto-indoeuropeo">protoindoeuropeo</a> y que es posible reconstruir parcialmente la <a href="/wiki/Protolengua" title="Protolengua">protolengua</a> de la que descienden estas familias.
</p>
<p>Sin embargo, estas hipótesis han encontrado un alto grado de críticas<sup class="reference separada" id="cite_ref-15"><a href="#cite_note-15"><span class="corchete-llamada">[</span>15<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-16"><a href="#cite_note-16"><span class="corchete-llamada">[</span>16<span class="corchete-llamada">]</span></a></sup>​ y no tienen aceptación general actualmente, aunque los partidarios de estas hipótesis han continuado con el trabajo comparativo en favor de la misma.
</p>
