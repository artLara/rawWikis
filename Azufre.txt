<p>El <b>azufre</b> es un <a href="/wiki/Elemento_qu%C3%ADmico" title="Elemento químico">elemento químico</a> de <a href="/wiki/N%C3%BAmero_at%C3%B3mico" title="Número atómico">número atómico</a> <b>16</b> y símbolo <b>S</b> (del <a href="/wiki/Lat%C3%ADn" title="Latín">latín</a> <i>sulphur</i>). Es un <a href="/wiki/No_metal" title="No metal">no metal</a> abundante con un color amarillo característico.<sup class="reference separada" id="cite_ref-1"><a href="#cite_note-1"><span class="corchete-llamada">[</span>1<span class="corchete-llamada">]</span></a></sup>​ Dicho elemento es generado en <a href="/wiki/Estrella" title="Estrella">estrellas masivas</a> en las que predominan temperaturas que provocan la fusión entre un núcleo de <a href="/wiki/Silicio" title="Silicio">silicio</a> y otro de <a href="/wiki/Helio" title="Helio">helio</a> en un proceso denominado <a href="/wiki/Nucleos%C3%ADntesis_de_supernovas" title="Nucleosíntesis de supernovas">nucleosíntesis de supernovas</a>.<sup class="reference separada" id="cite_ref-2"><a href="#cite_note-2"><span class="corchete-llamada">[</span>2<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El azufre se encuentra en forma nativa en regiones <a href="/wiki/Volc%C3%A1n" title="Volcán">volcánicas</a> y en sus formas reducidas formando <a href="/wiki/Sulfuro" title="Sulfuro">sulfuros</a> y <a class="new" href="/w/index.php?title=Sulfosales&amp;action=edit&amp;redlink=1" title="Sulfosales (aún no redactado)">sulfosales</a> o bien en sus formas oxidadas como <a href="/wiki/Sulfato" title="Sulfato">sulfatos</a>. Es un <a href="/wiki/Elemento_qu%C3%ADmico_esencial" title="Elemento químico esencial">elemento químico esencial</a> constituyente de los <a href="/wiki/Amino%C3%A1cido" title="Aminoácido">aminoácidos</a> <a class="mw-redirect" href="/wiki/Cisteina" title="Cisteina">cisteina</a> y <a href="/wiki/Metionina" title="Metionina">metionina</a><sup class="reference separada" id="cite_ref-3"><a href="#cite_note-3"><span class="corchete-llamada">[</span>3<span class="corchete-llamada">]</span></a></sup>​ y, por consiguiente, necesario para la síntesis de <a href="/wiki/Prote%C3%ADna" title="Proteína">proteínas</a> presentes en todos los organismos vivos. Se usa principalmente como <a href="/wiki/Fertilizante" title="Fertilizante">fertilizante</a>, pero también en la fabricación de <a href="/wiki/P%C3%B3lvora" title="Pólvora">pólvora</a>, <a href="/wiki/Laxante" title="Laxante">laxantes</a>, <a href="/wiki/F%C3%B3sforo_(utensilio)" title="Fósforo (utensilio)">fósforos</a> e <a href="/wiki/Insecticida" title="Insecticida">insecticidas</a>.
</p>
<p>Este <a href="/wiki/No_metal" title="No metal">no metal</a> tiene un color amarillento fuerte, amarronado o verde anaranjado y arde con llama de color azul, desprendiendo <a href="/wiki/Di%C3%B3xido_de_azufre" title="Dióxido de azufre">dióxido de azufre</a>. Es <a href="/wiki/Solubilidad" title="Solubilidad">insoluble</a> en <a href="/wiki/Agua" title="Agua">agua</a>, pero se disuelve en <a href="/wiki/Sulfuro_de_carbono" title="Sulfuro de carbono">disulfuro de carbono</a> y <a href="/wiki/Benceno" title="Benceno">benceno</a>. Es <a href="/wiki/Valencia_(qu%C3%ADmica)" title="Valencia (química)">multivalente</a>, y son comunes los estados de oxidación -3, 4, 34, -23.
</p>
<p>En todos los estados (<a href="/wiki/S%C3%B3lido" title="Sólido">sólido</a>, <a href="/wiki/L%C3%ADquido" title="Líquido">líquido</a> y <a href="/wiki/Gas" title="Gas">gaseoso</a>): según los químicos presenta <a href="/wiki/Alotrop%C3%ADa" title="Alotropía">formas alotrópicas</a> cuyas relaciones no son completamente conocidas. Las <a href="/wiki/Redes_de_Bravais" title="Redes de Bravais">estructuras cristalinas</a> más comunes son el <a href="/wiki/Octaedro" title="Octaedro">octaedro</a> ortorrómbico (azufre α) y el <a href="/wiki/Prisma_(geometr%C3%ADa)" title="Prisma (geometría)">prisma</a> monoclínico (azufre β), siendo la temperatura de transición de una a otra de 96 °C; en ambos casos el azufre se encuentra formando moléculas de S<sub>8</sub> con forma de anillo, y es la diferente disposición de estas moléculas la que provoca las distintas estructuras cristalinas. A temperatura ambiente, la transformación del azufre <a class="mw-redirect" href="/wiki/Monocl%C3%ADnico" title="Monoclínico">monoclínico</a> en <a class="mw-redirect" href="/wiki/Ortorr%C3%B3mbico" title="Ortorrómbico">ortorrómbico</a>, es más estable y muy lenta.
</p>
<p>Al <a href="/wiki/Fusi%C3%B3n_(cambio_de_estado)" title="Fusión (cambio de estado)">fundir</a> el azufre, se obtiene un líquido que fluye con facilidad formado por moléculas de S<sub>8</sub>. Sin embargo, si se calienta, el color se torna marrón algo rojizo, y se incrementa la <a href="/wiki/Viscosidad" title="Viscosidad">viscosidad</a>. Este comportamiento se debe a la ruptura de los anillos y la formación de largas cadenas de átomos de azufre, que pueden alcanzar varios miles de átomos de longitud, que se enredan entre sí disminuyendo la fluidez del líquido; el máximo de la viscosidad se alcanza en torno a los 200 °C. Enfriando rápidamente este líquido viscoso se obtiene una masa elástica, de consistencia similar a la de la <a href="/wiki/Goma" title="Goma">goma</a>, denominada «azufre plástico» (azufre γ) formada por cadenas que no han tenido tiempo de reordenarse para formar moléculas de S<sub>8</sub>; transcurrido cierto tiempo la masa pierde su elasticidad <a href="/wiki/Cristalizaci%C3%B3n" title="Cristalización">cristalizando</a> en el sistema rómbico. Estudios realizados con <a href="/wiki/Rayos_X" title="Rayos X">rayos X</a> muestran que esta forma deforme puede estar constituida por moléculas de S<sub>8</sub> con estructura de hélice espiral.
</p>
<p>En estado <a href="/wiki/Vapor_(estado)" title="Vapor (estado)">vapor</a> también forma moléculas de S<sub>8</sub>, pero a 780 °C ya se alcanza el equilibrio con moléculas diatómicas y por encima de aproximadamente 1800 °C la disociación es completa y se encuentran átomos de azufre.
</p>
<p>Además de en trozos, barras o polvo grueso, existe en el mercado una presentación en forma de polvo muy fino, llamada "Flor de azufre", que puede obtenerse por precipitación en medio líquido o por <a href="/wiki/Sublimaci%C3%B3n" title="Sublimación">sublimación</a> de su vapor sobre una placa metálica fría.
</p>
<p>El azufre se usa en multitud de <a href="/wiki/Industria" title="Industria">procesos industriales</a>, como la producción de <a href="/wiki/%C3%81cido_sulf%C3%BArico" title="Ácido sulfúrico">ácido sulfúrico</a> para <a class="mw-redirect" href="/wiki/Bater%C3%ADa_el%C3%A9ctrica" title="Batería eléctrica">baterías</a>, la fabricación de <a href="/wiki/P%C3%B3lvora" title="Pólvora">pólvora</a> y el <a class="mw-redirect" href="/wiki/Vulcanizado" title="Vulcanizado">vulcanizado</a> del <a href="/wiki/Caucho" title="Caucho">caucho</a>. 
</p>
<p>Los sulfitos se usan para blanquear el <a href="/wiki/Papel" title="Papel">papel</a> y en <a href="/wiki/F%C3%B3sforo_(utensilio)" title="Fósforo (utensilio)">fósforos</a>. El <a href="/wiki/Tiosulfato" title="Tiosulfato">tiosulfato</a> de <a href="/wiki/Sodio" title="Sodio">sodio</a> o <a href="/wiki/Amonio" title="Amonio">amonio</a> se emplea en la industria <a href="/wiki/Fotograf%C3%ADa" title="Fotografía">fotográfica</a> como «fijador», ya que disuelve el <a href="/wiki/Bromuro_de_plata" title="Bromuro de plata">bromuro de plata</a>; y el <a href="/wiki/Sulfato_de_magnesio" title="Sulfato de magnesio">sulfato de magnesio</a> (<a class="mw-redirect" href="/wiki/Sal_de_Epsom" title="Sal de Epsom">sal de Epsom</a>) tiene usos diversos como <a href="/wiki/Laxante" title="Laxante">laxante</a>, <a href="/wiki/Exfoliaci%C3%B3n_(mineralog%C3%ADa)" title="Exfoliación (mineralogía)">exfoliante</a>, o suplemento nutritivo para <a class="mw-redirect" href="/wiki/Planta" title="Planta">plantas</a>.
</p>
<p>También el azufre se emplea en la industria enológica como antiséptico. Uno de sus principales usos es como <a href="/wiki/Anh%C3%ADdrido_sulfuroso_en_vinos" title="Anhídrido sulfuroso en vinos">anhídrido sulfuroso</a>.
</p>
<p>El azufre tiene usos como <a href="/wiki/Fungicida" title="Fungicida">fungicida</a> y en la manufactura de <a href="/wiki/Fosfato" title="Fosfato">fosfatos</a> <a href="/wiki/Fertilizante" title="Fertilizante">fertilizantes</a> comunes.
</p>
<p>El azufre (del <a href="/wiki/Lat%C3%ADn" title="Latín">latín</a> <i>sulphur</i>, <i>sulfŭris</i>, vinculado con el <a href="/wiki/S%C3%A1nscrito" title="Sánscrito">sánscrito</a> <i>śulbāri</i>) es conocido desde la <a href="/wiki/Antig%C3%BCedad_cl%C3%A1sica" title="Antigüedad clásica">Antigüedad</a>, y ya los <a href="/wiki/Antiguo_Egipto" title="Antiguo Egipto">egipcios</a> lo utilizaban para purificar los <a href="/wiki/Templo" title="Templo">templos</a>.
</p>
<p>En el <i><a href="/wiki/G%C3%A9nesis" title="Génesis">Génesis</a></i> (19,24), los <a href="/wiki/Hebreos" title="Hebreos">hebreos</a> decían que Dios (<a class="mw-redirect" href="/wiki/Yahv%C3%A9" title="Yahvé">Yahvé</a>) hizo llover sobre <a href="/wiki/Sodoma" title="Sodoma">Sodoma</a> y <a href="/wiki/Gomorra" title="Gomorra">Gomorra</a> azufre y fuego desde el cielo.
</p>
<p><a href="/wiki/Homero" title="Homero">Homero</a> recomendaba, en el <a href="/wiki/Siglo_IX_a._C." title="Siglo IX a. C.">siglo <span style="font-variant:small-caps;text-transform:lowercase">IX</span> a. C.</a>, evitar la pestilencia mediante la quema de azufre (<i>zeio</i> en <a href="/wiki/Idioma_griego" title="Idioma griego">griego</a>, relacionado con <i><a href="/wiki/Teo" title="Teo">zeos</a>-<a href="/wiki/Zeus" title="Zeus">Zeus</a></i>).
</p>
<p>Según el <i>Diccionario sánscrito-inglés</i> (1899) de <a href="/wiki/Monier_Monier-Williams" title="Monier Monier-Williams">Monier Monier-Williams</a>, en <a href="/wiki/S%C3%A1nscrito" title="Sánscrito">sánscrito</a> al azufre se lo llamaba <i>śulbāri</i> (pronunciado <i>/shulbári/</i>), siendo <i>śulba</i> o <i>śulva:</i> ‘cobre’, y <i>a-rí</i> o <i>a-rís:</i> ‘enemigo, envidioso’ (<a class="new" href="/w/index.php?title=Literalmente&amp;action=edit&amp;redlink=1" title="Literalmente (aún no redactado)">lit.</a> ‘no liberal’).
</p>
<p>En <span class="plainlinks"><a class="external text" href="https://www.biblegateway.com/passage/?search=Apocalipsis+20%3A10&amp;version=DHH" rel="nofollow">Apocalipsis 20:10</a></span> se dice que el <a href="/wiki/Diablo" title="Diablo">diablo</a> será lanzado a un lago de fuego y azufre.
</p>
<p>Durante toda la <a href="/wiki/Edad_Media" title="Edad Media">Edad Media</a> se vinculó a <a href="/wiki/Satan%C3%A1s" title="Satanás">Satanás</a> con los olores de los gases sulfurosos que se desprendían de los <a href="/wiki/Volc%C3%A1n" title="Volcán">volcanes</a>, que se suponían eran entradas a los <a href="/wiki/Infierno" title="Infierno">infiernos</a> subterráneos).
</p>
<p>El azufre es un elemento muy abundante en la corteza terrestre, se encuentra en grandes cantidades combinado en forma de <a class="mw-redirect" href="/wiki/Sulfuros" title="Sulfuros">sulfuros</a> como (<a href="/wiki/Pirita" title="Pirita">pirita</a> y <a href="/wiki/Galena" title="Galena">galena</a>) y de sulfatos como (<a href="/wiki/Yeso" title="Yeso">yeso</a>). 
En forma nativa se encuentra en las cercanías de aguas termales, zonas volcánicas y en minas de <a href="/wiki/Cinabrio" title="Cinabrio">cinabrio</a>, <a href="/wiki/Galena" title="Galena">galena</a>, <a href="/wiki/Esfalerita" title="Esfalerita">esfalerita</a> y <a href="/wiki/Estibina_(mineral)" title="Estibina (mineral)">estibina</a>, y en Luisiana (Estados Unidos, primer productor mundial) se extrae mediante el proceso <a class="new" href="/w/index.php?title=Herman_Frasch&amp;action=edit&amp;redlink=1" title="Herman Frasch (aún no redactado)">Frasch</a> consistente en inyectar vapor de agua sobrecalentado para fundir el azufre que posteriormente es bombeado al exterior utilizando aire comprimido. También se obtiene separándolo del <a href="/wiki/Gas_natural" title="Gas natural">gas natural</a>, si bien su obtención anteriormente era a partir de depósitos de azufre puro impregnado en <a href="/wiki/Cenizas" title="Cenizas">cenizas</a> volcánicas (<a href="/wiki/Italia" title="Italia">Italia</a>, y más recientemente <a href="/wiki/Argentina" title="Argentina">Argentina</a>).
</p>
<p>También está presente, en pequeñas cantidades, en <a href="/wiki/Combustible_f%C3%B3sil" title="Combustible fósil">combustibles fósiles</a> (<a href="/wiki/Carb%C3%B3n" title="Carbón">carbón</a> y <a href="/wiki/Petr%C3%B3leo" title="Petróleo">petróleo</a>) cuya combustión produce <a href="/wiki/Di%C3%B3xido_de_azufre" title="Dióxido de azufre">dióxido de azufre</a> que combinado con agua produce la <a href="/wiki/Lluvia_%C3%A1cida" title="Lluvia ácida">lluvia ácida</a>; para evitarlo, las legislaciones de los países industrializados exigen la reducción del contenido de azufre de los combustibles, constituyendo este azufre, posteriormente refinado, un porcentaje importante del total producido en el mundo. También se extrae del <a href="/wiki/Gas_natural" title="Gas natural">gas natural</a> que contiene sulfuro de hidrógeno que una vez separado se quema para obtener azufre:
</p>
<p>El color distintivo de <a class="mw-redirect" href="/wiki/%C3%8Do_(luna)" title="Ío (luna)">Ío</a>, la luna volcánica de <a href="/wiki/J%C3%BApiter_(planeta)" title="Júpiter (planeta)">Júpiter</a>, se debe a la presencia de diferentes formas de azufre en estado líquido, sólido y gaseoso. El azufre se encuentra, además, en varios tipos de <a href="/wiki/Meteorito" title="Meteorito">meteoritos</a>, y se cree que la mancha oscura que puede observarse cerca del cráter <a href="/wiki/Luna" title="Luna">lunar</a> <a href="/wiki/Aristarco_(cr%C3%A1ter)" title="Aristarco (cráter)">Aristarco</a> puede ser un depósito de azufre.
</p>
<p>Fuente: <a class="external text" href="https://pubs.usgs.gov/periodicals/mcs2021/mcs2021-sulfur.pdf" rel="nofollow">USGS</a>.
</p>
<p>La variedad de compuestos azufrados son consecuencia de una gran variedad de posibles <a class="mw-redirect" href="/wiki/Estados_de_oxidaci%C3%B3n" title="Estados de oxidación">estados de oxidación</a> del átomo de azufre. En la Tabla<sup class="reference separada" id="cite_ref-4"><a href="#cite_note-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​ se recojen algunos ejemplos de familias de compuestos azufrados, en función del estado de oxidación del azufre.
</p>
<p>El olfato humano en general tiene una sensibilidad muy alta a los compuestos con azufre, con olores que resultan ser desagradables, de forma que detecta estos compuestos incluso cuando se encuentran en cantidades muy pequeñas. Así, por ejemplo, los olores a los que da lugar la descomposición de la <a href="/wiki/Materia_org%C3%A1nica" title="Materia orgánica">materia orgánica</a> tienen su origen en que en su composición se encuentran compuestos azufrados, tales como proteínas que contienen aminoácidos con azufre (metionina, cisteina, cistina), que contienen azufre. 
</p>
<p>El azufre disuelto en agua es ácido (pK<sub>a1</sub> = 7,00, pK<sub>a2</sub> = 12,92) y reacciona con los <a href="/wiki/Metal" title="Metal">metales</a>. Los sulfuros metálicos se encuentran en la naturaleza, sobre todo el de <a href="/wiki/Hierro" title="Hierro">hierro</a> (<a href="/wiki/Pirita" title="Pirita">pirita</a>) que puede presentar <a href="/wiki/Resistencia_negativa" title="Resistencia negativa">resistencia negativa</a> y la <a href="/wiki/Galena" title="Galena">galena</a>, <a class="mw-redirect" href="/wiki/Sulfuro_de_plomo" title="Sulfuro de plomo">sulfuro de plomo</a> que es un <a href="/wiki/Semiconductor" title="Semiconductor">semiconductor</a> natural que fue usado como <a href="/wiki/Rectificador" title="Rectificador">rectificador</a>.
</p>
<p>El nitruro de azufre polímero (SN)<sub>x</sub>, sintetizado en <a href="/wiki/1975" title="1975">1975</a> por <a href="/wiki/Alan_G._MacDiarmid" title="Alan G. MacDiarmid">Alan G. MacDiarmid</a> y <a href="/wiki/Alan_J._Heeger" title="Alan J. Heeger">Alan J. Heeger</a>, presenta propiedades metálicas, a pesar de estar constituido por no metales, e inusuales propiedades eléctricas y ópticas. Este trabajo sirvió de base para el posterior desarrollo, con <a href="/wiki/Hideki_Shirakawa" title="Hideki Shirakawa">Hideki Shirakawa</a>, de plásticos conductores y semiconductores que motivó la concesión del <a href="/wiki/Premio_Nobel_de_Qu%C3%ADmica" title="Premio Nobel de Química">Nobel de Química</a>, en 2000, a los tres investigadores.
</p>
<p>Los <a href="/wiki/%C3%93xido" title="Óxido">óxidos</a> más importantes son el <a href="/wiki/Di%C3%B3xido_de_azufre" title="Dióxido de azufre">dióxido de azufre</a>, SO<sub>2</sub> (formado por la combustión del azufre) que en agua forma una solución de <a href="/wiki/%C3%81cido_sulfuroso" title="Ácido sulfuroso">ácido sulfuroso</a>, y el trióxido de azufre, SO<sub>3</sub>, que en solución forma el <a href="/wiki/%C3%81cido_sulf%C3%BArico" title="Ácido sulfúrico">ácido sulfúrico</a>; siendo los <a href="/wiki/Sulfito" title="Sulfito">sulfitos</a> y <a href="/wiki/Sulfato" title="Sulfato">sulfatos</a> las <a href="/wiki/Sal_(qu%C3%ADmica)" title="Sal (química)">sales</a> respectivas.
</p>
<p>Se conocen 25 <a href="/wiki/Is%C3%B3topo" title="Isótopo">isótopos</a> del azufre, de los cuales cuatro son estables: S-32 (95,02 %), S-33 (0,75 %), S-34 (4,21 %) y S-36 (0,025 %). Aparte del S-35, formado al incidir la <a href="/wiki/Radiaci%C3%B3n_c%C3%B3smica" title="Radiación cósmica">radiación cósmica</a> sobre el <a href="/wiki/Arg%C3%B3n" title="Argón">argón</a>-40 atmosférico y que tiene un <a href="/wiki/Periodo_de_semidesintegraci%C3%B3n" title="Periodo de semidesintegración">periodo de semidesintegración</a> de 87 días, los demás isótopos radiactivos son de vida corta.
</p>
<p>El <a class="mw-redirect" href="/wiki/Disulfuro_de_carbono" title="Disulfuro de carbono">disulfuro de carbono</a>, el sulfuro de hidrógeno (comúnmente conocido como <a href="/wiki/%C3%81cido_sulfh%C3%ADdrico" title="Ácido sulfhídrico">ácido sulfhídrico</a>), y el <a href="/wiki/Di%C3%B3xido_de_azufre" title="Dióxido de azufre">dióxido de azufre</a> deben manejarse con precaución.
</p>
<p>El <a href="/wiki/%C3%81cido_sulfh%C3%ADdrico" title="Ácido sulfhídrico">ácido sulfhídrico</a> y algunos de sus derivados, los <a class="mw-redirect" href="/wiki/Mercaptano" title="Mercaptano">mercaptanos</a>, son muy tóxicos, pudiendo llegar a provocar la muerte en el hombre a concentraciones en el ambiente muy reducidas (del mismo orden de las del <a class="mw-redirect" href="/wiki/%C3%81cido_cianh%C3%ADdrico" title="Ácido cianhídrico">ácido cianhídrico</a>, usado en las ejecuciones con <a href="/wiki/C%C3%A1mara_de_gas" title="Cámara de gas">cámara de gas</a> en <a href="/wiki/Estados_Unidos" title="Estados Unidos">Estados Unidos</a>, o mucho más bajas que las del <a href="/wiki/Mon%C3%B3xido_de_carbono" title="Monóxido de carbono">monóxido de carbono</a> para provocar la muerte, y que es origen de numerosas intoxicaciones mortales con <a href="/wiki/Calefactor" title="Calefactor">calefactores</a> por combustión en lugares mal ventilados). Aunque provisto de "propiedad de aviso" por ser muy maloliente, incluso en concentraciones muy por debajo de la que provoca la muerte, se ha de tener en cuenta que cuando su concentración se incrementa el sentido del olfato rápidamente se satura o se narcotiza, <i>desapareciendo</i> el olor, por lo que a las víctimas potenciales de la exposición les puede pasar desapercibida su presencia en el aire hasta que se manifiestan sus efectos, posiblemente mortales.
</p>
<p>Igual que sucede con las sales del <a class="mw-redirect" href="/wiki/%C3%81cido_cianh%C3%ADdrico" title="Ácido cianhídrico">ácido cianhídrico</a>, los cianuros, las sales del ácido sulfhídrico, los sulfuros, han de manejarse con sumo cuidado, evitando que puedan entrar en contacto con ácidos o disoluciones ácidas (incluso ligeramente ácidas), que darían lugar a la emanación del tóxico <a href="/wiki/%C3%81cido_sulfh%C3%ADdrico" title="Ácido sulfhídrico">ácido sulfhídrico</a>.
</p>
<p>El dióxido de azufre reacciona con el agua atmosférica para producir la lluvia ácida. Irrita las mucosidades y los ojos y provoca tos al ser inhalado.
</p>
<p>Los vapores del ácido sulfúrico pueden provocar <a href="/wiki/Hemorragia" title="Hemorragia">hemorragias</a> en los <a class="mw-redirect" href="/wiki/Pulmones" title="Pulmones">pulmones</a>, llenándolos de <a href="/wiki/Sangre" title="Sangre">sangre</a> con la consiguiente <a href="/wiki/Asfixia" title="Asfixia">asfixia</a>.
</p>
<p>En la <a href="/wiki/Orfebrer%C3%ADa" title="Orfebrería">orfebrería</a> el uso del azufre está ampliamente extendido, en particular para la <a class="mw-redirect" href="/wiki/Oxidaci%C3%B3n" title="Oxidación">oxidación</a> de la <a href="/wiki/Plata" title="Plata">plata</a>, es decir, para la creación de la <a class="mw-disambig" href="/wiki/P%C3%A1tina" title="Pátina">pátina</a> (de color negro).
</p>
<p>Existen varias técnicas para este fin; una de estas es mezclar azufre en polvo con una materia grasa —<a href="/wiki/Vaselina" title="Vaselina">vaselina</a>, <a href="/wiki/Aceite" title="Aceite">aceite</a>—, aplicar el ungüento sobre la pieza de plata y, mediante el uso de un soplete, calentar el <a href="/wiki/Metal" title="Metal">metal</a> y la mezcla, hasta que obtenga un color negruzco. Posteriormente, lavar con <a href="/wiki/Agua" title="Agua">agua</a> y <a href="/wiki/Jab%C3%B3n" title="Jabón">jabón</a> neutro. El patinado es duradero.
</p>
<p>De igual manera se puede patinar la plata con <a href="/wiki/Sulfato_de_potasio" title="Sulfato de potasio">sulfato de potasio</a> y agua.
</p>
