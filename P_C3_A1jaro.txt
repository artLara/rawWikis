<p>Fila 1: <a href="/wiki/Tauraco_erythrolophus" title="Tauraco erythrolophus">Turaco crestirrojo</a>, <a href="/wiki/Balaeniceps_rex" title="Balaeniceps rex">picozapato</a>, <a href="/wiki/Phaethon_lepturus" title="Phaethon lepturus">rabijunco común </a><br/>
Fila 2: <a href="/wiki/Haliaeetus_pelagicus" title="Haliaeetus pelagicus">Pigargo gigante</a>, <a href="/wiki/Balearica_regulorum" title="Balearica regulorum">grulla coronada cuelligrís</a>, <a href="/wiki/Pavo_cristatus" title="Pavo cristatus">pavo real común</a><br/>
Fila 3: <a href="/wiki/Columba_livia" title="Columba livia">Paloma bravía</a>, <a href="/wiki/Calypte_anna" title="Calypte anna">colibrí de Ana</a>, <a href="/wiki/Fratercula_arctica" title="Fratercula arctica">frailecillo atlántico</a><br/>
Fila 4: <a href="/wiki/Casuarius_casuarius" title="Casuarius casuarius">Casuario común</a>, <a href="/wiki/Trichoglossus_moluccanus" title="Trichoglossus moluccanus">lori arcoíris</a>, <a href="/wiki/Phoenicopterus_ruber" title="Phoenicopterus ruber">flamenco del Caribe</a><br/>
Fila 5: <a href="/wiki/Pygoscelis_papua" title="Pygoscelis papua">Pingüino juanito</a>, <a href="/wiki/Ardea_cinerea" title="Ardea cinerea">garza real</a>, <a href="/wiki/Sula_nebouxii" title="Sula nebouxii">piquero camanay</a><br/>
</p>
<p>Las <b>aves</b> son <a href="/wiki/Animalia" title="Animalia">animales</a> <a class="mw-redirect" href="/wiki/Vertebrado" title="Vertebrado">vertebrados</a>, de <a href="/wiki/Homeotermia" title="Homeotermia">sangre caliente</a>, que <a href="/wiki/Bipedestaci%C3%B3n" title="Bipedestación">caminan, saltan o se mantienen solo sobre las extremidades posteriores</a>,<sup class="reference separada" id="cite_ref-4"><a href="#cite_note-4"><span class="corchete-llamada">[</span>3<span class="corchete-llamada">]</span></a></sup>​ mientras que las extremidades anteriores han evolucionado hasta convertirse en <a class="mw-redirect" href="/wiki/Ala_(animal)" title="Ala (animal)">alas</a> que, al igual que muchas otras características <a href="/wiki/Anatom%C3%ADa_de_las_aves" title="Anatomía de las aves">anatómicas</a> únicas, les permiten, en la mayoría de los casos, <a href="/wiki/Vuelo_de_las_aves" title="Vuelo de las aves">volar</a>, si bien <a href="/wiki/Ave_no_voladora" title="Ave no voladora">no todas vuelan</a>. Tienen el cuerpo cubierto de <a href="/wiki/Pluma" title="Pluma">plumas</a> y, las aves <i>sensu stricto</i>, un <a href="/wiki/Pico_(zoolog%C3%ADa)" title="Pico (zoología)">pico</a> <a href="/wiki/Queratina" title="Queratina">córneo</a> sin <a href="/wiki/Diente" title="Diente">dientes</a>. Para reproducirse <a class="mw-redirect" href="/wiki/Ov%C3%ADparo" title="Ovíparo">ponen</a> <a href="/wiki/Huevo_(biolog%C3%ADa)" title="Huevo (biología)">huevos</a> que incuban hasta su <a href="/wiki/Eclosi%C3%B3n" title="Eclosión">eclosión</a>.
</p>
<p>Su grupo <a class="mw-redirect" href="/wiki/Taxon%C3%B3mico" title="Taxonómico">taxonómico</a> se denomina <a href="/wiki/Clase_(biolog%C3%ADa)" title="Clase (biología)">clase</a> <b>Aves</b> (la palabra es latina y está en plural, en singular sería <i>avis</i>)<sup class="reference separada" id="cite_ref-5"><a href="#cite_note-5"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>​ para la <a href="/wiki/Sistem%C3%A1tica" title="Sistemática">sistemática</a> clásica, pero en la sistemática <a class="mw-redirect" href="/wiki/Filogen%C3%A9tica" title="Filogenética">filogenética</a> actual este <a href="/wiki/Clado" title="Clado">clado</a> no tiene rango, y es incluido a su vez sucesivamente dentro de los clados: <a href="/wiki/Theropoda" title="Theropoda">Theropoda</a>, <a href="/wiki/Dinosauria" title="Dinosauria">Dinosauria</a>, <a href="/wiki/Archosauria" title="Archosauria">Archosauria</a>, <a href="/wiki/Sauropsida" title="Sauropsida">Sauropsida</a>, <a href="/wiki/Tetrapoda" title="Tetrapoda">Tetrapoda</a>, etc., aunque hay <a class="mw-selflink-fragment" href="#Clasificación_cladística">más anidamientos intermedios con denominación</a>.
</p>
<p>Las aves <a href="/wiki/Origen_de_las_aves" title="Origen de las aves">se originaron</a> a partir de <a href="/wiki/Theropoda" title="Theropoda">dinosaurios carnívoros bípedos</a> del <a href="/wiki/Jur%C3%A1sico" title="Jurásico">Jurásico</a>, hace entre ciento cincuenta y doscientos millones de años y, de hecho, son los únicos dinosaurios que sobrevivieron a la <a class="mw-redirect" href="/wiki/Extinci%C3%B3n_masiva_del_Cret%C3%A1cico-Terciario" title="Extinción masiva del Cretácico-Terciario">extinción masiva producida al final del Mesozoico</a>. Por tanto, la <a href="/wiki/Sangre_caliente" title="Sangre caliente">sangre caliente</a>, que es la característica más notable que comparten con los <a class="mw-redirect" href="/wiki/Mam%C3%ADfero" title="Mamífero">mamíferos</a>, es un resultado de <a href="/wiki/Evoluci%C3%B3n_convergente" title="Evolución convergente">evolución convergente</a>, pues no hay un ancestro común a ambos grupos que tuviera este rasgo.
Su <a href="/wiki/Evoluci%C3%B3n_de_las_aves" title="Evolución de las aves">evolución</a> dio lugar, tras una fuerte <a href="/wiki/Radiaci%C3%B3n_adaptativa" title="Radiación adaptativa">radiación</a>, a las más de diez mil <a class="mw-redirect" href="/wiki/Especies" title="Especies">especies</a> actuales<sup class="reference separada" id="cite_ref-6"><a href="#cite_note-6"><span class="corchete-llamada">[</span>5<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-Clements_7-0"><a href="#cite_note-Clements-7"><span class="corchete-llamada">[</span>6<span class="corchete-llamada">]</span></a></sup>​(más 153 <a class="mw-redirect" href="/wiki/Extinta" title="Extinta">extintas</a> en tiempos históricos).<sup class="reference separada" id="cite_ref-Clements_7-1"><a href="#cite_note-Clements-7"><span class="corchete-llamada">[</span>6<span class="corchete-llamada">]</span></a></sup>​ Las aves son los <a href="/wiki/Tetrapoda" title="Tetrapoda">tetrápodos</a> más diversos; sin embargo, tienen una gran homogeneidad morfológica en comparación con los <a class="mw-redirect" href="/wiki/Mam%C3%ADfero" title="Mamífero">mamíferos</a>. Las relaciones de parentesco de las <a href="/wiki/Familia_(biolog%C3%ADa)" title="Familia (biología)">familias</a> de aves no siempre pueden definirse por <a href="/wiki/Morfolog%C3%ADa_(biolog%C3%ADa)" title="Morfología (biología)">morfología</a>, pero con el <a class="mw-redirect" href="/wiki/Secuenciaci%C3%B3n_de_ADN" title="Secuenciación de ADN">análisis de ADN</a> comenzaron a esclarecerse.
</p>
<p>Las aves habitan en todos los <a href="/wiki/Bioma" title="Bioma">biomas</a> terrestres y también en todos los océanos. Su tamaño puede ser desde 6,4 cm en el <a href="/wiki/Mellisuga_helenae" title="Mellisuga helenae">colibrí zunzuncito</a> hasta 2,74 metros en el <a href="/wiki/Struthio_camelus" title="Struthio camelus">avestruz</a>. Los <a class="mw-selflink-fragment" href="#Comportamiento">comportamientos</a> son diversos y notables, como en la <a class="mw-redirect" href="/wiki/Nido_de_las_aves" title="Nido de las aves">anidación</a>, la <a class="mw-selflink-fragment" href="#Dieta_y_alimentación">alimentación de las crías</a>, las <a href="/wiki/Migraci%C3%B3n_de_las_aves" title="Migración de las aves">migraciones</a>, el <a class="mw-selflink-fragment" href="#Reproducción">apareamiento</a> y la tendencia a la <a class="mw-selflink-fragment" href="#sociabilidad">asociación en grupos</a>. La comunicación entre las aves es variable y puede implicar señales visuales, <a href="/wiki/Vocalizaci%C3%B3n_de_las_aves" title="Vocalización de las aves">llamadas y cantos</a>. Algunas emiten gran diversidad de sonidos y destacan por su <a class="mw-selflink-fragment" href="#Inteligencia">inteligencia</a> y por la capacidad de transmisión cultural de conocimientos a nuevas generaciones.
</p>
<p>El ser humano ha tenido una intensa relación con las aves. En la economía humana las <a href="/wiki/Ave_de_corral" title="Ave de corral">aves de corral</a> y las <a class="mw-redirect" href="/wiki/Cineg%C3%A9tica" title="Cinegética">cinegéticas</a> son fuentes de alimento. Las <a class="mw-redirect" href="/wiki/Canora" title="Canora">canoras</a> y los <a href="/wiki/Psittacidae" title="Psittacidae">loros</a> son populares como mascotas. Se usa el <a href="/wiki/Plum%C3%B3n" title="Plumón">plumón</a> de <a href="/wiki/Pato" title="Pato">patos</a> y <a href="/wiki/Ganso" title="Ganso">gansos</a> domésticos para rellenar almohadas y se cazaban muchas aves para adornar sombreros con sus <a href="/wiki/Pluma" title="Pluma">plumas</a>. El <a href="/wiki/Guano" title="Guano">guano</a> de las aves se usa en la fertilización de suelos. Algunas aves son reverenciadas o repudiadas por motivos religiosos, supersticiones o por prejuicios erróneos. Muchas son símbolos culturales y referencia frecuente para el arte. En los últimos quinientos años se han <a href="/wiki/Anexo:Aves_extintas" title="Anexo:Aves extintas">extinguido</a> más de ciento cincuenta especies como consecuencia de actividades humanas<sup class="reference separada" id="cite_ref-bli2_8-0"><a href="#cite_note-bli2-8"><span class="corchete-llamada">[</span>7<span class="corchete-llamada">]</span></a></sup>​ y, actualmente,<sup>[<i><a class="mw-redirect" href="/wiki/Wikipedia:REFTEMPREL" title="Wikipedia:REFTEMPREL">¿cuándo?</a></i>]</sup> son más de mil doscientas las especies de <a class="mw-redirect" href="/wiki/Aves_amenazadas" title="Aves amenazadas">aves amenazadas</a> que necesitan esfuerzos para su <a href="/wiki/Conservaci%C3%B3n_de_las_aves" title="Conservación de las aves">conservación</a>.<sup class="reference separada" id="cite_ref-bli_9-0"><a href="#cite_note-bli-9"><span class="corchete-llamada">[</span>8<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La especialidad de la <a href="/wiki/Zoolog%C3%ADa" title="Zoología">zoología</a> que estudia específicamente a las aves se denomina <a href="/wiki/Ornitolog%C3%ADa" title="Ornitología">ornitología</a>.
</p>
<p>Las evidencias fósiles y los numerosos análisis biológicos han demostrado que las aves son <a href="/wiki/Dinosauria" title="Dinosauria">dinosaurios</a> <a class="mw-redirect" href="/wiki/Ter%C3%B3podo" title="Terópodo">terópodos</a>.<sup class="reference separada" id="cite_ref-10"><a href="#cite_note-10"><span class="corchete-llamada">[</span>9<span class="corchete-llamada">]</span></a></sup>​ Más específicamente, son miembros de <a href="/wiki/Maniraptora" title="Maniraptora">Maniraptora</a>, un grupo de terópodos que incluye también, entre otros, a <a class="mw-redirect" href="/wiki/Dromaeosaurus" title="Dromaeosaurus">dromeosaurios</a> y <a href="/wiki/Oviraptoridae" title="Oviraptoridae">oviraptóridos</a>.<sup class="reference separada" id="cite_ref-11"><a href="#cite_note-11"><span class="corchete-llamada">[</span>10<span class="corchete-llamada">]</span></a></sup>​ A medida que los científicos han descubierto más terópodos no-avianos que están cercanamente relacionados con las aves, la distinción antes clara entre no-aves y aves se ha vuelto borrosa. Los recientes descubrimientos en la provincia de <a href="/wiki/Liaoning" title="Liaoning">Liaoning</a> del noreste de <a class="mw-redirect" href="/wiki/China" title="China">China</a> demuestran que muchos pequeños <a href="/wiki/Dinosaurios_con_plumas" title="Dinosaurios con plumas">dinosaurios terópodos tenían plumas</a>, lo que contribuye a esta ambigüedad de límites.<sup class="reference separada" id="cite_ref-12"><a href="#cite_note-12"><span class="corchete-llamada">[</span>11<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La visión del consenso en la <a href="/wiki/Paleontolog%C3%ADa" title="Paleontología">paleontología</a> contemporánea es que las aves son el grupo más cercano a los <a class="mw-redirect" href="/wiki/Deinonicosaurio" title="Deinonicosaurio">deinonicosaurios</a>, que incluyen a <a href="/wiki/Dromaeosauridae" title="Dromaeosauridae">dromeosáuridos</a> y <a class="mw-redirect" href="/wiki/Trood%C3%B3ntido" title="Troodóntido">troodóntidos</a>. Juntas, estas tres forman el grupo <a href="/wiki/Paraves" title="Paraves">Paraves</a>. El dromeosaurio <a href="/wiki/Basal" title="Basal">basal</a> <i><a href="/wiki/Microraptor" title="Microraptor">Microraptor</a></i> tiene características que pueden haberle permitido planear o volar. Los deinonicosaurios más basales eran muy pequeños. Esta evidencia eleva la posibilidad de que el ancestro de todos los paravianos pudiera haber sido arbóreo, o pudiera haber sido capaz de <a href="/wiki/Planeador" title="Planeador">planear</a>.<sup class="reference separada" id="cite_ref-AHTetal07_13-0"><a href="#cite_note-AHTetal07-13"><span class="corchete-llamada">[</span>12<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-xuetal2003_14-0"><a href="#cite_note-xuetal2003-14"><span class="corchete-llamada">[</span>13<span class="corchete-llamada">]</span></a></sup>​
<i><a href="/wiki/Archaeopteryx" title="Archaeopteryx">Archaeopteryx</a></i>, del <a href="/wiki/Jur%C3%A1sico_Superior" title="Jurásico Superior">Jurásico Superior</a>, es muy conocido como uno de los primeros <a href="/wiki/Forma_transicional" title="Forma transicional">fósiles transicionales</a> encontrados y aportó apoyo a la teoría de la <a href="/wiki/Evoluci%C3%B3n_biol%C3%B3gica" title="Evolución biológica">evolución</a> a finales del siglo XIX. <i>Archaeopteryx</i> tiene caracteres claramente reptilianos: dientes, dedos de la mano con garras y una larga cola similar a la de los lagartos, pero tiene alas finamente preservadas con plumas de vuelo idénticas a las de las aves modernas. No se considera un ancestro directo de las aves modernas, pero sí el más antiguo y primitivo miembro de Aves o <a href="/wiki/Avialae" title="Avialae">Avialae</a>, y está probablemente muy cercano al ancestro real. Sin embargo, contradiciendo lo anterior, se ha sugerido por otros autores que <i>Archaeopteryx</i> fue un dinosaurio que no era más cercano a Aves de lo que fueran otros grupos de dinosaurios,<sup class="reference separada" id="cite_ref-Thulborn1984_15-0"><a href="#cite_note-Thulborn1984-15"><span class="corchete-llamada">[</span>14<span class="corchete-llamada">]</span></a></sup>​ y que <i><a href="/wiki/Avimimus" title="Avimimus">Avimimus</a></i> es un ancestro más plausible de todas las aves que <i>Archaeopteryx</i>.<sup class="reference separada" id="cite_ref-Kurzanov1987_16-0"><a href="#cite_note-Kurzanov1987-16"><span class="corchete-llamada">[</span>15<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Han existido algunas controversias con respecto al origen de las aves. Una de las primeras se encontraba relacionada con el posible origen de las aves a partir de <a class="mw-redirect" href="/wiki/Arcosaurios" title="Arcosaurios">arcosaurios</a> y no de <a class="mw-redirect" href="/wiki/Dinosaurio" title="Dinosaurio">dinosaurios</a> (estos descienden de los primeros). El consenso académico no toma enserio estas ideas, y a partir de nuevos descubrimientos y estudios, se ha cimentado como hecho científico el origen de las aves a partir dinosaurios no avianos.<sup class="reference separada" id="cite_ref-17"><a href="#cite_note-17"><span class="corchete-llamada">[</span>16<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Los dinosaurios ornitisquios (que en <a href="/wiki/Idioma_griego" title="Idioma griego">griego</a> significa «con cadera de ave») compartían con las aves modernas la estructura de la cadera. Se sostiene que las aves se originaron de un tipo de dinosaurios <a class="mw-redirect" href="/wiki/Saurisquios" title="Saurisquios">saurisquios</a> (gr. «con cadera de lagarto»), y por lo tanto sus caderas evolucionaron <a href="/wiki/Analog%C3%ADa_(biolog%C3%ADa)" title="Analogía (biología)">independientemente</a>.<sup class="reference separada" id="cite_ref-18"><a href="#cite_note-18"><span class="corchete-llamada">[</span>17<span class="corchete-llamada">]</span></a></sup>​ De hecho, una estructura de cadera aviana evolucionó en una tercera ocasión entre un grupo de terópodos peculiares conocidos como <a href="/wiki/Therizinosauridae" title="Therizinosauridae">Therizinosauridae</a>. Algunos científicos todavía sugieren que las aves no son dinosaurios, sino que evolucionaron de arcosaurios primitivos. Aunque no poseen ningún tipo de evidencia revisada por pares, para respaldar sus ideas.<sup class="reference separada" id="cite_ref-19"><a href="#cite_note-19"><span class="corchete-llamada">[</span>18<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las aves se diversificaron en una amplia variedad de formas durante el periodo <a href="/wiki/Cret%C3%A1cico" title="Cretácico">Cretácico</a>.<sup class="reference separada" id="cite_ref-chiappe2007_20-0"><a href="#cite_note-chiappe2007-20"><span class="corchete-llamada">[</span>19<span class="corchete-llamada">]</span></a></sup>​ Muchos grupos retuvieron sus <a href="/wiki/Simplesiomorfia" title="Simplesiomorfia">características primitivas</a>, como alas con garras, y dientes, aunque los dientes se perdieron de forma independiente en algunos grupos de aves, incluidas las <a href="/wiki/Neornithes" title="Neornithes">aves modernas</a>. Mientras las formas más primitivas, como <i>Archaeopteryx</i> y <i><a href="/wiki/Jeholornis" title="Jeholornis">Jeholornis</a></i>, retuvieron la cola larga ósea de sus ancestros,<sup class="reference separada" id="cite_ref-chiappe2007_20-1"><a href="#cite_note-chiappe2007-20"><span class="corchete-llamada">[</span>19<span class="corchete-llamada">]</span></a></sup>​ las colas de las aves más avanzadas se acortaron con la aparición del hueso <a href="/wiki/Pig%C3%B3stilo" title="Pigóstilo">pigóstilo</a> en el <a href="/wiki/Clado" title="Clado">clado</a> <a href="/wiki/Pygostylia" title="Pygostylia">Pygostylia</a>.
</p>
<p>El primer linaje grande y diverso de aves de cola corta que evolucionó fue <a href="/wiki/Enantiornithes" title="Enantiornithes">Enantiornithes</a> (significa «aves opuestas»), llamado así porque la construcción de sus huesos del hombro estaba invertida respecto a la de las aves modernas. Enantiornithes ocupó un amplio espectro de <a href="/wiki/Nicho_ecol%C3%B3gico" title="Nicho ecológico">nichos ecológicos</a>, desde sondeadoras en la arena, como las limícolas, y comedoras de pescado, hasta las formas arborícolas y comedoras de semillas.<sup class="reference separada" id="cite_ref-chiappe2007_20-2"><a href="#cite_note-chiappe2007-20"><span class="corchete-llamada">[</span>19<span class="corchete-llamada">]</span></a></sup>​ Otros linajes más avanzados se especializaron también en comer pescado, como la subclase <a class="mw-redirect" href="/wiki/Ichthyornithes" title="Ichthyornithes">Ichthyornithes</a> («aves-pez») con apariencia de <a href="/wiki/Laridae" title="Laridae">gaviota</a>.<sup class="reference separada" id="cite_ref-21"><a href="#cite_note-21"><span class="corchete-llamada">[</span>20<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Un orden de aves marinas del <a class="mw-redirect" href="/wiki/Mesozoico" title="Mesozoico">Mesozoico</a>, <a href="/wiki/Hesperornithiformes" title="Hesperornithiformes">Hesperornithiformes</a>, se adaptó tan bien a la pesca en ambientes marinos que sus miembros perdieron la capacidad de volar y se hicieron primariamente acuáticos. A pesar de su especialización extrema, Hesperornithiformes incluye los parientes más cercanos de las aves modernas.<sup class="reference separada" id="cite_ref-chiappe2007_20-3"><a href="#cite_note-chiappe2007-20"><span class="corchete-llamada">[</span>19<span class="corchete-llamada">]</span></a></sup>​
</p>

<p><small><a href="/wiki/Filogenia" title="Filogenia">Filogenia</a> de las aves <a href="/wiki/Basal" title="Basal">basales</a> según Chiappe, 2007<sup class="reference separada" id="cite_ref-chiappe2007_20-4"><a href="#cite_note-chiappe2007-20"><span class="corchete-llamada">[</span>19<span class="corchete-llamada">]</span></a></sup>​</small>
</p>
<p>Algunos de los linajes <a href="/wiki/Basal" title="Basal">basales</a> de Neornithes comenzaron a evolucionar hacia el final del Cretácico, como demuestra el descubrimiento de <i><a class="mw-redirect" href="/wiki/Vegavis" title="Vegavis">Vegavis</a></i>,<sup class="reference separada" id="cite_ref-22"><a href="#cite_note-22"><span class="corchete-llamada">[</span>21<span class="corchete-llamada">]</span></a></sup>​ y se dividieron en dos linajes, los <a class="mw-redirect" href="/wiki/Superorden" title="Superorden">superódenes</a> <a href="/wiki/Palaeognathae" title="Palaeognathae">Palaeognathae</a> y <a href="/wiki/Neognathae" title="Neognathae">Neognathae</a>. En Palaeognathae se incluyen <a class="mw-redirect" href="/wiki/Tinamiformes" title="Tinamiformes">Tinamiformes</a> y <a href="/wiki/Struthioniformes" title="Struthioniformes">Struthioniformes</a>. Se acepta que la rama Neognathae se dividió antes de finalizar el Cretácico, cuando evolucionó el clado basal <a href="/wiki/Galloanserae" title="Galloanserae">Galloanserae</a> (que contiene <a href="/wiki/Anseriformes" title="Anseriformes">patos</a>, <a href="/wiki/Galliformes" title="Galliformes">gallos</a> y formas afines). No existe acuerdo sobre cuándo ocurrió la división múltiple de las demás neognatas, o clado <a href="/wiki/Neoaves" title="Neoaves">Neoaves</a>, si antes o después del <a class="mw-redirect" href="/wiki/Extinci%C3%B3n_masiva_del_Cret%C3%A1cico-Terciario" title="Extinción masiva del Cretácico-Terciario">evento de extinción del límite Cretácico-Terciario</a> cuando desaparecieron los demás dinosaurios.<sup class="reference separada" id="cite_ref-Ericson_23-0"><a href="#cite_note-Ericson-23"><span class="corchete-llamada">[</span>22<span class="corchete-llamada">]</span></a></sup>​ Este desacuerdo se debe en parte a la divergencia de las evidencias. La <a class="mw-redirect" href="/wiki/Dataci%C3%B3n_molecular" title="Datación molecular">datación molecular</a> sugiere una radiación cretácica, mientras que las evidencias <a href="/wiki/F%C3%B3sil" title="Fósil">fósiles</a> solo prueban la radiación en el <a class="mw-redirect" href="/wiki/Era_Cenozoica" title="Era Cenozoica">Terciario</a>. Los intentos de reconciliar las evidencias moleculares y fósiles han sido controvertidos.<sup class="reference separada" id="cite_ref-Ericson_23-1"><a href="#cite_note-Ericson-23"><span class="corchete-llamada">[</span>22<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-24"><a href="#cite_note-24"><span class="corchete-llamada">[</span>23<span class="corchete-llamada">]</span></a></sup>​ Fósiles de pingüinos primitivos de 61 millones de años de antigüedad han servido para hacer una calibración de la datación molecular que implica que el <a class="mw-redirect" href="/wiki/Grupo_corona" title="Grupo corona">grupo corona</a> Neoaves ya se había diversificado antes del evento de extinción Cretácico-Terciario. Además se puede estimar a partir de esto que por lo menos hace 72 millones de años ya había ocurrido la separación del linaje de las aves acuáticas superiores del de las limícolas. Luego esos linajes ya diversificados de las Neornithes actuales tuvieron durante el <a href="/wiki/Paleoceno" title="Paleoceno">Paleoceno</a> una <a href="/wiki/Radiaci%C3%B3n_evolutiva" title="Radiación evolutiva">radiación</a> adaptativa explosiva cuando se desarrollaron 16 nuevos órdenes. En el final del <a href="/wiki/Mioceno" title="Mioceno">Mioceno</a> ya existían la mayoría de los géneros de aves actuales. El número de especies de aves puede haber llegado hasta 21 000 hacia el principio del <a href="/wiki/Pleistoceno" title="Pleistoceno">Pleistoceno</a> (hace 1,5 millones de años), pero se redujo a cerca de la mitad debido a los cambios climáticos, las <a href="/wiki/Glaciaci%C3%B3n" title="Glaciación">glaciaciones</a>, y los intercambios faunísticos entre continentes.<sup class="reference separada" id="cite_ref-biogeography_25-0"><a href="#cite_note-biogeography-25"><span class="corchete-llamada">[</span>24<span class="corchete-llamada">]</span></a></sup>​ El número de especies de aves vivientes se establece entre unas 9800<sup class="reference separada" id="cite_ref-Clements_7-2"><a href="#cite_note-Clements-7"><span class="corchete-llamada">[</span>6<span class="corchete-llamada">]</span></a></sup>​ y 10 050.<sup class="reference separada" id="cite_ref-26"><a href="#cite_note-26"><span class="corchete-llamada">[</span>25<span class="corchete-llamada">]</span></a></sup>​ Se estima que cuando se termine la caracterización por <a class="mw-redirect" href="/wiki/Secuenciaci%C3%B3n_de_ADN" title="Secuenciación de ADN">secuenciación de ADN</a> de todas las aves podrán identificarse numerosas nuevas especies mediante identificación de formas <a class="mw-redirect" href="/wiki/Especie_cr%C3%ADptica" title="Especie críptica">crípticas</a> dentro de especies reconocidas basándose en las diferencias de ADN.<sup class="reference separada" id="cite_ref-biogeography_25-1"><a href="#cite_note-biogeography-25"><span class="corchete-llamada">[</span>24<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La facultad de vuelo fue decisiva en la extraordinaria diversificación de las especies de aves respecto a otros <a href="/wiki/Tetrapoda" title="Tetrapoda">tetrápodos</a> (lo mismo ocurrió con los insectos respecto a los demás <a class="mw-redirect" href="/wiki/Artr%C3%B3podo" title="Artrópodo">artrópodos</a> o con los <a class="mw-redirect" href="/wiki/Murci%C3%A9lago" title="Murciélago">murciélagos</a> respecto al resto de mamíferos). La llegada casual de algunos individuos a un territorio geográficamente aislado puede ser el origen de una nueva <a class="mw-redirect" href="/wiki/Poblaci%C3%B3n_(biolog%C3%ADa)" title="Población (biología)">población</a> que en el transcurso del tiempo acumule diferencias <a href="/wiki/Gen%C3%A9tica" title="Genética">genéticas</a> respecto a la población madre originaria, por <a href="/wiki/Deriva_gen%C3%A9tica" title="Deriva genética">azar</a> o por <a href="/wiki/Adaptaci%C3%B3n_biol%C3%B3gica" title="Adaptación biológica">adaptación</a> a nuevos ambientes mediante la acción de la <a href="/wiki/Selecci%C3%B3n_natural" title="Selección natural">selección natural</a>. Algunas islas han desarrollado avifaunas diferenciadas por <a class="mw-redirect" href="/wiki/Vicarianza" title="Vicarianza">vicarianza</a> a partir de especies colonizadoras, o diversificadas a partir de pocas especies que se adaptaron por <a href="/wiki/Radiaci%C3%B3n_evolutiva" title="Radiación evolutiva">radiación</a> a la explotación de distintos <a href="/wiki/Nicho_ecol%C3%B3gico" title="Nicho ecológico">nichos ecológicos</a> sin los competidores y predadores habituales de sus territorios de origen.<sup class="reference separada" id="cite_ref-biogeography_25-2"><a href="#cite_note-biogeography-25"><span class="corchete-llamada">[</span>24<span class="corchete-llamada">]</span></a></sup>​ Este es el caso de los <a href="/wiki/Pinz%C3%B3n_de_Darwin" title="Pinzón de Darwin">pinzones de Darwin</a>,<sup class="reference separada" id="cite_ref-27"><a href="#cite_note-27"><span class="corchete-llamada">[</span>26<span class="corchete-llamada">]</span></a></sup>​ en las <a href="/wiki/Islas_Gal%C3%A1pagos" title="Islas Galápagos">Galápagos</a>, de los <a class="mw-redirect" href="/wiki/Drepanidinae" title="Drepanidinae">mieleros hawaianos</a><sup class="reference separada" id="cite_ref-28"><a href="#cite_note-28"><span class="corchete-llamada">[</span>27<span class="corchete-llamada">]</span></a></sup>​ o de las <a href="/wiki/Vangidae" title="Vangidae">vangas</a> en <a href="/wiki/Madagascar" title="Madagascar">Madagascar</a>.<sup class="reference separada" id="cite_ref-29"><a href="#cite_note-29"><span class="corchete-llamada">[</span>28<span class="corchete-llamada">]</span></a></sup>​ La diferenciación evolutiva de nuevas especies de aves no se detiene, y puede, en ocasiones, ocurrir en tiempo relativamente breve, como se comprueba en las islas volcánicas de corta <a class="mw-redirect" href="/wiki/Historia_geol%C3%B3gica" title="Historia geológica">historia geológica</a>.
</p>
<p>Las aves junto con los <a class="mw-redirect" href="/wiki/Reptiles" title="Reptiles">reptiles</a> y <a class="mw-redirect" href="/wiki/Mam%C3%ADfero" title="Mamífero">mamíferos</a> se clasifican en un clado llamado <a href="/wiki/Amniota" title="Amniota">Amniota</a>, actualmente corroborado por los análisis genéticos y su desarrollo embrionario. El taxón <a href="/wiki/Reptilia" title="Reptilia">Reptilia</a> es un grupo <a href="/wiki/Parafil%C3%A9tico" title="Parafilético">parafilético</a> del cual se originan las aves y mamíferos, ya que incluye a sus ancestros directos sin <a href="/wiki/Pluma" title="Pluma">plumas</a> o <a class="mw-redirect" href="/wiki/Pelos" title="Pelos">pelos</a>, por ello se usan las clases <a href="/wiki/Sauropsida" title="Sauropsida">Sauropsida</a> y <a href="/wiki/Synapsida" title="Synapsida">Synapsida</a> por ser monofiléticas. De acuerdo a con las evidencias fósiles las aves evolucionaron de <a class="mw-redirect" href="/wiki/Dinosaurios" title="Dinosaurios">dinosaurios</a> <a href="/wiki/Theropoda" title="Theropoda">terópodos</a> durante el <a href="/wiki/Jur%C3%A1sico" title="Jurásico">Jurásico</a>, lo cual indica que las aves son un grupo de dinosaurios vivos que alcanzaron un gran éxito evolutivo. Este escenario también esta respaldado por las secuencias proteicas obtenidas de <i><a href="/wiki/Tyrannosaurus_rex" title="Tyrannosaurus rex">Tyrannosaurus rex</a></i> y <i><a href="/wiki/Brachylophosaurus_canadensis" title="Brachylophosaurus canadensis">Brachylophosaurus canadensis</a></i>. Sus parientes más cercanos vivientes son los <a class="mw-redirect" href="/wiki/Cocodrilos" title="Cocodrilos">cocodrilos</a> (orden <a href="/wiki/Crocodilia" title="Crocodilia">Crocodilia</a>) con quienes conforman el clado <a href="/wiki/Archosauria" title="Archosauria">Archosauria</a>, el cual no solo esta respaldado molecularmente sino también morfológicamente.<sup class="reference separada" id="cite_ref-30"><a href="#cite_note-30"><span class="corchete-llamada">[</span>29<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Los estudios genéticos revelan las siguientes relaciones filogenéticas para las aves con respecto a otros tetrápodos vivos (incluyendo las secuencias proteicas obtenidas de <i><a href="/wiki/Tyrannosaurus_rex" title="Tyrannosaurus rex">Tyrannosaurus rex</a></i> y <i><a href="/wiki/Brachylophosaurus_canadensis" title="Brachylophosaurus canadensis">Brachylophosaurus canadensis</a></i>). Puede notarse que las aves se agrupan en lo profundo del árbol reptiliano:<sup class="reference separada" id="cite_ref-31"><a href="#cite_note-31"><span class="corchete-llamada">[</span>30<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-32"><a href="#cite_note-32"><span class="corchete-llamada">[</span>31<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-33"><a href="#cite_note-33"><span class="corchete-llamada">[</span>32<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-34"><a href="#cite_note-34"><span class="corchete-llamada">[</span>33<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-35"><a href="#cite_note-35"><span class="corchete-llamada">[</span>34<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-36"><a href="#cite_note-36"><span class="corchete-llamada">[</span>35<span class="corchete-llamada">]</span></a></sup>​
</p>

<p>La primera clasificación científica de las aves se debe a <a href="/wiki/Francis_Willughby" title="Francis Willughby">Francis Willughby</a> y <a href="/wiki/John_Ray" title="John Ray">John Ray</a> en su libro <i>Ornithologiae</i>, publicado en 1676.<sup class="reference separada" id="cite_ref-37"><a href="#cite_note-37"><span class="corchete-llamada">[</span>36<span class="corchete-llamada">]</span></a></sup>​ <a href="/wiki/Carlos_Linneo" title="Carlos Linneo">Carlos Linneo</a> modificó aquel trabajo en 1758 para crear la clasificación taxonómica aún en uso.<sup class="reference separada" id="cite_ref-38"><a href="#cite_note-38"><span class="corchete-llamada">[</span>37<span class="corchete-llamada">]</span></a></sup>​ Las aves están categorizadas como una <a href="/wiki/Clase_(biolog%C3%ADa)" title="Clase (biología)">clase</a> homónima en la <a class="mw-redirect" href="/wiki/Taxonom%C3%ADa_de_Linneo" title="Taxonomía de Linneo">Taxonomía de Linneo</a>. En la <a href="/wiki/Clad%C3%ADstica" title="Cladística">taxonomía filogenética</a>, las aves se ubican en el <a href="/wiki/Clado" title="Clado">clado</a> <a href="/wiki/Theropoda" title="Theropoda">Theropoda</a> (dinosaurios carnívoros bípedos).<sup class="reference separada" id="cite_ref-Theropoda_39-0"><a href="#cite_note-Theropoda-39"><span class="corchete-llamada">[</span>38<span class="corchete-llamada">]</span></a></sup>​
</p>

</p>
<p>Las aves, junto a su orden hermano (<a href="/wiki/Crocodilia" title="Crocodilia">Crocodilia</a>), representan los únicos sobrevivientes del clado reptiliano <a href="/wiki/Archosauria" title="Archosauria">Archosauria</a>. <a class="mw-redirect" href="/wiki/Filogen%C3%A9tica" title="Filogenética">Filogenéticamente</a>, el clado Aves se define comúnmente como todos los descendientes del ancestro común más reciente de las aves modernas y de <i><a class="mw-redirect" href="/wiki/Archaeopteryx_lithographica" title="Archaeopteryx lithographica">Archaeopteryx lithographica</a></i>.<sup class="reference separada" id="cite_ref-40"><a href="#cite_note-40"><span class="corchete-llamada">[</span>39<span class="corchete-llamada">]</span></a></sup>​ <i><a href="/wiki/Archaeopteryx" title="Archaeopteryx">Archaeopteryx</a></i>, perteneciente al <a href="/wiki/Jur%C3%A1sico_Superior" title="Jurásico Superior">Jurásico Superior</a> (entre 156-150 millones de años), es el animal más antiguo hasta ahora conocido que se clasifica como un ave. Otros autores han definido Aves para agrupar solamente a las más modernas, excluyendo a la mayoría de los grupos de aves solamente conocidos por sus fósiles,<sup class="reference separada" id="cite_ref-41"><a href="#cite_note-41"><span class="corchete-llamada">[</span>40<span class="corchete-llamada">]</span></a></sup>​ en parte para evitar las incertidumbres sobre la ubicación de <i>Archaeopteryx</i> en relación con los animales tradicionalmente conocidos como dinosaurios terópodos.<sup class="reference separada" id="cite_ref-42"><a href="#cite_note-42"><span class="corchete-llamada">[</span>41<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-43"><a href="#cite_note-43"><span class="corchete-llamada">[</span>42<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-Prum2015_44-0"><a href="#cite_note-Prum2015-44"><span class="corchete-llamada">[</span>43<span class="corchete-llamada">]</span></a></sup>​ Algunos <a href="/wiki/Clado" title="Clado">clado</a> según Yuri, T. et al. (2013).<sup class="reference separada" id="cite_ref-45"><a href="#cite_note-45"><span class="corchete-llamada">[</span>44<span class="corchete-llamada">]</span></a></sup>​

<p><a href="/wiki/Cathartiformes" title="Cathartiformes">Cathartiformes</a> (Cóndores y zopilotes)
</p>
<p>Hasta la década de 1990, los buitres americanos han sido considerados tradicionalmente como miembros del orden <a href="/wiki/Falconiformes" title="Falconiformes">Falconiformes</a>.<sup class="reference separada" id="cite_ref-Sibley_and_Ahlquist_1991_52-0"><a href="#cite_note-Sibley_and_Ahlquist_1991-52"><span class="corchete-llamada">[</span>51<span class="corchete-llamada">]</span></a></sup>​ Sin embargo, a finales del siglo xx algunos ornitólogos argumentaron que los catártidos están más estrechamente emparentados con las <a class="mw-redirect" href="/wiki/Cig%C3%BCe%C3%B1as" title="Cigüeñas">cigüeñas</a>, en base al <a href="/wiki/Cariotipo" title="Cariotipo">cariotipo</a>,<sup class="reference separada" id="cite_ref-53"><a href="#cite_note-53"><span class="corchete-llamada">[</span>52<span class="corchete-llamada">]</span></a></sup>​ datos morfológicos,<sup class="reference separada" id="cite_ref-54"><a href="#cite_note-54"><span class="corchete-llamada">[</span>53<span class="corchete-llamada">]</span></a></sup>​ y comportamiento.<sup class="reference separada" id="cite_ref-55"><a href="#cite_note-55"><span class="corchete-llamada">[</span>54<span class="corchete-llamada">]</span></a></sup>​  Por lo tanto algunas autoridades los colocan en el orden <a href="/wiki/Ciconiiformes" title="Ciconiiformes">Ciconiiformes</a> junto con las cigüeñas y <a class="mw-redirect" href="/wiki/Garza" title="Garza">garzas</a>;<sup class="reference separada" id="cite_ref-Sibley_y_Monroe_1990_56-0"><a href="#cite_note-Sibley_y_Monroe_1990-56"><span class="corchete-llamada">[</span>55<span class="corchete-llamada">]</span></a></sup>​ Esta posición ha sido cuestionada como una simplificación desmesurada.<sup class="reference separada" id="cite_ref-57"><a href="#cite_note-57"><span class="corchete-llamada">[</span>56<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-58"><a href="#cite_note-58"><span class="corchete-llamada">[</span>57<span class="corchete-llamada">]</span></a></sup>​ Se comprobó también que un estudio inicial de la secuencia de ADN estaba basado en datos erróneos y fue posteriormente retirado.<sup class="reference separada" id="cite_ref-59"><a href="#cite_note-59"><span class="corchete-llamada">[</span>58<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-60"><a href="#cite_note-60"><span class="corchete-llamada">[</span>59<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-61"><a href="#cite_note-61"><span class="corchete-llamada">[</span>60<span class="corchete-llamada">]</span></a></sup>​ 
</p>
<p>En consecuencia, existe una tendencia reciente de colocar los buitres americanos en el orden de los <a href="/wiki/Cathartiformes" title="Cathartiformes">Cathartiformes</a>, un orden independiente que no está estrechamente emparentado ni con las aves de rapiña ni con las cigüeñas o garzas.<sup class="reference separada" id="cite_ref-62"><a href="#cite_note-62"><span class="corchete-llamada">[</span>61<span class="corchete-llamada">]</span></a></sup>​ Eso es el caso con la lista provisional de aves de América del Sur de la AOU, que coloca la familia Cathartidae en el orden de los Cathartiformes.<sup class="reference separada" id="cite_ref-63"><a href="#cite_note-63"><span class="corchete-llamada">[</span>62<span class="corchete-llamada">]</span></a></sup>​<a class="image" href="/wiki/Archivo:Vintage_Vulture_Drawing_white_background.jpg"><img alt="Vintage Vulture Drawing white background.jpg" data-file-height="410" data-file-width="205" decoding="async" height="60" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/42/Vintage_Vulture_Drawing_white_background.jpg/30px-Vintage_Vulture_Drawing_white_background.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/42/Vintage_Vulture_Drawing_white_background.jpg/45px-Vintage_Vulture_Drawing_white_background.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/42/Vintage_Vulture_Drawing_white_background.jpg/60px-Vintage_Vulture_Drawing_white_background.jpg 2x" width="30"/></a>

<p>La clasificación de las aves es un asunto disputado. Como resumen <a class="new" href="/w/index.php?title=Bradley_Livezey&amp;action=edit&amp;redlink=1" title="Bradley Livezey (aún no redactado)">Livezey</a> y <a class="new" href="/w/index.php?title=Richard_Zusi&amp;action=edit&amp;redlink=1" title="Richard Zusi (aún no redactado)">Zusi</a>,<sup class="reference separada" id="cite_ref-Theropoda_39-1"><a href="#cite_note-Theropoda-39"><span class="corchete-llamada">[</span>38<span class="corchete-llamada">]</span></a></sup>​ en 20 años desde 1988, han sido numerosas y divergentes las <a href="/wiki/Filogenia" title="Filogenia">filogenias</a> de Aves propuestas. Han estado basadas en metodologías diversas y aplicadas con amplitudes dispares, que generalmente han usado alternativamente <a href="/wiki/Morfolog%C3%ADa_(biolog%C3%ADa)" title="Morfología (biología)">morfología</a>, <a class="mw-redirect" href="/wiki/Secuenciaci%C3%B3n_de_ADN" title="Secuenciación de ADN">secuencias</a> de <a class="mw-redirect" href="/wiki/ADN_mitocondrial" title="ADN mitocondrial">ADN mitocondrial</a> o de <a class="mw-redirect" href="/wiki/ADN_nuclear" title="ADN nuclear">ADN nuclear</a>; con y sin <a class="mw-redirect" href="/wiki/An%C3%A1lisis_clad%C3%ADstico" title="Análisis cladístico">análisis cladísticos</a>. Sin embargo, se han producido resultados discordantes incluso cuando los estudios se han basado en metodologías de un mismo tipo.<sup class="reference separada" id="cite_ref-Theropoda_39-2"><a href="#cite_note-Theropoda-39"><span class="corchete-llamada">[</span>38<span class="corchete-llamada">]</span></a></sup>​ Estos estudios dejan un panorama contradictorio donde los resultados están en constante fluidez y no hay ninguna versión de filogenia ampliamente aceptada, por lo que no es posible dar aquí una filogenia más o menos definitiva. La situación parece en vías de solución con los nuevos descubrimientos de fósiles que están permitiendo estudios más refinados basados en morfología,<sup class="reference separada" id="cite_ref-Theropoda_39-3"><a href="#cite_note-Theropoda-39"><span class="corchete-llamada">[</span>38<span class="corchete-llamada">]</span></a></sup>​ con el perfeccionamiento de los métodos estadísticos, con la expansión de las evidencias moleculares por el aumento de los <a href="/wiki/Locus" title="Locus">loci</a> analizados y con la aplicación de estos métodos a más amplias representaciones de grupos taxonómicos.<sup class="reference separada" id="cite_ref-Theropoda_39-4"><a href="#cite_note-Theropoda-39"><span class="corchete-llamada">[</span>38<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-Hackett_65-0"><a href="#cite_note-Hackett-65"><span class="corchete-llamada">[</span>64<span class="corchete-llamada">]</span></a></sup>​ Con estas progresivas mejoras se considera cuestión de poco tiempo que se alcancen resultados que logren un consenso de la comunidad científica sobre la evolución de los órdenes y familias de aves modernas. Estos resultados quizás puedan ser acelerados con estudios basados en «evidencia total», con el análisis combinado de todas las fuentes de evidencia.<sup class="reference separada" id="cite_ref-Theropoda_39-5"><a href="#cite_note-Theropoda-39"><span class="corchete-llamada">[</span>38<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La clasificación filogenética de los grupos de <a href="/wiki/Neornithes" title="Neornithes">aves modernas</a> que a continuación se presenta, se basa en un <a href="/wiki/An%C3%A1lisis_moleculares_de_ADN" title="Análisis moleculares de ADN">estudio de las secuencias de ADN</a> de Hackett y colaboradores (2008),<sup class="reference separada" id="cite_ref-Hackett_65-1"><a href="#cite_note-Hackett-65"><span class="corchete-llamada">[</span>64<span class="corchete-llamada">]</span></a></sup>​ realizado mediante el examen de cerca de 32 <a href="/wiki/Kilobase" title="Kilobase">kilobases</a> de secuencias alineadas provenientes de 19 <i><a href="/wiki/Locus" title="Locus">loci</a></i> independientes de ADN nuclear en 169 especies de aves, que representan a todos los grupos mayores existentes. En ella se confirma la <a href="/wiki/Monofil%C3%A9tico" title="Monofilético">monofilia</a> de <a href="/wiki/Galloanserae" title="Galloanserae">Galloanserae</a> y <a href="/wiki/Neoaves" title="Neoaves">Neoaves</a>. Se refuerza, a su vez, la existencia de diversos clados propuestos a lo largo de los últimos años dentro de Neoaves, aunque las relaciones entre la gran mayoría aún están indeterminadas, lo cual genera una serie de politomías. Hackett y colaboradores consolidan en gran medida los estudios de Fain &amp; Houde (2004) y Ericson <i>et ál</i>. (2006), y se destaca en todos ellos la división de Neoaves en dos ramas: <a href="/wiki/Metaves" title="Metaves">Metaves</a> y <a class="new" href="/w/index.php?title=Coronave&amp;action=edit&amp;redlink=1" title="Coronave (aún no redactado)">Coronaves</a>.<sup class="reference separada" id="cite_ref-autogenerated1_66-0"><a href="#cite_note-autogenerated1-66"><span class="corchete-llamada">[</span>65<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-67"><a href="#cite_note-67"><span class="corchete-llamada">[</span>66<span class="corchete-llamada">]</span></a></sup>​ Aunque esta división se basa en un único gen (beta-fibrinógeno) y, además, se contrapone con los resultados obtenidos a partir de estudios del <a class="mw-redirect" href="/wiki/ADN_mitocondrial" title="ADN mitocondrial">ADN mitocondrial</a>.<sup class="reference separada" id="cite_ref-68"><a href="#cite_note-68"><span class="corchete-llamada">[</span>67<span class="corchete-llamada">]</span></a></sup>​ La validez de Metaves y Coronaves dependerá de futuros estudios. A continuación se definen nuevos clados (identificados solo con letras) y se señala el carácter <a href="/wiki/Parafil%C3%A9tico" title="Parafilético">parafilético</a> de algunos grupos (que deben ser redefinidos). Véase la <a class="mw-redirect" href="/wiki/Anexo:Aves" title="Anexo:Aves">lista de familias de aves</a> para más detalles dentro de los órdenes.

<p><b>Subclase <a href="/wiki/Neornithes" title="Neornithes">Neornithes</a> </b>
</p>
<p>Existen <a href="/wiki/Filogenia" title="Filogenia">filogenias</a> de aves muy diferentes, como la <a class="new" href="/w/index.php?title=Filogenia_de_Livezey_y_Zusi_(2007)&amp;action=edit&amp;redlink=1" title="Filogenia de Livezey y Zusi (2007) (aún no redactado)">filogenia de Livezey y Zusi (2007)</a> basada en un análisis cladístico de datos morfológicos también muy amplio,<sup class="reference separada" id="cite_ref-Theropoda_39-6"><a href="#cite_note-Theropoda-39"><span class="corchete-llamada">[</span>38<span class="corchete-llamada">]</span></a></sup>​ o la de la <a class="new" href="/w/index.php?title=Clasificaci%C3%B3n_tradicional_de_las_aves&amp;action=edit&amp;redlink=1" title="Clasificación tradicional de las aves (aún no redactado)">clasificación tradicional</a> (llamada de <a href="/wiki/James_Clements" title="James Clements">Clements</a>),<sup class="reference separada" id="cite_ref-Clements_7-3"><a href="#cite_note-Clements-7"><span class="corchete-llamada">[</span>6<span class="corchete-llamada">]</span></a></sup>​ o la de <a href="/wiki/Charles_Sibley" title="Charles Sibley">Sibley</a> y Monroe (<a href="/wiki/Taxonom%C3%ADa_de_Sibley-Ahlquist" title="Taxonomía de Sibley-Ahlquist">Taxonomía de Sibley-Ahlquist</a>),<sup class="reference separada" id="cite_ref-69"><a href="#cite_note-69"><span class="corchete-llamada">[</span>68<span class="corchete-llamada">]</span></a></sup>​ basada en datos de <a href="/wiki/Hibridaci%C3%B3n_(biolog%C3%ADa_molecular)" title="Hibridación (biología molecular)">hibridación del ADN</a>, que tuvo amplia aceptación en unos pocos aspectos, como por ejemplo la aceptación de <a href="/wiki/Galloanserae" title="Galloanserae">Galloanserae</a>.
</p>
<p>Las aves viven y crían en la mayoría de los <a href="/wiki/H%C3%A1bitat" title="Hábitat">hábitats</a> terrestres y están presentes en todos los continentes, incluso en el territorio antártico donde anidan las colonias de <a href="/wiki/Pagodroma_nivea" title="Pagodroma nivea">petreles níveos</a>, las aves más australes.<sup class="reference separada" id="cite_ref-71"><a href="#cite_note-71"><span class="corchete-llamada">[</span>70<span class="corchete-llamada">]</span></a></sup>​
La mayor diversidad de aves se da en las regiones tropicales, y el país con el mayor número de especies en el mundo es <a class="mw-redirect" href="/wiki/Aves_de_Colombia" title="Aves de Colombia">Colombia</a>,<sup class="reference separada" id="cite_ref-72"><a href="#cite_note-72"><span class="corchete-llamada">[</span>71<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-73"><a href="#cite_note-73"><span class="corchete-llamada">[</span>72<span class="corchete-llamada">]</span></a></sup>​ seguido por <a class="mw-redirect" href="/wiki/Aves_del_Per%C3%BA" title="Aves del Perú">Perú</a> y <a class="mw-redirect" href="/wiki/Aves_de_Brasil" title="Aves de Brasil">Brasil</a>. Otras avifaunas notables por su cantidad de <a href="/wiki/Endemismo" title="Endemismo">endemismos</a> son las <a href="/wiki/Aves_end%C3%A9micas_de_Nueva_Zelanda" title="Aves endémicas de Nueva Zelanda">de Nueva Zelanda</a>,<sup class="reference separada" id="cite_ref-74"><a href="#cite_note-74"><span class="corchete-llamada">[</span>73<span class="corchete-llamada">]</span></a></sup>​ <a class="mw-redirect" href="/wiki/Aves_end%C3%A9micas_de_Madagascar" title="Aves endémicas de Madagascar">Madagascar</a><sup class="reference separada" id="cite_ref-75"><a href="#cite_note-75"><span class="corchete-llamada">[</span>74<span class="corchete-llamada">]</span></a></sup>​ y <a class="mw-redirect" href="/wiki/Aves_end%C3%A9micas_de_Australia" title="Aves endémicas de Australia">Australia</a>, las cuales, a diferencia de las de países sudamericanos, cuentan además con un considerable número de taxones superiores endémicos.<sup class="reference separada" id="cite_ref-76"><a href="#cite_note-76"><span class="corchete-llamada">[</span>75<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-77"><a href="#cite_note-77"><span class="corchete-llamada">[</span>76<span class="corchete-llamada">]</span></a></sup>​ La <a class="mw-redirect" href="/wiki/Regi%C3%B3n_biogeogr%C3%A1fica" title="Región biogeográfica">región biogeográfica</a> con mayor número de especies, con unas 3700 (más de la tercera parte mundial), es el <a href="/wiki/Neotr%C3%B3pico" title="Neotrópico">Neotrópico</a> (que incluye América del Sur, América Central, tierras bajas de <a href="/wiki/M%C3%A9xico" title="México">México</a> y las <a href="/wiki/Antillas" title="Antillas">Antillas</a>). Además, 31 familias son <a class="new" href="/w/index.php?title=Aves_end%C3%A9micas_del_Neotr%C3%B3pico&amp;action=edit&amp;redlink=1" title="Aves endémicas del Neotrópico (aún no redactado)">endémicas del Neotrópico</a>, más del doble que en cualquier otra región biogeográfica.<sup class="reference separada" id="cite_ref-biogeography_25-3"><a href="#cite_note-biogeography-25"><span class="corchete-llamada">[</span>24<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Tradicionalmente se ha considerado que la alta diversidad tropical era resultado de unas mayores tasas de <a href="/wiki/Especiaci%C3%B3n" title="Especiación">especiación</a>; sin embargo, estudios recientes descubrieron que en las altas latitudes hay mayores tasas de especiación que son compensadas por tasas de <a href="/wiki/Extinci%C3%B3n" title="Extinción">extinción</a> más altas.<sup class="reference separada" id="cite_ref-78"><a href="#cite_note-78"><span class="corchete-llamada">[</span>77<span class="corchete-llamada">]</span></a></sup>​ Numerosas familias de aves se han adaptado a vivir en el <a href="/wiki/Mar" title="Mar">mar</a>; algunas especies de aves marinas solo recalan en tierra para criar,<sup class="reference separada" id="cite_ref-Burger_79-0"><a href="#cite_note-Burger-79"><span class="corchete-llamada">[</span>78<span class="corchete-llamada">]</span></a></sup>​ y se sabe que algunos <a href="/wiki/Spheniscidae" title="Spheniscidae">pingüinos</a> llegan a bucear hasta a 300 m de profundidad.<sup class="reference separada" id="cite_ref-80"><a href="#cite_note-80"><span class="corchete-llamada">[</span>79<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Muchas especies de aves se han establecido en regiones donde han sido <a href="/wiki/Especie_introducida" title="Especie introducida">introducidas</a> por el hombre. Algunas de estas introducciones han sido deliberadas; el <a href="/wiki/Phasianus_colchicus" title="Phasianus colchicus">faisán común</a>, por ejemplo, ha sido introducido como especie para la caza en buena parte del mundo.<sup class="reference separada" id="cite_ref-81"><a href="#cite_note-81"><span class="corchete-llamada">[</span>80<span class="corchete-llamada">]</span></a></sup>​ Otras introducciones han sido accidentales; este es el caso de varias especies de <a href="/wiki/Psittacidae" title="Psittacidae">loros</a>, como la <a href="/wiki/Myiopsitta_monachus" title="Myiopsitta monachus">cotorra argentina</a> que a partir de ejemplares cautivos escapados se ha establecido en numerosas ciudades de Norteamérica,<sup class="reference separada" id="cite_ref-82"><a href="#cite_note-82"><span class="corchete-llamada">[</span>81<span class="corchete-llamada">]</span></a></sup>​ Sudamérica y Europa.<sup class="reference separada" id="cite_ref-83"><a href="#cite_note-83"><span class="corchete-llamada">[</span>82<span class="corchete-llamada">]</span></a></sup>​
Algunas especies, como la <a href="/wiki/Bubulcus_ibis" title="Bubulcus ibis">garcilla bueyera</a>,<sup class="reference separada" id="cite_ref-84"><a href="#cite_note-84"><span class="corchete-llamada">[</span>83<span class="corchete-llamada">]</span></a></sup>​ el <a href="/wiki/Milvago_chimachima" title="Milvago chimachima">chimachimá</a>,<sup class="reference separada" id="cite_ref-85"><a href="#cite_note-85"><span class="corchete-llamada">[</span>84<span class="corchete-llamada">]</span></a></sup>​ o la <a href="/wiki/Eolophus_roseicapilla" title="Eolophus roseicapilla">cacatúa galah</a>,<sup class="reference separada" id="cite_ref-86"><a href="#cite_note-86"><span class="corchete-llamada">[</span>85<span class="corchete-llamada">]</span></a></sup>​ se han introducido de forma natural en regiones fuera de sus áreas de distribución original gracias a que la agricultura ha creado ecosistemas adecuados para ellas.
</p>
<p>La <a href="/wiki/Anatom%C3%ADa" title="Anatomía">anatomía</a> de las aves presenta un <a href="/wiki/Plan_corporal" title="Plan corporal">plan corporal</a> que exhibe un gran número de adaptaciones inusuales en comparación con otros vertebrados, en su mayor parte para facilitar el <a href="/wiki/Vuelo_de_las_aves" title="Vuelo de las aves">vuelo</a>.
</p>
<p>El <a href="/wiki/Esqueleto" title="Esqueleto">esqueleto</a> está formado por <a href="/wiki/Hueso" title="Hueso">huesos</a> huecos, pero de estructura resistente, lo que les confiere ligereza. Estas cavidades óseas están llenas de <a href="/wiki/Aire" title="Aire">aire</a> y conectan con el <a href="/wiki/Aparato_respiratorio" title="Aparato respiratorio">aparato respiratorio</a>.<sup class="reference separada" id="cite_ref-87"><a href="#cite_note-87"><span class="corchete-llamada">[</span>86<span class="corchete-llamada">]</span></a></sup>​ Los huesos del <a href="/wiki/Cr%C3%A1neo" title="Cráneo">cráneo</a> están fusionados, sin presentar <a class="new" href="/w/index.php?title=Sutura_craneal&amp;action=edit&amp;redlink=1" title="Sutura craneal (aún no redactado)">suturas craneales</a>.<sup class="reference separada" id="cite_ref-Gill_88-0"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ Las <a class="mw-redirect" href="/wiki/%C3%93rbita_(anatom%C3%ADa)" title="Órbita (anatomía)">órbitas</a> son grandes y separadas por un <a href="/wiki/Septo" title="Septo">septo</a> óseo. La <a href="/wiki/Columna_vertebral" title="Columna vertebral">columna vertebral</a> de las aves presenta un gran contraste entre las zonas superiores y las inferiores. El número de <a href="/wiki/V%C3%A9rtebra" title="Vértebra">vértebras</a> cervicales es muy variable, aunque siempre numeroso y el cuello es especialmente flexible, pero en las vértebras torácicas anteriores la movilidad es reducida, y en todas las posteriores la movilidad es nula, dado que están fusionadas.<sup class="reference separada" id="cite_ref-89"><a href="#cite_note-89"><span class="corchete-llamada">[</span>88<span class="corchete-llamada">]</span></a></sup>​ Las pocas vértebras posteriores están fusionadas con la <a href="/wiki/Pelvis" title="Pelvis">pelvis</a> para formar el <a href="/wiki/Sinsacro" title="Sinsacro">sinsacro</a>.<sup class="reference separada" id="cite_ref-Gill_88-1"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ Las <a href="/wiki/Costilla" title="Costilla">costillas</a> son aplastadas y el <a href="/wiki/Estern%C3%B3n" title="Esternón">esternón</a> es <a href="/wiki/Quilla_(ave)" title="Quilla (ave)">aquillado</a> para el anclaje de los <a href="/wiki/M%C3%BAsculo" title="Músculo">músculos</a> del vuelo, excepto en los órdenes de aves terrestres no voladoras. Las extremidades anteriores están modificadas en forma de alas.<sup class="reference separada" id="cite_ref-90"><a href="#cite_note-90"><span class="corchete-llamada">[</span>89<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Los pies de las aves están clasificados según la disposición de sus dedos en <a class="mw-redirect" href="/wiki/Anisod%C3%A1ctilo" title="Anisodáctilo">anisodáctilos</a>, <a class="mw-redirect" href="/wiki/Zigod%C3%A1ctilo" title="Zigodáctilo">zigodáctilos</a>, <a class="mw-redirect" href="/wiki/Heterod%C3%A1ctilo" title="Heterodáctilo">heterodáctilos</a>, <a class="mw-redirect" href="/wiki/Sind%C3%A1ctilo" title="Sindáctilo">sindáctilos</a> y <a class="mw-redirect" href="/wiki/Pamprod%C3%A1ctilo" title="Pamprodáctilo">pamprodáctilos</a>.<sup class="reference separada" id="cite_ref-91"><a href="#cite_note-91"><span class="corchete-llamada">[</span>90<span class="corchete-llamada">]</span></a></sup>​ La mayor parte de las aves tienen cuatro dedos (aunque hay muchas especies tridáctilas y algunas didáctilas) que se organizan en torno a un ancho y fuerte metatarso.<sup class="reference separada" id="cite_ref-huellas_92-0"><a href="#cite_note-huellas-92"><span class="corchete-llamada">[</span>91<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Como los <a class="mw-redirect" href="/wiki/Reptil" title="Reptil">reptiles</a>, las aves son primariamente <a href="/wiki/Excreci%C3%B3n" title="Excreción">uricotélicos</a>, es decir, sus <a class="mw-redirect" href="/wiki/Ri%C3%B1ones" title="Riñones">riñones</a> extraen desechos nitrogenados de su sangre y los excretan como <a href="/wiki/%C3%81cido_%C3%BArico" title="Ácido úrico">ácido úrico</a>, en vez de <a href="/wiki/Urea" title="Urea">urea</a> o <a href="/wiki/Amon%C3%ADaco" title="Amoníaco">amoníaco</a>, a través de los <a href="/wiki/Ur%C3%A9ter" title="Uréter">uréteres</a> hacia el intestino. Las aves no tienen <a href="/wiki/Vejiga_urinaria" title="Vejiga urinaria">vejiga urinaria</a> o apertura <a href="/wiki/Uretra" title="Uretra">urétrica</a> externa y el ácido úrico se excreta junto con las <a class="mw-redirect" href="/wiki/Heces" title="Heces">heces</a> como desperdicio semisólido.<sup class="reference separada" id="cite_ref-93"><a href="#cite_note-93"><span class="corchete-llamada">[</span>92<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-94"><a href="#cite_note-94"><span class="corchete-llamada">[</span>93<span class="corchete-llamada">]</span></a></sup>​ Sin embargo, aves como los colibríes pueden ser facultativamente <a href="/wiki/Excreci%C3%B3n" title="Excreción">amoniotélicos</a>, al excretar la mayor parte de los desechos nitrogenados en forma de amoníaco.<sup class="reference separada" id="cite_ref-95"><a href="#cite_note-95"><span class="corchete-llamada">[</span>94<span class="corchete-llamada">]</span></a></sup>​ Las razones de esto son diversas y no están del todo claras, aunque sus dietas basadas en el néctar, por lo tanto con grandes aportes de agua, juegan un papel clave. También se debe a que sus metabolismos requieren poco nitrógeno, y a las bajas ingestiones de proteínas y sal. Cuando estas condiciones cambian, se reduce la ingesta de néctar o suben las proteínas y sales obtenidas, estas aves pueden pasar a ser uricotélicas.<sup class="reference separada" id="cite_ref-96"><a href="#cite_note-96"><span class="corchete-llamada">[</span>95<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-97"><a href="#cite_note-97"><span class="corchete-llamada">[</span>96<span class="corchete-llamada">]</span></a></sup>​ Pueden excretar también <a href="/wiki/Creatina" title="Creatina">creatina</a>, en vez de <a href="/wiki/Creatinina" title="Creatinina">creatinina</a> como los mamíferos.<sup class="reference separada" id="cite_ref-Gill_88-2"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ Esta materia, así como la fecal de los <a href="/wiki/Intestino" title="Intestino">intestinos</a>, es expulsada a través de la <a class="mw-redirect" href="/wiki/Cloaca_(biolog%C3%ADa)" title="Cloaca (biología)">cloaca</a> del ave.<sup class="reference separada" id="cite_ref-98"><a href="#cite_note-98"><span class="corchete-llamada">[</span>97<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-99"><a href="#cite_note-99"><span class="corchete-llamada">[</span>98<span class="corchete-llamada">]</span></a></sup>​ La cloaca es una abertura multifuncional: por ella se expulsan los desechos, las aves se aparean juntando sus cloacas y las hembras ponen huevos a través de ella. Adicionalmente muchas especies regurgitan <a href="/wiki/Egagr%C3%B3pila" title="Egagrópila">egagrópilas</a>.<sup class="reference separada" id="cite_ref-100"><a href="#cite_note-100"><span class="corchete-llamada">[</span>99<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El <a href="/wiki/Aparato_digestivo" title="Aparato digestivo">aparato digestivo</a> de las aves es único, con un <a href="/wiki/Buche" title="Buche">buche</a> para almacenamiento de lo ingerido y una <a href="/wiki/Molleja" title="Molleja">molleja</a> que contiene piedras que el ave ha tragado y que sirven para triturar el alimento para compensar la ausencia de dientes.<sup class="reference separada" id="cite_ref-101"><a href="#cite_note-101"><span class="corchete-llamada">[</span>100<span class="corchete-llamada">]</span></a></sup>​ La mayoría de las aves están adaptadas a una rápida digestión para ayudar al vuelo.<sup class="reference separada" id="cite_ref-Attenborough_102-0"><a href="#cite_note-Attenborough-102"><span class="corchete-llamada">[</span>101<span class="corchete-llamada">]</span></a></sup>​ Algunas aves migratorias se han adaptado a usar <a href="/wiki/Prote%C3%ADna" title="Proteína">proteínas</a> de muchas partes del cuerpo, incluidas proteínas de los intestinos, como fuente adicional de energía durante la migración.<sup class="reference separada" id="cite_ref-Battley_103-0"><a href="#cite_note-Battley-103"><span class="corchete-llamada">[</span>102<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las aves son animales <a class="mw-redirect" href="/wiki/Homeot%C3%A9rmico" title="Homeotérmico">homeotérmicos</a>, es decir, que la temperatura interna se mantiene regulada, por encima de la temperatura exterior, lo que les permite tener un elevado <a href="/wiki/Metabolismo" title="Metabolismo">metabolismo</a>; el plumaje participa en su regulación. La temperatura media interna de las aves adultas es bastante alta, en general entre 40 y 43 °C, con variaciones entre especies. Algunas <a href="/wiki/Apodiformes" title="Apodiformes">Apodiformes</a> tienen temperaturas nocturnas notablemente menores. Ciertas aves, como los <a href="/wiki/Regulidae" title="Regulidae">reyezuelos</a>, cuando son recién nacidos mantienen la temperatura ambiental (<a class="mw-redirect" href="/wiki/Poiquilotermia" title="Poiquilotermia">poiquilotermia</a>), y adquieren la capacidad de regularla pocos días después.<sup class="reference separada" id="cite_ref-Storer_104-0"><a href="#cite_note-Storer-104"><span class="corchete-llamada">[</span>103<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las aves tienen uno de los <a class="mw-redirect" href="/wiki/Sistema_respiratorio" title="Sistema respiratorio">aparatos respiratorios</a> más complejos del reino animal.<sup class="reference separada" id="cite_ref-Gill_88-3"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ Tras la inhalación, el 75 % del aire fresco pasa de largo de los pulmones y fluye directo a los <a href="/wiki/Saco_a%C3%A9reo" title="Saco aéreo">sacos aéreos</a> posteriores, que se extienden desde los pulmones y conectan con los espacios en los huesos, y los llenan con aire. El otro 25 % del aire va directamente a los pulmones. Cuando el ave exhala, el aire usado fluye fuera de los pulmones y el aire almacenado de los sacos aéreos posteriores es simultáneamente forzado a entrar en los pulmones. De este modo los pulmones de un ave reciben un suministro constante de aire fresco tanto en la inhalación como en la exhalación.<sup class="reference separada" id="cite_ref-105"><a href="#cite_note-105"><span class="corchete-llamada">[</span>104<span class="corchete-llamada">]</span></a></sup>​ La producción de sonidos se logra usando la <a href="/wiki/Siringe" title="Siringe">siringe</a>, una cámara muscular con varias membranas timpánicas que está situada en el extremo inferior de la <a href="/wiki/Tr%C3%A1quea" title="Tráquea">tráquea</a>, desde la cual se separa.<sup class="reference separada" id="cite_ref-Suthers_106-0"><a href="#cite_note-Suthers-106"><span class="corchete-llamada">[</span>105<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El <a href="/wiki/Coraz%C3%B3n" title="Corazón">corazón</a> de las aves tiene cuatro cámaras separadas (dos <a class="mw-redirect" href="/wiki/Aur%C3%ADcula_card%C3%ADaca" title="Aurícula cardíaca">aurículas</a> y dos <a href="/wiki/Ventr%C3%ADculo" title="Ventrículo">ventrículos</a>) y es el <a class="new" href="/w/index.php?title=Arco_a%C3%B3rtico_derecho&amp;action=edit&amp;redlink=1" title="Arco aórtico derecho (aún no redactado)">arco aórtico derecho</a> el que da lugar a la <a href="/wiki/Circulaci%C3%B3n_sist%C3%A9mica" title="Circulación sistémica">circulación sistémica</a> (al contrario que en los mamíferos, en los que el involucrado es el arco aórtico izquierdo).<sup class="reference separada" id="cite_ref-Gill_88-4"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ La <a href="/wiki/Vena_cava_inferior" title="Vena cava inferior">vena cava inferior</a> (única) recibe <a href="/wiki/Sangre" title="Sangre">sangre</a> de las patas por vía del <a class="new" href="/w/index.php?title=Sistema_porta_renal&amp;action=edit&amp;redlink=1" title="Sistema porta renal (aún no redactado)">sistema porta renal</a> (muy reducido). La mayor parte de esta sangre proveniente de la <a class="mw-redirect" href="/wiki/Cintura_p%C3%A9lvica" title="Cintura pélvica">cintura pélvica</a> y la cola, llega al corazón sin pasar por los <a href="/wiki/Capilar_sangu%C3%ADneo" title="Capilar sanguíneo">capilares</a> renales. Los <a class="mw-redirect" href="/wiki/Gl%C3%B3bulo_rojo" title="Glóbulo rojo">glóbulos rojos</a> tienen <a href="/wiki/N%C3%BAcleo_celular" title="Núcleo celular">núcleo</a>, a diferencia de los mamíferos, y son ovales y biconvexos.<sup class="reference separada" id="cite_ref-107"><a href="#cite_note-107"><span class="corchete-llamada">[</span>106<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El <a href="/wiki/Sistema_nervioso" title="Sistema nervioso">sistema nervioso</a> es grande en relación con el tamaño del ave.<sup class="reference separada" id="cite_ref-Gill_88-5"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ La parte más desarrollada del <a href="/wiki/Enc%C3%A9falo" title="Encéfalo">encéfalo</a> es la que controla las funciones relacionadas con el vuelo, mientras el <a href="/wiki/Cerebelo" title="Cerebelo">cerebelo</a> coordina el movimiento, y los <a class="mw-redirect" href="/wiki/Hemisferios_cerebrales" title="Hemisferios cerebrales">hemisferios cerebrales</a> controlan patrones de comportamiento, la orientación, el apareamiento y la construcción del nido.
</p>
<p>Unas pocas especies son capaces de usar defensas químicas contra sus depredadores. Algunos <a href="/wiki/Procellariiformes" title="Procellariiformes">Procellariiformes</a> pueden expulsar <a href="/wiki/Aceite" title="Aceite">aceites</a> repulsivos contra sus atacantes,<sup class="reference separada" id="cite_ref-108"><a href="#cite_note-108"><span class="corchete-llamada">[</span>107<span class="corchete-llamada">]</span></a></sup>​ y algunas especies en el género <i><a href="/wiki/Pitohui" title="Pitohui">Pitohui</a></i> de <a href="/wiki/Nueva_Guinea" title="Nueva Guinea">Nueva Guinea</a> tienen una potente <a href="/wiki/Neurotoxina" title="Neurotoxina">neurotoxina</a> en su piel y plumas.<sup class="reference separada" id="cite_ref-109"><a href="#cite_note-109"><span class="corchete-llamada">[</span>108<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Los <a href="/wiki/Sentido_(percepci%C3%B3n)" title="Sentido (percepción)">sentidos</a> de las aves no deberían diferir básicamente de los mamíferos, pero para algunos de ellos quedan incógnitas: no se sabía muy bien, por ejemplo, cómo logran orientarse en sus migraciones. Estudios más recientes confirma la presencia de <a href="/wiki/Magnetita" title="Magnetita">magnetita</a> en el cráneo de las aves, mineral que ayuda a la orientación ya que funciona como una brújula.
</p>
<p>El <a href="/wiki/Visi%C3%B3n_de_las_aves" title="Visión de las aves">sistema visual de las aves</a> suele estar altamente desarrollado. Las <a class="mw-redirect" href="/wiki/Rapaz" title="Rapaz">rapaces</a> en especial tienen una gran agudeza visual, dos o tres veces mejor que la del ser humano.<sup class="reference separada" id="cite_ref-Ritchison2_110-0"><a href="#cite_note-Ritchison2-110"><span class="corchete-llamada">[</span>109<span class="corchete-llamada">]</span></a></sup>​ La <a href="/wiki/F%C3%B3vea" title="Fóvea">fóvea</a> de una especie del género <i><a href="/wiki/Buteo" title="Buteo">Buteo</a></i> posee alrededor de 100 000 <a class="mw-redirect" href="/wiki/Cono_(biolog%C3%ADa)" title="Cono (biología)">conos</a> por mm², frente a los 20 000 en el hombre,<sup class="reference separada" id="cite_ref-GrEncScTec_111-0"><a href="#cite_note-GrEncScTec-111"><span class="corchete-llamada">[</span>110<span class="corchete-llamada">]</span></a></sup>​ cinco veces más. Los ojos de las aves son muy voluminosos. Por ejemplo, los del <a href="/wiki/Sturnus" title="Sturnus">estornino</a> tienen un volumen correspondiente a 15 % del volumen craneal (como comparación, en el hombre representan el 1 % de dicho volumen).<sup class="reference separada" id="cite_ref-GrEncScTec_111-1"><a href="#cite_note-GrEncScTec-111"><span class="corchete-llamada">[</span>110<span class="corchete-llamada">]</span></a></sup>​
Las aves acuáticas tienen lentes flexibles especiales, lo que les permite la acomodación para la visión en el aire y en el agua.<sup class="reference separada" id="cite_ref-Gill_88-6"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ Algunas especies tienen fóveas duales (por ejemplo: <a href="/wiki/Hirundinidae" title="Hirundinidae">golondrinas</a>, <a class="mw-redirect" href="/wiki/Charr%C3%A1n" title="Charrán">charranes</a>, <a href="/wiki/Nycticorax_nycticorax" title="Nycticorax nycticorax">martinetes</a>, <a href="/wiki/Falco" title="Falco">halcones</a>, <a class="mw-redirect" href="/wiki/Colibr%C3%AD" title="Colibrí">colibríes</a>, etc.).<sup class="reference separada" id="cite_ref-GrEncScTec_111-2"><a href="#cite_note-GrEncScTec-111"><span class="corchete-llamada">[</span>110<span class="corchete-llamada">]</span></a></sup>​ Las aves nocturnas tienen generalmente un campo visual restringido, pero una gran movilidad de la cabeza (que a veces puede girar más de 250°).<sup class="reference separada" id="cite_ref-GrEncScTec_111-3"><a href="#cite_note-GrEncScTec-111"><span class="corchete-llamada">[</span>110<span class="corchete-llamada">]</span></a></sup>​ Las aves son <a href="/wiki/Tetracromatismo" title="Tetracromatismo">tetracromáticas</a>, al poseer en la <a href="/wiki/Retina" title="Retina">retina</a> <a class="mw-redirect" href="/wiki/C%C3%A9lula_cono" title="Célula cono">conos</a> sensibles al <a class="mw-redirect" href="/wiki/Luz_ultravioleta" title="Luz ultravioleta">ultravioleta</a>, además de las sensibles a verde, rojo y azul.<sup class="reference separada" id="cite_ref-Ritchison2_110-1"><a href="#cite_note-Ritchison2-110"><span class="corchete-llamada">[</span>109<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-112"><a href="#cite_note-112"><span class="corchete-llamada">[</span>111<span class="corchete-llamada">]</span></a></sup>​ Esto les permite percibir la luz ultravioleta, la cual está involucrada en el cortejo. Muchas aves muestran patrones ultravioleta en los plumajes que son invisibles al ojo humano. Algunas aves cuyos sexos parecen similares a simple vista se pueden distinguir con visión ultravioleta por ciertas manchas en sus plumas que reflejan esa luz. Los machos de <a class="mw-redirect" href="/wiki/Parus_caeruleus" title="Parus caeruleus">herrerillo europeo</a> tienen un parche en la coronilla que refleja el ultravioleta que es mostrado en el cortejo cambiando la postura y erizando las plumas de la nuca.<sup class="reference separada" id="cite_ref-113"><a href="#cite_note-113"><span class="corchete-llamada">[</span>112<span class="corchete-llamada">]</span></a></sup>​ La luz ultravioleta se usa en la detección del alimento; se ha observado que los <a class="mw-redirect" href="/wiki/Cern%C3%ADcalo" title="Cernícalo">cernícalos</a> buscan la presa por medio de la detección de los rastros de orina, que reflejan el ultravioleta, dejados en el suelo por los roedores.<sup class="reference separada" id="cite_ref-114"><a href="#cite_note-114"><span class="corchete-llamada">[</span>113<span class="corchete-llamada">]</span></a></sup>​ Los párpados de un ave no se usan para pestañear. En vez de eso, el ojo es lubricado por la <a href="/wiki/Membrana_nictitante" title="Membrana nictitante">membrana nictitante</a>, un tercer párpado que se mueve horizontalmente.<sup class="reference separada" id="cite_ref-115"><a href="#cite_note-115"><span class="corchete-llamada">[</span>114<span class="corchete-llamada">]</span></a></sup>​ La membrana nictitante también cubre el ojo y actúa como una <a href="/wiki/Lente_de_contacto" title="Lente de contacto">lente de contacto</a> en muchas aves acuáticas.<sup class="reference separada" id="cite_ref-Gill_88-7"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ La <a href="/wiki/Retina" title="Retina">retina</a> de las aves tiene un sistema de suministro de sangre en forma de abanico llamado <a href="/wiki/Pecten" title="Pecten">pecten</a>.<sup class="reference separada" id="cite_ref-Gill_88-8"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ La mayoría de las aves no pueden mover sus ojos, aunque hay excepciones, como el <a href="/wiki/Phalacrocorax_carbo" title="Phalacrocorax carbo">cormorán grande</a>.<sup class="reference separada" id="cite_ref-116"><a href="#cite_note-116"><span class="corchete-llamada">[</span>115<span class="corchete-llamada">]</span></a></sup>​ Las aves con ojos a los lados de la cabeza tienen un amplio <a href="/wiki/Campo_visual" title="Campo visual">campo visual</a>, mientras que las que tienen los ojos en el frente, como los <a href="/wiki/Strigiformes" title="Strigiformes">búhos</a>, tienen <a href="/wiki/Visi%C3%B3n_binocular" title="Visión binocular">visión binocular</a> y pueden estimar mejor la profundidad del campo visual.<sup class="reference separada" id="cite_ref-117"><a href="#cite_note-117"><span class="corchete-llamada">[</span>116<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La mayoría de las aves tienen un pobre <a class="mw-redirect" href="/wiki/Sentido_del_olfato" title="Sentido del olfato">sentido del olfato</a>, pero hay excepciones notables como los <a href="/wiki/Apteryx" title="Apteryx">kiwis</a>,<sup class="reference separada" id="cite_ref-118"><a href="#cite_note-118"><span class="corchete-llamada">[</span>117<span class="corchete-llamada">]</span></a></sup>​ las aves <a href="/wiki/Cathartidae" title="Cathartidae">carroñeras americanas</a>,<sup class="reference separada" id="cite_ref-GrEncScTec_111-4"><a href="#cite_note-GrEncScTec-111"><span class="corchete-llamada">[</span>110<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-Avian_Sense_of_Smell_119-0"><a href="#cite_note-Avian_Sense_of_Smell-119"><span class="corchete-llamada">[</span>118<span class="corchete-llamada">]</span></a></sup>​ y los <a href="/wiki/Procellariiformes" title="Procellariiformes">albatros y petreles</a>.<sup class="reference separada" id="cite_ref-120"><a href="#cite_note-120"><span class="corchete-llamada">[</span>119<span class="corchete-llamada">]</span></a></sup>​ Se ha comprobado que estos últimos son capaces de localizar a sus presas, en particular el aceite de pescado, por el olfato.<sup class="reference separada" id="cite_ref-121"><a href="#cite_note-121"><span class="corchete-llamada">[</span>120<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-122"><a href="#cite_note-122"><span class="corchete-llamada">[</span>121<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El <a href="/wiki/O%C3%ADdo" title="Oído">oído</a> de las aves está bien desarrollado; aunque carece de <a href="/wiki/Pabell%C3%B3n_auricular" title="Pabellón auricular">pabellón auricular</a>, está cubierto por plumas, y en algunas, como en los géneros de <a href="/wiki/Strigidae" title="Strigidae">búhos</a> <i><a href="/wiki/Asio_(g%C3%A9nero)" title="Asio (género)">Asio</a></i>, <i><a href="/wiki/Bubo" title="Bubo">Bubo</a></i> y <i><a href="/wiki/Otus" title="Otus">Otus</a></i>, forman penachos que parecen orejas. Ciertas especies de búho pueden localizar una presa en completa oscuridad solo con la audición.<sup class="reference separada" id="cite_ref-Ritchison2_110-2"><a href="#cite_note-Ritchison2-110"><span class="corchete-llamada">[</span>109<span class="corchete-llamada">]</span></a></sup>​ Sin embargo la ausencia de orejas les obliga a realizar rotaciones de la cabeza para percibir los sonidos provenientes de diferentes direcciones. Las <a href="/wiki/Collocaliini" title="Collocaliini">salanganas</a> y el <a href="/wiki/Steatornis_caripensis" title="Steatornis caripensis">guácharo de las cavernas</a> son capaces de desplazarse en la oscuridad gracias a que sus oídos están adaptados a la <a href="/wiki/Ecolocalizaci%C3%B3n" title="Ecolocalización">ecolocalización</a>. Al contrario que los mamíferos, el <a href="/wiki/O%C3%ADdo_medio" title="Oído medio">oído medio</a> de las aves tiene un solo <a class="mw-redirect" href="/wiki/Huesecillo" title="Huesecillo">huesecillo</a>, la <a class="new" href="/w/index.php?title=Columela_(huesecillo)&amp;action=edit&amp;redlink=1" title="Columela (huesecillo) (aún no redactado)">columela</a>. En el <a href="/wiki/O%C3%ADdo_interno" title="Oído interno">oído interno</a>, la <a href="/wiki/C%C3%B3clea" title="Cóclea">cóclea</a> no es espiralada sino recta, al contrario que en los mamíferos.<sup class="reference separada" id="cite_ref-GrEncScTec_111-5"><a href="#cite_note-GrEncScTec-111"><span class="corchete-llamada">[</span>110<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-123"><a href="#cite_note-123"><span class="corchete-llamada">[</span>122<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las <a href="/wiki/Papilas_gustativas" title="Papilas gustativas">papilas gustativas</a> no se encuentran en el extremo de la <a href="/wiki/Lengua_(anatom%C3%ADa)" title="Lengua (anatomía)">lengua</a> sino en el fondo y en la garganta; además son poco numerosas (200 en una <a href="/wiki/Anatinae" title="Anatinae">Anatinae</a>, frente a las 9000 en el hombre), pero otros mecanismos pueden ser accionados para la degustación, como el sentido del tacto (notable a nivel del pico) que, en muchas aves, parece intervenir durante la búsqueda de alimento.
En el sentido del tacto intervienen diversos corpúsculos: los <a class="mw-redirect" href="/wiki/Corp%C3%BAsculo_de_Merkel" title="Corpúsculo de Merkel">corpúsculos encapsulados de Merkel</a> (en la piel y en el interior de la boca) y los de <a class="mw-redirect" href="/wiki/Corp%C3%BAsculo_de_Grandry" title="Corpúsculo de Grandry">Grandry</a> (en el <a href="/wiki/Paladar" title="Paladar">paladar</a>) participan en la sensibilidad táctil general; estos serían los correspondientes al <a href="/wiki/Corp%C3%BAsculo_de_Meissner" title="Corpúsculo de Meissner">corpúsculos de Meissner</a> en los mamíferos.<sup class="reference separada" id="cite_ref-Tanzarella_124-0"><a href="#cite_note-Tanzarella-124"><span class="corchete-llamada">[</span>123<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Los <a class="mw-redirect" href="/wiki/Corp%C3%BAsculo_de_Herbst" title="Corpúsculo de Herbst">corpúsculos de Herbst</a> (que corresponden a los <a class="mw-redirect" href="/wiki/Corp%C3%BAsculo_de_Pacini" title="Corpúsculo de Pacini">corpúsculos de Pacini</a> de los mamíferos<sup class="reference separada" id="cite_ref-Tanzarella_124-1"><a href="#cite_note-Tanzarella-124"><span class="corchete-llamada">[</span>123<span class="corchete-llamada">]</span></a></sup>​) son preferentemente sensibles a las vibraciones; son particularmente numerosos en el pico y las patas, particularmente en las especies que deben encontrar su alimento «a tientas»: lengua en <a href="/wiki/Picidae" title="Picidae">Picidae</a>, pico en <a href="/wiki/Anatidae" title="Anatidae">Anatidae</a> y en numerosos <a href="/wiki/Scolopacidae" title="Scolopacidae">Scolopacidae</a>, pero también en los rebordes coloreados que bordean los picos de las crías de numerosos <a class="mw-redirect" href="/wiki/Paseriformes" title="Paseriformes">paseriformes</a> y <a href="/wiki/Picinae" title="Picinae">Picinae</a>.<sup class="reference separada" id="cite_ref-GrEncScTec_111-6"><a href="#cite_note-GrEncScTec-111"><span class="corchete-llamada">[</span>110<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las aves poseen varios órganos de equilibrio independientes; el oído interno, como en los mamíferos, y un órgano situado en la <a href="/wiki/Pelvis" title="Pelvis">pelvis</a>.
</p>
<p>Uno de los sentidos más misteriosos es la detección del <a href="/wiki/Campo_magn%C3%A9tico_terrestre" title="Campo magnético terrestre">campo magnético terrestre</a>; el órgano que lo detecta se sitúa en el pico o cerca de los ojos.<sup class="reference separada" id="cite_ref-Ritchison2_110-3"><a href="#cite_note-Ritchison2-110"><span class="corchete-llamada">[</span>109<span class="corchete-llamada">]</span></a></sup>​ La existencia de este sentido fue demostrada experimentalmente por primera vez en 1967 por <a class="new" href="/w/index.php?title=Wolfgang_Wiltschko&amp;action=edit&amp;redlink=1" title="Wolfgang Wiltschko (aún no redactado)">Wolfgang Wiltschko</a> en los <a href="/wiki/Erithacus_rubecula" title="Erithacus rubecula">petirrojos europeos</a>.<sup class="reference separada" id="cite_ref-125"><a href="#cite_note-125"><span class="corchete-llamada">[</span>124<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El sexo en las aves se <a class="mw-redirect" href="/wiki/Determinaci%C3%B3n_del_sexo" title="Determinación del sexo">determina</a> por <a class="mw-redirect" href="/wiki/Cromosomas_sexuales" title="Cromosomas sexuales">cromosomas sexuales</a>, denominados Z y W. Las hembras son <a class="mw-redirect" href="/wiki/Heterogam%C3%A9tico" title="Heterogamético">heterogaméticas</a> y los machos <a class="mw-redirect" href="/wiki/Homogam%C3%A9tico" title="Homogamético">homogaméticos</a>. Esto significa que la hembra tiene sus cromosomas sexuales diferentes (ZW) y produce dos tipos de <a href="/wiki/%C3%93vulo" title="Óvulo">óvulos</a>, lo que determina el sexo del futuro <a href="/wiki/Cigoto" title="Cigoto">cigoto</a>. En el macho, en cambio, sus cromosomas sexuales son idénticos (ZZ) y sus <a href="/wiki/Espermatozoide" title="Espermatozoide">espermatozoides</a>, todos portadores de cromosoma Z, no determinan el sexo de la descendencia. Al contrario de lo que ocurre en los mamíferos, donde los cromosomas sexuales, llamados en este caso X e Y, son diferentes en el macho (XY) y similares en la hembra (XX).<sup class="reference separada" id="cite_ref-Gill_88-9"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>En todas las especies de aves el sexo del individuo se determina en la fertilización. Sin embargo un estudio reciente ha demostrado que en el caso de los <a href="/wiki/Alectura_lathami" title="Alectura lathami">talégalos cabecirrojos</a> la temperatura durante la incubación influye en la proporción de sexos en la descendencia; a mayores temperaturas, más cantidad de hembras y viceversa;<sup class="reference separada" id="cite_ref-126"><a href="#cite_note-126"><span class="corchete-llamada">[</span>125<span class="corchete-llamada">]</span></a></sup>​ pero esto no se debe a que la temperatura altere los mecanismos genéticos de determinación del sexo, sino a que con temperaturas extremas de incubación, el desarrollo y mortalidad de los pollos es diferente según su sexo: a altas temperaturas los machos se desarrollan peor y mueren más, y a bajas temperaturas le ocurre lo mismo a las hembras.<sup class="reference separada" id="cite_ref-127"><a href="#cite_note-127"><span class="corchete-llamada">[</span>126<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>A diferencia de la <a href="/wiki/Piel" title="Piel">piel</a> de los mamíferos, la de las aves es delgada y seca y no posee <a class="mw-redirect" href="/wiki/Gl%C3%A1ndulas_sudor%C3%ADparas" title="Glándulas sudoríparas">glándulas sudoríparas</a>; de hecho, la única <a href="/wiki/Gl%C3%A1ndula" title="Glándula">glándula</a> cutánea de las aves es la <a href="/wiki/Gl%C3%A1ndula_urop%C3%ADgea" title="Glándula uropígea">glándula uropígea</a>, situada en la base de la cola, que secreta <a href="/wiki/Grasa" title="Grasa">grasa</a> que el ave esparce por su plumaje con el pico; dicha glándula está especialmente desarrollada en las aves acuáticas, con lo que consiguen una mayor impermeabilización.<sup class="reference separada" id="cite_ref-you_128-0"><a href="#cite_note-you-128"><span class="corchete-llamada">[</span>127<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las <a href="/wiki/Pluma" title="Pluma">plumas</a> son una característica única de las aves. Les permiten <a href="/wiki/Vuelo_de_las_aves" title="Vuelo de las aves">volar</a>, proporcionan aislamiento térmico al impedir la circulación del aire que ayuda en la <a href="/wiki/Termorregulaci%C3%B3n" title="Termorregulación">termorregulación</a>, y son usadas para la exhibición, camuflaje, e identificación.<sup class="reference separada" id="cite_ref-Gill_88-10"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ Hay varios tipos de plumas, y cada una tiene unas funciones y características determinadas: las <a href="/wiki/Pluma_de_vuelo" title="Pluma de vuelo">plumas de vuelo</a> o rémiges (primarias, secundarias y terciarias); las <a class="mw-redirect" href="/wiki/Rectriz" title="Rectriz">rectrices</a> (plumas de la cola, que sirven como timón en el vuelo); las <a class="new" href="/w/index.php?title=Cobertera_(pluma)&amp;action=edit&amp;redlink=1" title="Cobertera (pluma) (aún no redactado)">coberteras</a> (que cubren parcialmente las rémiges y también las rectrices); las <a class="new" href="/w/index.php?title=Tectriz&amp;action=edit&amp;redlink=1" title="Tectriz (aún no redactado)">tectrices</a> (que cubren todo el cuerpo y lo protegen frente a agentes adversos) y el <a href="/wiki/Plum%C3%B3n" title="Plumón">plumón</a> (que evita la pérdida de calor).<sup class="reference separada" id="cite_ref-huellas_92-1"><a href="#cite_note-huellas-92"><span class="corchete-llamada">[</span>91<span class="corchete-llamada">]</span></a></sup>​ Las plumas son formaciones <a href="/wiki/Epidermis" title="Epidermis">epidérmicas</a> de <a href="/wiki/Queratina" title="Queratina">queratina</a> unidas a la piel y surgen solo en series específicas de la piel llamadas <a class="mw-redirect" href="/wiki/Pterilo" title="Pterilo">pterilos</a>. El patrón de distribución de estas series de plumas (<a href="/wiki/Pterilosis" title="Pterilosis">pterilosis</a>) se usa en taxonomía y sistemática. El ordenamiento y el aspecto de las plumas en el cuerpo, llamado <a href="/wiki/Plumaje" title="Plumaje">plumaje</a>, puede variar dentro de la especie por edad, posición social,<sup class="reference separada" id="cite_ref-129"><a href="#cite_note-129"><span class="corchete-llamada">[</span>128<span class="corchete-llamada">]</span></a></sup>​ y <a href="/wiki/Dimorfismo_sexual" title="Dimorfismo sexual">sexo</a>.<sup class="reference separada" id="cite_ref-130"><a href="#cite_note-130"><span class="corchete-llamada">[</span>129<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El plumaje es <a href="/wiki/Muda_(biolog%C3%ADa)" title="Muda (biología)">mudado</a> regularmente. El plumaje característico de un ave que ha mudado tras la reproducción se conoce como plumaje post-reproductivo, o en la <a href="/wiki/Terminolog%C3%ADa_Humphrey-Parkes" title="Terminología Humphrey-Parkes">terminología Humphrey-Parkes</a> plumaje «básico». Los plumajes reproductivos o las variaciones del plumaje básico se conocen en el sistema Humphrey-Parkes como plumajes «alternativos».<sup class="reference separada" id="cite_ref-131"><a href="#cite_note-131"><span class="corchete-llamada">[</span>130<span class="corchete-llamada">]</span></a></sup>​ La muda es anual en la mayoría de las especies, aunque algunas pueden tener dos mudas al año, y las grandes aves de presa pueden mudar solo una vez cada pocos años. Los patrones de muda varían entre especies. En los paseriformes, las plumas de vuelo son reemplazadas de una en una iniciando el ciclo con la primaria más interna. Cuando la quinta de las seis primarias es mudada, la terciaria más externa comienza a desprenderse. Después que las terciarias más internas son mudadas, las secundarias comienzan a mudarse desde la más interna y esto prosigue hasta las plumas más externas (muda centrífuga). Las coberteras primarias mayores se mudan sincrónicamente con las primarias con las que se superponen.<sup class="reference separada" id="cite_ref-pettingill_132-0"><a href="#cite_note-pettingill-132"><span class="corchete-llamada">[</span>131<span class="corchete-llamada">]</span></a></sup>​ Un pequeño número de especies, como los patos y gansos, pierden todas las plumas de vuelo a la vez, y quedan temporalmente sin capacidad de volar.<sup class="reference separada" id="cite_ref-debeeretal_133-0"><a href="#cite_note-debeeretal-133"><span class="corchete-llamada">[</span>132<span class="corchete-llamada">]</span></a></sup>​ Como regla general, las plumas de la cola se mudan y reemplazan comenzando desde el par más interior;<sup class="reference separada" id="cite_ref-pettingill_132-1"><a href="#cite_note-pettingill-132"><span class="corchete-llamada">[</span>131<span class="corchete-llamada">]</span></a></sup>​ sin embargo, se observan mudas centrípetas de plumas de la cola en <a href="/wiki/Phasianidae" title="Phasianidae">Phasianidae</a>.<sup class="reference separada" id="cite_ref-134"><a href="#cite_note-134"><span class="corchete-llamada">[</span>133<span class="corchete-llamada">]</span></a></sup>​ La muda centrífuga es diferente en las plumas de la cola de los <a href="/wiki/Picidae" title="Picidae">pájaros carpinteros</a> y los <a href="/wiki/Certhiidae" title="Certhiidae">trepatroncos</a> en los que comienza por el segundo par de plumas más internas y termina con el par central, de modo que el ave mantiene la capacidad de ayudarse con su cola para trepar.<sup class="reference separada" id="cite_ref-pettingill_132-2"><a href="#cite_note-pettingill-132"><span class="corchete-llamada">[</span>131<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-135"><a href="#cite_note-135"><span class="corchete-llamada">[</span>134<span class="corchete-llamada">]</span></a></sup>​ El patrón general que se ve en <a class="mw-redirect" href="/wiki/Paseriformes" title="Paseriformes">paseriformes</a> es que las primarias son reemplazadas hacia afuera, las secundarias hacia adentro, y la cola desde el centro hacia afuera.<sup class="reference separada" id="cite_ref-136"><a href="#cite_note-136"><span class="corchete-llamada">[</span>135<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Antes de anidar, en las hembras de la mayoría de las especies de aves se produce lo que se llama una <a href="/wiki/Placa_de_incubaci%C3%B3n" title="Placa de incubación">placa de incubación</a>, es decir, una zona libre de plumas cerca del abdomen. La piel está allí bien irrigada con vasos sanguíneos y ayuda al ave en la incubación.<sup class="reference separada" id="cite_ref-137"><a href="#cite_note-137"><span class="corchete-llamada">[</span>136<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las plumas requieren mantenimiento y las aves las acicalan o peinan diariamente, tomándose en promedio un 9 % de su tiempo diario en ello.<sup class="reference separada" id="cite_ref-138"><a href="#cite_note-138"><span class="corchete-llamada">[</span>137<span class="corchete-llamada">]</span></a></sup>​ El pico se usa para extraer partículas extrañas y para aplicar secreciones <a href="/wiki/Cera" title="Cera">cerosas</a> provenientes de la <a class="mw-redirect" href="/wiki/Gl%C3%A1ndula_uropigial" title="Glándula uropigial">glándula uropigial</a>. Estas secreciones protegen la flexibilidad de la pluma y actúan como agente antimicrobiano, inhibiendo el crecimiento de <a href="/wiki/Bacteria" title="Bacteria">bacterias</a> degradadoras de la pluma.<sup class="reference separada" id="cite_ref-139"><a href="#cite_note-139"><span class="corchete-llamada">[</span>138<span class="corchete-llamada">]</span></a></sup>​ Esto puede complementarse con secreciones de <a href="/wiki/%C3%81cido_f%C3%B3rmico" title="Ácido fórmico">ácido fórmico</a> de las hormigas, que reciben mediante un comportamiento conocido como «<a href="/wiki/Ba%C3%B1o_de_hormigas" title="Baño de hormigas">baño de hormigas</a>», para quitarse los parásitos de las plumas.<sup class="reference separada" id="cite_ref-140"><a href="#cite_note-140"><span class="corchete-llamada">[</span>139<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las <a href="/wiki/Escama" title="Escama">escamas</a> de las aves se componen de la misma queratina que las plumas, el pico, las garras y espolones. Se encuentran principalmente en los dedos del pie y en el <a href="/wiki/Metatarso" title="Metatarso">metatarso</a>, pero pueden encontrarse más arriba hasta el talón en algunas aves. La mayoría de las escamas de las aves no se superponen significativamente, excepto en los casos de los <a href="/wiki/Alcedines" title="Alcedines">martín-pescadores</a> y los <a href="/wiki/Picidae" title="Picidae">carpinteros</a>. Se piensa que las escamas de las aves son <a href="/wiki/Homolog%C3%ADa_(biolog%C3%ADa)" title="Homología (biología)">homólogas</a> a las de los reptiles y mamíferos.<sup class="reference separada" id="cite_ref-141"><a href="#cite_note-141"><span class="corchete-llamada">[</span>140<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La mayor parte de las aves pueden volar, lo que las distingue de casi todo el resto de vertebrados. Volar es el principal modo de locomoción para la mayoría de las aves y lo usan para reproducirse, alimentarse y huir de sus depredadores. Para volar, las aves han desarrollado diversas adaptaciones fisionómicas que incluyen un <a href="/wiki/Anatom%C3%ADa_de_las_aves#Sistema_esquelético" title="Anatomía de las aves">esqueleto ligero</a>, dos grandes músculos de vuelo (el <a class="new" href="/w/index.php?title=M%C3%BAsculo_pectoral&amp;action=edit&amp;redlink=1" title="Músculo pectoral (aún no redactado)">pectoral</a> que es el 15 % de la masa total del ave, y el <a class="new" href="/w/index.php?title=M%C3%BAsculo_supracoracoideo&amp;action=edit&amp;redlink=1" title="Músculo supracoracoideo (aún no redactado)">supracoracoideo</a>), y dos miembros modificados (<a href="/wiki/Ala_(zoolog%C3%ADa)" title="Ala (zoología)">alas</a>) que sirven como <a href="/wiki/Perfil_alar" title="Perfil alar">perfiles alares</a>.<sup class="reference separada" id="cite_ref-Gill_88-11"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ La forma y el tamaño de las alas determinan el tipo de vuelo de cada ave; muchas especies combinan un estilo de vuelo basado en fuertes aleteos, con un vuelo de planeo que requiere menos energía. La <a href="/wiki/Anexo:Aves_de_gran_altitud_de_vuelo" title="Anexo:Aves de gran altitud de vuelo">altitud de vuelo de las aves</a> presenta un rango enorme, pues varía desde el nivel del mar hasta por encima de los 11 000 m.<sup class="reference separada" id="cite_ref-laybourne_142-0"><a href="#cite_note-laybourne-142"><span class="corchete-llamada">[</span>141<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-Carwardine_143-0"><a href="#cite_note-Carwardine-143"><span class="corchete-llamada">[</span>142<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Alrededor de 60 especies de aves son <a class="mw-redirect" href="/wiki/Aves_no_voladoras" title="Aves no voladoras">no voladoras</a>, también un buen número de especies extintas carecían de la capacidad de volar.<sup class="reference separada" id="cite_ref-144"><a href="#cite_note-144"><span class="corchete-llamada">[</span>143<span class="corchete-llamada">]</span></a></sup>​ Las aves no voladoras a menudo se encuentran en islas aisladas, probablemente debido a una escasez de recursos y a la ausencia de depredadores terrestres.<sup class="reference separada" id="cite_ref-145"><a href="#cite_note-145"><span class="corchete-llamada">[</span>144<span class="corchete-llamada">]</span></a></sup>​ A pesar de que no pueden volar, los pingüinos usan una musculatura y unos movimientos similares para <i>volar</i> a través del agua; así lo hacen también los <a href="/wiki/Alcidae" title="Alcidae">álcidos</a>, las <a href="/wiki/Procellariidae" title="Procellariidae">pardelas</a> y los <a href="/wiki/Cinclidae" title="Cinclidae">mirlos acuáticos</a>.<sup class="reference separada" id="cite_ref-146"><a href="#cite_note-146"><span class="corchete-llamada">[</span>145<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La menor de las aves es el macho del <a href="/wiki/Mellisuga_helenae" title="Mellisuga helenae">colibrí zunzuncito</a> de unos 64 mm de largo,<sup class="reference separada" id="cite_ref-147"><a href="#cite_note-147"><span class="corchete-llamada">[</span>146<span class="corchete-llamada">]</span></a></sup>​ y 2,8 g,<sup class="reference separada" id="cite_ref-Storer_104-1"><a href="#cite_note-Storer-104"><span class="corchete-llamada">[</span>103<span class="corchete-llamada">]</span></a></sup>​ y envergadura alar de unos 78 mm.<sup class="reference separada" id="cite_ref-148"><a href="#cite_note-148"><span class="corchete-llamada">[</span>147<span class="corchete-llamada">]</span></a></sup>​ El ave actual más grande es el <a href="/wiki/Struthio_camelus" title="Struthio camelus">avestruz</a> con una altura de hasta 2,74 metros y 155 kg de peso, pone también los huevos más grandes, con promedios de 15 por 13 cm y un peso de 1,4 kg.<sup class="reference separada" id="cite_ref-149"><a href="#cite_note-149"><span class="corchete-llamada">[</span>148<span class="corchete-llamada">]</span></a></sup>​ Las aves más altas que hayan existido y de las cuales se tengan registros eran las <a href="/wiki/Dinornis" title="Dinornis">moas gigantes</a> que medían hasta 3,6 m, pesaban más de 230 kg y habitaban en <a href="/wiki/Nueva_Zelanda" title="Nueva Zelanda">Nueva Zelanda</a> hasta, al menos, el siglo XVI.<sup class="reference separada" id="cite_ref-150"><a href="#cite_note-150"><span class="corchete-llamada">[</span>149<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-151"><a href="#cite_note-151"><span class="corchete-llamada">[</span>150<span class="corchete-llamada">]</span></a></sup>​ El ave más pesada que haya coexistido con el hombre es la <a class="mw-redirect" href="/wiki/Extinta" title="Extinta">extinta</a> <a class="mw-redirect" href="/wiki/Aepyornis_maximus" title="Aepyornis maximus">ave elefante</a> de <a href="/wiki/Madagascar" title="Madagascar">Madagascar</a>, que llegaba a medir hasta tres metros de altura y unos 500 kg de peso.<sup class="reference separada" id="cite_ref-152"><a href="#cite_note-152"><span class="corchete-llamada">[</span>151<span class="corchete-llamada">]</span></a></sup>​ De dimensiones similares era <i><a href="/wiki/Dromornis_stirtoni" title="Dromornis stirtoni">Dromornis stirtoni</a></i> de Australia.<sup class="reference separada" id="cite_ref-153"><a href="#cite_note-153"><span class="corchete-llamada">[</span>152<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Los <a href="/wiki/Diomedea_exulans" title="Diomedea exulans">albatros viajeros</a> poseen la mayor envergadura alar de todos los animales voladores actuales, pueden exceder los 340 cm, con peso de más de 10 kg.<sup class="reference separada" id="cite_ref-154"><a href="#cite_note-154"><span class="corchete-llamada">[</span>153<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-teratorns_155-0"><a href="#cite_note-teratorns-155"><span class="corchete-llamada">[</span>154<span class="corchete-llamada">]</span></a></sup>​ La gigantesca ave fósil <i><a href="/wiki/Argentavis_magnificens" title="Argentavis magnificens">Argentavis magnificens</a></i> es la mayor ave voladora que se ha descubierto; se estima que su envergadura era de más de 5,8 metros y un peso de 60–80 kg.<sup class="reference separada" id="cite_ref-teratorns_155-1"><a href="#cite_note-teratorns-155"><span class="corchete-llamada">[</span>154<span class="corchete-llamada">]</span></a></sup>​ Son varias las especies actuales de aves voladoras consideradas como las más pesadas, las cuales no suelen superar los 20 kg, entre ellas la <a href="/wiki/Otis_tarda" title="Otis tarda">avutarda común</a> y la <a href="/wiki/Ardeotis_kori" title="Ardeotis kori">avutarda kori</a>.<sup class="reference separada" id="cite_ref-156"><a href="#cite_note-156"><span class="corchete-llamada">[</span>155<span class="corchete-llamada">]</span></a></sup>​ El <a href="/wiki/Vultur_gryphus" title="Vultur gryphus">cóndor andino</a> no supera a las anteriores pero es un ave voladora de dimensiones considerables, con una longitud de hasta 120 cm, envergadura que llega a los 320 cm y un peso de hasta 15 kg.<sup class="reference separada" id="cite_ref-teratorns_155-2"><a href="#cite_note-teratorns-155"><span class="corchete-llamada">[</span>154<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-157"><a href="#cite_note-157"><span class="corchete-llamada">[</span>156<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La vida de las aves se organiza en función de varios ritmos biológicos. El más común, como en otros vertebrados, es el <a href="/wiki/Ritmo_circadiano" title="Ritmo circadiano">ritmo circadiano</a>. La mayoría de las aves son diurnas, pero algunas, como la mayor parte de las <a href="/wiki/Strigiformes" title="Strigiformes">rapaces nocturnas</a> y los <a href="/wiki/Caprimulgidae" title="Caprimulgidae">chotacabras</a> son nocturnas o crepusculares. Otras especies, como la mayoría de las <a class="mw-redirect" href="/wiki/Lim%C3%ADcola" title="Limícola">limícolas</a>, siguen un ritmo basado en las <a href="/wiki/Marea" title="Marea">mareas</a>.<sup class="reference separada" id="cite_ref-158"><a href="#cite_note-158"><span class="corchete-llamada">[</span>157<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las aves, debido a la existencia de estaciones, siguen también un <a class="mw-redirect" href="/wiki/Ritmo_circanual" title="Ritmo circanual">ritmo circanual</a>. Las <a href="/wiki/Migraci%C3%B3n_de_las_aves" title="Migración de las aves">aves que migran grandes distancias</a> sufren generalmente cambios anatómicos o de comportamiento (como, por ejemplo, el <a href="/wiki/Zugunruhe" title="Zugunruhe">zugunruhe</a>) o una <a class="mw-redirect" href="/wiki/Muda_de_las_aves" title="Muda de las aves">muda</a> para preparase para el viaje. Según la estación, ciertas especies pueden realizar igualmente migraciones diarias, altitudinales por ejemplo, o para llegar a sus zonas de aprovisionamiento.
</p>
<p>Los ritmos circadianos y estacionales parecen estar ligados a la duración del día. Los ciclos de reproducción son anuales, pero en ciertas especies particularmente prolíficas pueden sacar adelante varias nidadas una misma estación.
</p>
<p>La duración de la vida de las aves es muy variable según las especies, puede ser de tres o cuatro años para algunas paseriformes, a más de 50 años para los <a href="/wiki/Diomedeidae" title="Diomedeidae">albatros</a>, <a href="/wiki/Procellariidae" title="Procellariidae">petreles y pardelas</a>, o incluso más de 60 años para ciertas especies raras como el <a class="mw-redirect" href="/wiki/Strigops_habroptila" title="Strigops habroptila">kakapo</a>.<sup class="reference separada" id="cite_ref-159"><a href="#cite_note-159"><span class="corchete-llamada">[</span>158<span class="corchete-llamada">]</span></a></sup>​ Para deducir aproximadamente la edad de un ave hay que tener buenos conocimientos sobre fenómenos como las variaciones de la muda según la edad (además de la estación),<sup class="reference separada" id="cite_ref-160"><a href="#cite_note-160"><span class="corchete-llamada">[</span>159<span class="corchete-llamada">]</span></a></sup>​ o la <a class="new" href="/w/index.php?title=Neumatizaci%C3%B3n&amp;action=edit&amp;redlink=1" title="Neumatización (aún no redactado)">neumatización</a> del esqueleto al envejecer.
</p>
<p>Aunque <i>tener cerebro de pájaro</i> o <i>ser un cabeza de chorlito</i> significa no tener <a href="/wiki/Inteligencia" title="Inteligencia">inteligencia</a> en varias <a href="/wiki/Cultura" title="Cultura">culturas</a>, ciertas especies de aves dan pruebas de capacidades cognitivas relativamente elevadas. Las especies de <a href="/wiki/Corvidae" title="Corvidae">Corvidae</a> tienen fama de ser las aves más inteligentes;<sup class="reference separada" id="cite_ref-161"><a href="#cite_note-161"><span class="corchete-llamada">[</span>160<span class="corchete-llamada">]</span></a></sup>​ los <a href="/wiki/Psittaciformes" title="Psittaciformes">loros</a> son también capaces de demostraciones sorprendentes, pero con bastante disparidad entre las especies. Por otra parte, es difícil definir el término «inteligencia» y también el distinguir aquello que es parte del dominio de lo <a class="mw-redirect" href="/wiki/Innato_o_adquirido" title="Innato o adquirido">innato</a> o del dominio de lo adquirido, y por lo tanto de evaluar sus capacidades de razonamiento.
</p>
<p>Las aves son capaces de aprender; se sabe por ejemplo que los cucos aprenden los cantos de sus padres adoptivos o que los cuervos hacen su aprendizaje imitando a sus semejantes.<sup class="reference separada" id="cite_ref-162"><a href="#cite_note-162"><span class="corchete-llamada">[</span>161<span class="corchete-llamada">]</span></a></sup>​
Sus capacidades más comunes son ciertamente la representación espacial (que les permite orientarse, reencontrar sus fuentes de alimento o construir nidos sofisticados) y la capacidad de comunicación.
</p>
<p>Una de las capacidades más sorprendentes es la aptitud bastante difundida de servirse de un objeto como utensilio.<sup class="reference separada" id="cite_ref-163"><a href="#cite_note-163"><span class="corchete-llamada">[</span>162<span class="corchete-llamada">]</span></a></sup>​ El <a href="/wiki/Corvus_moneduloides" title="Corvus moneduloides">cuervo neocaledoniano</a>, por ejemplo, es capaz de usar un palo para sacar de los troncos los insectos con los que se nutre.
Ciertas aves son también capaces de contar, como los <a href="/wiki/Psittacidae" title="Psittacidae">loros</a>, que son también conocidos no solo por reproducir la voz humana, sino también por comprender lo que dicen y utilizar su vocabulario con acierto.
</p>
<p>Se ha observado igualmente a aves capaces de <i>medicarse</i>, por ejemplo al ingerir arcilla para combatir los efectos nefastos de toxinas alimentarias.<sup class="reference separada" id="cite_ref-164"><a href="#cite_note-164"><span class="corchete-llamada">[</span>163<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Ciertas facultades son prácticamente únicas, el <a href="/wiki/Merops_orientalis" title="Merops orientalis">abejaruco verde chico</a> es capaz de «meterse en la cabeza» de sus predadores, es decir de calcular lo que el predador puede ver o no.<sup class="reference separada" id="cite_ref-165"><a href="#cite_note-165"><span class="corchete-llamada">[</span>164<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La mayoría de las aves son diurnas, pero algunas especies, sobre todo <a href="/wiki/Strigiformes" title="Strigiformes">búhos</a> y <a href="/wiki/Caprimulgidae" title="Caprimulgidae">chotacabras</a>, son nocturnas o crepusculares; y muchas <a class="mw-redirect" href="/wiki/Ave_lim%C3%ADcola" title="Ave limícola">aves limícolas</a> costeras se alimentan cuando las mareas les son propicias ya sea de día o de noche.<sup class="reference separada" id="cite_ref-166"><a href="#cite_note-166"><span class="corchete-llamada">[</span>165<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La dieta de las aves incluye una gran cantidad de tipos de alimentos como <a class="mw-redirect" href="/wiki/N%C3%A9ctar_(Bot%C3%A1nica)" title="Néctar (Botánica)">néctar</a>, <a href="/wiki/Fruta" title="Fruta">frutas</a>, <a href="/wiki/Plantae" title="Plantae">plantas</a>, <a href="/wiki/Semilla" title="Semilla">semillas</a>, <a href="/wiki/Carro%C3%B1a" title="Carroña">carroña</a>, y diversos animales pequeños, incluidas otras aves.<sup class="reference separada" id="cite_ref-Gill_88-12"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ Como las aves no tienen dientes, su <a href="/wiki/Aparato_digestivo" title="Aparato digestivo">aparato digestivo</a> está adaptado a procesar alimentos <a href="/wiki/Masticaci%C3%B3n" title="Masticación">sin masticar</a> que el ave traga enteros.
</p>
<p>Las aves llamadas generalistas son las que emplean muchas y diferentes estrategias para conseguir alimentos de una amplia variedad de tipos, mientras que las que se concentran en un espectro reducido de alimentos o tienen una única estrategia para conseguir comida son consideradas especialistas.<sup class="reference separada" id="cite_ref-Gill_88-13"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ Las estrategias de alimentación de las aves varían según la especie. Algunas cazan <a class="mw-redirect" href="/wiki/Insectos" title="Insectos">insectos</a> lanzándose sorpresivamente desde una rama. Las especies que se alimentan de néctar, como los <a href="/wiki/Trochilidae" title="Trochilidae">colibríes</a>, los <a href="/wiki/Nectariniidae" title="Nectariniidae">suimangas</a> y los <a href="/wiki/Loriinae" title="Loriinae">loris</a>, tienen lenguas pelosas y formas de pico especialmente adaptadas para ajustarse a las plantas de las que se alimentan.<sup class="reference separada" id="cite_ref-167"><a href="#cite_note-167"><span class="corchete-llamada">[</span>166<span class="corchete-llamada">]</span></a></sup>​ Los <a href="/wiki/Apteryx" title="Apteryx">kiwis</a> y las <a class="mw-redirect" href="/wiki/Lim%C3%ADcola" title="Limícola">aves limícolas</a> tienen largos picos que usan para sondear el suelo en busca de invertebrados; en el caso de las limícolas, sus picos presentan diferentes longitudes y curvaturas, ya que cada especie tiene un <a href="/wiki/Nicho_ecol%C3%B3gico" title="Nicho ecológico">nicho ecológico</a> diferente.<sup class="reference separada" id="cite_ref-Gill_88-14"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-168"><a href="#cite_note-168"><span class="corchete-llamada">[</span>167<span class="corchete-llamada">]</span></a></sup>​ Los <a class="mw-redirect" href="/wiki/Colimbo" title="Colimbo">colimbos</a>, <a href="/wiki/Aythynae" title="Aythynae">patos buceadores</a>, <a class="mw-redirect" href="/wiki/Ping%C3%BCino" title="Pingüino">pingüinos</a> y <a class="mw-redirect" href="/wiki/%C3%81lcido" title="Álcido">álcidos</a> persiguen a sus presas bajo el agua, usando sus alas y/o sus pies para propulsarse,<sup class="reference separada" id="cite_ref-Burger_79-1"><a href="#cite_note-Burger-79"><span class="corchete-llamada">[</span>78<span class="corchete-llamada">]</span></a></sup>​ mientras que los <a href="/wiki/Sulidae" title="Sulidae">alcatraces</a>, <a href="/wiki/Alcedines" title="Alcedines">martines pescadores</a> y <a href="/wiki/Sternidae" title="Sternidae">charranes</a> son predadores aéreos que se sumergen en picado en busca de su presa. Los flamencos, tres especies de <a href="/wiki/Pachyptila" title="Pachyptila">petreles</a>, y algunos patos, se alimentan filtrando el agua.<sup class="reference separada" id="cite_ref-169"><a href="#cite_note-169"><span class="corchete-llamada">[</span>168<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-170"><a href="#cite_note-170"><span class="corchete-llamada">[</span>169<span class="corchete-llamada">]</span></a></sup>​ Otras aves, como los <a href="/wiki/Ganso" title="Ganso">gansos</a> y los <a href="/wiki/Anatinae" title="Anatinae">patos nadadores</a>, se alimentan principalmente pastando. Algunas especies, entre las que se incluyen las <a href="/wiki/Fregata" title="Fregata">fragatas</a>, las <a href="/wiki/Laridae" title="Laridae">gaviotas</a>,<sup class="reference separada" id="cite_ref-171"><a href="#cite_note-171"><span class="corchete-llamada">[</span>170<span class="corchete-llamada">]</span></a></sup>​ y los <a href="/wiki/Stercorariidae" title="Stercorariidae">págalos</a>,<sup class="reference separada" id="cite_ref-172"><a href="#cite_note-172"><span class="corchete-llamada">[</span>171<span class="corchete-llamada">]</span></a></sup>​ son <a href="/wiki/Cleptoparasitismo" title="Cleptoparasitismo">cleptoparásitas</a>, es decir, roban comida a otras aves. Se supone que con esto logran un suplemento adicional, pero no una parte importante de su dieta general; un caso estudiado de cleptoparasitismo de la <a href="/wiki/Fregata_minor" title="Fregata minor">fragata grande</a> sobre el <a href="/wiki/Sula_dactylatra" title="Sula dactylatra">alcatraz enmascarado</a> determinó que obtenían en promedio solo el 5 % de su comida, y como máximo un 40 %.<sup class="reference separada" id="cite_ref-173"><a href="#cite_note-173"><span class="corchete-llamada">[</span>172<span class="corchete-llamada">]</span></a></sup>​ Hay otras aves que son <a class="mw-redirect" href="/wiki/Carro%C3%B1era" title="Carroñera">carroñeras</a>, algunas de las cuales, como los buitres, están especializados en comer cadáveres, mientras que otras, como las gaviotas, los <a class="mw-redirect" href="/wiki/C%C3%B3rvido" title="Córvido">córvidos</a> o algunas <a href="/wiki/Ave_de_presa" title="Ave de presa">aves de presa</a> lo hacen solo como oportunistas.<sup class="reference separada" id="cite_ref-174"><a href="#cite_note-174"><span class="corchete-llamada">[</span>173<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La mayoría de las aves necesitan beber, aunque su demanda fisiológica de agua se ve reducida por la <a href="/wiki/Excreci%C3%B3n" title="Excreción">excreción uricotélica</a> y la ausencia de <a href="/wiki/Gl%C3%A1ndula_sudor%C3%ADpara" title="Glándula sudorípara">glándulas sudoríparas</a>.<sup class="reference separada" id="cite_ref-175"><a href="#cite_note-175"><span class="corchete-llamada">[</span>174<span class="corchete-llamada">]</span></a></sup>​ Además, pueden refrescarse moviéndose a la sombra, metiéndose en el agua, jadeando, agitando su garganta, o con algunos comportamientos especiales como la <a href="/wiki/Urohidrosis" title="Urohidrosis">urohidrosis</a>. Algunas aves del <a href="/wiki/Desierto" title="Desierto">desierto</a> pueden obtener toda el agua que necesitan de su alimento. También pueden presentar otras adaptaciones, como permitir que la temperatura de su cuerpo se eleve, lo que evita la pérdida de humedad que se produciría mediante el enfriamiento por evaporación o por jadeo.<sup class="reference separada" id="cite_ref-176"><a href="#cite_note-176"><span class="corchete-llamada">[</span>175<span class="corchete-llamada">]</span></a></sup>​ Las <a href="/wiki/Ave_marina" title="Ave marina">aves marinas</a> pueden beber agua del mar, ya que tienen unas glándulas en la cabeza que usan para eliminar el exceso de sal, que expulsan a través de las fosas nasales.<sup class="reference separada" id="cite_ref-177"><a href="#cite_note-177"><span class="corchete-llamada">[</span>176<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La mayor parte de las aves recogen el agua con el pico y después elevan su cabeza para dejar que el agua caiga por la garganta. Todas las <a href="/wiki/Columbidae" title="Columbidae">palomas</a> y algunas especies, especialmente en las zonas áridas, pertenecientes a las familias de los <a href="/wiki/Estrildidae" title="Estrildidae">pinzones tejedores</a>, los <a href="/wiki/Coliidae" title="Coliidae">cólidos</a>, los <a href="/wiki/Turnicidae" title="Turnicidae">turnícidos</a> y las <a class="mw-redirect" href="/wiki/Otididae" title="Otididae">avutardas</a>, son capaces de beber agua sin necesidad de echar para atrás la cabeza.<sup class="reference separada" id="cite_ref-178"><a href="#cite_note-178"><span class="corchete-llamada">[</span>177<span class="corchete-llamada">]</span></a></sup>​ Algunas especies del desierto dependen de fuentes de agua; tal es el caso de las <a class="mw-redirect" href="/wiki/Pteroclididae" title="Pteroclididae">gangas</a> que se concentran diariamente en grandes números en los abrevaderos, y transportan el agua para sus pollos en las plumas mojadas del vientre.<sup class="reference separada" id="cite_ref-179"><a href="#cite_note-179"><span class="corchete-llamada">[</span>178<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Muchas especies de aves migran para aprovecharse de las diferencias estacionales de temperatura en el mundo, con lo que optimizan la disponibilidad de fuentes de alimento y de hábitats reproductivos. Las migraciones varían mucho según la especie. Muchas realizan largas migraciones anuales, por lo general provocadas por los cambios en la duración del día así como por las condiciones meteorológicas. Estas aves se caracterizan por pasar la temporada de cría en regiones <a href="/wiki/Clima_templado" title="Clima templado">templadas</a> o <a href="/wiki/Regiones_polares" title="Regiones polares">polares</a>, e invernar en regiones templadas más cálidas, <a href="/wiki/Tr%C3%B3pico" title="Trópico">tropicales</a>, o en el hemisferio contrario.
</p>
<p>Antes de la migración incrementan sustancialmente sus grasas y reservas corporales, y reducen el tamaño de algunos de sus órganos.<sup class="reference separada" id="cite_ref-Battley_103-1"><a href="#cite_note-Battley-103"><span class="corchete-llamada">[</span>102<span class="corchete-llamada">]</span></a></sup>​<sup class="reference separada" id="cite_ref-Klaassen_180-0"><a href="#cite_note-Klaassen-180"><span class="corchete-llamada">[</span>179<span class="corchete-llamada">]</span></a></sup>​ La migración es una actividad que consume mucha energía, sobre todo cuando el ave debe cruzar desiertos y océanos sin poder reabastecerse. Las aves terrestres tienen una autonomía de vuelo de unos 2500 km y las aves limícolas de unos 4000 km,<sup class="reference separada" id="cite_ref-Gill_88-15"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ aunque la <a href="/wiki/Limosa_lapponica" title="Limosa lapponica">aguja colipinta</a> puede llegar a volar 10 200 km sin parar.<sup class="reference separada" id="cite_ref-181"><a href="#cite_note-181"><span class="corchete-llamada">[</span>180<span class="corchete-llamada">]</span></a></sup>​ Las <a href="/wiki/Ave_marina" title="Ave marina">aves marinas</a> también llevan a cabo largas migraciones; la migración anual más larga es la realizada por la <a class="mw-redirect" href="/wiki/Puffinus_griseus" title="Puffinus griseus">pardela sombría</a>, que cría en <a href="/wiki/Nueva_Zelanda" title="Nueva Zelanda">Nueva Zelanda</a> y <a href="/wiki/Chile" title="Chile">Chile</a> y pasa el verano del <a class="mw-redirect" href="/wiki/Hemisferio_Norte" title="Hemisferio Norte">Hemisferio Norte</a> alimentándose en el <a class="mw-redirect" href="/wiki/Pac%C3%ADfico_Norte" title="Pacífico Norte">Pacífico Norte</a> (<a href="/wiki/Jap%C3%B3n" title="Japón">Japón</a>, <a href="/wiki/Alaska" title="Alaska">Alaska</a> y <a href="/wiki/California" title="California">California</a>); en total hace unos 64 000 km al año.<sup class="reference separada" id="cite_ref-182"><a href="#cite_note-182"><span class="corchete-llamada">[</span>181<span class="corchete-llamada">]</span></a></sup>​ Otras aves marinas se dispersan después de la época de cría, viajando mucho pero sin una ruta establecida. Los <a href="/wiki/Diomedeidae" title="Diomedeidae">albatros</a>, que crían en el <a class="mw-redirect" href="/wiki/Oc%C3%A9ano_Ant%C3%A1rtico" title="Océano Antártico">océano Antártico</a>, a menudo realizan viajes circumpolares en los períodos no reproductivos.<sup class="reference separada" id="cite_ref-183"><a href="#cite_note-183"><span class="corchete-llamada">[</span>182<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Algunas especies de aves realizan migraciones más cortas, viajando solo lo necesario para evitar el mal tiempo o conseguir comida. Especies como algunos <a class="mw-redirect" href="/wiki/Fring%C3%ADlidos" title="Fringílidos">fringílidos</a> boreales tienen comportamientos <a class="mw-redirect" href="/wiki/Irrupci%C3%B3n" title="Irrupción">irruptivos</a>, al estar presentes en un lugar un año y ausentes al siguiente. Este tipo de desplazamiento suele estar asociado a disponibilidad de alimentos.<sup class="reference separada" id="cite_ref-184"><a href="#cite_note-184"><span class="corchete-llamada">[</span>183<span class="corchete-llamada">]</span></a></sup>​ También pueden realizar desplazamientos pequeños dentro de su área de distribución, con individuos de latitudes más altas que se mueven hacia los territorios sureños de sus congéneres; otras aves realizan migraciones parciales, en las cuales solo una parte de la población (normalmente hembras y machos subdominantes) migran.<sup class="reference separada" id="cite_ref-185"><a href="#cite_note-185"><span class="corchete-llamada">[</span>184<span class="corchete-llamada">]</span></a></sup>​ En algunas regiones, la migración parcial puede ser un gran porcentaje de las formas de migración de sus avifaunas; en <a href="/wiki/Australia" title="Australia">Australia</a>, algunos estudios han calculado que en torno al 44 % de las aves no-paseriformes y el 32 % de las paseriformes son parcialmente migratorias.<sup class="reference separada" id="cite_ref-186"><a href="#cite_note-186"><span class="corchete-llamada">[</span>185<span class="corchete-llamada">]</span></a></sup>​ La migración altitudinal es un tipo de migración de corta distancia, en la que las aves pasan la época de cría a mayores altitudes, y en épocas menos favorables se desplazan hacia altitudes menores. Por lo general están provocadas por cambios de temperatura, y normalmente ocurren cuando los territorios se vuelven inhóspitos por la falta de comida.<sup class="reference separada" id="cite_ref-187"><a href="#cite_note-187"><span class="corchete-llamada">[</span>186<span class="corchete-llamada">]</span></a></sup>​ Algunas especies pueden ser nómadas, sin tener un territorio fijo y se desplazan en función del tiempo meteorológico y de la disponibilidad de alimentos. La mayoría de los <a href="/wiki/Psittacidae" title="Psittacidae">loros</a> no son ni migradores, ni sedentarios, sino que son dispersivos, irruptivos, nómadas, o realizan desplazamientos pequeños e irregulares.<sup class="reference separada" id="cite_ref-188"><a href="#cite_note-188"><span class="corchete-llamada">[</span>187<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>La capacidad de las aves de volver a un lugar concreto tras recorrer grandes distancias se conoce desde hace cierto tiempo; en un experimento realizado en la década de 1950, una <a href="/wiki/Puffinus_puffinus" title="Puffinus puffinus">pardela pichoneta</a> liberada en <a href="/wiki/Boston" title="Boston">Boston</a>, volvió a su colonia en <a href="/wiki/Skomer" title="Skomer">Skomer</a> (<a href="/wiki/Gales" title="Gales">Gales</a>) en 13 días, cubriendo una distancia de 5150 km.<sup class="reference separada" id="cite_ref-189"><a href="#cite_note-189"><span class="corchete-llamada">[</span>188<span class="corchete-llamada">]</span></a></sup>​ Las aves se orientan durante la migración usando diversos métodos; durante el día usan el sol para guiarse, y las estrellas son la referencia por la noche. Las especies que se orientan por el sol usan un <a href="/wiki/Cronobiolog%C3%ADa" title="Cronobiología">reloj interno</a> para compensar los cambios de posición del astro a lo largo del día.<sup class="reference separada" id="cite_ref-Gill_88-16"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ La orientación a partir de las estrellas se basa en la posición de las <a href="/wiki/Constelaci%C3%B3n" title="Constelación">constelaciones</a> alrededor de los <a href="/wiki/Polo_celeste" title="Polo celeste">polos celestes</a>.<sup class="reference separada" id="cite_ref-190"><a href="#cite_note-190"><span class="corchete-llamada">[</span>189<span class="corchete-llamada">]</span></a></sup>​ En algunas especies, esto es reforzado además con su capacidad de sentir el <a href="/wiki/Campo_magn%C3%A9tico_terrestre" title="Campo magnético terrestre">magnetismo de la Tierra</a> a través de <a href="/wiki/Fotorreceptor" title="Fotorreceptor">fotorreceptores</a> especializados.<sup class="reference separada" id="cite_ref-191"><a href="#cite_note-191"><span class="corchete-llamada">[</span>190<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las aves se comunican principalmente a través de señales visuales y auditivas. Las señales pueden ser interespecíficas (entre especies distintas) o intraespecíficas (de una sola especie).
</p>
<p>En ocasiones usan su plumaje para establecer o reafirmar su posición social;<sup class="reference separada" id="cite_ref-192"><a href="#cite_note-192"><span class="corchete-llamada">[</span>191<span class="corchete-llamada">]</span></a></sup>​ para indicar su receptividad sexual, o para intimidar, como en el caso de la exhibición de la <a href="/wiki/Eurypyga_helias" title="Eurypyga helias">tigana</a> que busca ahuyentar a sus predadores y proteger a sus pollos.<sup class="reference separada" id="cite_ref-193"><a href="#cite_note-193"><span class="corchete-llamada">[</span>192<span class="corchete-llamada">]</span></a></sup>​ Las variaciones del plumaje permiten la identificación de las aves, sobre todo entre especies. La comunicación visual en las aves incluye exhibiciones rituales, que se conforman de acciones <i>habituales</i> como acicalarse las plumas, picotazos y otros. Estas demostraciones pueden ser señales de amenaza o de sumisión, o contribuir a la formación de parejas.<sup class="reference separada" id="cite_ref-Gill_88-17"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ Las exhibiciones más elaboradas se dan en el <a href="/wiki/Cortejo_sexual" title="Cortejo sexual">cortejo</a>, a menudo compuesto de complejas combinaciones de muchos movimientos distintos;<sup class="reference separada" id="cite_ref-194"><a href="#cite_note-194"><span class="corchete-llamada">[</span>193<span class="corchete-llamada">]</span></a></sup>​ el éxito reproductivo de los machos puede depender de la calidad de dichas exhibiciones.<sup class="reference separada" id="cite_ref-195"><a href="#cite_note-195"><span class="corchete-llamada">[</span>194<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las <a href="/wiki/Vocalizaci%C3%B3n_de_las_aves" title="Vocalización de las aves">vocalizaciones de las aves</a>, es decir, sus cantos y reclamos, ocurren en la <a href="/wiki/Siringe" title="Siringe">siringe</a> y son el principal medio que usan para comunicarse mediante el sonido. Esta comunicación puede ser muy compleja; algunas especies pueden usar los dos lados de la siringe independientemente, y logran así producir simultáneamente dos sonidos.<sup class="reference separada" id="cite_ref-Suthers_106-1"><a href="#cite_note-Suthers-106"><span class="corchete-llamada">[</span>105<span class="corchete-llamada">]</span></a></sup>​ Los reclamos se usan para una amplia variedad de propósitos: para el cortejo (atracción de la pareja,<sup class="reference separada" id="cite_ref-Gill_88-18"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ evaluar las posibles parejas<sup class="reference separada" id="cite_ref-196"><a href="#cite_note-196"><span class="corchete-llamada">[</span>195<span class="corchete-llamada">]</span></a></sup>​); para proteger y marcar el territorio;<sup class="reference separada" id="cite_ref-Gill_88-19"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ para la identificación de otros individuos (como cuando los padres buscan a sus pollos en las colonias, o cuando las parejas se quieren reunir);<sup class="reference separada" id="cite_ref-197"><a href="#cite_note-197"><span class="corchete-llamada">[</span>196<span class="corchete-llamada">]</span></a></sup>​ o para alertar de un depredador potencial. Los reclamos de alerta, en ocasiones, incluyen información específica de la naturaleza de la amenaza;<sup class="reference separada" id="cite_ref-198"><a href="#cite_note-198"><span class="corchete-llamada">[</span>197<span class="corchete-llamada">]</span></a></sup>​ y además algunas aves son capaces de reconocer las llamadas de alerta que realizan otras especies.<sup class="reference separada" id="cite_ref-199"><a href="#cite_note-199"><span class="corchete-llamada">[</span>198<span class="corchete-llamada">]</span></a></sup>​ Aparte de las vocalizaciones, algunas aves usan métodos mecánicos para la comunicación auditiva. Las agachadizas neozelandesas del género <i><a href="/wiki/Coenocorypha" title="Coenocorypha">Coenocorypha</a></i> producen sonidos cuando el aire pasa entre sus plumas,<sup class="reference separada" id="cite_ref-Miskelly_200-0"><a href="#cite_note-Miskelly-200"><span class="corchete-llamada">[</span>199<span class="corchete-llamada">]</span></a></sup>​ los <a href="/wiki/Picidae" title="Picidae">pájaros carpinteros</a> realizan tamborileos para marcar territorio,<sup class="reference separada" id="cite_ref-Attenborough_102-1"><a href="#cite_note-Attenborough-102"><span class="corchete-llamada">[</span>101<span class="corchete-llamada">]</span></a></sup>​ y la <a href="/wiki/Probosciger_aterrimus" title="Probosciger aterrimus">cacatúa enlutada</a> hacen llamados a base de golpes usando herramientas.<sup class="reference separada" id="cite_ref-201"><a href="#cite_note-201"><span class="corchete-llamada">[</span>200<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Mientras que algunas aves son esencialmente solitarias o viven en pequeños grupos familiares, otras pueden formar grandes <a href="/wiki/Bandada" title="Bandada">bandadas</a>. Los beneficios principales de agruparse son mayor seguridad y un incremento de la eficiencia en la búsqueda de alimento.<sup class="reference separada" id="cite_ref-Gill_88-20"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ Defenderse contra los depredadores es especialmente importante en hábitats cerrados como los bosques, donde las emboscadas son comunes, y una gran cantidad de ojos contribuyen a un buen sistema de alerta. Esto ha llevado al desarrollo de bandadas compuestas por un pequeño número de diferentes especies unidas para la alimentación; estas bandadas aumentan la seguridad y reducen la competencia potencial por los recursos.<sup class="reference separada" id="cite_ref-203"><a href="#cite_note-203"><span class="corchete-llamada">[</span>202<span class="corchete-llamada">]</span></a></sup>​ No todo son beneficios, entre los costes que presentan las asociaciones en bandadas están las intimidaciones y el acoso por parte de las aves dominantes hacia las subordinadas, y en algunos casos la reducción de la eficiencia en la búsqueda de alimento.<sup class="reference separada" id="cite_ref-204"><a href="#cite_note-204"><span class="corchete-llamada">[</span>203<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>También, en ocasiones, las aves forman asociaciones con especies que no son aves. Algunas aves marinas se asocian con los <a class="mw-redirect" href="/wiki/Delf%C3%ADn" title="Delfín">delfines</a> y <a class="mw-redirect" href="/wiki/At%C3%BAn" title="Atún">atunes</a>, que empujan los bancos de peces hacia la superficie y las aves se zambullen para pescarlos.<sup class="reference separada" id="cite_ref-AU_205-0"><a href="#cite_note-AU-205"><span class="corchete-llamada">[</span>204<span class="corchete-llamada">]</span></a></sup>​ Los <a class="new" href="/w/index.php?title=Bucerotidae&amp;action=edit&amp;redlink=1" title="Bucerotidae (aún no redactado)">cálaos</a> tienen una <a href="/wiki/Mutualismo_(biolog%C3%ADa)" title="Mutualismo (biología)">relación mutualista</a> con la <a href="/wiki/Helogale_parvula" title="Helogale parvula">mangosta enana</a>, se alimentan juntos y se alertan de la cercanía de <a href="/wiki/Ave_de_presa" title="Ave de presa">aves de presa</a> y otros depredadores.<sup class="reference separada" id="cite_ref-206"><a href="#cite_note-206"><span class="corchete-llamada">[</span>205<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las altas tasas metabólicas de las aves durante sus momentos de actividad diurna están contrarrestadas por sus momentos de descanso. Mientras duermen, las aves a menudo realizan un tipo de sueño llamado vigilante, donde se intercalan períodos de descanso con leves y rápidos vistazos, que les permiten estar atentos a cualquier ruido y escapar de las amenazas;<sup class="reference separada" id="cite_ref-207"><a href="#cite_note-207"><span class="corchete-llamada">[</span>206<span class="corchete-llamada">]</span></a></sup>​ los <a class="mw-redirect" href="/wiki/Vencejo" title="Vencejo">vencejos</a> son capaces de dormir mientras vuelan, y usan el viento estratégicamente (generalmente encarándose a él) para no alejarse en exceso de sus territorios.<sup class="reference separada" id="cite_ref-208"><a href="#cite_note-208"><span class="corchete-llamada">[</span>207<span class="corchete-llamada">]</span></a></sup>​ Se ha sugerido que algunos tipos de sueño podrían ser posibles en vuelo.<sup class="reference separada" id="cite_ref-209"><a href="#cite_note-209"><span class="corchete-llamada">[</span>208<span class="corchete-llamada">]</span></a></sup>​ Algunas especies tienen la capacidad de caer en un sueño de onda lenta con un solo <a href="/wiki/Hemisferio_cerebral" title="Hemisferio cerebral">hemisferio del cerebro</a> cada vez. Las aves tienden a usar esta habilidad según su posición relativa dentro de la bandada; las que se encuentran en los laterales del grupo mantienen así abierto y alerta el ojo que vigila las cercanías de la bandada; esta adaptación también se presenta en <a class="mw-redirect" href="/wiki/Mam%C3%ADfero_marino" title="Mamífero marino">mamíferos marinos</a>.<sup class="reference separada" id="cite_ref-210"><a href="#cite_note-210"><span class="corchete-llamada">[</span>209<span class="corchete-llamada">]</span></a></sup>​ Es bastante normal que las aves se junten a la hora de dormir, lo que reduce la pérdida de <a class="mw-redirect" href="/wiki/Calor_corporal" title="Calor corporal">calor corporal</a> y disminuye los riesgos asociados a los depredadores.<sup class="reference separada" id="cite_ref-211"><a href="#cite_note-211"><span class="corchete-llamada">[</span>210<span class="corchete-llamada">]</span></a></sup>​ Los dormideros son a menudo elegidos en atención a esos dos factores: seguridad y <a href="/wiki/Termorregulaci%C3%B3n" title="Termorregulación">termorregulación</a>.<sup class="reference separada" id="cite_ref-212"><a href="#cite_note-212"><span class="corchete-llamada">[</span>211<span class="corchete-llamada">]</span></a></sup>​ Muchas aves, cuando duermen, doblan sus cabezas hacia el dorso y meten el pico debajo de las plumas, mientras que lo meten bajo las plumas del pecho. Es muy común que las aves descansen sobre una pata, y en algunos casos, sobre todo en climas fríos, meten la pata entre sus plumas. Los <a class="mw-redirect" href="/wiki/Paseriforme" title="Paseriforme">paseriformes</a> tienen un mecanismo de bloqueo en el tendón que les permite mantenerse en la percha sin caer mientras duermen. Muchas aves terrestres, como los <a class="mw-redirect" href="/wiki/Fais%C3%A1n" title="Faisán">faisanes</a> y las <a href="/wiki/Phasianidae" title="Phasianidae">perdices</a>, descansan subidos a los árboles. Varias especies del género <i><a href="/wiki/Loriculus" title="Loriculus">Loriculus</a></i> duermen boca abajo,<sup class="reference separada" id="cite_ref-213"><a href="#cite_note-213"><span class="corchete-llamada">[</span>212<span class="corchete-llamada">]</span></a></sup>​ como hacen también los <a class="mw-redirect" href="/wiki/Murci%C3%A9lago" title="Murciélago">murciélagos</a>. Algunas especies por la noche entran en un estado de letargo que se ve acompañado de una reducción de sus tasas metabólicas; hay unas 100 especies que tienen esta <a href="/wiki/Adaptaci%C3%B3n_biol%C3%B3gica" title="Adaptación biológica">adaptación fisiológica</a>: colibríes,<sup class="reference separada" id="cite_ref-214"><a href="#cite_note-214"><span class="corchete-llamada">[</span>213<span class="corchete-llamada">]</span></a></sup>​ <a class="mw-redirect" href="/wiki/Egotelo" title="Egotelo">egotelos</a> y <a class="mw-redirect" href="/wiki/Chotacabras" title="Chotacabras">chotacabras</a>. Una especie, el <a href="/wiki/Phalaenoptilus_nuttallii" title="Phalaenoptilus nuttallii">chotacabras pachacua</a>, incluso entra en un estado de <a href="/wiki/Hibernaci%C3%B3n" title="Hibernación">hibernación</a>.<sup class="reference separada" id="cite_ref-215"><a href="#cite_note-215"><span class="corchete-llamada">[</span>214<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las aves han desarrollado un comportamiento reproductor más complejo que la mayoría de los vertebrados. Durante la época de reproducción realizan una serie de rituales, algunos de ellos muy elaborados, como el cortejo del macho para aparearse con la hembra, o la construcción de nidos para llevar a cabo la puesta de huevos.
</p>
<p>Las aves se reproducen mediante <a href="/wiki/Fecundaci%C3%B3n" title="Fecundación">fecundación</a> interna y ponen huevos provistos de una cubierta calcárea dura (el <a href="/wiki/Huevo_(biolog%C3%ADa)" title="Huevo (biología)">cascarón</a>).<sup class="reference separada" id="cite_ref-216"><a href="#cite_note-216"><span class="corchete-llamada">[</span>215<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El noventa y cinco por ciento de las especies de aves son <a href="/wiki/Monogamia" title="Monogamia">monógamas sociales</a>. Las parejas se mantienen al menos durante toda la temporada de cría, pero pueden durar varios años o incluso hasta la muerte de uno de los miembros de la pareja.<sup class="reference separada" id="cite_ref-218"><a href="#cite_note-218"><span class="corchete-llamada">[</span>217<span class="corchete-llamada">]</span></a></sup>​ La monogamia permite el <a href="/wiki/Inversi%C3%B3n_parental" title="Inversión parental">cuidado biparental</a> que es especialmente importante en las especies en las que se necesita a dos adultos para sacar adelante la nidada.<sup class="reference separada" id="cite_ref-219"><a href="#cite_note-219"><span class="corchete-llamada">[</span>218<span class="corchete-llamada">]</span></a></sup>​ En muchas especies monógamas, las cópulas fuera de la pareja («infidelidades») son comunes.<sup class="reference separada" id="cite_ref-220"><a href="#cite_note-220"><span class="corchete-llamada">[</span>219<span class="corchete-llamada">]</span></a></sup>​ Este comportamiento es muy típico entre machos dominantes y hembras emparejadas con machos subordinados, pero también puede ser el resultado de cópulas forzadas, como en el caso de los patos y otras <a href="/wiki/Anatidae" title="Anatidae">anátidas</a>.<sup class="reference separada" id="cite_ref-221"><a href="#cite_note-221"><span class="corchete-llamada">[</span>220<span class="corchete-llamada">]</span></a></sup>​ Los beneficios de estas cópulas fuera de la pareja incluyen, para las hembras, conseguir mejores <a href="/wiki/Gen" title="Gen">genes</a> para sus descendientes y asegurarse frente a la posibilidad de que su pareja sea infértil, para los machos, aumentar el número de descendientes sin coste de cuidado parental.<sup class="reference separada" id="cite_ref-222"><a href="#cite_note-222"><span class="corchete-llamada">[</span>221<span class="corchete-llamada">]</span></a></sup>​ En las especies en las que las cópulas fuera de la pareja son comunes, los machos vigilan estrechamente a sus parejas, esta adaptación aumenta la probabilidad de que los pollos que crían tengan sus genes.<sup class="reference separada" id="cite_ref-223"><a href="#cite_note-223"><span class="corchete-llamada">[</span>222<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Otros sistemas de emparejamiento, como la <a href="/wiki/Poliginia" title="Poliginia">poliginia</a>, <a href="/wiki/Poliandria" title="Poliandria">poliandria</a>, <a href="/wiki/Poligamia" title="Poligamia">poligamia</a>, <a href="/wiki/Poliginandria" title="Poliginandria">poliginandria</a> y la <a href="/wiki/Promiscuidad" title="Promiscuidad">promiscuidad</a> también se dan en aves.<sup class="reference separada" id="cite_ref-Gill_88-21"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ Los sistemas poligámicos se dan en especies donde las hembras son capaces de criar a sus pollos sin la necesidad de los machos.<sup class="reference separada" id="cite_ref-Gill_88-22"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ Algunas especies usan más de uno de estos sistemas según las circunstancias.
</p>
<p>En la reproducción normalmente se realiza alguna forma de exhibición de cortejo, por lo general realizada por el macho.<sup class="reference separada" id="cite_ref-224"><a href="#cite_note-224"><span class="corchete-llamada">[</span>223<span class="corchete-llamada">]</span></a></sup>​ La mayor parte de estas exhibiciones son bastante simples e incluyen algún tipo de <a class="mw-redirect" href="/wiki/Canto_de_las_aves" title="Canto de las aves">canto</a>. Sin embargo otras están muy elaboradas. Dentro de los despliegues nupciales más llamativos se encuentran los realizados por las <a href="/wiki/Paradisaeidae" title="Paradisaeidae">aves del paraíso</a> de <a href="/wiki/Nueva_Guinea" title="Nueva Guinea">Nueva Guinea</a><sup class="reference separada" id="cite_ref-225"><a href="#cite_note-225"><span class="corchete-llamada">[</span>224<span class="corchete-llamada">]</span></a></sup>​ y los <a href="/wiki/Pipridae" title="Pipridae">bailarines</a> del <a href="/wiki/Neotr%C3%B3pico" title="Neotrópico">Neotrópico</a>.<sup class="reference separada" id="cite_ref-226"><a href="#cite_note-226"><span class="corchete-llamada">[</span>225<span class="corchete-llamada">]</span></a></sup>​ Según la especie pueden incluir golpeteos y tamborileos con las alas o la cola, bailes y vuelos acrobáticos en arenas de combate (<i><a href="/wiki/Lek_(comportamiento_animal)" title="Lek (comportamiento animal)">leks</a></i>). Las hembras suelen ser las que eligen a su pareja,<sup class="reference separada" id="cite_ref-227"><a href="#cite_note-227"><span class="corchete-llamada">[</span>226<span class="corchete-llamada">]</span></a></sup>​ aunque en algunas especies poliándricas, como el <a href="/wiki/Phalaropus_lobatus" title="Phalaropus lobatus">falaropo picofino</a>, es al revés: los machos, que son de colores poco llamativos, son los que eligen a las hembras de plumaje colorido y brillante.<sup class="reference separada" id="cite_ref-228"><a href="#cite_note-228"><span class="corchete-llamada">[</span>227<span class="corchete-llamada">]</span></a></sup>​ Los acicalamientos mutuos, las cebas de cortejo, y los roces y «besos» con los picos son comportamientos comunes, generalmente después de que se hayan emparejado.<sup class="reference separada" id="cite_ref-Attenborough_102-2"><a href="#cite_note-Attenborough-102"><span class="corchete-llamada">[</span>101<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Muchas aves defienden activamente un territorio de las intromisiones de sus congéneres durante la época de cría; al mantener su territorio aseguran las fuentes de alimento para sus pollos. Las especies que no defienden un territorio, como las <a href="/wiki/Ave_marina" title="Ave marina">aves marinas</a> o los <a class="mw-redirect" href="/wiki/Vencejo" title="Vencejo">vencejos</a>, a menudo crían en colonias; al criar en colonias se consigue cierta protección contra los depredadores. En las colonias las aves defienden sus lugares de anidamiento y la competencia por estos lugares puede ser intensa.<sup class="reference separada" id="cite_ref-229"><a href="#cite_note-229"><span class="corchete-llamada">[</span>228<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Todas las aves ponen <a class="mw-redirect" href="/wiki/Huevo_amni%C3%B3tico" title="Huevo amniótico">huevos amnióticos</a> con cáscaras duras compuestas en su mayor parte por <a class="mw-redirect" href="/wiki/Carbonato_c%C3%A1lcico" title="Carbonato cálcico">carbonato cálcico</a>.<sup class="reference separada" id="cite_ref-Gill_88-23"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ Los huevos de las especies que anidan en agujeros o madrigueras suelen ser blancos o de colores claros, mientras que los huevos de las que anidan en el suelo o entre la vegetación por lo general se <a href="/wiki/Cripsis" title="Cripsis">camuflan</a> con el entorno. Hay, sin embargo, muchas excepciones a esta regla; por ejemplo los chotacabras anidan en el suelo, pero sus huevos son blancos y el camuflaje se consigue por su plumaje. En las aves que son víctimas de los <a href="/wiki/Parasitismo_de_puesta" title="Parasitismo de puesta">parásitos de puesta</a> sus huevos tienen patrones de color variantes, adaptación que aumenta las probabilidades de descubrir el huevo del parásito en el nido, esto a su vez dirigió una adaptación en las hembras parásitas que ajusta los colores de sus huevos a los de sus <a href="/wiki/Hu%C3%A9sped_(biolog%C3%ADa)" title="Huésped (biología)">hospedadores</a>.<sup class="reference separada" id="cite_ref-230"><a href="#cite_note-230"><span class="corchete-llamada">[</span>229<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Los huevos suelen ser incubados en un <a class="mw-redirect" href="/wiki/Nido_de_las_aves" title="Nido de las aves">nido</a>. La mayor parte de las especies construyen un nido más o menos trabajado, que puede ser una copa, una bóveda, una plataforma, un montículo, una madriguera o una simple escarbadura en el suelo.<sup class="reference separada" id="cite_ref-Hansell_231-0"><a href="#cite_note-Hansell-231"><span class="corchete-llamada">[</span>230<span class="corchete-llamada">]</span></a></sup>​ Algunos nidos son, en cambio muy sencillos; los albatros ponen los huevos casi directamente sobre el suelo. La mayor parte de las aves ubican sus nidos en lugares protegidos y ocultos para evitar a los depredadores, pero en las especies coloniales, que tienen mayor capacidad de defensa, los nidos se sitúan en zonas más expuestas. Durante la construcción, algunas especies recogen plantas provistas de <a href="/wiki/Toxina" title="Toxina">toxinas</a> dañinas para los parásitos, lo que favorece la supervivencia de sus pollos,<sup class="reference separada" id="cite_ref-232"><a href="#cite_note-232"><span class="corchete-llamada">[</span>231<span class="corchete-llamada">]</span></a></sup>​ y a menudo se usan las plumas como <a href="/wiki/Aislamiento_t%C3%A9rmico" title="Aislamiento térmico">aislamiento térmico</a>.<sup class="reference separada" id="cite_ref-Hansell_231-1"><a href="#cite_note-Hansell-231"><span class="corchete-llamada">[</span>230<span class="corchete-llamada">]</span></a></sup>​ Varias especies de aves no tienen nidos; el <a href="/wiki/Uria_aalge" title="Uria aalge">arao común</a> cría en acantilados donde deposita los huevos directamente sobre la roca, y el <a href="/wiki/Aptenodytes_forsteri" title="Aptenodytes forsteri">pingüino emperador</a> guarda e incuba sus huevos entre sus pies y su cuerpo. La ausencia de nidos es más común en especies que anidan en el suelo, ya que sus pollos suelen ser <a href="/wiki/Precocidad" title="Precocidad">precoces</a>.
</p>
<p>La incubación generalmente comienza cuando se ha puesto el último huevo y tiene el fin de optimizar su temperatura para el correcto desarrollo del embrión.<sup class="reference separada" id="cite_ref-Gill_88-24"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ En las especies monógamas la incubación suele ser una tarea compartida, mientras que en las especies polígamas el encargado es solo uno de los progenitores. El calor de los padres pasa a los huevos a través de unas zonas concretas del vientre o del abdomen del ave que han perdido las plumas y tienen la piel descubierta. La incubación puede ser un proceso que demanda mucha energía; los albatros adultos, por ejemplo, pierden hasta 83 g de peso por día de incubación.<sup class="reference separada" id="cite_ref-233"><a href="#cite_note-233"><span class="corchete-llamada">[</span>232<span class="corchete-llamada">]</span></a></sup>​ El calor en la incubación de los huevos de los <a href="/wiki/Megapodiidae" title="Megapodiidae">talégalos</a> proviene del sol, de la descomposición de la vegetación o del calor del suelo en zonas volcánicas.<sup class="reference separada" id="cite_ref-234"><a href="#cite_note-234"><span class="corchete-llamada">[</span>233<span class="corchete-llamada">]</span></a></sup>​ Los períodos de incubación varían mucho, desde 10 días en los <a href="/wiki/Cuculidae" title="Cuculidae">cucos</a>, <a href="/wiki/Picidae" title="Picidae">pájaros carpinteros</a> y <a class="mw-redirect" href="/wiki/Paseriforme" title="Paseriforme">paseriformes</a>, hasta más de 80 días en los albatros y en los <a href="/wiki/Apteryx" title="Apteryx">kiwis</a>.<sup class="reference separada" id="cite_ref-Gill_88-25"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Al salir del cascarón los pollos pueden ser desde indefensos a independientes, incluidos varios estados intermedios, según la especie. Las que nacen indefensas se llaman <a href="/wiki/Altricial" title="Altricial">altriciales</a> o nidícolas y suelen ser pequeñas, ciegas y desnudas; los pollos que nacen más formados se llaman <a href="/wiki/Precocidad" title="Precocidad">precoces</a> o nidífugos, están cubiertos de <a href="/wiki/Plum%C3%B3n" title="Plumón">plumón</a> y son capaces de seguir a sus padres (como es el caso de los <a href="/wiki/Anseriformes" title="Anseriformes">anseriformes</a> y <a href="/wiki/Galliformes" title="Galliformes">galliformes</a>).<sup class="reference separada" id="cite_ref-235"><a href="#cite_note-235"><span class="corchete-llamada">[</span>234<span class="corchete-llamada">]</span></a></sup>​ Los pollos altriciales necesitan la ayuda de sus padres para <a href="/wiki/Termorregulaci%C3%B3n" title="Termorregulación">termorregularse</a> y su periodo de cría dura más tiempo que el de los precoces.
</p>
<p>La duración y naturaleza del cuidado parental varía mucho entre los diferentes órdenes y especies. En un extremo están los <a href="/wiki/Megapodiidae" title="Megapodiidae">talégalos</a> que dejan de cuidar a sus descendientes al romper el cascarón; el pollo recién nacido es capaz de salir del cascarón y del nido sin ayuda de los padres y valerse por sí mismo inmediatamente.<sup class="reference separada" id="cite_ref-236"><a href="#cite_note-236"><span class="corchete-llamada">[</span>235<span class="corchete-llamada">]</span></a></sup>​ En el otro extremo muchas aves marinas cuidan a sus pollos durante largos períodos; el más largo es el de la <a href="/wiki/Fregata_minor" title="Fregata minor">fragata grande</a>, sus pollos tardan seis meses en emplumar y son alimentados por los padres durante otros catorce meses más.<sup class="reference separada" id="cite_ref-237"><a href="#cite_note-237"><span class="corchete-llamada">[</span>236<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>En algunas especies los dos padres se ocupan de la cría, en otras solo uno de los sexos carga con la responsabilidad. En ocasiones otros congéneres, generalmente parientes cercanos de la pareja, como los juveniles de años pasados, ayudan en la cría.<sup class="reference separada" id="cite_ref-238"><a href="#cite_note-238"><span class="corchete-llamada">[</span>237<span class="corchete-llamada">]</span></a></sup>​ Este comportamiento es bastante común entre los <a class="mw-redirect" href="/wiki/C%C3%B3rvido" title="Córvido">córvidos</a>, y especies cercanas a estos como la <a href="/wiki/Gymnorhina_tibicen" title="Gymnorhina tibicen">urraca australiana</a> o el género <i><a href="/wiki/Malurus" title="Malurus">Malurus</a></i>,<sup class="reference separada" id="cite_ref-239"><a href="#cite_note-239"><span class="corchete-llamada">[</span>238<span class="corchete-llamada">]</span></a></sup>​ pero se ha observado también en especies tan diferentes como el <a href="/wiki/Acanthisitta_chloris" title="Acanthisitta chloris">acantisita verdoso</a> o el <a href="/wiki/Milvus_milvus" title="Milvus milvus">milano real</a>. En la mayor parte de los grupos animales el cuidado de las crías por parte de los machos es raro. Sin embargo, en las aves es bastante común, más que en cualquier otra clase de vertebrado.<sup class="reference separada" id="cite_ref-Gill_88-26"><a href="#cite_note-Gill-88"><span class="corchete-llamada">[</span>87<span class="corchete-llamada">]</span></a></sup>​ Aunque las labores relacionadas con la reproducción y la cría son a menudo compartidas, en ocasiones hay una división del trabajo en la que uno de los miembros de la pareja lleva a cabo toda o la mayor parte de una determinada tarea.<sup class="reference separada" id="cite_ref-240"><a href="#cite_note-240"><span class="corchete-llamada">[</span>239<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El momento en que los pollos abandonan el nido varía de manera muy acusada. Los pollos de los <a href="/wiki/Alcidae" title="Alcidae">álcidos</a> del género <i><a href="/wiki/Synthliboramphus" title="Synthliboramphus">Synthliboramphus</a></i>, como el <a href="/wiki/Synthliboramphus_antiquus" title="Synthliboramphus antiquus">mérgulo antiguo</a>, dejan el nido la noche después de salir del cascarón, siguen a sus padres al mar donde se desarrollan a salvo de los depredadores terrestres.<sup class="reference separada" id="cite_ref-241"><a href="#cite_note-241"><span class="corchete-llamada">[</span>240<span class="corchete-llamada">]</span></a></sup>​ Otras especies, como los patos, también abandonan el nido a una edad temprana. En la mayor parte de los casos, los pollos dejan el nido cuando son capaces de volar. Tras dejar el nido hay especies, por ejemplo los albatros, que no continúan cuidando de su nidada, mientras que otras siguen alimentándoles.<sup class="reference separada" id="cite_ref-242"><a href="#cite_note-242"><span class="corchete-llamada">[</span>241<span class="corchete-llamada">]</span></a></sup>​ Los juveniles pueden también seguir a sus padres durante su primera <a href="/wiki/Migraci%C3%B3n_de_las_aves" title="Migración de las aves">migración</a>.<sup class="reference separada" id="cite_ref-243"><a href="#cite_note-243"><span class="corchete-llamada">[</span>242<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>El <a href="/wiki/Parasitismo_de_puesta" title="Parasitismo de puesta">parasitismo de puesta</a>, en el que una especie deja sus huevos entre los de otra especie para que esta los críe, es más común entre las aves que en cualquier otro tipo de organismo.<sup class="reference separada" id="cite_ref-brood_244-0"><a href="#cite_note-brood-244"><span class="corchete-llamada">[</span>243<span class="corchete-llamada">]</span></a></sup>​ Después de que el ave parásita deposite sus huevos en el nido de otra, por lo general son aceptados y criados por los padres adoptivos a expensas de su propia nidada. Entre las especies que usan este modo de parasitismo hay unas que son incapaces de sacar adelante una nidada propia y por ello están obligadas a parasitar, y hay otras que en ocasiones ponen sus huevos en nidos de individuos de su misma especie para incrementar su rendimiento reproductivo, incluso aunque hayan criado a sus propios pollos.<sup class="reference separada" id="cite_ref-245"><a href="#cite_note-245"><span class="corchete-llamada">[</span>244<span class="corchete-llamada">]</span></a></sup>​ Unas cien especies, entre las que se incluyen <a href="/wiki/Indicatoridae" title="Indicatoridae">indicadores</a>, <a href="/wiki/Icteridae" title="Icteridae">ictéridos</a>, <a href="/wiki/Estrildidae" title="Estrildidae">estríldidos</a> y <a href="/wiki/Heteronetta_atricapilla" title="Heteronetta atricapilla">patos</a> son parásitos obligados, aunque los más famosos son los <a href="/wiki/Cuculidae" title="Cuculidae">cucos</a>.<sup class="reference separada" id="cite_ref-brood_244-1"><a href="#cite_note-brood-244"><span class="corchete-llamada">[</span>243<span class="corchete-llamada">]</span></a></sup>​ En algunos parásitos de puesta sus huevos eclosionan antes que los del hospedador, lo que le permite al parásito destruir los huevos empujándolos fuera del nido o matar a los pollos que han tenido menos tiempo para desarrollarse; esto les asegura que toda la comida que traigan los padres-hospedadores sea para ellos.<sup class="reference separada" id="cite_ref-246"><a href="#cite_note-246"><span class="corchete-llamada">[</span>245<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las aves ocupan un amplio espectro de nichos ecológicos.<sup class="reference separada" id="cite_ref-flycatcher_202-1"><a href="#cite_note-flycatcher-202"><span class="corchete-llamada">[</span>201<span class="corchete-llamada">]</span></a></sup>​ Mientras algunas aves son generalistas, otras están altamente especializadas en su <a href="/wiki/H%C3%A1bitat" title="Hábitat">hábitat</a> o en su alimentación. Incluso en un solo hábitat, como por ejemplo un bosque, los nichos ecológicos ocupados por diferentes aves varían; algunas especies se alimentan en la copa de los árboles, otras por debajo del dosel arbóreo, y algunas en el suelo del bosque. Las aves forestales pueden ser <a class="mw-redirect" href="/wiki/Insect%C3%ADvora" title="Insectívora">insectívoras</a>, <a class="mw-redirect" href="/wiki/Frug%C3%ADvora" title="Frugívora">frugívoras</a> y <a class="mw-redirect" href="/wiki/Nectar%C3%ADvora" title="Nectarívora">nectarívoras</a>. Las <a href="/wiki/Ave_acu%C3%A1tica" title="Ave acuática">aves acuáticas</a> por lo general se alimentan pescando, comiendo plantas acuáticas, o como <a href="/wiki/Cleptoparasitismo" title="Cleptoparasitismo">cleptoparásitas</a>. Las <a href="/wiki/Ave_de_presa" title="Ave de presa">aves de presa</a> están especializadas en cazar mamíferos, otras aves y otros animales, mientras que los buitres son aves <a class="mw-redirect" href="/wiki/Carro%C3%B1era" title="Carroñera">carroñeras</a> especializadas.
</p>
<p>Algunas aves nectarívoras son importantes <a href="/wiki/Polinizador" title="Polinizador">polinizadoras</a>, y muchas especies frugívoras juegan un papel clave en la <a href="/wiki/Zoocoria" title="Zoocoria">dispersión de las semillas</a>.<sup class="reference separada" id="cite_ref-Clout_247-0"><a href="#cite_note-Clout-247"><span class="corchete-llamada">[</span>246<span class="corchete-llamada">]</span></a></sup>​ Las plantas y las aves que las polinizan, a menudo <a href="/wiki/Coevoluci%C3%B3n" title="Coevolución">coevolucionan</a>,<sup class="reference separada" id="cite_ref-248"><a href="#cite_note-248"><span class="corchete-llamada">[</span>247<span class="corchete-llamada">]</span></a></sup>​ y en algunos casos el principal polinizador de la planta es el único capaz de llegar a su <a class="mw-redirect" href="/wiki/N%C3%A9ctar_(Bot%C3%A1nica)" title="Néctar (Botánica)">néctar</a>.<sup class="reference separada" id="cite_ref-249"><a href="#cite_note-249"><span class="corchete-llamada">[</span>248<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las aves son importantes en la ecología de las islas. Alcanzan islas a donde los mamíferos no han podido llegar; en estas islas las aves desempeñan roles ecológicos que en zonas continentales ocupan animales de mayor tamaño. Por ejemplo, en <a href="/wiki/Nueva_Zelanda" title="Nueva Zelanda">Nueva Zelanda</a> las <a href="/wiki/Dinornithidae" title="Dinornithidae">moas</a> eran importantes <a class="new" href="/w/index.php?title=Ramoneador&amp;action=edit&amp;redlink=1" title="Ramoneador (aún no redactado)">ramoneadoras</a> y frugívoras, como lo son el <a href="/wiki/Hemiphaga_novaeseelandiae" title="Hemiphaga novaeseelandiae">kereru</a> y el <a href="/wiki/Callaeas_cinerea" title="Callaeas cinerea">kokako</a> en la actualidad.<sup class="reference separada" id="cite_ref-Clout_247-1"><a href="#cite_note-Clout-247"><span class="corchete-llamada">[</span>246<span class="corchete-llamada">]</span></a></sup>​ Hoy en día las plantas en Nueva Zelanda mantienen las adaptaciones defensivas que desarrollaron para protegerse de las extintas moas.<sup class="reference separada" id="cite_ref-250"><a href="#cite_note-250"><span class="corchete-llamada">[</span>249<span class="corchete-llamada">]</span></a></sup>​ Las aves marinas también afectan la ecología de las islas en que nidifican, sobre todo a través de la concentración y acumulación de grandes cantidades de <a href="/wiki/Guano" title="Guano">guano</a>, que enriquecen con nutrientes los suelos,<sup class="reference separada" id="cite_ref-251"><a href="#cite_note-251"><span class="corchete-llamada">[</span>250<span class="corchete-llamada">]</span></a></sup>​ y los mares circundantes.<sup class="reference separada" id="cite_ref-252"><a href="#cite_note-252"><span class="corchete-llamada">[</span>251<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Los <a href="/wiki/Parasitismo" title="Parasitismo">parásitos</a> más corrientes de las aves pertenecen a los grupos de los <a href="/wiki/Acari" title="Acari">ácaros</a>, los <a href="/wiki/Ischnocera" title="Ischnocera">piojos aviares</a> y los <a class="mw-redirect" href="/wiki/Verme" title="Verme">vermes</a>. Otros parásitos microscópicos, como <a class="mw-redirect" href="/wiki/Hongo" title="Hongo">hongos</a>, <a href="/wiki/Protozoo" title="Protozoo">protozoos</a>, <a href="/wiki/Bacteria" title="Bacteria">bacterias</a> y <a href="/wiki/Virus" title="Virus">virus</a>, también les provocan enfermedades.
</p>
<p>Al menos 2500 especies de ácaros repartidos en 40 familias viven en relación estrecha con las aves, ocupan sus nidos, sus plumas, o incluso sus picos, como ciertos ácaros de los <a href="/wiki/Trochilidae" title="Trochilidae">colibríes</a>. Estos ácaros pueden tener una relación simplemente <a href="/wiki/Comensalismo" title="Comensalismo">forética</a> o pueden perturbar a sus hospedadores y provocarles desnutrición, pero pueden también ser verdaderos parásitos como <i><a href="/wiki/Dermanyssus" title="Dermanyssus">Dermanyssus</a></i> y <i><a href="/wiki/Ornithonyssus" title="Ornithonyssus">Ornithonyssus</a></i>. Todas las especies de aves se ven afectadas, incluso los <a class="mw-redirect" href="/wiki/Ping%C3%BCino" title="Pingüino">pingüinos</a> portan <a class="mw-redirect" href="/wiki/Garrapata" title="Garrapata">garrapatas</a>.<sup class="reference separada" id="cite_ref-Ritchison_253-0"><a href="#cite_note-Ritchison-253"><span class="corchete-llamada">[</span>252<span class="corchete-llamada">]</span></a></sup>​ El modo de vida de una garrapata de ave depende, por supuesto, de su especie; sin embargo, la larva vive por lo común en el nido. Los ácaros tienen ciclos de reproducción cortos y son capaces de multiplicarse muy rápidamente. Algunos ácaros se nutren de piel muerta, otros como los de los colibríes, se hacen transportar de flor en flor y se nutren de néctar. En los nidos, incluso se han descubierto garrapatas enanas parásitas de garrapatas aviarias.<sup class="reference separada" id="cite_ref-Ritchison_253-1"><a href="#cite_note-Ritchison-253"><span class="corchete-llamada">[</span>252<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Un número muy importante de garrapatas puede perjudicar a la nidada e incluso a la vida de los pollos. Por tanto, ciertos estudios podrían sugerir que este <a href="/wiki/Comensalismo" title="Comensalismo">comensalismo</a> no es únicamente desfavorable a las aves.<sup class="reference separada" id="cite_ref-Ritchison_253-2"><a href="#cite_note-Ritchison-253"><span class="corchete-llamada">[</span>252<span class="corchete-llamada">]</span></a></sup>​
Los piojos de las aves (<a href="/wiki/Ischnocera" title="Ischnocera">Ischnocera</a>) infestan, la mayoría de ocasiones, a una especie concreta.
Varias especies de <a href="/wiki/Platyhelminthes" title="Platyhelminthes">platelmintos</a>, entre los cuales están los <a href="/wiki/Cestoda" title="Cestoda">cestodos</a>, o los <a href="/wiki/Trematoda" title="Trematoda">trematodos</a>, pueden infectar las aves que los pueden transportar de un continente a otro. Por ejemplo, las <a href="/wiki/Ave_marina" title="Ave marina">aves marinas</a>, al comer los <a href="/wiki/Cerastoderma" title="Cerastoderma">berberechos</a>, favorecen un parasitismo de trematodos (géneros <i><a class="new" href="/w/index.php?title=Meiogymnophalus&amp;action=edit&amp;redlink=1" title="Meiogymnophalus (aún no redactado)">Meiogymnophalus</a></i>, <i><a href="/wiki/Himasthla" title="Himasthla">Himasthla</a></i>, etc.) que pueden luego afectar a varias especies de hospedadores, ya sean aves u otros moluscos.<sup class="reference separada" id="cite_ref-254"><a href="#cite_note-254"><span class="corchete-llamada">[</span>253<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Además de parásitos, las aves pueden sufrir otras enfermedades infecciosas como:
</p>
<p>Dado que las aves son animales muy visibles y muy comunes, los humanos han tenido una intensa relación con ellas desde el comienzo de la humanidad.<sup class="reference separada" id="cite_ref-255"><a href="#cite_note-255"><span class="corchete-llamada">[</span>254<span class="corchete-llamada">]</span></a></sup>​ Algunas veces estas relaciones son <a href="/wiki/Mutualismo_(biolog%C3%ADa)" title="Mutualismo (biología)">mutualísticas</a>, como la existente entre los <a href="/wiki/Indicatoridae" title="Indicatoridae">indicadores</a> y pueblos africanos como los <a href="/wiki/Borana" title="Borana">borana</a> que se ayudan de estas aves a la hora de recolectar miel.<sup class="reference separada" id="cite_ref-256"><a href="#cite_note-256"><span class="corchete-llamada">[</span>255<span class="corchete-llamada">]</span></a></sup>​ En otras ocasiones la relación puede ser de <a href="/wiki/Comensalismo" title="Comensalismo">comensalismo</a>, situación que se da cuando una especie se beneficia de las actividades humanas, como por ejemplo el <a href="/wiki/Passer_domesticus" title="Passer domesticus">gorrión común</a>.<sup class="reference separada" id="cite_ref-257"><a href="#cite_note-257"><span class="corchete-llamada">[</span>256<span class="corchete-llamada">]</span></a></sup>​ Muchas especies se han convertido en plagas económicamente significativas para la agricultura,<sup class="reference separada" id="cite_ref-258"><a href="#cite_note-258"><span class="corchete-llamada">[</span>257<span class="corchete-llamada">]</span></a></sup>​ y algunas generan riesgos para la aviación.<sup class="reference separada" id="cite_ref-259"><a href="#cite_note-259"><span class="corchete-llamada">[</span>258<span class="corchete-llamada">]</span></a></sup>​ Las actividades humanas también han perjudicado a las aves, y han extinguido y puesto en peligro de extinción a numerosas especies.
</p>
<p>Las aves juegan papeles prominentes y diversos en el <a href="/wiki/Folclore" title="Folclore">folclore</a>, la <a href="/wiki/Religi%C3%B3n" title="Religión">religión</a> y la cultura popular. En la religión, las aves pueden servir tanto como mensajeras, representantes o portadoras de una <a href="/wiki/Deidad" title="Deidad">deidad</a>, como en el culto de <a href="/wiki/Make-Make" title="Make-Make">Make-Make</a>, en que los <a class="mw-redirect" href="/wiki/Tangata_Manu" title="Tangata Manu">Tangata Manu</a> de la <a href="/wiki/Isla_de_Pascua" title="Isla de Pascua">Isla de Pascua</a> servían de jefes,<sup class="reference separada" id="cite_ref-260"><a href="#cite_note-260"><span class="corchete-llamada">[</span>259<span class="corchete-llamada">]</span></a></sup>​ o como asistentes, como en el caso de <a href="/wiki/Hugin_y_Munin" title="Hugin y Munin">Hugin y Munin</a>, dos <a href="/wiki/Corvus_corax" title="Corvus corax">cuervos comunes</a> que susurraban las noticias al oído del <a href="/wiki/Mitolog%C3%ADa_n%C3%B3rdica" title="Mitología nórdica">dios noruego</a> <a href="/wiki/Od%C3%ADn" title="Odín">Odín</a>.<sup class="reference separada" id="cite_ref-261"><a href="#cite_note-261"><span class="corchete-llamada">[</span>260<span class="corchete-llamada">]</span></a></sup>​ También pueden servir de símbolos religiosos, como cuando <a href="/wiki/Jon%C3%A1s_(profeta)" title="Jonás (profeta)">Jonás</a> (Hebreo: יוֹנָה, paloma) corporizaba el miedo, la pasividad, el lamento, y la belleza, asociados tradicionalmente a las palomas.<sup class="reference separada" id="cite_ref-262"><a href="#cite_note-262"><span class="corchete-llamada">[</span>261<span class="corchete-llamada">]</span></a></sup>​ Las aves han sido por sí mismas deificadas, como en el caso del <a href="/wiki/Pavo_cristatus" title="Pavo cristatus">pavo real común</a>, que es percibido como la madre tierra por los <a class="mw-redirect" href="/wiki/Dr%C3%A1vida" title="Drávida">drávidas</a> de la <a href="/wiki/India" title="India">India</a>.<sup class="reference separada" id="cite_ref-263"><a href="#cite_note-263"><span class="corchete-llamada">[</span>262<span class="corchete-llamada">]</span></a></sup>​ Algunas aves han sido percibidas también como monstruos, como el mitológico <a class="mw-redirect" href="/wiki/Roc" title="Roc">Roc</a> y el <a href="/wiki/Pouakai" title="Pouakai">Pouakai</a> legendario de los <a href="/wiki/Maor%C3%AD" title="Maorí">maoríes</a>, un ave gigante capaz de levantar en vuelo a humanos agarrados.<sup class="reference separada" id="cite_ref-264"><a href="#cite_note-264"><span class="corchete-llamada">[</span>263<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las aves han sido representadas en la cultura y el arte desde tiempos prehistóricos, cuando eran <a href="/wiki/Pintura_rupestre" title="Pintura rupestre">pintadas en cuevas</a>.<sup class="reference separada" id="cite_ref-265"><a href="#cite_note-265"><span class="corchete-llamada">[</span>264<span class="corchete-llamada">]</span></a></sup>​ Fueron usadas luego como arte y diseños religiosos o simbólicos, como el magnífico <a href="/wiki/Trono_del_Pavo_real" title="Trono del Pavo real">Trono del Pavo real</a> de los <a class="mw-redirect" href="/wiki/Mogol" title="Mogol">Mogoles</a> y los emperadores de <a class="mw-redirect" href="/wiki/Persia" title="Persia">Persia</a>.<sup class="reference separada" id="cite_ref-266"><a href="#cite_note-266"><span class="corchete-llamada">[</span>265<span class="corchete-llamada">]</span></a></sup>​ Con el surgimiento del interés científico por las aves, muchas pinturas de aves fueron encargadas para libros. Uno de los pintores de aves más famoso fue <a href="/wiki/John_James_Audubon" title="John James Audubon">John James Audubon</a>, cuyas obras de aves norteamericanas tuvieron un gran éxito comercial en Europa, y quien luego prestó su nombre a la <a href="/wiki/National_Audubon_Society" title="National Audubon Society">National Audubon Society</a>.<sup class="reference separada" id="cite_ref-267"><a href="#cite_note-267"><span class="corchete-llamada">[</span>266<span class="corchete-llamada">]</span></a></sup>​ Las aves son también personajes importantes en la <a href="/wiki/Poes%C3%ADa" title="Poesía">poesía</a>; por ejemplo, <a href="/wiki/Homero" title="Homero">Homero</a> incorporó a los <a href="/wiki/Luscinia_megarhynchos" title="Luscinia megarhynchos">ruiseñores</a> en su <i><a href="/wiki/Odisea" title="Odisea">Odisea</a></i>, y <a href="/wiki/Catulo" title="Catulo">Catulo</a> usó al <a href="/wiki/Passer_domesticus" title="Passer domesticus">gorrión</a> como un símbolo erótico en su <i><a href="/wiki/Catullus_2" title="Catullus 2">Catullus 2</a></i>.<sup class="reference separada" id="cite_ref-268"><a href="#cite_note-268"><span class="corchete-llamada">[</span>267<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>En español, existen algunos nombres de aves con sentido metafórico para describir o representar comportamientos y características humanas. Pero las percepciones de una misma ave a menudo varían entre distintas culturas. Los <a href="/wiki/B%C3%BAho" title="Búho">búhos</a> se asocian con la mala suerte, la <a href="/wiki/Brujer%C3%ADa" title="Brujería">brujería</a> y la <a href="/wiki/Muerte" title="Muerte">muerte</a> en zonas de <a href="/wiki/%C3%81frica" title="África">África</a>,<sup class="reference separada" id="cite_ref-269"><a href="#cite_note-269"><span class="corchete-llamada">[</span>268<span class="corchete-llamada">]</span></a></sup>​ pero son relacionados con la sabiduría en gran parte de <a href="/wiki/Europa" title="Europa">Europa</a>.<sup class="reference separada" id="cite_ref-270"><a href="#cite_note-270"><span class="corchete-llamada">[</span>269<span class="corchete-llamada">]</span></a></sup>​ Las <a href="/wiki/Upupa_epops" title="Upupa epops">abubillas</a> eran consideras sagradas en el <a class="mw-redirect" href="/wiki/Egipto_Antiguo" title="Egipto Antiguo">Egipto Antiguo</a>, y símbolos de virtud en <a class="mw-redirect" href="/wiki/Persia" title="Persia">Persia</a>, pero eran percibidas como ladronas en gran parte de <a href="/wiki/Europa" title="Europa">Europa</a>, y como presagio de guerra en <a href="/wiki/Escandinavia" title="Escandinavia">Escandinavia</a>.<sup class="reference separada" id="cite_ref-271"><a href="#cite_note-271"><span class="corchete-llamada">[</span>270<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las <a class="mw-redirect" href="/wiki/Ave_dom%C3%A9stica" title="Ave doméstica">aves domésticas</a> criadas para carne y huevos, también llamadas <a class="mw-redirect" href="/wiki/Aves_de_corral" title="Aves de corral">aves de corral</a>, son la mayor fuente de <a href="/wiki/Prote%C3%ADna" title="Proteína">proteína</a> animal en la alimentación humana: en 2003, se produjeron 76 millones de toneladas de carne de aves y 61 millones de toneladas de huevos en todo el mundo.<sup class="reference separada" id="cite_ref-272"><a href="#cite_note-272"><span class="corchete-llamada">[</span>271<span class="corchete-llamada">]</span></a></sup>​ Los <a class="mw-redirect" href="/wiki/Gallina" title="Gallina">pollos</a> son la mayor proporción de la carne de aves domésticas que se consume, aunque [[Meleagris
|pavos]], <a href="/wiki/Pato" title="Pato">patos</a> y <a href="/wiki/Ganso" title="Ganso">gansos</a> son también relativamente comunes. Muchas especies de aves son cazadas por su carne. La cacería de aves es principalmente una actividad recreativa, pero en áreas poco desarrolladas la caza sigue aportando parte de la dieta. Las especies <a class="mw-redirect" href="/wiki/Cineg%C3%A9tica" title="Cinegética">cinegéticas</a> de aves más importantes de <a href="/wiki/Am%C3%A9rica_del_Sur" title="América del Sur">América del Sur</a> y <a href="/wiki/Am%C3%A9rica_del_Norte" title="América del Norte">del Norte</a> son las <a href="/wiki/Anatidae" title="Anatidae">anátidas</a>, pero también se cazan mucho los <a href="/wiki/Phasianidae" title="Phasianidae">faisanes</a>, <a href="/wiki/Meleagris" title="Meleagris">pavos silvestres</a>, <a href="/wiki/Coturnix" title="Coturnix">codornices</a>, <a href="/wiki/Columbidae" title="Columbidae">palomas</a>, <a class="new" href="/w/index.php?title=Perdicinae&amp;action=edit&amp;redlink=1" title="Perdicinae (aún no redactado)">perdices</a>, <a class="mw-redirect" href="/wiki/Tetraonidae" title="Tetraonidae">gallos silvestres, grévoles</a>, <a href="/wiki/Gallinago" title="Gallinago">agachadizas</a> y <a href="/wiki/Scolopax" title="Scolopax">chochas</a>.<sup class="reference separada" id="cite_ref-273"><a href="#cite_note-273"><span class="corchete-llamada">[</span>272<span class="corchete-llamada">]</span></a></sup>​ La caza de pollos de <a href="/wiki/Puffinus" title="Puffinus">pardelas</a> (<i>muttonbirding</i>) es popular en Australia y Nueva Zelanda.<sup class="reference separada" id="cite_ref-274"><a href="#cite_note-274"><span class="corchete-llamada">[</span>273<span class="corchete-llamada">]</span></a></sup>​ Aunque alguna cacería, como la de pichones de pardelas, pueda ser sostenible, la caza ha llevado a la extinción o a la amenaza de extinción a decenas de especies.<sup class="reference separada" id="cite_ref-275"><a href="#cite_note-275"><span class="corchete-llamada">[</span>274<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Otros productos de aves comercialmente valiosos incluyen las plumas (especialmente el <a href="/wiki/Plum%C3%B3n" title="Plumón">plumón</a> de gansos y patos), usados en el aislamiento de ropas y acolchado de camas, y el guano (las heces de las aves), que es una rica fuente de <a href="/wiki/F%C3%B3sforo" title="Fósforo">fósforo</a> y <a href="/wiki/Nitr%C3%B3geno" title="Nitrógeno">nitrógeno</a>. La <a href="/wiki/Guerra_del_Pac%C3%ADfico" title="Guerra del Pacífico">Guerra del Pacífico</a>, llamada a veces Guerra del Guano, se libró en parte por el control de grandes depósitos de esta sustancia.<sup class="reference separada" id="cite_ref-276"><a href="#cite_note-276"><span class="corchete-llamada">[</span>275<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las aves han sido domesticadas por humanos como <a href="/wiki/Animal_de_compa%C3%B1%C3%ADa" title="Animal de compañía">mascotas</a> y para propósitos prácticos. Aves coloridas como los <a href="/wiki/Psittaciformes" title="Psittaciformes">loros</a> y las <a href="/wiki/Sturnidae" title="Sturnidae">minas</a> son criadas en cautiverio y mantenidas como <a href="/wiki/Animal_de_compa%C3%B1%C3%ADa" title="Animal de compañía">animales de compañía</a>, práctica que ha llevado al <a href="/wiki/Comercio_de_animales_salvajes" title="Comercio de animales salvajes">tráfico ilegal</a> de un número de especies amenazadas.<sup class="reference separada" id="cite_ref-277"><a href="#cite_note-277"><span class="corchete-llamada">[</span>276<span class="corchete-llamada">]</span></a></sup>​ Los <a href="/wiki/Falconidae" title="Falconidae">halcones</a> y <a href="/wiki/Phalacrocorax" title="Phalacrocorax">cormoranes</a> han sido desde muy antiguo usados para la caza (<a href="/wiki/Cetrer%C3%ADa" title="Cetrería">cetrería</a>) y la pesca respectivamente. La <a href="/wiki/Paloma_mensajera" title="Paloma mensajera">paloma mensajera</a>, usada al menos desde el año 1 d. C., siguió siendo importante hasta épocas tan recientes como la <a href="/wiki/Segunda_Guerra_Mundial" title="Segunda Guerra Mundial">Segunda Guerra Mundial</a>. Hoy en día, tales actividades son más comunes como afición, entretenimiento, turismo,<sup class="reference separada" id="cite_ref-278"><a href="#cite_note-278"><span class="corchete-llamada">[</span>277<span class="corchete-llamada">]</span></a></sup>​ o deporte, como en las <a href="/wiki/Carrera_de_palomas" title="Carrera de palomas">carreras de palomas</a>.
</p>
<p>Los entusiastas aficionados llamados <a href="/wiki/Observaci%C3%B3n_de_aves" title="Observación de aves">observadores de aves</a> o pajareros se cuentan por millones.<sup class="reference separada" id="cite_ref-279"><a href="#cite_note-279"><span class="corchete-llamada">[</span>278<span class="corchete-llamada">]</span></a></sup>​ Muchas personas disponen comederos de aves cerca de sus viviendas para atraer varias especies. La alimentación para aves se ha convertido en un negocio multimillonario. Por ejemplo, se estima que el 75 % de las amas de casa de Gran Bretaña provee alimento para las aves en algún momento durante el invierno.<sup class="reference separada" id="cite_ref-280"><a href="#cite_note-280"><span class="corchete-llamada">[</span>279<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Las aves pueden jugar un rol sanitario importante al ser <a class="mw-redirect" href="/wiki/Vector_(biolog%C3%ADa)" title="Vector (biología)">vectores</a> de enfermedades humanas, al propagarlas a largas distancias, como la <a href="/wiki/Psitacosis" title="Psitacosis">psitacosis</a>, <a href="/wiki/Salmonelosis" title="Salmonelosis">salmonelosis</a>, <a href="/wiki/Campilobacteriosis" title="Campilobacteriosis">campilobacteriosis</a>, <a class="new" href="/w/index.php?title=Micobacteriosis&amp;action=edit&amp;redlink=1" title="Micobacteriosis (aún no redactado)">micobacteriosis</a> (<a href="/wiki/Tuberculosis" title="Tuberculosis">tuberculosis</a> aviar), <a class="mw-redirect" href="/wiki/Gripe_aviar" title="Gripe aviar">gripe aviar</a>, <a href="/wiki/Giardiasis" title="Giardiasis">giardiasis</a>, y <a href="/wiki/Criptosporidiosis" title="Criptosporidiosis">criptosporidiosis</a>.<sup class="reference separada" id="cite_ref-281"><a href="#cite_note-281"><span class="corchete-llamada">[</span>280<span class="corchete-llamada">]</span></a></sup>​ Por ello estas <a href="/wiki/Zoonosis" title="Zoonosis">zoonosis</a> son estudiadas y su propagación es cuidadosamente vigilada. El descubrimiento de reservorios de enfermedades aviares puede llevar a las autoridades locales a tomar medidas radicales respecto a las aves de corral, lo cual puede representar un fuerte impacto económico. Así, en septiembre de 2007, 205 000 aves fueron sacrificadas en <a href="/wiki/Baviera" title="Baviera">Baviera</a>,<sup class="reference separada" id="cite_ref-282"><a href="#cite_note-282"><span class="corchete-llamada">[</span>281<span class="corchete-llamada">]</span></a></sup>​ y otras 160 000 en <a href="/wiki/Banglad%C3%A9s" title="Bangladés">Bangladés</a> en junio de 2007,<sup class="reference separada" id="cite_ref-283"><a href="#cite_note-283"><span class="corchete-llamada">[</span>282<span class="corchete-llamada">]</span></a></sup>​ tras el descubrimiento de la cepa de la gripe aviar, entre otros ejemplos. Ciertas enfermedades pueden ser más específicas de un orden, como la <a href="/wiki/Enfermedad_de_Pacheco" title="Enfermedad de Pacheco">enfermedad de Pacheco</a> producida por un <a class="mw-redirect" href="/wiki/Herpesvirus" title="Herpesvirus">herpesvirus</a> en <a href="/wiki/Psittaciformes" title="Psittaciformes">Psittaciformes</a>.<sup class="reference separada" id="cite_ref-284"><a href="#cite_note-284"><span class="corchete-llamada">[</span>283<span class="corchete-llamada">]</span></a></sup>​
</p>
<p>Los humanos han tenido un gran impacto sobre muchas especies de aves. Las actividades humanas han permitido en algunos casos expandir drásticamente su territorio a algunas especies; otras, en cambio, han reducido su área de distribución, lo que ha conducido a muchas <a class="mw-redirect" href="/wiki/Extinci%C3%B3n_de_especies" title="Extinción de especies">extinciones</a>. Más de 120 especies de aves se han <a href="/wiki/Anexo:Aves_extintas" title="Anexo:Aves extintas">extinguido</a> desde el siglo XVII,<sup class="reference separada" id="cite_ref-full_286-0"><a href="#cite_note-full-286"><span class="corchete-llamada">[</span>285<span class="corchete-llamada">]</span></a></sup>​ aunque las extinciones más dramáticas causadas por el hombre ocurrieron durante la colonización humana de las islas de <a href="/wiki/Melanesia" title="Melanesia">Melanesia</a>, <a href="/wiki/Polinesia" title="Polinesia">Polinesia</a> y <a href="/wiki/Micronesia" title="Micronesia">Micronesia</a> en el <a href="/wiki/Oc%C3%A9ano_Pac%C3%ADfico" title="Océano Pacífico">océano Pacífico</a>, durante la cual se estima que se extinguieron de 750 a 1800 especies de aves.<sup class="reference separada" id="cite_ref-stea_287-0"><a href="#cite_note-stea-287"><span class="corchete-llamada">[</span>286<span class="corchete-llamada">]</span></a></sup>​ Muchas poblaciones de aves están en declive en todo el mundo; en condición de <a href="/wiki/Especie_amenazada" title="Especie amenazada">amenazadas</a> se cuentan 1253 especies (año 2011) en las listas de <a class="mw-redirect" href="/wiki/Birdlife_International" title="Birdlife International">Birdlife International</a> y la <a href="/wiki/Uni%C3%B3n_Internacional_para_la_Conservaci%C3%B3n_de_la_Naturaleza" title="Unión Internacional para la Conservación de la Naturaleza">UICN</a>.<sup class="reference separada" id="cite_ref-bli_9-1"><a href="#cite_note-bli-9"><span class="corchete-llamada">[</span>8<span class="corchete-llamada">]</span></a></sup>​ La causa más frecuentemente citada involucra la pérdida de hábitat.<sup class="reference separada" id="cite_ref-norr_288-0"><a href="#cite_note-norr-288"><span class="corchete-llamada">[</span>287<span class="corchete-llamada">]</span></a></sup>​ Otras amenazas incluyen la caza excesiva, la mortalidad accidental por colisión con edificaciones o debida al enganche por la pesca con sedal largo,<sup class="reference separada" id="cite_ref-brot_289-0"><a href="#cite_note-brot-289"><span class="corchete-llamada">[</span>288<span class="corchete-llamada">]</span></a></sup>​ por contaminación (incluidos derrames de <a href="/wiki/Petr%C3%B3leo" title="Petróleo">petróleo</a> y uso de <a class="mw-redirect" href="/wiki/Pesticida" title="Pesticida">pesticidas</a>),<sup class="reference separada" id="cite_ref-wurs_290-0"><a href="#cite_note-wurs-290"><span class="corchete-llamada">[</span>289<span class="corchete-llamada">]</span></a></sup>​ competencia y depredación por <a href="/wiki/Especie_invasora" title="Especie invasora">especies invasoras</a> (como <a class="mw-redirect" href="/wiki/Rata" title="Rata">ratas</a>, <a class="mw-redirect" href="/wiki/Gato" title="Gato">gatos</a> y <a class="mw-redirect" href="/wiki/Mangosta" title="Mangosta">mangostas</a>),<sup class="reference separada" id="cite_ref-blac_291-0"><a href="#cite_note-blac-291"><span class="corchete-llamada">[</span>290<span class="corchete-llamada">]</span></a></sup>​ y el <a href="/wiki/Cambio_clim%C3%A1tico" title="Cambio climático">cambio climático</a>. La familia que cuenta con más especies amenazadas (más de 70) es la de los <a href="/wiki/Psittacidae" title="Psittacidae">loros</a>. Los gobiernos, junto a organizaciones de conservación, trabajan para proteger a las aves mediante de leyes, <a href="/wiki/Conservaci%C3%B3n_in_situ" title="Conservación in situ">preservando</a> y <a href="/wiki/Restauraci%C3%B3n_ecol%C3%B3gica" title="Restauración ecológica">restaurando</a> sus hábitats, o manteniendo <a href="/wiki/Conservaci%C3%B3n_ex_situ" title="Conservación ex situ">poblaciones en cautiverio</a> para ulteriores <a href="/wiki/Reintroducci%C3%B3n" title="Reintroducción">reintroducciones</a>. Los esfuerzos de conservación biológica han conseguido algunos éxitos; un estudio estimó que <a href="/wiki/Anexo:Aves_salvadas_de_la_extinci%C3%B3n_inminente" title="Anexo:Aves salvadas de la extinción inminente">entre 1994 y 2004 fueron salvadas 16 especies de aves</a> que se habrían extinguido si no se hubieran realizado estas acciones.<sup class="reference separada" id="cite_ref-butc_292-0"><a href="#cite_note-butc-292"><span class="corchete-llamada">[</span>291<span class="corchete-llamada">]</span></a></sup>​
</p>
