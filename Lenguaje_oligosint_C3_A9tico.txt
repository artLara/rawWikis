<p>Una <b>lengua oligosintética</b> (del <a href="/wiki/Idioma_griego" title="Idioma griego">griego</a> ὀλίγος <i>olígos</i>, que quiere decir "poco" o "pequeño") es cualquier <a href="/wiki/Lengua_natural" title="Lengua natural">lengua natural</a> que use muy pocos <a href="/wiki/Morfema" title="Morfema">morfemas</a>, tal vez unos pocos cientos, lo que combina una forma <a href="/wiki/Lengua_sint%C3%A9tica" title="Lengua sintética">sintética</a> para crear oraciones. Esto contrasta con lo que sería una <a href="/wiki/Lengua_polisint%C3%A9tica" title="Lengua polisintética">lengua polisintética</a>. La oligosíntesis es en su mayoría teórica y dependería básicamente en la creación de largas palabras compuestas, en una mayor medida en comparación a las <a href="/wiki/Lengua_sint%C3%A9tica" title="Lengua sintética">lenguas sintéticas</a>.
</p>
<p>Las lenguas nativas americanas <a href="/wiki/N%C3%A1huatl" title="Náhuatl">náhuatl</a> y <a href="/wiki/Idioma_siksik%C3%A1" title="Idioma siksiká">siksiká</a> han mostrado en el pasado tener cualidades oligosintéticas (mencionado por <a class="mw-redirect" href="/wiki/Benjamin_Whorf" title="Benjamin Whorf">Benjamin Whorf</a>). Sin embargo, la comunidad lingüística ha rechazado dichos estudios, prefiriendo categorizar el náhuatl y el siksika como polisintéticas.
</p>
<p>Debido a que no existen lenguas naturales que muestren propiedades oligosintéticas, algunos lingüistas sostienen que una verdadera lengua oligosintética es imposible o nada práctica para un uso productivo entre humanos.
</p>
<p>Algunas <a href="/wiki/Lengua_auxiliar" title="Lengua auxiliar">lenguas auxiliares</a>, por ejemplo la <a href="/wiki/Lengua_sona" title="Lengua sona">lengua sona</a>, el <a href="/wiki/Lengua_aUI" title="Lengua aUI">aUI</a>, el <a class="external text" href="https://web.archive.org/web/20070220130336/http://www.medianet.pl/~andrew/ygyde/ygyde.htm" rel="nofollow">Ygyde</a>, el <a class="external text" href="https://web.archive.org/web/20100117181304/http://www.langmaker.com/kalisise.htm" rel="nofollow">Kali-sise</a>, el <a class="external text" href="https://web.archive.org/web/20070930182738/http://www.langmaker.com/db/Vuyamu" rel="nofollow">Vuyamu</a> y el <a class="external text" href="http://www.kwesho.com" rel="nofollow">Kah</a>, deberían ser considerados oligosintéticos.
</p>
<p>A diferencia de las lenguas oligosintéticas, una lengua oligoanalítica o oligoaislada es aquella que tienen algunos morfemas pero tienden a un origen <a href="/wiki/Lengua_aislada" title="Lengua aislada">aislado</a>. Por ejemplo, la lengua auxiliar <a class="mw-redirect" href="/wiki/Toki_Pona" title="Toki Pona">Toki Pona</a> ha sido descrita como oligoaislada.<a class="external autonumber" href="http://groups.yahoo.com/group/tokipona/message/521" rel="nofollow">[1]</a>
</p>
<p>La <a href="/wiki/Neolengua" title="Neolengua">neolengua</a> hablada en <a href="/wiki/Ocean%C3%ADa" title="Oceanía">Oceanía</a> en la novela <a href="/wiki/1984_(novela)" title="1984 (novela)">1984</a> tiende <i>teóricamente</i> a ser una lengua oligosintética, ya que el objetivo era reducir los morfemas.
</p>
